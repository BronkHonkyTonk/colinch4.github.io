이번 포스트에서는 플러터(Flutter)에서 사용되는 **위젯(Widget)**과 **UI 컴포넌트**에 대해 알아보겠습니다.

## 위젯(Widget)이란?

플러터에서 **위젯**은 화면에 무언가를 표시하는 데 사용됩니다. 화면에 보이는 모든 것, 예를 들어 버튼, 이미지, 레이아웃, 애니메이션 등은 모두 위젯으로 구성됩니다. 

플러터 위젯은 크게 **StatelessWidget**과 **StatefulWidget**으로 나뉩니다. **StatelessWidget**은 변하지 않는 화면 구성을 표현할 때 사용되고, **StatefulWidget**은 상태가 바뀔 수 있는 화면 구성을 표현할 때 사용됩니다.

## UI 컴포넌트

플러터에서는 여러가지 **UI 컴포넌트**를 제공합니다. 이러한 UI 컴포넌트들은 사용자 인터페이스를 구성하는 데 필요한 다양한 기능과 스타일을 제공합니다. 버튼, 텍스트 필드, 아이콘, 이미지 등 다양한 UI 요소들이 있으며, 미리 디자인된 UI를 쉽게 구현할 수 있도록 도와줍니다.

위젯과 UI 컴포넌트는 플러터 개발 시 핵심적인 역할을 하므로, 이들을 잘 이해하고 활용하는 것이 필수적입니다.

이상으로 플러터 위젯과 UI 컴포넌트에 대해 간략히 살펴보았습니다. 다음 포스트에서는 각 위젯과 UI 컴포넌트에 대해 자세히 알아보겠습니다.
