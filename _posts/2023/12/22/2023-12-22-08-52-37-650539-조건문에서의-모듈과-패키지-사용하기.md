파이썬은 모듈과 패키지를 사용하여 코드를 모듈화하고 조건문에서 활용할 수 있습니다. 모듈과 패키지를 사용하면 코드를 구조화하여 유지 및 보수가 용이하고, 중복을 방지할 수 있습니다. 이 글에서는 파이썬 조건문에서 모듈과 패키지를 활용하는 방법에 대해 알아봅니다.

## 모듈(Module)과 패키지(Package)

**모듈(Module)**은 파이썬 코드를 담고 있는 파일입니다. 간단하게 다른 파이썬 파일에서 코드를 재사용하고 싶을 때 모듈을 이용할 수 있습니다. **패키지(Package)**는 여러 모듈을 묶은 것으로, 디렉토리 구조를 가지고 있습니다.

## 모듈과 패키지의 조건문 사용 예시

다음은 간단한 조건문을 사용하여 모듈을 불러와 활용하는 예시입니다.

```python
# 모듈 불러오기
import mymodule

# 모듈의 함수 호출
mymodule.my_function()
```

위 코드에서는 `mymodule`이라는 모듈을 불러와 `my_function` 함수를 호출하는 예시입니다.

## 패키지의 조건문 사용 예시

패키지 역시 조건문에서 활용할 수 있습니다. 다음은 패키지의 모듈을 불러와 사용하는 예시입니다.

```python
# 패키지의 모듈 불러오기
from mypackage import mymodule

# 모듈의 함수 호출
mymodule.my_function()
```

위 코드에서는 `mypackage`라는 패키지 안에 있는 `mymodule` 모듈을 불러와 `my_function` 함수를 호출하는 예시입니다.

## 결론

모듈과 패키지는 파이썬에서 코드를 구조화하고 재사용하기 위한 강력한 도구입니다. 조건문에서도 모듈과 패키지를 활용하여 코드를 간결하게 유지하고, 유지보수를 용이하게 할 수 있습니다.

이상으로, 파이썬 조건문에서 모듈과 패키지를 활용하는 방법에 대해 알아보았습니다.
