## 중단점 설정

코드 편집기에서 실행을 중지하고 싶은 줄에 커서를 두고 `F9` 키를 누르거나, 해당 줄의 왼쪽 여백을 클릭하여 중단점을 설정할 수 있습니다.

또는 코드를 실행 중단 시키고 싶은 지점에서 `System.Diagnostics.Debugger.Break()` 메서드를 호출할 수도 있습니다.

이렇게 하면 프로그램이 해당 줄에서 실행을 중단하고 디버거가 활성화됩니다.

## 중단점 제거

중단점이 설정된 코드 줄의 왼쪽 여백을 클릭하거나, 디버깅 중인 상태에서 `F9` 키를 다시 눌러 해당 줄의 중단점을 제거할 수 있습니다.

중단점을 모두 제거하려면 디버깅 중단 상태에서 `Ctrl` + `Shift` + `F9` 키를 누릅니다.

이제 중단점을 설정하고 제거하는 방법을 숙지하셨으니, 디버깅을 좀 더 효과적으로 수행할 수 있게 되었을 것입니다.
