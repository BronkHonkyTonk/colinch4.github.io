CSS를 사용하여 HTML 문서의 특정 요소를 스타일링하려면 해당 요소의 선택자를 식별해야 합니다. 가장 일반적인 선택자 유형 중 하나가 요소 선택자입니다.

```css
p {
  color: blue;
  font-size: 16px;
}
```

위의 예제에서 `p`는 `p` 태그를 선택하는 요소 선택자입니다. 따라서 이 스타일은 HTML 문서 내의 모든 `p` 태그에 적용됩니다.

다른 예를 보겠습니다. `h1` 태그를 선택하여 스타일을 적용하려면 다음과 같이 작성할 수 있습니다.

```css
h1 {
  color: red;
  font-size: 24px;
}
```

이것은 `h1` 태그 요소에 적용되는 스타일을 정의합니다. 요소 선택자를 사용하여 HTML 요소에 쉽게 스타일을 적용할 수 있습니다.

페이지의 여러 요소에 스타일을 적용하려면 요소 선택자를 사용하여 해당 요소의 모든 인스턴스에 스타일을 일괄 적용할 수 있습니다.

이렇게 하면 HTML 문서의 특정 요소에 일관된 스타일을 적용할 수 있으며, 유지 및 관리도 더 쉬워집니다.

이를테면, 본문 텍스트에 적용할 기본 스타일을 설정하여 모든 본문 텍스트에 적용하는 것이 가능합니다.
