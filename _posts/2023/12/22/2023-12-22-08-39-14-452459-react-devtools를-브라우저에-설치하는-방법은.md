1. 먼저, Chrome 또는 Firefox 브라우저를 엽니다.
2. React 애플리케이션을 실행합니다.
3. 브라우저에서 확장 기능 스토어로 이동하여 "React DevTools"를 검색합니다.
4. React DevTools 확장을 설치합니다.
5. 브라우저의 확장 목록에서 React DevTools를 활성화합니다.

이제 React 애플리케이션을 디버깅하고 프로파일링하는 데 도움이 되는 React DevTools가 브라우저에 설치되었습니다.
