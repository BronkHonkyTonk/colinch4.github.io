CSS를 사용하여 웹 페이지의 배경을 디자인하는 것은 매우 중요합니다. 배경 이미지와 색상을 설정하여 웹 페이지를 더욱 매력적이고 시각적으로 매력적으로 만들 수 있습니다. 이 글에서는 CSS를 사용하여 배경 이미지와 색상을 설정하는 방법에 대해 알아보겠습니다.

## 배경 이미지 설정하기

웹 페이지의 배경에 이미지를 추가하려면 `background-image` 속성을 사용합니다. 이미지 파일의 경로를 지정하여 배경 이미지를 설정할 수 있습니다.

예시:
```css
body {
  background-image: url('example.jpg');
}
```

`background-image` 속성을 사용하여 웹 페이지의 배경에 이미지를 추가합니다.

## 배경 색상 설정하기

`background-color` 속성을 사용하여 웹 페이지의 배경 색상을 변경할 수 있습니다. 색상의 이름, HEX, RGB 또는 HSL 값을 사용하여 배경 색상을 정의할 수 있습니다.

예시:
```css
body {
  background-color: #f2f2f2;
}
```

`background-color` 속성을 사용하여 웹 페이지의 배경 색상을 변경합니다.

## 배경 이미지 및 색상 조합하기

배경 이미지와 색상을 함께 사용하여 웹 페이지의 배경을 디자인할 수 있습니다.

예시:
```css
body {
  background-image: url('example.jpg');
  background-color: #f2f2f2;
}
```

`background-image` 및 `background-color` 속성을 함께 사용하여 배경 이미지와 색상을 조합합니다.

배경 이미지 및 색상을 조합하여 웹 페이지의 배경을 디자인함으로써 사용자의 시선을 사로잡을 수 있습니다.

이렇게 CSS를 사용하여 배경 이미지와 색상을 설정하는 방법에 대해 알아보았습니다. 웹 디자인에 있어서 배경은 매우 중요한 부분이므로 다양한 속성들을 활용하여 웹 페이지를 더욱 매력적으로 꾸밀 수 있습니다.
