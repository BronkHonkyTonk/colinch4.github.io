블루투스 네트워킹을 위해서는 우선 먼저 시스템에 블루투스 어댑터가 필요합니다. 블루투스 어댑터가 시스템에 연결되어 있는지 확인하여야 합니다. 

리눅스 환경에서 블루투스 어댑터가 제대로 인식되었는지 확인하려면 터미널에서 다음 명령어를 실행합니다.

```bash
hciconfig
```

이 명령은 블루투스 어댑터에 대한 상세 정보를 보여줍니다. 어댑터가 제대로 인식되지 않거나 활성화되어 있지 않은 경우, 추가적인 조치가 필요할 수 있습니다.

다음으로, **블루투스 서비스**를 설치하고 활성화해야 합니다. 대부분의 리눅스 배포판에는 블루투스 관련 패키지가 포함되어 있지만, 설치되어 있지 않은 경우 아래와 같은 명령어로 설치할 수 있습니다.

```bash
sudo apt install bluez
```

블루투스 서비스 설치가 완료되면, 서비스를 시작하고 **활성화**하여야 합니다.

```bash
sudo systemctl start bluetooth
sudo systemctl enable bluetooth
```

이제 블루투스 서비스가 활성화되었으므로, 다른 블루투스 장치들을 스캔하고 페어링할 수 있습니다. 이를 위해 다음 명령어를 사용합니다.

```bash
bluetoothctl
```

위 명령을 실행하면 블루투스 커맨드 라인 유틸리티가 열립니다. 여기서 `scan on` 명령을 사용하여 주변의 장치들을 스캔하고, `pair` 명령을 사용하여 원하는 장치를 페어링할 수 있습니다.

이제 블루투스 네트워킹이 설정되었으며, 페어링한 디바이스와 데이터를 주고받을 수 있게 되었습니다.

이렇게하여 리눅스에서 블루투스 네트워킹을 설정하는 과정을 알아보았습니다.
