C언어는 변수와 상수를 선언하여 데이터를 저장하고 조작하는 데 사용됩니다. 변수는 값을 저장하고 나중에 편집하거나 검색할 수 있는 메모리 위치입니다. 반면에 상수는 값을 한 번 정의하고 나중에 수정할 수 없는 데이터입니다.

## 변수

변수를 선언할 때는 데이터 유형과 변수 이름을 지정해야 합니다. 변수 이름은 향후 코드에서 해당 변수를 참조할 때 사용됩니다.

```c
int age;  // 정수형 변수 선언
float salary;  // 실수형 변수 선언
char gender;  // 문자형 변수 선언
```

변수를 초기화하려면 변수를 선언하는 동시에 값을 할당할 수 있습니다.

```c
int age = 30;  // 정수형 변수 초기화
float salary = 1000.50;  // 실수형 변수 초기화
char gender = 'M';  // 문자형 변수 초기화
```

## 상수

상수는 선언될 때 값을 할당 받으며, 이후로는 변경할 수 없습니다. 상수를 선언하는 방법은 변수와 유사하지만, `const` 키워드를 사용하여 선언된다는 점이 다릅니다.

```c
const int MONTHS_IN_YEAR = 12;  // 연 당 월 수 상수 선언
const float PI = 3.14159;  // 원주율 상수 선언
```

변수와 상수는 C 프로그램에서 데이터를 저장하고 관리하는 데 필수적입니다. 올바르게 사용하면 프로그램의 가독성과 유지 관리가 향상되며, 프로그램에 유연성과 안정성을 부여할 수 있습니다.

변수와 상수에 대한 자세한 내용은 [C언어 공식 문서](https://www.gnu.org/software/gnu-c-manual/gnu-c-manual.html)를 확인하시기 바랍니다.
