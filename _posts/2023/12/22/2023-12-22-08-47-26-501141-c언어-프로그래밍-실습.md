## 1. 개요
C 언어는 프로그래밍 언어 중에서 가장 기본적이면서도 다양한 분야에서 활용되고 있는 언어입니다. C 언어 실습을 통해 기초적인 문법과 프로그래밍 기법을 익히고 응용력을 키울 수 있습니다.

## 2. 실습 환경 설정
C 언어를 학습하고 실습하기 위해서는 컴파일러가 필요합니다. 대표적인 C 언어 컴파일러로는 GCC (GNU Compiler Collection)가 있으며, 다양한 플랫폼에서 지원됩니다. GCC를 설치하고 환경 변수를 설정하여 C 언어 프로그래밍을 실습할 수 있습니다.

## 3. 실습 예제
다음은 간단한 "Hello, world!"를 출력하는 C 언어의 예제 코드입니다.

```c
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
```

위 코드를 텍스트 에디터로 작성하고, 확장자를 `.c`로 저장한 뒤 GCC를 이용하여 컴파일하면 실행 파일이 생성됩니다.
