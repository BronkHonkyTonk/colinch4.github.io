또는 다음과 같은 방법으로 설정할 수도 있습니다.
```go
package main

func main() {
    // 프로그램의 본문
}
```
위 코드에서 `package main`은 실행 파일의 이름을 결정하기 위해 필요한 정보를 포함하고 있습니다. 코드를 빌드하면 실행 파일의 이름은 소스 파일의 이름과 동일하게 됩니다.
