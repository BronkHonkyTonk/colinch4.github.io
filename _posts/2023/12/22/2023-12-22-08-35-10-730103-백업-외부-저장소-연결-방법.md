리눅스에서 시스템 또는 데이터를 백업하는 것은 매우 중요합니다. 중요한 파일을 보호하기 위해 외부 저장소와 연결하여 안전하게 백업하는 방법을 알고 계십니다. 이 글에서는 리눅스에서 외부 저장소에 백업하는 방법에 대해 설명하겠습니다.

## USB 외장 하드 드라이브 연결

우선 USB 외장 하드 드라이브를 리눅스 시스템에 연결해야 합니다. 외장 하드 드라이브를 USB 포트에 연결한 후에는 시스템이 자동으로 인식할 것입니다. 대부분의 리눅스 배포판은 USB 외장 하드 드라이브를 자동으로 마운트하고 파일 관리자에 표시합니다.

## 마운트 포인트 확인

외장 하드 드라이브가 마운트되었는지 확인해야 합니다. `df -h` 명령을 사용하여 시스템에 마운트된 디스크를 확인할 수 있습니다. `/dev/sdX`나 `/dev/sdXX`와 같은 디바이스 이름을 기억해 두세요.

```bash
df -h
```

## 외부 저장소에 백업

이제 외부 저장소에 백업할 준비가 되었습니다. `rsync`나 `cp`와 같은 명령을 사용하여 원하는 파일이나 디렉토리를 외장 하드 드라이브로 복사할 수 있습니다. 예를 들어, 현재 디렉토리의 모든 파일을 외장 하드 드라이브로 복사하려면 다음과 같이 하면 됩니다.

```bash
rsync -avz . /마운트_포인트
```

## 외장 하드 드라이브 분리

백업이 완료되면 외장 하드 드라이브를 시스템에서 분리합니다. 이때 운영체제가 외장 하드 드라이브를 사용 중이지 않는지 확인하고 `umount` 명령으로 디스크 마운트를 해제한 후에 안전하게 분리해야 합니다.

이제 외부 저장소에 리눅스 시스템의 백업을 안전하게 저장할 수 있게 되었습니다. 백업 주기를 정하고 정기적으로 외장 하드 드라이브에 백업하여 시스템과 데이터를 안전하게 보호하세요.
