아래는 아이디 선택자를 사용하여 요소를 선택하는 예시입니다.

```css
#myElement {
  color: blue;
}
```

위 예시에서 "myElement"는 아이디이며, 이 아이디를 가진 요소에 대해 색상을 파란색으로 지정하는 CSS 규칙을 정의하고 있습니다.

아이디 선택자는 문서 내에서 고유한 식별자로 사용되어야 하며, 한 페이지 내에서 특정 아이디는 유일해야 합니다. 이를 통해 CSS에서 해당 아이디를 가진 요소를 정확하게 선택할 수 있습니다.

아이디 선택자를 사용하여 요소에 스타일을 적용할 때, 해당 아이디를 가진 요소 하나에만 특정 스타일을 적용하므로, 유용하게 활용될 수 있습니다.
