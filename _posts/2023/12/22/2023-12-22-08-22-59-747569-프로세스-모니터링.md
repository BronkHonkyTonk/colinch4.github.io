1. **top 명령어:**
   `top` 명령어를 사용하여 실시간으로 실행 중인 프로세스의 목록을 볼 수 있습니다. 이 명령어는 CPU 및 메모리 사용량과 함께 프로세스의 상태를 표시합니다.

   ```bash
   top
   ```

2. **ps 명령어:**
   `ps` 명령어를 사용하면 실행 중인 프로세스의 상세 정보를 표시할 수 있습니다. `-e` 옵션은 모든 프로세스를, `-f` 옵션은 전체 형식으로 표시합니다.

   ```bash
   ps -ef
   ```

3. **htop 유틸리티:**
   `htop`은 `top` 명령어의 확장된 버전으로, 프로세스를 쉽게 모니터링할 수 있도록 도와줍니다. 사용자 친화적인 인터페이스를 제공하며 프로세스를 종료하거나 우선순위를 변경하는 등의 작업을 할 수 있습니다.

   ```bash
   htop
   ```

프로세스 모니터링은 시스템 안정성을 유지하고 성능을 최적화하는데 도움이 됩니다. 위에서 언급한 방법들로 원하는 프로세스를 식별하고 추적할 수 있습니다.
