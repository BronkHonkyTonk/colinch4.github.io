조건문을 구성하는 if-else 문은 다음과 같은 형식을 갖습니다.

```shell
if [ 조건 ]; then
    # 조건이 참일 때 수행되는 코드
else
    # 조건이 거짓일 때 수행되는 코드
fi
```

이와 같은 조건문을 사용하여 프로그램이 특정 조건에 따라 다른 동작을 수행하도록 제어할 수 있습니다. 각 언어별로 구체적인 문법 및 사용 방법은 해당 언어의 공식 문서를 참고하시기 바랍니다.
