Go 언어로 작성된 프로그램을 실행하기 위해서는 몇 가지 옵션을 사용할 수 있습니다. 이 포스트에서는 Go 언어의 다양한 실행 옵션을 살펴보겠습니다.

## 명령 실행 구문

Go 언어로 작성된 프로그램을 실행하는 기본적인 구문은 다음과 같습니다.

```go
go run main.go
```

이 명령은 `main.go` 파일을 실행합니다. 만약에 다른 파일을 실행해야 한다면, 해당 파일명을 사용하면 됩니다.

## 실행 옵션

### 1. 종속성 다운로드

만약 프로그램에 의존하는 패키지가 있다면, `go run` 명령을 실행할 때 해당 패키지를 먼저 다운로드할 수 있습니다.

```go
go run -d main.go
```

### 2. 출력 경로 지정

실행 결과를 특정 경로로 저장할 수 있습니다.

```go
go run -o output main.go
```

이렇게 하면 실행 결과가 `output` 파일에 저장됩니다.

### 3. 환경 변수 설정

실행 시 환경 변수를 설정할 수 있습니다.

```go
go run -e GOMAXPROCS=2 main.go
```

위의 예시는 `GOMAXPROCS` 환경 변수를 `2`로 설정하여 프로그램을 실행합니다.

## 결론

Go 언어에서는 `go run` 명령을 통해 프로그램을 실행할 때 다양한 옵션을 활용할 수 있습니다. 위에서 소개한 옵션들을 활용하여 효율적으로 프로그램을 실행해 보세요.
