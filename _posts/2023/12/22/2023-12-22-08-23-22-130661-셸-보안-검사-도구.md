## 개요
셸(Shell)은 운영체제와 사용자간 상호작용을 도와주는 프로그램이다. 셸 보안 검사 도구는 셸 스크립트나 셸 환경에서 사용되는 명령어들을 분석하고 셸 스크립트의 취약점을 발견하기 위한 도구이다. 셸 보안 검사 도구를 이용하여 보다 안전한 셸 환경을 유지할 수 있다.

## 셸 보안 검사 도구 종류
### 1. ShellCheck
ShellCheck는 셸 스크립트의 문법 오류와 일반적인 문제를 식별하고 더 좋은 코딩 관행을 제시하는 도구이다. ShellCheck는 bash, sh 및 ksh와 같은 셸 언어를 다루는 훌륭한 도구이다.

### 2. Lynis
Lynis는 리눅스 및 UNIX 계열 시스템의 보안 측면을 검사할 수 있는 도구이다. 이는 셸 보안 검사 도구 중 하나로서 셸의 사용자 및 권한 관리 등에 대한 문제점을 식별하여 제시한다.

## 장점
- 보다 안전한 셸 스크립트 및 환경을 구축할 수 있다.
- 셸 스크립트의 오류와 취약점을 식별하여 보다 안정적인 운영을 할 수 있다.

## 결론
셸 보안 검사 도구를 이용하여 셸 스크립트 및 셸 환경을 검사하고 보안 취약점을 개선함으로써 시스템을 더 안전하게 운영할 수 있다.
