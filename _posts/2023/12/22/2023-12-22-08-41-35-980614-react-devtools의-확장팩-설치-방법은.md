1. Chrome 브라우저를 열고 Chrome 웹 스토어에 접속합니다.
2. 검색 창에 "React DevTools"를 입력하여 해당 확장 프로그램을 찾습니다.
3. "추가" 버튼을 클릭하여 확장팩을 설치합니다.
4. 설치가 완료되면 브라우저의 오른쪽 상단에 있는 확장 아이콘을 클릭하여 React DevTools를 활성화합니다.
5. React 애플리케이션을 실행하고, 브라우저 개발자 도구(DevTools)를 열면 React 탭이 나타나고 React 구성 요소 트리와 상태를 확인할 수 있습니다.

이렇게 하면 React DevTools 확장팩을 설치하고 활성화하여 React 애플리케이션을 디버깅할 수 있습니다.
