쉘 스크립트에서 함수를 정의하는 방법에 대해 설명드리겠습니다.

함수를 정의하려면 함수명을 지정하고 괄호로 둘러싼 후 중괄호로 감싸는 방식을 사용합니다. 함수 본문은 중괄호 안에 작성됩니다.

```sh
함수명() {
    # 함수 본문
    명령어1
    명령어2
    ...
}
```

여기서 "함수명"은 사용자가 원하는 함수 이름으로 대체됩니다. 함수를 호출할 때는 해당 함수명을 사용하여 호출할 수 있습니다.

예를 들어, "hello" 함수를 정의하고 호출하는 코드는 다음과 같습니다.

```sh
hello() {
    echo "안녕하세요!"
}

# 함수 호출
hello
```

이제 쉘 스크립트에서 함수를 정의하고 호출하는 방법에 대해 이해하셨습니다!
