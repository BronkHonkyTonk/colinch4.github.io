## 3. 예시 코드 및 설명
아래는 C언어를 사용하여 모바일 애플리케이션의 보안을 강화하는 간단한 예시 코드입니다.

```c
#include <stdio.h>

int main() {
    char password[10] = "secure123";
    char input[10];

    printf("Enter the password: ");
    scanf("%s", input);

    if (strcmp(password, input) == 0) {
        printf("Access granted.");
    } else {
        printf("Access denied.");
    }

    return 0;
}
```

위 코드는 사용자로부터 입력받은 비밀번호와 저장된 비밀번호를 비교하여 접근 허용 여부를 결정하는 간단한 예시입니다.
