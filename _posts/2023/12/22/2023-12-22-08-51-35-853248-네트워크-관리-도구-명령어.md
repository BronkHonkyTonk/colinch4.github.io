리눅스 운영 체제는 다양한 네트워크 관리 명령어를 제공하여 네트워크 설정, 연결 및 해제, 네트워크 상태 확인 등을 수행할 수 있습니다. 이 글에서는 리눅스에서 자주 사용되는 네트워크 관리 도구 명령어를 살펴보겠습니다.

## **ifconfig**
```bash
ifconfig
```
**ifconfig** 명령어는 네트워크 인터페이스의 설정을 보거나 변경하기 위해 사용됩니다. 이를 통해 네트워크 인터페이스의 IP 주소, 서브넷 마스크, 맥 주소 등을 확인하거나 설정할 수 있습니다.

## **ip**
```bash
ip addr show
```
**ip** 명령어는 네트워크 인터페이스 및 라우팅 설정을 제어하는 데 사용됩니다. **ip addr show** 명령어를 사용하여 네트워크 인터페이스의 IP 주소를 확인할 수 있습니다.

## **ping**
```bash
ping www.example.com
```
**ping** 명령어는 다른 호스트에 TCP/IP 네트워크 연결 품질을 확인하는 데 사용됩니다. 호스트와의 통신이 가능한지, 대기 시간, 패킷 손실 등을 확인할 수 있습니다.

## **netstat**
```bash
netstat -r
```
**netstat** 명령어는 네트워크 연결, 라우팅 테이블, 인터페이스 상태 등의 정보를 출력하는 데 사용됩니다. **-r** 옵션을 사용하여 라우팅 테이블을 확인할 수 있습니다.

## **traceroute**
```bash
traceroute www.example.com
```
**traceroute** 명령어는 목적지 호스트까지의 경로를 추적하는 데 사용됩니다. 패킷이 이동하는 경로와 각 호스트까지의 대기 시간을 확인할 수 있습니다.

위에서 소개한 명령어들은 리눅스 시스템에서 네트워크 관리를 위해 자주 사용되는 도구들입니다. 네트워크 설정, 상태 확인, 문제 해결에 유용하니 익숙해지는 것이 좋습니다.
