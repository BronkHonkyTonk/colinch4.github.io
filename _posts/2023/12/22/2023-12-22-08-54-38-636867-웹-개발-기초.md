## HTML 소개
HTML은 월드 와이드 웹에서 정보를 표시하고 구조화하기 위한 가장 기본적인 언어입니다. HTML은 다른 웹 기술과 조합하여 멋진 웹 페이지를 만들 수 있게 해줍니다.

## 기본 HTML 구조
모든 HTML 문서는 기본적인 구조를 갖습니다. 아래는 기본적인 HTML 문서의 구조입니다.

```html
<!DOCTYPE html>
<html>
<head>
    <title>문서 제목</title>
</head>
<body>
    <h1>제목</h1>
    <p>내용</p>
</body>
</html>
```

## 주요 HTML 태그
### `<html>`
문서의 루트 요소를 정의합니다. 모든 다른 요소는 이 요소 안에 포함되어야 합니다.

### `<head>`
웹 페이지의 메타데이터를 정의합니다. 스타일 시트 링크, 스크립트 링크, 문서 제목 등을 포함합니다.

### `<title>`
웹 페이지의 제목을 정의합니다. 브라우저 탭에 표시됩니다.

### `<body>`
문서의 본문을 정의합니다. 화면에 보여지는 모든 콘텐츠는 이 요소 안에 정의되어야 합니다.

### `<h1>, <h2>, <h3>, ...`
제목을 표시하는 데 사용됩니다. 숫자가 클수록 제목의 크기가 작아집니다.

### `<p>`
단락을 표시하는 데 사용됩니다.
