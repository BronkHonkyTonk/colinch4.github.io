### 감시 대시보드란 무엇인가요?

**감시 대시보드**는 시스템의 여러 측면을 실시간으로 모니터링하고 시각화하는 도구입니다. CPU, 메모리, 디스크, 네트워크 등의 리소스 사용량 및 시스템 로그 등을 표시하여 시스템 상태를 쉽게 파악할 수 있습니다.

### 감시 대시보드 선정

리눅스 시스템에는 여러 감시 대시보드 도구가 있습니다. 가장 인기 있는 것 중 하나는 **Grafana**입니다. Grafana는 다양한 데이터원천으로부터 정보를 시각화하고 모니터링하는 데 사용됩니다.

### 감시 대시보드 설치

먼저, 시스템에 Grafana를 설치해야 합니다. <br>
```
$ sudo apt-get update
$ sudo apt-get install -y software-properties-common
$ sudo add-apt-repository "deb https://packages.grafana.com/oss/deb stable main"
$ sudo apt-get update
$ sudo apt-get install grafana
```

### 감시 대시보드 설정

Grafana를 설치한 후, 웹 브라우저를 열어 `http://서버주소:3000`에 접속합니다. 초기 로그인 정보는 admin/admin입니다. 로그인한 후, 데이터베이스 및 데이터소스를 추가하여 감시 대시보드를 설정할 수 있습니다.

이제, 리눅스 시스템에 감시 대시보드를 설정하기 위한 기본적인 단계를 완료했습니다. 감시 대시보드를 통해 리눅스 시스템의 상태를 실시간으로 모니터링하고 문제를 예방하고 대응할 수 있습니다.

감시 대시보드 설정에 대한 더 자세한 정보는 [Grafana 공식 문서](https://grafana.com/docs/)를 참고하시기 바랍니다.
