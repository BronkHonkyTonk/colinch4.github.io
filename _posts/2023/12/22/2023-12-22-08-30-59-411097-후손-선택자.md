CSS에서 후손 선택자는 부모 요소의 하위 요소를 스타일링하는 데 사용됩니다. 후손 선택자는 공백 문자로 표시되며, 부모 요소와 하위 요소를 연결하는데 사용됩니다.

```css
부모요소 하위요소 {
    속성: 값;
}
```

예를 들어, 아래의 HTML 요소가 있다고 가정해봅시다.

```html
<div class="부모">
    <p>후손 요소</p>
</div>
```
우리는 다음과 같이 후손 선택자를 사용하여 스타일을 지정할 수 있습니다.

```css
.부모 p {
    color: blue;
}
```

이렇게 하면 클래스가 "부모"인 div 요소에 포함된 모든 p 태그의 텍스트가 파란색으로 표시됩니다.

이렇게 하위 요소를 스타일링함으로써 CSS의 유연함을 최대한 활용할 수 있습니다.
