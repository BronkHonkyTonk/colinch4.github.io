이 자습서에서는 MINGW(미닉스 유사 환경용 Windows GNU 컴파일러)와 Git에 대해 다룹니다.

## MINGW란 무엇인가요?

MINGW는 Windows 운영 체제에서 Unix 환경과 유사한 도구를 지원하는 소프트웨어입니다. MINGW를 사용하면 Windows에서 Unix 스타일의 개발 환경을 만들 수 있습니다. 주로 GCC(GNU 컴파일러 콜렉션)를 사용하여 C 및 C++ 프로그램을 컴파일하기 위해 사용됩니다.

## MINGW 설치하기

MINGW를 설치하려면 공식 웹 사이트에서 다운로드할 수 있습니다. 설치 프로그램을 실행한 후 필요한 컴포넌트를 선택하여 설치합니다. 설치가 완료되면 커맨드 라인에서 `g++` 또는 `gcc`를 실행하여 컴파일러의 버전을 확인할 수 있습니다.

```sh
g++ --version
```

## Git이란 무엇인가요?

Git은 버전 관리 시스템으로, 프로젝트의 변경 사항을 추적하고 여러 명의 개발자가 협업하는 데 사용됩니다. 소스 코드의 변경 사항을 추적하고 이전 버전으로 돌아갈 수 있게 해줍니다. 또한 여러 명의 개발자가 동시에 작업할 수 있도록 지원하며, 변경 사항을 효과적으로 병합할 수 있습니다.

## Git 설치하기

Git을 설치하려면 [공식 Git 웹 사이트](https://git-scm.com/)에서 다운로드하고 설치합니다. 설치가 완료되면 커맨드 라인에서 `git --version`을 실행하여 설치된 Git의 버전을 확인할 수 있습니다.

```sh
git --version
```

MINGW 및 Git을 설치하고 나면 우리는 Windows에서 C++ 프로그램을 개발하고 Git을 사용하여 버전 관리할 수 있습니다.
