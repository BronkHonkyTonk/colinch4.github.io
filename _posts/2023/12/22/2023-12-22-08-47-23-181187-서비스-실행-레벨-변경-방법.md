리눅스에서 서비스 실행 레벨을 변경하는 방법은 `systemctl` 명령어를 사용하는 것입니다. 

서비스를 실행 레벨로 변경하려면 다음과 같은 단계를 따릅니다.

### 1. 현재 실행 레벨 확인

먼저, 현재의 실행 레벨을 확인합니다. 이를 위해서는 다음 명령어를 사용합니다.
```bash
runlevel
```

### 2. 원하는 실행 레벨에서 서비스 활성화

`systemctl enable` 명령어를 사용하여 특정 서비스를 특정 실행 레벨에서 활성화합니다.
```bash
sudo systemctl enable servicename.service
```

서비스명은 실제 서비스의 이름으로 변경해야 합니다.

### 3. 시스템 재부팅

서비스가 적절한 레벨에서 시작되도록 하려면 시스템을 재부팅해야 합니다.
```bash
sudo reboot
```

이제 원하는 실행 레벨에서 해당 서비스가 성공적으로 실행되는지 확인할 수 있습니다.

이와 같이 `systemctl` 명령어를 사용하여 리눅스에서 서비스 실행 레벨을 변경할 수 있습니다. 이를 통해 시스템의 부팅 동작을 커스터마이징하고 효율적으로 관리할 수 있습니다.
