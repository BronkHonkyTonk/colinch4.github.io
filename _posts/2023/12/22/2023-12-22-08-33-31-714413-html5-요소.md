# HTML5 요소 목록

1. [header](#header)
2. [nav](#nav)
3. [section](#section)
4. [article](#article)
5. [aside](#aside)
6. [footer](#footer)

## <a name="header">header</a>
`<header>` 요소는 섹션의 머리말을 나타냅니다. 일반적으로 제목, 로고, 검색 폼, 작성자 이름 등을 포함합니다.

## <a name="nav">nav</a>
`<nav>` 요소는 다른 페이지로의 링크 또는 사이트 내 탐색 링크를 담고 있는 섹션을 나타냅니다.

## <a name="section">section</a>
`<section>` 요소는 문서의 독립적인 섹션을 정의합니다. 주제별로 그룹핑하여 스타일링하거나 스크립팅하기 위해 사용됩니다.

## <a name="article">article</a>
`<article>` 요소는 독립적으로 구분해 배포하거나 재사용할 수 있는 콘텐츠를 나타냅니다. 블로그 글, 뉴스 기사, 포럼 게시물 등이 포함됩니다.

## <a name="aside">aside</a>
`<aside>` 요소는 주변 콘텐츠와의 관련성이 높은 콘텐츠 (광고, 사이드바, 코멘트)를 나타냅니다.

## <a name="footer">footer</a>
`<footer>` 요소는 섹션의 바닥글을 나타냅니다. 보통 저작권 정보, 연락처 정보, 사이트 맵, 사이트 제작자 등이 위치합니다.

이러한 HTML5 요소들은 웹 문서를 효과적으로 구조화하고 의미론적으로 표현하는 데 유용하게 활용될 수 있습니다.
