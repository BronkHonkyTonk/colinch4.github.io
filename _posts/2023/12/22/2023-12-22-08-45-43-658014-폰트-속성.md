웹 디자인에서 폰트는 중요한 역할을 합니다. 웹사이트의 폰트를 선택하고 스타일링하는 것은 사용자 경험과 시각적 효과에 영향을 미치는 중요한 작업입니다. CSS를 사용하여 텍스트의 폰트를 스타일링하는 방법에 대해 알아보겠습니다.

## 1. 폰트 패밀리 지정

HTML 요소에 적용되는 기본 폰트 패밀리는 다음과 같이 지정할 수 있습니다.

```css
font-family: "Times New Roman", Times, serif;
```

시스템에 지정된 폰트가 없을 때 대체할 수 있는 폰트 패밀리를 순서대로 나열합니다.

## 2. 폰트 굵기 및 스타일

폰트의 굵기와 스타일을 지정할 수 있습니다.

```css
font-weight: bold;
font-style: italic;
```

## 3. 텍스트 장식

텍스트에 밑줄이나 중간선을 추가하여 장식할 수 있습니다.

```css
text-decoration: underline;
text-decoration: line-through;
```

## 4. 폰트 크기

폰트의 크기를 픽셀, 백분율 등의 단위로 지정할 수 있습니다.

```css
font-size: 16px;
font-size: 80%;
```

## 5. 줄 간격과 문자 간격

텍스트의 줄 간격과 문자 간격을 조정할 수 있습니다.

```css
line-height: 1.5;
letter-spacing: 1px;
```

폰트 속성을 적절히 조절하여 웹사이트의 텍스트를 더 잘 보이도록 만들 수 있습니다. 폰트 스타일링은 사용자 경험을 향상시키는 데 중요한 역할을 합니다.

이상으로 CSS를 사용한 폰트 속성에 대한 간략한 소개였습니다. 더 많은 정보를 원하시는 경우 [CSS 폰트 속성](https://developer.mozilla.org/ko/docs/Web/CSS/font)에 대한 MDN 웹 문서를 참고하시기 바랍니다.
