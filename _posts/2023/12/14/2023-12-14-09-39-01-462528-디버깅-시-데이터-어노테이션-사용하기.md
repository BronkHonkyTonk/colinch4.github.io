다음은 간단한 예제를 사용하여 데이터 어노테이션의 활용 방법을 보여줍니다.

```swift
var age: Int // 변수의 데이터 타입을 설명하는 주석을 추가합니다.
// age: 사용자의 나이를 저장하는 변수

let name: String // 상수의 데이터 타입을 설명하는 주석을 추가합니다.
// name: 사용자의 이름을 저장하는 상수
```

위의 코드에서 변수 `age`와 상수 `name`에 간단한 데이터 어노테이션을 추가했습니다. 이러한 주석을 통해 코드를 읽는 사람은 각 데이터 요소의 용도와 의미를 더 쉽게 이해할 수 있습니다.

데이터 어노테이션은 코드의 가독성을 높이고 디버깅을 용이하게 만들어줍니다. 많은 개발자들이 데이터 어노테이션을 사용하여 코드를 작성하고 유지보수할 때 매우 유용하다고 평가하고 있습니다.

더 자세한 내용은 Apple 공식 문서인 "The Swift Programming Language"를 참고하시기 바랍니다.
