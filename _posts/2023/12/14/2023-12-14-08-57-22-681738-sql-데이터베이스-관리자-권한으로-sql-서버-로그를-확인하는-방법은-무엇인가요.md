1. SQL Server Management Studio(SSMS)를 엽니다.
2. "서버 유형"에서 "Database Engine"을 선택합니다.
3. 로그인 화면에서 Windows 인증 또는 SQL Server 인증을 사용하여 로그인합니다.
4. 객체 탐색기에서 "Management" 폴더를 확장하고 "SQL Server 로그"를 클릭합니다.
5. 로그 파일을 선택한 후, **오른쪽** 클릭하여 "로그 보기"를 선택하면 해당 로그 파일의 내용이 표시됩니다.

이러한 단계를 통해 SQL 데이터베이스 관리자 권한으로 SQL Server 로그를 확인할 수 있습니다.

더 자세한 내용은 Microsoft 공식 문서를 확인하시기 바랍니다.  
- Microsoft 공식 문서: https://docs.microsoft.com/en-us/sql/relational-databases/sql-server-management-studio-ssms/view-the-sql-server-log?view=sql-server-ver15
