## 객체 생성하기

Python에서 객체를 생성하는 것은 매우 간단합니다. 예를 들어, 새로운 리스트 객체를 생성하고 값을 할당하는 방법은 다음과 같습니다.

```python
# 주피터 노트북에서 객체 생성하기
my_list = [1, 2, 3, 4, 5]
```

위의 코드를 주피터 노트북 셀에 입력하여 실행하면, `my_list`라는 이름의 리스트 객체가 생성되고 값 `[1, 2, 3, 4, 5]`가 할당됩니다.

## 객체 속성 및 메서드

주피터 노트북에서 객체를 생성하면 해당 객체의 속성과 메서드를 사용하여 작업할 수 있습니다. 예를 들어, 리스트 객체의 `append` 메서드를 사용하여 값을 추가할 수 있습니다.

```python
# 주피터 노트북에서 객체 메서드 호출하기
my_list.append(6)
```

위의 코드를 실행하면 `my_list`에 `6`이 추가됩니다.

## 정리

이제 주피터 노트북에서 Python 객체를 생성하고 메서드를 호출하는 방법을 살펴보았습니다. 객체를 생성하고 관리하는 방법을 익히면 데이터 분석 및 시각화 작업을 보다 효과적으로 수행할 수 있을 것입니다.
