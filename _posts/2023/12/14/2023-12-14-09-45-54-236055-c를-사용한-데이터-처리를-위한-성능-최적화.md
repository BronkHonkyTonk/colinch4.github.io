## 데이터 구조 최적화

**데이터 구조의 선택**은 데이터 처리 성능에 큰 영향을 미칩니다. 예를 들어, 배열은 메모리에 연속적으로 저장되어 효율적인 액세스를 제공하므로 데이터 접근 시간을 최소화합니다. 또한, 불필요한 메모리 할당 및 복사를 피하기 위해 **적절한 데이터 구조**를 선택해야 합니다.

다음은 배열을 사용하여 데이터 구조를 최적화하는 예시입니다.
```c++
#include <vector>

// Vector를 사용한 데이터 저장
std::vector<int> data;
data.reserve(1000000); // 불필요한 재할당을 피하기 위해 초기 용량을 할당

// 데이터 액세스
for (int i = 0; i < data.size(); ++i) {
    // 데이터 처리
}
```

## 알고리즘 최적화

**알고리즘의 선택**은 데이터 처리 과정에서 상당히 중요합니다. 데이터를 효율적으로 처리하기 위해 **적절한 알고리즘**을 사용해야 합니다. 시간복잡도와 공간복잡도를 고려하여 가장 효율적인 알고리즘을 선택하는 것이 필요합니다.

다음은 선택 정렬을 사용하여 데이터 처리를 최적화하는 예시입니다.
```c++
// 선택 정렬 알고리즘
void selectionSort(std::vector<int>& data) {
    int n = data.size();
    for (int i = 0; i < n - 1; ++i) {
        int minIndex = i;
        for (int j = i + 1; j < n; ++j) {
            if (data[j] < data[minIndex]) {
                minIndex = j;
            }
        }
        std::swap(data[i], data[minIndex]);
    }
}
```
