1. **Xcode**: Core ML 모델을 만들고 통합하기 위해서는 Xcode 개발 환경이 필요합니다. 최신 버전의 Xcode를 설치해야 합니다.

2. **모델**: Core ML은 다양한 머신러닝 프레임워크에서 만들어진 모델을 지원합니다. 따라서 구축하려는 모델을 선택하고 해당 모델을 머신러닝 프레임워크로부터 변환해야 합니다.

3. **모델 변환 도구**: 선택한 머신러닝 프레임워크에서 Core ML 형식으로 모델을 변환해주는 변환 도구가 필요합니다. 예를 들어, TensorFlow에서는 TensorFlow 변환 도구를 사용할 수 있습니다.

4. **학습된 가중치**: Core ML 모델을 구축하려면 머신러닝 모델에 대한 학습된 가중치 또는 파라미터가 필요합니다.

이러한 요구 사항을 충족시키면 Core ML 모델을 구축하여 iOS 앱에서 머신러닝을 활용할 수 있습니다.
