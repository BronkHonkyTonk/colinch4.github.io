유닉스/리눅스 시스템 호출을 사용하여 파일을 이동시키려면 `mv` 명령어를 사용해야 합니다. `mv` 명령어는 파일을 다른 디렉터리로 이동하거나 파일의 이름을 변경하는 데 사용됩니다.

아래는 두 가지 주요 사용 사례에 대한 예제와 함께 `mv` 명령어를 사용하는 방법입니다.

### 파일을 다른 디렉터리로 이동하기
```bash
mv /원본/파일/경로 /대상/디렉터리/경로
```

예를 들어, `example.txt` 파일을 `/home/user/documents`로 이동하려면 다음과 같이 입력하면 됩니다.
```bash
mv example.txt /home/user/documents
```

### 파일 이름 변경하기
```bash
mv 이전_파일_이름 새로운_파일_이름
```

예를 들어, `oldfile.txt` 파일을 `newfile.txt`로 이름을 변경하려면 다음과 같이 입력하면 됩니다.
```bash
mv oldfile.txt newfile.txt
```

이렇게 하면 유닉스/리눅스 시스템 호출을 사용하여 파일을 이동하거나 이름을 변경할 수 있습니다. 이러한 명령어는 파일 및 디렉터리 관리를 간편하게 해주는 강력한 도구입니다.

자세한 정보는 [mv 명령어 매뉴얼 페이지](https://linux.die.net/man/1/mv)를 참고하시기 바랍니다.
