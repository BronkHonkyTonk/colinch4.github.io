트위터에서 트윗을 북마크하고 싶을 때가 있습니다. 북마크를 통해 트윗을 나중에 다시 보거나 나눌 수 있습니다. 이 기능을 iOS 앱에서 활용하는 방법에 대해 알아보겠습니다.

## 1. 트윗 북마크하기

트위터 iOS 앱에서 트윗을 북마크하는 것은 매우 간단합니다. 트윗을 선택한 후, 우측 하단에 위치한 공유 버튼을 누릅니다. 그런 다음, "북마크" 옵션을 선택하면 해당 트윗이 북마크됩니다.

```swift
// Swift 코드 예시
func bookmarkTweet() {
    // 트윗을 북마크하는 코드 작성
}
```

## 2. 북마크된 트윗 보기

북마크한 트윗을 보려면 트위터 iOS 앱의 하단 탭 바에서 "북마크" 항목을 선택합니다. 그러면 북마크된 모든 트윗을 확인할 수 있습니다.

## 3. 북마크 관리

트윗을 북마크하는 것 외에도, 북마크된 트윗을 관리할 수 있습니다. 트위터 iOS 앱에서 "북마크" 화면에서 트윗을 삭제하거나, 공유하거나, 다시 보내는 등의 작업을 수행할 수 있습니다.

트위터의 트윗 북마크 기능을 활용하여 중요한 정보나 관심 있는 내용을 쉽게 저장하고 다시 찾을 수 있습니다.
