자바 프로그래밍 언어에서 배열(Array)은 동일한 유형의 데이터를 저장하는 데 사용되는 자료구조입니다. 배열은 **인덱스를 사용하여 개별 요소에 접근**할 수 있고, **배열 길이는 고정**되어 있습니다. 

## 배열의 선언과 초기화

자바에서 배열을 선언하고 초기화 하는 방법은 다음과 같습니다.

```java
// 데이터 타입이 int이고 길이가 5인 배열 선언
int[] numbers = new int[5];

// 배열 요소에 값 할당
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;
```

## 배열의 활용

배열은 여러 데이터를 다룰 때 유용하게 활용될 수 있습니다. **반복문을 활용해 배열의 모든 요소에 접근**하고 연산을 수행하거나, **배열을 파라미터로 갖는 메소드를 정의**하여 배열을 효과적으로 활용할 수 있습니다.

## 다차원 배열

자바에서는 **다차원 배열**을 생성하여 복잡한 데이터를 다룰 수 있습니다. 다차원 배열을 다루는 방법을 확인하는 것도 중요합니다.

이러한 배열 자료구조는 자바 프로그래밍에서 빈번하게 활용되므로, **배열에 대한 이해가 필수**입니다.
