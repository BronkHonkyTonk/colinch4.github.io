## 1. TCP 사용

TCP(Transmission Control Protocol)는 데이터를 전송할 때 일련의 확인 메시지를 사용하여 데이터 유실을 방지하는 프로토콜입니다. TCP를 사용하면 데이터가 순서대로 전달되고, 유실된 데이터를 재전송하여 안정적인 데이터 전송이 가능합니다.

```swift
let task = URLSession.shared.dataTask(with: url) { (data, response, error) in
    // TCP를 사용하여 안정적인 데이터 전송
}
task.resume()
```
