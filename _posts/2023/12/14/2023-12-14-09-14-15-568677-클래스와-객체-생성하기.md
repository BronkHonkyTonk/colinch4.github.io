Kotlin은 자바와 완벽하게 상호운용이 가능한 프로그래밍 언어로, 클래스와 객체를 생성하는 방법을 살펴보겠습니다.

## 클래스 정의하기

```kotlin
class Car {
   // 속성과 함수 정의
}
```

위의 코드는 `Car` 클래스를 정의하는 예시입니다. 

## 객체 생성하기

```kotlin
val myCar = Car()
```

위의 코드는 `Car` 클래스의 인스턴스를 생성하는 예시입니다.
