1. 먼저 SQL Server Management Studio(SSMS)에 데이터베이스 관리자로 로그인합니다.
2. '보안' 노드를 확장하고 '로그인'을 마우스 오른쪽 버튼으로 클릭한 다음 '새 로그인'을 선택합니다.
3. 새 로그인 대화상자에서 새 로그인의 이름을 지정하고, 'SQL Server 인증'을 선택한 후 암호를 입력합니다.
4. '서버 역할' 탭으로 이동하여 해당 사용자에게 적절한 서버 역할을 할당합니다.
5. 필요에 따라 '사용자 매핑' 탭으로 이동하여 해당 로그인에게 데이터베이스 액세스 및 역할을 부여합니다.
6. 변경을 적용하기 위해 '확인'을 클릭합니다.

이렇게 하면 SQL Server의 데이터베이스 관리자 권한으로 SQL 서버 보안 설정을 할 수 있습니다.
