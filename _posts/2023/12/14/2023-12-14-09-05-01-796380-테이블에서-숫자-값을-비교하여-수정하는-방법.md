SQL을 사용하여 테이블에서 숫자 값을 비교하고 필요한 경우 수정하는 방법에 대해 알아봅시다.

### 숫자 값을 비교하기

테이블에서 숫자 값을 비교하려면 비교 연산자를 사용합니다. 일반적인 비교 연산자로는 `<` (작다), `>` (크다), `<=` (작거나 같다), `>=` (크거나 같다), `=` (같다), `!=` (같지 않다) 등이 있습니다.

예를 들어, `salary` 열이 50000보다 큰 모든 레코드를 선택하려면 다음과 같이 SQL을 작성할 수 있습니다.

```sql
SELECT * FROM employees WHERE salary > 50000;
```

### 숫자 값을 수정하기

선택한 레코드의 숫자 값을 수정하려면 `UPDATE` 문을 사용합니다. 이때 `SET` 절을 사용하여 새로운 값을 지정합니다. 

예를 들어, `salary`가 50000보다 큰 모든 레코드의 `salary` 값을 55000으로 업데이트하려면 다음과 같이 SQL을 작성할 수 있습니다.

```sql
UPDATE employees SET salary = 55000 WHERE salary > 50000;
```

위의 예제에서 `employees`는 테이블 이름, `salary`는 열 이름을 나타냅니다.

### 결론

SQL에서 숫자 값을 비교하고 수정하는 방법을 살펴보았습니다. 이러한 기본적인 개념을 활용하여 테이블에서 원하는 조건에 따라 숫자 값을 비교하고 수정할 수 있습니다.
