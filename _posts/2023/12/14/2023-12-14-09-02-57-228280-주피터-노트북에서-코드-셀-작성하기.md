## 코드 셀 작성하기

주피터 노트북에서 코드 셀을 작성하려면 다음 단계를 따르세요:

1. **셀 추가**: 새 코드 셀을 추가하려면 메뉴 바에서 "Insert"를 클릭한 후 "Insert Cell Below"를 선택하거나, 키보드 단축키 `B`를 누르세요.
2. **코드 입력**: 추가된 코드 셀에 Python 코드를 입력하세요. 다음은 간단한 예제입니다:
    ```python
    # 간단한 덧셈 예제
    a = 3
    b = 5
    result = a + b
    print(result)
    ```

## 코드 실행하기

셀에 코드를 입력한 후에는 해당 코드를 실행할 수 있습니다. 코드 셀을 실행하는 방법은 다음과 같습니다:

- **단축키**: 코드 셀을 실행하려면 해당 셀을 선택한 후 `Shift + Enter`를 누르세요.
- **메뉴 바**: 코드 셀을 실행하려면 메뉴 바에서 "Cell"을 클릭한 후 "Run Cells"를 선택하세요.

이렇게 하면 코드가 실행되고 결과가 출력됩니다.

위의 단계를 따라하면 주피터 노트북에서 코드 셀을 작성하고 실행할 수 있습니다.
