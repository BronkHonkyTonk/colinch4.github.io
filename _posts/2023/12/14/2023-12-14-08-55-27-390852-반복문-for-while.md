# for-in 반복문
```swift
for item in collection {
    // 동작 수행
}
```
**for-in 반복문**은 주어진 컬렉션의 각 요소에 대해 지정된 동작을 반복적으로 수행합니다. 예를 들어, 배열이나 범위에 있는 요소에 접근하여 작업을 수행하는 데 사용됩니다.

# while 반복문
```swift
while condition {
    // 동작 수행
}
```
**while 반복문**은 주어진 조건이 참인 동안 여러 번 동작을 반복적으로 실행합니다. 조건이 거짓이 되면 반복문이 종료됩니다.

이러한 반복문을 사용하여 코드를 보다 효율적으로 작성할 수 있으며, 작업을 반복해서 수행해야 하는 경우 유용하게 활용할 수 있습니다.
