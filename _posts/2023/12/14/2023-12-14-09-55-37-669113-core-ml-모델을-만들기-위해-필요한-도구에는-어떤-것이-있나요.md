1. **Machine Learning 모델:** Core ML 모델을 만들기 위해서는 먼저 모델을 학습시켜야 합니다. 이를 위해서는 주로 Python 기반의 라이브러리인 TensorFlow, Keras, PyTorch 등을 사용하여 모델을 개발하고 학습시킵니다.

2. **Core ML Tools:** Core ML 모델은 `.mlmodel` 형식으로 제공되며, 이를 생성하고 변환하기 위한 Core ML Tools가 필요합니다. 이를 통해 기존의 머신러닝 모델을 Core ML 모델로 변환할 수 있습니다.

3. **Xcode:** iOS 앱에 Core ML 모델을 통합하려면 Xcode가 필요합니다. Xcode는 Core ML 모델을 앱에 통합하고 모델을 사용하여 인공지능 애플리케이션을 빌드하는 데 사용됩니다.

4. **Apple Developer 계정:** Core ML을 사용하여 앱을 빌드하고 배포하기 위해서는 Apple Developer 계정이 필요합니다.

Core ML 모델을 만들기 위해서는 이러한 도구와 리소스들을 숙지하고 활용할 수 있어야 합니다.
