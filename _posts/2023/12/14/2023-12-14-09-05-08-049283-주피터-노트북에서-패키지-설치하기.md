파이썬을 사용하다보면 외부 패키지를 설치하여 사용해야 할 때가 많습니다. 주피터 노트북에서는 패키지를 명령어를 통해 간편하게 설치할 수 있습니다. 이 글에서는 주피터 노트북에서 **패키지를 설치하는 방법**을 알아보겠습니다.

## 패키지 설치하기

주피터 노트북에서 패키지를 설치하려면 `!` 기호를 사용하여 셀에 명령어를 입력합니다.
먼저, 다음과 같이 `!pip install` 명령어를 사용하여 원하는 패키지를 설치할 수 있습니다.

```python
!pip install 패키지명
```

## 예시

아래는 `matplotlib` 패키지를 주피터 노트북에서 설치하는 예시입니다.

```python
!pip install matplotlib
```
