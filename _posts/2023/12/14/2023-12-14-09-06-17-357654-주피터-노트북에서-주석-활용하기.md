주석은 코드를 이해하고 유지 보수하는 데 도움이 되는 중요한 도구입니다. 주피터 노트북에서는 코드 셀과 마크다운 셀에 모두 주석을 작성할 수 있습니다.

## 코드 셀에 주석 작성하기

Python 코드 셀에서 주석은 `#` 기호를 사용하여 작성됩니다. 코드 셀에서 주석을 작성하면 해당 코드의 실행을 방해하지 않으면서 코드를 설명할 수 있습니다. 

예시:

```python
# 이 코드는 두 수를 더하는 예제입니다
a = 5
b = 3
c = a + b  # 두 수를 더한 결과를 c에 저장
print(c)  # c 값을 출력합니다
```

## 마크다운 셀에 주석 작성하기

마크다운 셀에서는 주석을 작성하여 코드 블록이나 설명을 추가로 제공할 수 있습니다. 주석은 `<!-- 주석 내용 -->`와 같은 형식으로 작성됩니다.

예시:

```markdown
<!-- 이 부분은 마크다운 셀에서의 주석입니다 -->
이 문장은 일반적인 마크다운 텍스트입니다.
```

주석은 코드의 가독성과 이해도를 높이는 데 도움이 됩니다. 주피터 노트북에서 주석을 적절히 활용하여 코드를 보기 좋고 이해하기 쉽게 작성할 수 있습니다.
