### 관리자 권한 확인
먼저, 해당 데이터베이스 서버에 관리자 권한이 있어야 합니다. 관리자 권한이 없다면 시스템 관리자에게 요청해야 합니다.

### SQL Server Management Studio 열기
SQL Server Management Studio(SSMS)를 열고 관리자 권한으로 로그인합니다.

### 새로운 데이터베이스 생성
SSMS에서 "데이터베이스"를 마우스 오른쪽 버튼으로 클릭하고 "새 데이터베이스"를 선택합니다. 여기서 데이터베이스의 세부 정보를 입력하고 "확인"을 클릭하여 새 데이터베이스를 생성합니다.

### 데이터베이스 속성 설정
생성된 데이터베이스의 속성을 필요에 따라 구성합니다. 이는 데이터베이스의 파일 경로, 크기, 로그 백업 등을 포함합니다.

이것으로 데이터베이스 서버 인스턴스를 생성할 수 있습니다.
