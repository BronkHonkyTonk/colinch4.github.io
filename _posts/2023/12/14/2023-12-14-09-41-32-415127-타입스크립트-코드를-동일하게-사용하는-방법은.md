1. **타입스크립트 프로젝트 초기화**: 일단 타입스크립트 프로젝트를 초기화합니다. 이를 위해서는 다음 명령어를 실행합니다.
   ```sh
   npm init -y
   ```

2. **타입스크립트 패키지 설치**: 타입스크립트 패키지를 설치합니다.
   ```sh
   npm install typescript
   ```

3. **타입스크립트 설정 파일 생성**: `tsconfig.json` 파일을 프로젝트 루트 디렉토리에 생성합니다. 이 파일은 타입스크립트 컴파일러에 대한 설정을 포함합니다.

4. **코드 작성**: 공유하려는 타입스크립트 코드를 작성합니다.

5. **코드 컴파일**: 타입스크립트 코드를 ECMAScript 버전으로 변환하려면 다음 명령어를 실행합니다.
   ```sh
   npx tsc
   ```

이제 이 코드를 사용할 수 있습니다. 이 방법을 통해 타입스크립트 코드를 공유하고 동일하게 사용할 수 있게 됩니다.

위의 단계에서 언급된 대로 `npm`을 사용하였습니다. `yarn`을 사용하고 싶다면, `npm install` 대신 `yarn add` 명령어를 사용하면 됩니다.
