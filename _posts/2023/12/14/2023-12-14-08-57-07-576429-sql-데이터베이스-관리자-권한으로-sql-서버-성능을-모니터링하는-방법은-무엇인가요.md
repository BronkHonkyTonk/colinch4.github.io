## 쿼리 실행 계획 확인
SQL Server Management Studio(SSMS)에서 쿼리 실행 계획을 확인하여 쿼리의 성능 향상을 위한 인덱스나 조인 방법 등을 분석할 수 있습니다. 쿼리를 작성한 후, '쿼리' 메뉴에서 '쿼리 실행 계획'을 클릭하여 확인할 수 있습니다.

```sql
-- 예시: 쿼리 실행 계획 확인
SELECT * FROM YourTableName
```

## 성능 모니터링
SSMS의 '성능 모니터링' 기능을 사용하여 SQL Server의 성능을 실시간으로 모니터링할 수 있습니다. 이를 통해 데이터베이스의 자원 사용률 및 성능을 실시간으로 확인할 수 있습니다.

## 시스템 뷰 및 동적 관리 뷰 활용
시스템 뷰와 동적 관리 뷰를 쿼리하여 SQL Server의 성능 관련 정보를 확인할 수 있습니다. 예를 들어, sys.dm_os_performance_counters를 사용하여 데이터베이스 서버의 성능 카운터를 확인할 수 있습니다.

이러한 방법들을 활용하여 SQL 데이터베이스의 성능을 모니터링할 수 있으며, 각각의 기능을 사용하여 성능에 영향을 미치는 요소들을 식별하고 최적화할 수 있습니다.
