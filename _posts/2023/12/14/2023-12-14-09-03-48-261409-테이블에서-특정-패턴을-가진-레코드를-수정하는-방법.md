```sql
UPDATE 테이블명
SET 열이름 = '새로운값'
WHERE 조건;
```

여기서 '테이블명'은 해당 테이블의 이름, '열이름'은 수정하려는 열의 이름, '새로운값'은 해당 열에 적용할 새로운 값, '조건'은 특정 패턴이 있는 레코드를 식별하는 조건입니다. 

예를 들어, '이름' 열에서 '홍길동'을 '임꺽정'으로 바꾸고 싶다면 다음과 같이 작성할 수 있습니다.

```sql
UPDATE 회원
SET 이름 = '임꺽정'
WHERE 이름 = '홍길동';
```
