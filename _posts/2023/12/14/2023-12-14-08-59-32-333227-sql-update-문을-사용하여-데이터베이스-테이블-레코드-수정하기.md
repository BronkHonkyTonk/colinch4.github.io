데이터베이스 테이블에서 레코드를 수정하는 것은 매우 일반적인 작업입니다. SQL UPDATE 문은 테이블의 레코드를 수정하는 데 사용됩니다. 이 문을 사용하여 레코드의 값을 업데이트하거나 변경할 수 있습니다.

## UPDATE 문의 구문

아래는 SQL UPDATE 문의 기본 구문입니다.

```sql
UPDATE 테이블명
SET 컬럼1 = 값1, 컬럼2 = 값2, ...
WHERE 조건;
```

- `UPDATE`: 레코드를 수정할 때 사용하는 키워드
- `테이블명`: 레코드를 수정할 데이터베이스 테이블의 이름
- `SET`: 수정할 컬럼과 값을 지정하는 키워드
- `컬럼1 = 값1, 컬럼2 = 값2, ...`: 수정할 컬럼과 해당 값을 지정
- `WHERE`: 수정할 레코드를 선택하기 위한 조건을 지정하는 키워드
- `조건`: 수정할 레코드를 선택하기 위한 조건을 지정

## 예제

다음은 `employees` 테이블에서 `id`가 1인 레코드의 `name`을 'John'으로, `age`를 30으로 수정하는 예제입니다.

```sql
UPDATE employees
SET name = 'John', age = 30
WHERE id = 1;
```

이렇게 하면 `employees` 테이블에서 `id`가 1인 레코드의 `name`이 'John'으로, `age`가 30으로 수정됩니다.

위와 같이 SQL UPDATE 문을 사용하여 데이터베이스 테이블의 레코드를 효과적으로 수정할 수 있습니다.

## 결론

SQL UPDATE 문은 데이터베이스 테이블에서 레코드를 수정하는 데 사용되며, 매우 유용한 기능입니다. 이 문을 사용하여 데이터의 정확성과 일관성을 유지하고 필요에 따라 데이터를 업데이트할 수 있습니다.
