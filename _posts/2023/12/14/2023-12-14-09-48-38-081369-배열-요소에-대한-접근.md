## 1. 배열 요소에 대한 접근

C++에서 배열의 요소에 접근하려면 배열 이름 뒤에 대괄호를 사용하여 해당 요소의 인덱스를 지정합니다. 예를 들어, 다음과 같이 5개의 정수를 저장하는 배열을 만들고, 각 요소에 접근할 수 있습니다.

```cpp
int arr[5] = {10, 20, 30, 40, 50};
int firstElement = arr[0]; // 첫 번째 요소에 접근
int thirdElement = arr[2]; // 세 번째 요소에 접근
```

위의 코드에서 `arr[0]`은 배열 `arr`의 첫 번째 요소에 접근하고, `arr[2]`는 세 번째 요소에 접근합니다.

## 2. 배열 요소 수정

배열 요소를 수정하려면 해당 요소에 새 값으로 할당하면 됩니다.

```cpp
arr[1] = 200; // 두 번째 요소를 수정하여 200으로 설정
```

위의 코드에서 `arr[1]`은 두 번째 요소를 나타내며, 새로운 값으로 수정되었습니다.

C++에서 배열 요소에 접근하고 수정하는 방법을 사용하여 배열을 효과적으로 다룰 수 있습니다.
