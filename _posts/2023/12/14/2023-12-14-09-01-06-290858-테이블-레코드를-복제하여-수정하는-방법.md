먼저, 원본 레코드를 복제하여 새로운 레코드를 만듭니다. 다음 SQL 문은 원본 레코드를 복제하여 새로운 레코드를 만드는 예시입니다.

```sql
INSERT INTO 테이블명 (컬럼1, 컬럼2, 컬럼3, ...)
SELECT 컬럼1, 컬럼2, 컬럼3, ...
FROM 테이블명
WHERE 조건;
```

그런 다음, 새로 만든 레코드를 수정할 수 있습니다.

```sql
UPDATE 테이블명
SET 컬럼1 = '수정값1',
    컬럼2 = '수정값2',
    ...
WHERE 조건;
```

위의 SQL 쿼리에서, 테이블명은 해당 테이블의 이름으로 바꿔야 합니다. 컬럼1, 컬럼2 등은 실제 컬럼 이름으로 대체되어야 하며, 수정값1, 수정값2는 각각 해당 컬럼에 적용할 수정값으로 바꿔주어야 합니다.

이러한 SQL 쿼리를 사용하여 테이블 레코드를 복제하고 수정할 수 있습니다.
