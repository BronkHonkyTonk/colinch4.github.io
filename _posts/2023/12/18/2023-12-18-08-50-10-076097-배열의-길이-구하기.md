다음은 몇 가지 프로그래밍 언어에서 배열의 길이를 구하는 방법에 대한 예시입니다.

## JavaScript
JavaScript에서 배열의 길이를 구하는 방법은 `length` 속성을 사용하는 것입니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```javascript
const arr = [1, 2, 3, 4, 5];
const length = arr.length;
console.log(length); // 출력: 5
```

## Python
Python에서는 `len()` 함수를 사용하여 배열 또는 리스트의 길이를 구할 수 있습니다. 예를 들어,

```python
arr = [1, 2, 3, 4, 5]
length = len(arr)
print(length)  # 출력: 5
```

## Java
Java에서는 배열의 길이를 `length` 속성을 통해 구할 수 있습니다.

```java
int[] arr = {1, 2, 3, 4, 5};
int length = arr.length;
System.out.println(length); // 출력: 5
```

다양한 프로그래밍 언어에서 배열의 길이를 구하기 위한 다양한 방법이 있지만, 위의 예시들은 일반적으로 사용되는 방법들입니다.
