다음은 stdout을 사용하는 간단한 예시입니다:

```c
#include <stdio.h>

int main() {
    fprintf(stdout, "Hello, stdout!\n");
    return 0;
}
```

이 코드는 "Hello, stdout!"을 화면에 출력합니다. 출력을 파일로 리디렉션하거나 다른 프로그램으로 파이핑하는 등 다양한 용도로 stdout을 사용할 수 있습니다.
