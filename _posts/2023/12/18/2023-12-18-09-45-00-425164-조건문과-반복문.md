## 조건문
조건문은 주어진 조건이 참일 때 특정한 코드 블록을 실행합니다. 주로 `if`, `else if`, `else` 키워드를 사용하여 구성됩니다.

```javascript
if (조건) {
  // 조건이 참일 때 실행될 코드
} else if (다른 조건) {
  // 다른 조건이 참일 때 실행될 코드
} else {
  // 위의 조건들이 모두 거짓일 때 실행될 코드
}
```

## 반복문
반복문은 주어진 조건이 참일 경우 코드 블록을 반복해서 실행합니다. 주로 `for`, `while`, `do-while`을 사용하여 구성됩니다.

### for문
```javascript
for (초깃값; 조건; 증감식) {
  // 조건이 참일 때 실행될 코드
}
```

### while문
```javascript
while (조건) {
  // 조건이 참일 때 실행될 코드
}
```

### do-while문
```javascript
do {
  // 일단 실행하고 조건을 검사
} while (조건);
```

조건문과 반복문을 잘 활용하면 효과적인 프로그램을 작성할 수 있습니다. 코드를 보다 간결하고 효율적으로 작성하기 위해 이러한 개념들을 잘 숙지하는 것이 중요합니다.
