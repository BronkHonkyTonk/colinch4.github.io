예를 들어, 버튼 클릭 이벤트를 처리하는 람다식은 다음과 같이 작성할 수 있습니다:

```csharp
button1.Click += (s, e) => {
    // 버튼 클릭 이벤트 처리 로직
};
```

위 코드에서 `(s, e)`는 이벤트 핸들러의 매개변수를 나타내며, 중괄호 내부에는 해당 이벤트가 발생했을 때 실행될 코드를 작성합니다.

이와 같이 람다식을 사용하면 별도의 메서드를 정의하지 않고도 간단하게 이벤트를 처리할 수 있습니다.
