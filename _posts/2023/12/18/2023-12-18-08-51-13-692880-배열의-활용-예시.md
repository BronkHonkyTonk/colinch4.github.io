다음은 자바스크립트를 사용하여 배열을 선언하고 활용하는 예시입니다:

```javascript
// 배열 선언
let users = ['John', 'Jane', 'Bob', 'Alice'];

// 배열 요소에 접근
console.log(users[0]); // 'John' 출력
console.log(users[2]); // 'Bob' 출력

// 배열 길이 확인
console.log(users.length); // 4 출력

// 배열에 요소 추가
users.push('Eve'); // ['John', 'Jane', 'Bob', 'Alice', 'Eve'] 출력

// 배열 순회
for (let i = 0; i < users.length; i++) {
  console.log(users[i]); // 모든 사용자 출력
}
```

이러한 배열 활용 예시를 통해 백엔드에서 데이터를 구조화하고 관리하는 데 어떻게 배열을 사용할 수 있는지 이해할 수 있습니다.
