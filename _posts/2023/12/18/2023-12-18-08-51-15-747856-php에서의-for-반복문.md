PHP에서 for 반복문은 특정 횟수만큼 코드 블록을 반복 실행하기 위해 사용됩니다. 

## 기본 문법

```php
for ($i = 0; $i < 5; $i++) {
    // 반복 실행할 코드 블록
}
```

위의 예제에서 `$i = 0`은 초기화 구문으로, 반복문이 시작할 때 한 번 실행됩니다. 그 후에는 `$i < 5`로 지정된 조건이 참이면 코드 블록을 실행하고, `$i++`는 반복 실행 후에 변수 `$i`를 1씩 증가시킵니다.

`for`문 안에 다른 변수를 사용할 수도 있으며, 배열과 같은 데이터 구조를 반복할 때 유용하게 사용됩니다.

## 반복 횟수 제어

`for`문은 초기화 구문, 조건문, 증감식으로 구성되어 있어 반복 횟수를 자유롭게 제어할 수 있습니다. 이를 통해 특정 조건에 따라 다양한 반복 동작을 수행할 수 있습니다.

## 예제

아래는 1부터 10까지의 숫자를 출력하는 예제입니다.

```php
for ($i = 1; $i <= 10; $i++) {
    echo $i . "<br>";
}
```

위의 예제는 `$i`가 1부터 10까지 증가하면서 각 숫자를 출력합니다. 

`for`문은 코드를 간결하게 작성할 수 있는 장점이 있으며, 특히 일정한 규칙에 따라 연속적인 작업을 수행해야할 때 유용하게 활용됩니다.

`for` 반복문을 사용하여 반복 작업을 수행할 때는 초기화, 조건, 증감식을 잘 조합해 원하는 동작을 제어할 수 있도록 유의해야합니다.
