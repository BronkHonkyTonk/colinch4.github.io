```c
int a = 10;
int b = 5;
int sum = a + b;  // 더하기 연산
int difference = a - b;  // 빼기 연산
int product = a * b;  // 곱하기 연산
int quotient = a / b;  // 나누기 연산
int remainder = a % b;  // 나머지 연산
```

더하기와 빼기 연산은 각각 두 수를 더하거나 빼는 데 사용됩니다. 곱하기 연산은 두 수를 곱하고, 나누기 연산은 한 수를 다른 수로 나눈 결과를 반환합니다. 나머지 연산은 두 수를 나눈 후의 나머지 값을 반환합니다.

산술 연산자는 C 언어뿐만 아니라 다른 프로그래밍 언어에서도 사용되며, 프로그램에서 숫자를 다룰 때 기본적으로 필요한 연산자입니다.
