C 프로그래밍에서 `printf` 함수는 화면에 텍스트를 출력하는 데 사용됩니다. 이 함수는 서식 지정자를 사용하여 변수의 값을 특정 형식에 맞게 출력할 수 있습니다.

## 기본 사용법

`printf` 함수는 다음과 같은 형식으로 사용됩니다.

```c
printf("형식 지정자", 변수);
```

이때, `"형식 지정자"`는 출력할 내용의 형식을 지정하는데 사용되며, `변수`는 출력할 값을 가진 변수입니다.

예를 들어, 정수를 출력하려면 `%d` 형식 지정자를 사용하고, 실수를 출력하려면 `%f` 형식 지정자를 사용합니다.

## 예제

다음은 `printf` 함수의 간단한 예제입니다.

```c
#include <stdio.h>

int main() {
    int num = 10;
    printf("정수 값: %d\n", num);

    float val = 3.14;
    printf("실수 값: %f\n", val);

    return 0;
}
```

위 예제에서는 먼저 정수 변수 `num`의 값을 정수 형식 지정자 `%d`를 사용하여 출력하고, 그 다음으로 실수 변수 `val`의 값을 실수 형식 지정자 `%f`를 사용하여 출력합니다.

`printf` 함수를 사용하면 변수의 값이나 특정 텍스트를 포맷에 맞게 출력할 수 있으며, 이는 C 프로그래밍에서 매우 유용하게 사용됩니다.
