예를 들어, 다음과 같이 변수에 값을 할당할 수 있습니다.

```c
int x;
x = 5;
```

위의 예제에서는 변수 x에 5라는 값을 할당하고 있습니다. C 언어에서는 이런 방식으로 변수에 값을 할당하고 사용할 수 있습니다.

할당 연산자를 사용하여 변수에 값을 할당할 때는 변수의 데이터 타입과 일치하는 값을 할당해야 합니다. 타입 불일치 문제는 프로그램 오류의 원인이 될 수 있으므로 주의해야 합니다.

할당 연산자를 사용하면 변수에 값을 저장하거나 변경할 수 있으며, 프로그램의 다양한 기능을 구현할 수 있습니다.
