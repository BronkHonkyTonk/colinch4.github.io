간단한 예를 들어 설명하면, 다음과 같습니다. 

```python
def add(a, b):  # a, b는 매개변수
    return a + b

result = add(3, 5)  # 3, 5는 인수
``` 

위의 예제에서 `add` 함수를 정의할 때, `a`와 `b`는 매개변수로 사용되었고, 함수를 호출할 때 전달된 3과 5는 각각 `a`와 `b`에 대입되는 인수로 사용되었습니다.

이러한 매개변수와 인수의 개념을 이해하고 활용하면 함수를 보다 유연하게 활용할 수 있습니다.
