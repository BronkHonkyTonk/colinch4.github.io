## 매개변수
매개변수는 함수나 메서드의 정의에서 사용되는 변수를 가리킵니다. 함수나 메서드를 정의할 때, 매개변수를 선언하여 함수가 실행될 때 필요한 값을 받을 수 있습니다. 

```python
def add(a, b):
    return a + b
```

위의 예시에서 `a`와 `b`는 함수 `add`의 매개변수입니다.

## 인수
인수는 함수를 호출할 때 전달되는 값 혹은 표현식을 의미합니다. 함수를 호출할 때, 매개변수에 전달할 값을 지정합니다.

```python
result = add(3, 5)
```

위의 예시에서 `3`과 `5`는 함수 `add`에 전달되는 인수입니다.

매개변수와 인수는 함수나 메서드를 사용할 때 중요한 부분이며, 올바른 값을 전달하여 원하는 작업을 수행할 수 있도록 해줍니다.
