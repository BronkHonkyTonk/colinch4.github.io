## 1. 소개

PHP는 문자열을 다루는데 매우 강력한 기능을 제공한다. **반복문**을 사용하여 문자열을 조작하면 매우 효율적으로 원하는 결과를 얻을 수 있다. 이 글에서는 PHP의 반복문을 사용하여 문자열을 조작하는 간단한 예시를 살펴볼 것이다.

## 2. 문자열 반복문 조작 예시

아래 예시는 문자열을 반복문을 사용하여 조작하는 간단한 PHP 코드이다.

```php
<?php
$string = "Hello";
$reversedString = "";
for ($i = strlen($string) - 1; $i >= 0; $i--) {
    $reversedString .= $string[$i];
}
echo $reversedString; // 출력 결과: olleH
?>
```

위 코드는 "Hello"라는 문자열을 반복문을 사용하여 역순으로 변경하는 예시이다. 

## 3. 결론

PHP에서 반복문을 사용하여 문자열을 조작하는 것은 매우 유용하다. 이를 통해 문자열을 뒤집는 등 다양한 조작이 가능하며, 효과적인 문자열 처리를 할 수 있다.
