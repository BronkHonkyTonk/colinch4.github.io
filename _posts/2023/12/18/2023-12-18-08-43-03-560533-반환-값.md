함수가 실행을 완료하고 반환하는 값은 **반환 값(return value)** 이라고 합니다. 함수가 호출될 때, 해당 함수의 코드 블록이 실행되며, 그 결과로 하나의 값이 반환됩니다. 이 값은 함수를 호출한 쪽으로 반환되어 후속 처리에 사용됩니다.

```c
int add(int a, int b) {
    return a + b;
}
```

위의 코드에서 `add` 함수는 `a`와 `b`를 더한 결과인 `a + b`를 반환합니다. 이 값을 함수를 호출한 곳에서 사용할 수 있습니다.

반환 값은 함수가 성공적으로 수행되었는지 여부와 수행 결과를 나타내는 데 중요한 역할을 합니다. 함수의 반환 값을 적절히 활용하는 것은 프로그램의 효율성을 향상시키는 데 도움이 됩니다.
