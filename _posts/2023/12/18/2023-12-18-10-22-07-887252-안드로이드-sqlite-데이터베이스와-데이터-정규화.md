## SQLite 데이터베이스란?

SQLite는 경량의 디스크 기반 데이터베이스 시스템으로, 안드로이드 플랫폼에서 널리 사용됩니다. 안드로이드에서 SQLite를 사용하면 구조화된 데이터를 저장하고 관리하는 데 효율적입니다. **SQLiteOpenHelper** 클래스를 사용하여 데이터베이스를 생성하고 관리할 수 있습니다. 

## 데이터 정규화의 이점

데이터 정규화는 데이터베이스 설계의 과정 중 하나로, 중복을 최소화하고 데이터 무결성을 유지하는 데 도움이 됩니다. 이를 통해 **데이터의 일관성, 효율성 및 유지보수성을 향상**시킬 수 있습니다. 

## 안드로이드 앱에서 SQLite 데이터베이스 구현

SQLite 데이터베이스를 사용하려면 **SQLiteOpenHelper** 클래스를 확장하여 데이터베이스를 생성하고 업그레이드하는 기능을 구현해야 합니다. 또한, 데이터베이스 작업을 처리하는 DAO(Data Access Object) 클래스를 만들어야 합니다.

```java
public class DBHelper extends SQLiteOpenHelper {
    // 데이터베이스 생성, 업그레이드 등의 로직 구현
}

public class DAO {
    // 데이터베이스 작업을 처리하는 메서드들 구현
}
```

## 데이터 정규화 적용

1. **1NF(제 1 정규형)**: 모든 필드가 원자값(Atomic value)을 갖도록 테이블을 수정합니다.
2. **2NF(제 2 정규형)**: 부분 함수 종속을 제거하여 테이블의 속성들을 분해합니다.
3. **3NF(제 3 정규형)**: 이행적 함수 종속성을 제거하여 중복을 피합니다.
