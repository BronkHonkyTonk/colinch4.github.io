# Using 기본문

`.NET Framework`나 `NuGet 패키지` 등에서 제공하는 네임스페이스를 사용하려면 `using` 키워드를 사용하여 해당 네임스페이스를 **import** 해야 합니다. 

예를 들어, `System` 네임스페이스를 사용하려면 다음과 같이 코드를 작성합니다.

```csharp
using System;
```

위 코드를 통해 `System` 네임스페이스에 포함된 클래스, 구조체, 메서드 등을 사용할 수 있습니다.

그러면 추가적인 도움이 필요하시거나 다른 질문이 있으시면 언제든지 물어보세요!
