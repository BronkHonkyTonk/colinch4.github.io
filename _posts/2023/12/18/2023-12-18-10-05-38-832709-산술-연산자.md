```c
int a = 10;
int b = 5;
int sum = a + b; // 덧셈 연산
int difference = a - b; // 뺄셈 연산
int product = a * b; // 곱셈 연산
int quotient = a / b; // 나눗셈 연산
int remainder = a % b; // 나머지 연산
```

산술 연산자는 C 프로그래밍 언어에서도 매우 기본적이고 중요한 부분입니다.
