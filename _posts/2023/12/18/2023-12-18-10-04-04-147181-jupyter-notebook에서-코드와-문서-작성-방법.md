## 1. 코드 셀과 마크다운 셀
Jupyter Notebook은 코드 셀과 마크다운 셀을 포함하고 있습니다. 코드 셀에서는 Python, R, Julia 등 다양한 프로그래밍 언어를 사용하여 코드를 작성하고 실행할 수 있으며, 마크다운 셀에서는 문서를 작성할 수 있습니다.

## 2. 코드 작성
Python 코드를 작성해보겠습니다.

```python
# 덧셈 연산
a = 3
b = 4
sum = a + b
print(sum)
```

위 코드를 실행하면 두 변수를 더한 결과가 출력됩니다.

## 3. 마크다운 셀을 이용한 문서 작성
마크다운을 사용하여 문서를 작성할 수 있습니다. 마크다운을 사용하면 헤더, 목록, 링크, 이미지 등을 쉽게 작성할 수 있습니다. 아래는 마크다운 예제입니다.

```markdown
# 제목
## 소제목

- 항목 1
- 항목 2

[구글 링크](https://www.google.com)
```

## 4. 마무리
이상으로 Jupyter Notebook에서 코드와 문서를 작성하는 방법에 대해 알아보았습니다. 코드와 문서를 함께 작성하여 분석 보고서나 프로젝트 문서를 작성하는 데 Jupyter Notebook이 얼마나 효과적인지 알 수 있습니다.
