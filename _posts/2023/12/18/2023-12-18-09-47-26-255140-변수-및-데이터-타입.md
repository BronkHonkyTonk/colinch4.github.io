## 변수
변수는 데이터를 저장하는 공간으로, 변수를 사용하여 데이터를 읽고 쓸 수 있습니다. 변수는 메모리에 할당되며, 그 값을 변경할 수 있습니다. 변수를 사용하여 값을 저장하고 해당 값을 프로그램의 다른 부분에서 사용할 수 있습니다.

예를 들어 JavaScript에서 변수를 선언하는 방법은 다음과 같습니다:

```javascript
let 변수명 = 값;
```

## 데이터 타입
데이터 타입은 변수에 저장될 수 있는 값의 종류를 의미합니다. 다양한 데이터 타입이 있으며, 각각의 데이터 타입은 특정한 종류의 데이터만을 포함할 수 있습니다.

간단한 데이터 타입의 예시:
- 정수형 데이터 타입: 5
- 실수형 데이터 타입: 3.14
- 문자열 데이터 타입: "안녕하세요"
- 불리언 데이터 타입: true 또는 false

프로그래밍 언어마다 변수와 데이터 타입을 사용하는 방식이 다를 수 있으므로, 각 언어의 문법에 따라 변수와 데이터 타입을 이해하고 활용해야 합니다.
