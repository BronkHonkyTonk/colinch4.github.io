Kotlin에서 예외를 처리하는 것은 중요합니다. 예외가 발생하면 예외 메시지를 출력하여 디버깅에 도움이 될 수 있습니다. 이 글에서는 Kotlin에서 예외가 발생했을 때 예외 메시지를 출력하는 방법에 대해 알아보겠습니다.

## 예외가 발생한 경우 예외 메시지 출력하기

```kotlin
fun main() {
    try {
        // 예외가 발생할 수 있는 코드
        val result = 10 / 0
    } catch (e: Exception) {
        // 예외가 발생했을 때 예외 메시지를 출력
        println("예외가 발생했습니다: ${e.message}")
    }
}
```

위 코드에서 `try` 블록 안에 예외가 발생할 수 있는 코드를 작성하고 `catch` 블록에서 해당 예외를 잡아서 예외 메시지를 출력하고 있습니다.

여기서 `e.message`는 발생한 예외의 메시지를 가져오는 속성입니다. 이를 통해 예외가 발생했을 때 해당 메시지를 출력할 수 있습니다.

이렇게 예외가 발생했을 때 예외 메시지를 출력함으로써 프로그램의 안정성을 높일 수 있습니다.

프로그래머는 이러한 예외 처리 기능을 통해 프로그램이 예상치 못한 문제에 직면했을 때 이를 탐지하고 조치를 취할 수 있습니다.

## 결론

이 글에서는 Kotlin에서 예외가 발생했을 때 예외 메시지를 출력하는 방법에 대해 알아봤습니다. 예외 처리는 프로그램의 안정성을 높이는 데 중요한 요소이며, 적절한 예외 메시지 출력은 디버깅을 보다 효율적으로 할 수 있도록 도와줍니다.
