## SwiftUI란 무엇인가요?

SwiftUI는 **Apple**이 제공하는 사용자 인터페이스(UI)를 만들기 위한 **선언적 프레임워크**입니다. SwiftUI를 사용하면 간단하고 직관적인 방식으로 다양한 플랫폼(iOS, iPadOS, macOS, watchOS)에 대한 멋진 UI를 작성할 수 있습니다.

## SwiftUI의 장점

SwiftUI는 **간결하고 직관적**이며, **실시간 미리보기(Real-time preview)** 기능을 제공하여 UI를 빠르게 테스트하고 개발할 수 있습니다. 또한 SwiftUI는 **다양한 장치에 대한 호환성**을 제공하여 코드의 재사용성과 생산성을 향상시킵니다.

## SwiftUI 예시

아래는 SwiftUI를 사용하여 간단한 "Hello, World!" 메시지를 표시하는 코드 예시입니다.

```swift
import SwiftUI

struct ContentView: View {
    var body: some View {
        Text("Hello, World!")
    }
}
```
