배열을 생성하고 초기화하는 방법은 여러 가지가 있습니다. 이번 글에서는 각 방법에 대해 알아보겠습니다.

1. **배열 선언 후 원소 하나하나 대입하기**
   
   ```c
   int arr[5];
   arr[0] = 1;
   arr[1] = 2;
   arr[2] = 3;
   arr[3] = 4;
   arr[4] = 5;
   ```

2. **배열 선언과 동시에 초기화하기**

   - **정적 배열 초기화**
   
     ```c
     int arr[5] = {1, 2, 3, 4, 5};
     ```

   - **동적 배열 초기화 (C99 이상)**
   
     ```c
     int arr[] = {1, 2, 3, 4, 5};
     ```

3. **memset 함수 사용하기**

   ```c
   #include <string.h>
   int arr[5];
   memset(arr, 0, sizeof(arr)); // 모든 원소를 0으로 초기화
   ```

4. **반복문을 이용하여 초기화하기**

   ```c
   int arr[5];
   for (int i = 0; i < 5; i++) {
       arr[i] = i + 1;
   }
   ```

5. **문자열을 이용하여 초기화하기**

   ```c
   char str[] = "Hello";
   ```

이렇게 다양한 방법을 통해 배열을 초기화할 수 있습니다. 각 상황에 맞게 적절한 방법을 선택하여 사용하시면 됩니다.
