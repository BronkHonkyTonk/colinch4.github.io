PHP는 다양한 라이브러리를 사용하여 기능을 확장할 수 있습니다. 라이브러리를 설치하는 방법에 대해 알아봅시다.

## 1. Composer를 사용한 라이브러리 설치

Composer를 사용하면 PHP 프로젝트에 라이브러리를 손쉽게 추가할 수 있습니다.

먼저, 프로젝트 루트 디렉토리에 `composer.json` 파일을 생성합니다.

```json
{
    "require": {
        "vendor/package": "version"
    }
}
```

여기서 `vendor/package`는 설치할 라이브러리의 이름이고 `version`은 원하는 버전입니다. 예를 들어, `monolog/monolog`은 유명한 로깅 라이브러리입니다.

그런 다음 터미널에서 다음 명령을 실행하여 라이브러리를 설치합니다.

```sh
composer install
```

## 2. 수동 다운로드와 포함

일부 라이브러리는 Composer를 사용하지 않고 수동으로 다운로드하여 포함해야 할 수 있습니다. 이 경우에는 다음과 같이 하면 됩니다.

- 라이브러리를 다운로드하고 압축을 해제합니다.
- 프로젝트의 적절한 디렉토리에 라이브러리 파일을 추가합니다.
- 라이브러리를 코드에서 require 또는 include 문을 사용하여 포함시킵니다.

## 3. 페키지 관리자를 사용하는 라이브러리

특정 라이브러리는 페키지 관리자를 사용하여 설치해야 할 수 있습니다. 예를 들어, 페키지 관리자가 있는 경우 `apt-get`을 사용하여 PHP 라이브러리를 설치할 수도 있습니다.

이제 PHP 라이브러리를 설치하는 방법에 대해 이해했습니다. 선택한 방법으로 필요한 라이브러리를 설치하여 PHP 프로젝트를 보다 강력하게 만들어 보세요!
