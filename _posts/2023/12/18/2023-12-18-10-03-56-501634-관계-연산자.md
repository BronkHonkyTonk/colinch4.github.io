관계 연산자는 두 개의 값을 비교하여 그들 간의 관계를 평가하는 데 사용됩니다. 주로 참과 거짓을 판단하는데 사용되며, 일반적으로 조건문에서 자주 볼 수 있습니다. 다음은 주요 관계 연산자들입니다.

- `==` : 값이 같은지 비교
- `!=` : 값이 다른지 비교
- `<` : 작은지 비교
- `>` : 큰지 비교
- `<=` : 작거나 같은지 비교
- `>=` : 크거나 같은지 비교

### 예제

```c
#include <stdio.h>

int main() {
    int a = 5;
    int b = 3;

    if (a == b) {
        printf("a와 b는 같습니다.");
    } else {
        printf("a와 b는 다릅니다.");
    }

    return 0;
}
```

위의 예제에서, `a`와 `b`가 같지 않기 때문에 "a와 b는 다릅니다."가 출력됩니다.

이러한 관계 연산자들은 프로그래밍에서 매우 유용하며, 조건문과 논리적인 판단에 중요한 역할을 합니다.
