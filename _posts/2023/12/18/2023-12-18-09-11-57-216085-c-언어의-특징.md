C 언어는 다음과 같은 특징을 가지고 있습니다.

## 1. 저수준 언어
C 언어는 저수준 언어로, 기계어에 가깝게 동작합니다. 이를 통해 메모리와 하드웨어에 대한 직접적인 제어가 가능합니다.

## 2. 이식성
C 언어는 이식성이 뛰어나며, 다양한 플랫폼과 운영체제에서 동작할 수 있습니다.

## 3. 간결하고 빠른 실행
C 언어는 간결한 문법과 뛰어난 성능을 가지고 있으며, 시스템 프로그래밍이나 임베디드 시스템에서 널리 사용됩니다.

## 4. 구조화된 프로그래밍
C 언어는 모듈화된 프로그래밍과 재사용성을 강조하는 구조화된 프로그래밍을 지원합니다.

C 언어는 이러한 특징으로 많은 프로그래머들에게 사랑받고 있으며, 다양한 영역에서 활용되고 있습니다.
