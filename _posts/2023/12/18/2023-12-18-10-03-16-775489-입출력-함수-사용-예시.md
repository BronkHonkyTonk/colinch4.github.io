다음은 간단한 예시 코드입니다.

```c
#include <stdio.h>

int main() {
    int num;
    
    printf("정수를 입력하세요: ");
    scanf("%d", &num);
    
    printf("입력한 숫자는 %d입니다.", num);
    
    return 0;
}
```

이 예시 코드는 사용자로부터 정수를 입력받고, 그 값을 화면에 출력하는 기본적인 기능을 수행합니다. 

입력받은 값을 사용하여 계산하거나 화면에 출력하는 등 다양한 목적으로 사용할 수 있습니다.
