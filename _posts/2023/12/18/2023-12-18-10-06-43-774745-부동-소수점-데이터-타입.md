부동 소수점 데이터 타입은 많은 프로그래밍 언어에서 사용되며, 실수를 표현하는 데 사용됩니다. 이 데이터 타입은 정수와 소수 부분으로 구성되며, 숫자를 더 정밀하게 나타낼 수 있습니다.

## 부동 소수점 데이터 타입의 종류

대부분의 프로그래밍 언어는 두 가지 주요한 부동 소수점 데이터 타입을 제공합니다.

1. **float**: 이 데이터 타입은 단정밀도를 제공하며, 일반적으로 4바이트의 메모리를 사용합니다.
2. **double**: 두배의 정밀도를 제공하며, 일반적으로 8바이트의 메모리를 사용합니다.

## 부동 소수점 데이터 타입의 사용

부동 소수점 데이터 타입은 실수를 저장하고 다루는 데 사용됩니다. 이러한 데이터 타입을 사용하면 정밀한 계산이 가능하며, 과학 및 공학 분야에서 더욱 정확한 결과를 얻을 수 있습니다.

```go
package main

import "fmt"

func main() {
    var num1 float32 = 3.14
    var num2 float64 = 3.1415
    fmt.Println(num1, num2)
}
```

위의 예시에서는 Go 언어를 사용하여 float32와 float64 데이터 타입을 사용한 예시를 보여줍니다.
