```c
#include <stdio.h>

int main() {
   FILE *file = fopen("파일경로", "모드");
   // 파일 작업 수행
   fclose(file);
   return 0;
}
```

위 코드에서 `fopen` 함수를 사용하여 파일을 열고, 작업을 수행한 후 `fclose` 함수를 호출하여 파일을 닫을 수 있습니다. 이렇게 하면 파일 핸들이 닫히고, 시스템 자원이 해제됩니다.
