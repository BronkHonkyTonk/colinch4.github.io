## 연산자 우선순위란?

연산자 우선순위는 수학적 연산의 우선순위를 결정하는 규칙입니다. 예를 들어, 곱셈은 덧셈보다 우선순위가 높기 때문에 곱셈 연산이 먼저 수행됩니다.

## 효율적인 코드 작성법

### 괄호 사용

연산자 우선순위를 명확히 하기 위해 괄호를 사용하는 것이 좋습니다. 가독성을 높이고 코드의 의도를 분명히 전달할 수 있습니다.

```c
int result = (a + b) * c;
```

### 복잡한 표현식 분해

운영자 우선순위가 혼동스러운 복잡한 표현식은 여러 단계로 분해하여 작성하는 것이 좋습니다. 이렇게 하면 다른 개발자가 코드를 빠르게 이해할 수 있습니다.

```c
int firstResult = a * b;
int secondResult = firstResult + c;
```

연산자 우선순위를 올바르게 이해하고 코드를 작성하면 코드의 가독성을 높이고 버그를 줄일 수 있습니다.
