이번에는 배열과 함수 호출에 대해 알아보겠습니다.

## 배열

배열은 동일한 유형의 데이터를 여러 개 저장할 수 있는 자료 구조입니다. 다음은 C 언어에서 배열을 선언하고 사용하는 예제입니다.

```c
int numbers[5]; // 5개의 정수를 저장할 배열 선언
numbers[0] = 1; // 첫 번째 요소에 값 할당
numbers[1] = 2; // 두 번째 요소에 값 할당
// ...
```

배열을 사용하면 동일한 종류의 데이터를 쉽게 관리할 수 있습니다.

## 함수 호출

함수 호출은 프로그램에서 작성한 함수를 실행하는 것을 말합니다. 다음은 C 언어에서 함수를 호출하는 예제입니다.

```c
int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 4); // add 함수 호출
    return 0;
}
```

함수를 호출하여 원하는 작업을 수행할 수 있습니다.

## 결론

배열은 여러 데이터를 쉽게 관리할 수 있게 해주고, 함수 호출을 통해 필요한 기능을 모듈화하여 코드를 구조화할 수 있습니다.
