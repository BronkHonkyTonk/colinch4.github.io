## 파일 읽기
텍스트 파일을 읽기 위해서는 `open()` 함수를 사용합니다. 

```python
with open('파일이름.txt', 'r') as file:
    data = file.read()
    print(data)
```

위 코드에서 `open()` 함수의 두 번째 매개변수는 파일을 여는 방식을 지정합니다. 'r'은 읽기 모드를 의미합니다. 또한, `with` 문을 사용하여 파일을 열고 작업을 마친 뒤 자동으로 파일을 닫도록 할 수 있습니다.

## 파일 쓰기
새로운 내용을 파일에 쓰려면 `open()` 함수를 다음과 같이 사용합니다.

```python
with open('파일이름.txt', 'w') as file:
    file.write('내용을 입력하세요.')
```

파일을 쓰기 모드('w')로 열면, 파일이 이미 존재하면 그 내용이 모두 지워지고 새로운 내용으로 대체됩니다.

파일 작업을 할 때는 항상 예외 처리를 고려해야 합니다. 또한 파일을 다룬 뒤에는 항상 파일을 닫아야 합니다.

텍스트 파일 입출력은 파이썬 프로그래밍의 핵심 부분이며, 데이터 처리 작업에 매우 유용하게 사용됩니다.
