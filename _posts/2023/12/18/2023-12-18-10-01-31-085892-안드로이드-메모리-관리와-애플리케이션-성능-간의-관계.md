## 내부 메모리 관리

안드로이드 애플리케이션은 내부 메모리를 효율적으로 활용해야 합니다. 메모리 누수를 방지하기 위해 **가비지 컬렉션**을 효과적으로 수행하여 사용되지 않는 객체를 정리해야 합니다. 또한, 이미지나 파일과 같은 리소스를 메모리에 적재할 때 **메모리 캐싱**을 이용하여 효율적으로 관리해야 합니다.

```java
// 가비지 컬렉션 수행

// 메모리 캐싱
```

애플리케이션은 외부 메모리 사용 시 반드시 **파일 핸들링**과 **리소스 회수**에 신경을 써야 합니다. 파일 핸들링 시에는 적절한 예외 처리 및 자원의 해제를 고려해야 합니다. 또한, 외부 리소스를 사용할 때는 불필요한 리소스는 즉시 반환하여 메모리 누수를 방지해야 합니다.

```java
// 파일 핸들링
try {
    FileInputStream fileInputStream = new FileInputStream("file.txt");
    // 파일 읽기 작업 수행
    fileInputStream.close(); // 파일 핸들 닫기

    // 예외 처리
}
```
