## 작업 분배 알고리즘 소개
작업 분배 알고리즘은 대규모 작업을 여러 개의 작은 작업으로 분할하여 처리하는 방법을 말합니다. 이를 통해 병렬 처리 및 시간 단축을 이룰 수 있습니다. 예를 들어, 여러 대상의 연산을 동시에 처리하거나, 대용량 데이터를 여러 대상에게 분배하여 처리하는 등의 작업에 적용할 수 있습니다.

## 파이썬을 이용한 작업 분배 알고리즘 구현
파이썬에서 작업 분배 알고리즘을 구현하는 방법에는 다양한 방식이 있지만, 가장 간단한 예제 중 하나를 살펴보겠습니다.

다음은 **병렬 처리**를 위한 간단한 작업 분배 예제입니다. 

```python
import multiprocessing

def worker(num):
    """작업 분배를 위한 워커 함수"""
    print(f'Worker: {num}')

if __name__ == '__main__':
    jobs = []
    for i in range(5):
        p = multiprocessing.Process(target=worker, args=(i,))
        jobs.append(p)
        p.start()
```

위 예제는 `multiprocessing` 모듈을 사용하여 여러 프로세스를 생성하여 작업을 분배하는 방법을 보여줍니다.
