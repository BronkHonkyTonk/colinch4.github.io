조건문은 프로그램이 특정 조건을 판단하여 특정 동작을 수행하도록 하는 구문입니다. 

## 조건문 구조
조건문은 다음과 같은 기본적인 구조를 갖습니다:

```go
if 조건식 {
    // 조건식이 참일 때 실행되는 코드
} else {
    // 조건식이 거짓일 때 실행되는 코드 (선택사항)
}
```

위 구조를 보면, `if` 키워드 다음에 조건식이 오며, 조건식이 참일 경우 해당 블록의 코드가 실행됩니다. `else` 키워드 뒤에는 조건식이 거짓일 경우 실행될 코드 블록을 지정할 수 있습니다.

## 예제
```go
age := 18

if age >= 18 {
    fmt.Println("성인입니다.")
} else {
    fmt.Println("미성년자입니다.")
}
```

위 예제에서는 `age` 변수의 값이 18 이상인지를 판단하여 `fmt.Println`으로 해당 메시지를 출력하는 코드입니다. 

조건문은 프로그램에서 특정 상황에 따라 다른 동작을 수행하도록 하며, 제어 구문 중에서 중요한 역할을 합니다.

[Go 언어 공식 문서](https://golang.org/doc/effective_go.html#if)에서도 `if` 문에 대한 자세한 내용을 확인할 수 있습니다.
