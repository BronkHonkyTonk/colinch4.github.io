```c
#include <stdio.h>

int main() {
   FILE *file;
   int ch;
   
   file = fopen("example.txt", "r");
   if (file == NULL) {
      perror("Error opening file");
      return -1;
   }
   
   while ((ch = fgetc(file)) != EOF) {
      putchar(ch);
   }
   
   fclose(file);
   return 0;
}
```

위 코드는 "example.txt" 파일을 열어서 각 글자를 읽어 화면에 출력하는 간단한 프로그램입니다. 파일 끝에 도달하면 `fgetc`는 `EOF`를 반환하여 루프를 종료합니다.
