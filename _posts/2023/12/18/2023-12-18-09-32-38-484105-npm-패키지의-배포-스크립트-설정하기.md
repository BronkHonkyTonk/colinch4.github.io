NPM(Node Package Manager)은 JavaScript 패키지를 관리하고 공유하는 도구이며, 사용자들은 NPM을 통해 패키지를 다운로드하여 사용할 수 있습니다. 이번 글에서는 NPM 패키지를 작성하고 배포하는 과정 중 배포 스크립트를 설정하는 방법에 대해 살펴보겠습니다.

## 1. 패키지 초기화

먼저, 새로운 NPM 패키지를 생성하기 위해 프로젝트 폴더에서 다음 명령어를 실행하여 초기화합니다.

```bash
npm init
```

위 명령어를 실행하면 패키지의 이름, 버전, 설명 등을 입력할 수 있는 대화형 프롬프트가 표시됩니다.

## 2. 배포 스크립트 추가

패키지를 배포하기 위해서는 NPM 공식 레지스트리 또는 사용자가 설치한 프라이빗 레지스트리에 업로드해야 합니다. 이를 위해 `package.json` 파일에 배포 스크립트를 추가합니다.

```json
"scripts": {
  "publish": "npm publish"
}
```

위와 같이 `scripts` 항목에 `publish` 스크립트를 추가하고, 해당 스크립트가 실행될 때 실행할 명령어를 지정합니다. 

## 3. 패키지 배포

배포 스크립트를 추가한 후에는 다음 명령어를 통해 패키지를 배포할 수 있습니다.

```bash
npm run publish
```

위와 같이 `npm publish` 명령어를 직접 실행하는 대신에 추가한 배포 스크립트를 통해 패키지를 배포할 수 있습니다.

## 결론

이제 NPM 패키지의 배포 스크립트를 설정하는 방법에 대해 알아보았습니다. 이를 통해 사용자는 더 효율적으로 패키지를 배포하고 관리할 수 있습니다.

[Node.js 공식 문서](https://nodejs.org/)에서 NPM 패키지 관련 정보를 확인할 수 있습니다.
