Anaconda를 사용하지 않고도 Jupyter Notebook을 설치하고 사용할 수 있습니다. 아래는 Python을 기반으로 한 Jupyter Notebook을 설치하고 실행하는 방법에 대한 안내입니다.

## 1. Python과 pip 설치

우선 가장 최신 버전의 Python을 다운로드하여 설치해야 합니다. 또한, pip도 함께 설치되어 있어야 합니다. Python 및 pip이 이미 설치되어 있는 경우 이 단계는 건너뛰실 수 있습니다. [Python 공식 사이트](https://www.python.org/downloads/)에서 Python을 다운로드하여 설치할 수 있습니다.

## 2. Jupyter Notebook 설치

터미널 또는 명령 프롬프트에서 다음 명령을 사용하여 Jupyter Notebook을 설치합니다.

```python
pip install jupyter
```

위 명령을 실행하여 Jupyter Notebook을 설치합니다.

## 3. Jupyter Notebook 실행

설치가 완료되었으면 터미널 또는 명령 프롬프트에서 다음 명령을 사용하여 Jupyter Notebook을 실행합니다.

```python
jupyter notebook
```

그러면 웹 브라우저가 열리고 Jupyter Notebook 대시보드가 표시됩니다. 여기서 새 노트북을 만들거나 기존의 노트북을 열어 작업할 수 있습니다.

이제 설치 및 실행이 완료되었으니 원하는 Python 코드를 실행하여 Jupyter Notebook을 사용할 수 있습니다.

이러한 간단한 단계를 따라하면 Anaconda 없이도 Jupyter Notebook을 설치하고 사용할 수 있습니다.깨지지 않도록 주의하여 Python 및 Jupyter Notebook을 설치하고 실행하시기 바랍니다.
