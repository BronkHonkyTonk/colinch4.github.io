## 현재 읽고 있는 이 글에서 {#intro}

NPM을 사용하다 보면 패키지의 최신 버전이 아닌 특정 버전으로 업그레이드해야 하는 경우가 있습니다. 일부 프로젝트에서는 최신 버전과 호환되지 않는 경우가 발생할 수 있고, 이때 특정 버전으로 업그레이드하는 것이 필요합니다.

## 특정 버전으로 패키지 업그레이드하기 {#upgrade}

NPM을 사용하여 특정 버전의 패키지로 업그레이드하려면 다음의 명령을 사용합니다.

```bash
npm install package-name@version
```

위 명령어에서 `package-name`은 업그레이드할 패키지의 이름을, `version`은 업그레이드할 특정 버전을 나타냅니다. 예를 들어, `lodash` 패키지의 `4.17.21` 버전으로 업그레이드하려면 다음과 같이 명령합니다.

```bash
npm install lodash@4.17.21
```

이제 NPM을 통해 원하는 패키지의 특정 버전으로 업그레이드할 수 있게 되었습니다.
