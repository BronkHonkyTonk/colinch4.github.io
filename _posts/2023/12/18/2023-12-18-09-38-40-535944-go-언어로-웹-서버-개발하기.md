## Go 언어란?

**Go 언어**는 2009년 구글에서 개발된 오픈소스 프로그래밍 언어로, C와 유닉스 계열 운영체제에서의 경험이 있는 개발자들이 개발했습니다. Go는 C 언어의 직관적이고 간결한 문법을 계승하면서도, 메모리 안정성과 동시성을 보장하기 위해 노력했습니다. 이러한 특징 덕분에 Go 언어는 대규모 시스템의 개발자들 사이에서 인기를 얻고 있습니다.

## 웹 서버 개발을 위한 패키지 가져오기

Go 언어로 웹 서버를 개발하기 위해서는 표준 라이브러리에 포함된 `net/http` 패키지를 사용할 수 있습니다. 이 패키지를 이용하여 HTTP 요청을 처리하고 응답하는 웹 서버를 구현할 수 있습니다.

```go
import "net/http"
```

## 간단한 웹 서버 만들기

다음은 Go 언어로 간단한 "Hello, World!"를 응답하는 웹 서버를 만드는 예제 코드입니다.

```go
func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, World!")
}

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}
```

위 코드에서 `handler` 함수는 HTTP 요청을 처리하고, `main` 함수에서는 `/` 경로로 요청이 오면 `handler` 함수를 실행하도록 합니다.

## 웹 서버 실행하기

위 예제 코드를 `server.go` 파일로 저장한 뒤, 터미널에서 다음 명령어를 통해 웹 서버를 실행할 수 있습니다.

```bash
go run server.go
```

이제 웹 브라우저를 열고 `http://localhost:8080`에 접속하면 "Hello, World!"가 출력되는 것을 확인할 수 있습니다.
