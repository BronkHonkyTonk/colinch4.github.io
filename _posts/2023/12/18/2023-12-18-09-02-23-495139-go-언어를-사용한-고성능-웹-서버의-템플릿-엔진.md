## Go 언어 소개

**Go 언어**는 구글에서 개발한 언어로, 2009년에 공식적으로 발표되었습니다. Go 언어는 컴파일 언어이며 간결하고 병행 처리를 잘 지원하는 등의 특징을 가지고 있습니다. 또한, 빠른 컴파일 속도와 안정성 때문에 고성능 서버 개발에 적합한 언어로 평가받고 있습니다.

## 웹 서버 개발을 위한 Go 언어의 장점

Go 언어는 뛰어난 성능과 안정성을 제공하여 웹 서버 개발에 적합한 언어입니다. 또한, 병행 처리를 위한 고루틴(Goroutine) 및 풍부한 표준 라이브러리 등의 기능을 제공하여 웹 서버를 효율적으로 구현할 수 있습니다.

## 템플릿 엔진의 필요성

**템플릿 엔진**은 동적으로 웹 페이지를 생성하기 위해 사용되는 도구로, 정적인 HTML 파일에 동적인 데이터를 삽입하여 클라이언트에게 제공할 수 있습니다. 템플릿 엔진을 사용하면 웹 페이지의 디자인과 데이터를 분리하여 유지보수 및 개발의 편의성을 높일 수 있습니다.

## Go 언어에서의 템플릿 엔진 사용

Go 언어에서는 템플릿을 다루기 위한 표준 라이브러리인 `html/template`을 제공하고 있습니다. 이 라이브러리를 사용하여 웹 서버에서 템플릿을 렌더링하고 클라이언트에게 동적인 웹 페이지를 제공할 수 있습니다.

아래는 Go 언어에서의 템플릿 엔진을 사용하는 간단한 예시 코드입니다.

```go
package main

import (
    "html/template"
    "net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
    data := struct {
        Title   string
        Message string
    }{
        Title:   "템플릿 엔진 예제",
        Message: "템플릿 엔진을 사용하여 동적인 웹 페이지를 렌더링합니다.",
    }
    tmpl, err := template.New("example").Parse(`
        <html>
        <head>
            <title>{{.Title}}</title>
        </head>
        <body>
            <h1>{{.Message}}</h1>
        </body>
        </html>
    `)
    if err != nil {
        http.Error(w, err.Error(), http.StatusInternalServerError)
        return
    }
    tmpl.Execute(w, data)
}

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}
```

위 코드는 간단한 웹 서버를 구현하고, 요청이 들어올 때마다 템플릿을 렌더링하여 동적인 웹 페이지를 반환하는 예시입니다.
