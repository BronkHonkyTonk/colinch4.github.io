다차원 배열을 생성하고 접근하는 방법은 언어에 따라 다르지만, 보통 각 차원마다 인덱스를 사용하여 원하는 요소에 접근할 수 있습니다. 

예를 들어, Python에서는 numpy 라이브러리를 사용하여 다차원 배열을 생성하고 다룰 수 있습니다. 아래는 2차원 배열을 생성하고 접근하는 예제입니다.

```python
import numpy as np

# 2차원 배열 생성
arr = np.array([[1, 2, 3], [4, 5, 6]])

# 배열 요소 접근
print(arr[0, 0])  # 첫 번째 행, 첫 번째 열의 요소 출력
print(arr[1, 2])  # 두 번째 행, 세 번째 열의 요소 출력
```

이와 같이 다차원 배열은 데이터를 효율적으로 표현하고 다루는 것에 도움을 주는 중요한 자료 구조입니다.
