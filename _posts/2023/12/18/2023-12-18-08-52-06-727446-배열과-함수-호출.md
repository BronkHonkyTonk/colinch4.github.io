배열은 프로그래밍에서 쉽게 여러 개의 데이터를 다룰 수 있는 자료 구조입니다. 함수 호출은 프로그램에서 특정한 동작을 실행하기 위해 사용됩니다.

## 배열

배열은 동일한 데이터 타입을 가진 여러 요소를 담을 수 있는 구조입니다. 예를 들어, 정수형 배열을 선언하려면 다음과 같이 작성합니다.

```c
int numbers[5] = {1, 2, 3, 4, 5};
```

위 코드는 `numbers`라는 이름의 정수형 배열을 선언하고 1부터 5까지의 값을 요소로 가지고 있습니다.

## 함수 호출

함수 호출은 프로그램의 특정한 기능을 실행하기 위해 함수를 호출하는 것을 의미합니다. 함수 호출은 다음과 같이 작성됩니다.

```c
int result = addNumbers(3, 4);
```

위 코드는 `addNumbers`라는 함수를 호출하고, 3과 4라는 인수를 전달하여 결과를 반환합니다.

배열과 함수 호출은 프로그래밍에서 중요한 요소이며, 올바르게 다루는 것이 중요합니다.
