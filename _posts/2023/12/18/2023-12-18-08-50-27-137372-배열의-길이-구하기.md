아래는 몇 가지 프로그래밍 언어에서 배열의 길이를 구하는 방법을 보여줍니다.

### JavaScript

JavaScript에서 배열의 길이를 구하는 방법은 `length` 속성을 사용하는 것입니다. 다음은 예시 코드입니다.

```javascript
let arr = [1, 2, 3, 4, 5];
let length = arr.length; // 배열의 길이는 5
```

### Python

Python에서는 내장 함수 `len`을 사용하여 배열의 길이를 구할 수 있습니다. 아래는 예시 코드입니다.

```python
arr = [1, 2, 3, 4, 5]
length = len(arr)  # 배열의 길이는 5
```

### Java

Java에서 배열의 길이는 배열의 `length` 속성을 통해 구할 수 있습니다. 아래는 예시 코드입니다.

```java
int[] arr = {1, 2, 3, 4, 5};
int length = arr.length; // 배열의 길이는 5
```

배열의 길이를 구하는 방법은 언어마다 조금씩 다를 수 있지만, 대부분의 경우 내장된 함수나 속성을 사용하여 간단하게 구할 수 있습니다.
