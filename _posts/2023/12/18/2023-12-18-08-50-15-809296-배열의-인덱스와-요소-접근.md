인덱스를 사용하여 배열의 요소에 접근할 수 있습니다. 예를 들어, 다음과 같이 배열에서 인덱스를 사용하여 요소를 읽고 쓸 수 있습니다.

```javascript
const fruits = ['apple', 'banana', 'orange'];

// 배열 요소에 접근
const firstFruit = fruits[0];
console.log(firstFruit); // 출력: 'apple'

// 배열 요소 업데이트
fruits[1] = 'grape';
console.log(fruits); // 출력: ['apple', 'grape', 'orange']
```

배열의 인덱스와 요소에 대한 이해는 프로그래밍에서 매우 중요합니다. 이를 통해 데이터를 효과적으로 조작하고 다룰 수 있습니다.
