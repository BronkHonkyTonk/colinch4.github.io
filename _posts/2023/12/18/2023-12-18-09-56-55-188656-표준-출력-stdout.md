C 언어에서는 `printf` 함수를 사용하여 표준 출력을 수행합니다. 아래는 간단한 예시입니다:

```c
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
```

위 코드에서 `printf` 함수를 사용하여 "Hello, world!"를 출력하고, `\n`을 통해 줄바꿈을 수행합니다. "Hello, world!"는 표준 출력을 통해 화면에 출력됩니다.
