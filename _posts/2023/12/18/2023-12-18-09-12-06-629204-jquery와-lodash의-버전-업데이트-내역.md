## 1. jQuery

### 1.1. jQuery 버전 3.6.0 업데이트 내역
jQuery 3.6.0은 2021년 4월에 릴리스되었습니다. 이 업데이트에서는 주요한 기능 추가나 변경은 없고, 주로 보안 문제 및 버그 수정에 중점을 두었습니다. 자세한 내역은 [공식 릴리스 노트](https://github.com/jquery/jquery/releases/tag/3.6.0)에서 확인할 수 있습니다.

```javascript
// Example code
$(document).ready(function(){
  // jQuery code here
});
```

## 2. Lodash

### 2.1. Lodash 버전 4.17.21 업데이트 내역
Lodash 4.17.21은 2021년 6월에 릴리스되었습니다. 이 업데이트에서는 메모리 누수와 관련된 일부 문제를 해결하였으며, 추가적으로 주요 변경 사항은 없습니다. 더 많은 정보는 [공식 홈페이지](https://lodash.com)에서 확인할 수 있습니다.

```javascript
// Example code
const array = [1, 2, 3];
_.reverse(array);
// → [3, 2, 1]
```

위 내용은 jQuery와 Lodash의 최신 버전 업데이트 내역에 관한 내용을 담고 있습니다. 관련하여 추가 질문이 있으시면 언제든지 연락해 주세요.
