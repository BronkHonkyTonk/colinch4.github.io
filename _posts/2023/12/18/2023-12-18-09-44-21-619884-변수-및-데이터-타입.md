# 변수

변수는 데이터를 저장하는 데 사용됩니다. 변수를 사용하면 데이터를 반복해서 사용하고 조작할 수 있습니다. 변수를 사용하려면 변수를 지정하고 값을 할당해야 합니다.

예를 들어, JavaScript에서 변수를 선언하고 값을 할당하는 방법은 다음과 같습니다.

```javascript
let age = 25;
```

여기서 `age`는 변수의 이름이고, `25`는 변수에 저장된 값입니다. 변수를 사용하면 값을 다시 사용하거나 변경할 수 있습니다.

# 데이터 타입

프로그래밍 언어는 여러 종류의 데이터 타입을 지원합니다. 각 데이터 타입은 다른 종류의 데이터를 나타냅니다. 가장 일반적인 데이터 타입에는 정수, 부동 소수점 숫자, 문자열, 불리언 등이 있습니다.

예를 들어, JavaScript에서 데이터 타입을 확인하고 싶다면 다음과 같이 할 수 있습니다.

```javascript
typeof age; // 결과는 "number"가 반환됩니다.
```

이 코드는 `age` 변수의 데이터 타입을 확인하는 것으로, 결과는 숫자 타입이라는 것을 알려줍니다.

데이터 타입을 이해하고 변수를 사용하는 것은 프로그래밍에서 중요합니다. 변수를 올바르게 사용하여 데이터를 저장하고 조작하는 것은 프로그래밍의 기본 사항 중 하나입니다.

변수 및 데이터 타입에 대한 더 자세한 정보는 각 프로그래밍 언어의 공식 문서를 참조하시기 바랍니다.
