앱의 외관을 꾸미는데 앱 아이콘 및 스플래시 화면은 중요한 부분입니다. **플러터(Flutter)** 를 사용하여 이 두 가지를 설정하는 방법에 대해 알아보겠습니다.

## 앱 아이콘 설정하기

앱 아이콘을 설정하려면 먼저 원하는 이미지를 포함한 `icon` 폴더를 앱 프로젝트의 루트 디렉토리에 만듭니다. 그런 다음, `pubspec.yaml` 파일을 열고 아래와 같이 `flutter` 섹션을 업데이트합니다.

```yaml
flutter:
  # ...
  assets:
    - icon/   # 여기에 아이콘 이미지가 있는 폴더 이름을 작성합니다.
```

이제 아래와 같이 **터미널**을 열어 아래 명령어를 실행하여 앱의 아이콘을 업데이트합니다.

```bash
flutter pub get
```

### 앱 아이콘 이미지 크기 설정

플러터는 다양한 크기의 아이콘 이미지를 지원합니다. 다양한 크기에 대한 아이콘을 설정하려면 `icon` 폴더 내에 아이콘 이미지를 저장하고 `pubspec.yaml` 파일에 아이콘 이미지의 경로를 선언합니다.

```yaml
flutter:
  # ...
  assets:
    - icon/icon-36x36.png
    - icon/icon-48x48.png
    - icon/icon-72x72.png
    # ... 다른 크기의 아이콘 이미지 경로들을 추가합니다.
```

## 스플래시 화면 설정하기

스플래시 화면은 앱이 로드되는 동안 사용자에게 표시되는 시작 화면입니다. 이를 설정하려면, 원하는 스플래시 이미지를 `images` 폴더에 저장하고 `pubspec.yaml` 파일에서 **승인된 아이콘** 설정과 같은 방법으로 스플래시 이미지를 설정합니다.

```yaml
flutter:
  # ...
  assets:
    - images/   # 여기에 스플래시 이미지가 있는 폴더 이름을 작성합니다.
```

이제 **터미널**을 열고 아래 명령어를 실행하여 앱의 스플래시 이미지를 업데이트합니다.

```bash
flutter pub get
```

위와 같이 설정하면 **플러터(Flutter)** 앱의 아이콘 및 스플래시 이미지를 쉽게 설정할 수 있습니다. 앱의 외관을 더욱 멋지게 꾸며보세요!
