다음은 네임스페이스를 선언하는 간단한 예제입니다.

```csharp
namespace MyNamespace
{
    class MyClass
    {
        // 클래스 멤버와 메서드들
    }
}
```

위의 예제에서 `MyNamespace`는 새로운 네임스페이스를 나타내며, `MyClass`는 해당 네임스페이스 내에 포함된 클래스를 나타냅니다.
