PHP를 사용하여 데이터 배열의 합계를 계산하는 방법은 여러 가지가 있습니다. 그 중에서도 가장 일반적인 방법은 **반복문**을 이용하여 배열의 각 요소를 순회하면서 값을 누적하는 것입니다.

## 배열 생성 및 초기화

먼저, 배열을 생성하고 초기화해야 합니다. 다음은 예제 데이터 배열의 생성과 초기화하는 방법입니다.

```php
<?php
$data = [10, 20, 30, 40, 50];
?>
```

## 반복문을 사용한 합계 계산

이제 생성한 배열을 반복문을 사용하여 합을 계산할 수 있습니다. 아래는 `for` 반복문을 사용하여 배열의 합을 구하는 예제입니다.

```php
<?php
$sum = 0;
for ($i = 0; $i < count($data); $i++) {
    $sum += $data[$i];
}
echo "배열의 합계: " . $sum;
?>
```

위 코드에서 `for` 반복문을 통해 각 요소를 더해 `sum` 변수에 누적시킵니다. 마지막으로 `echo` 문을 사용하여 계산된 배열의 합을 출력합니다.

## foreach 문을 사용한 합계 계산

또한, PHP의 `foreach` 문을 사용하여 배열 요소를 순회하면서 합을 계산할 수도 있습니다. 아래는 `foreach` 문을 이용한 예제입니다.

```php
<?php
$sum = 0;
foreach ($data as $value) {
    $sum += $value;
}
echo "배열의 합계: " . $sum;
?>
```

`foreach` 문은 배열의 각 요소를 순회하면서 `$value` 변수에 현재 요소의 값을 할당하고, 합계를 누적해나갑니다.

## 결론

PHP를 사용하여 데이터 배열의 합을 계산하는 방법에 대해 알아보았습니다. `for` 반복문과 `foreach` 문을 이용하여 배열의 합을 계산하는 방법을 살펴보았으며, 실제 상황에 맞게 적절한 방법을 선택하여 사용할 수 있습니다.
