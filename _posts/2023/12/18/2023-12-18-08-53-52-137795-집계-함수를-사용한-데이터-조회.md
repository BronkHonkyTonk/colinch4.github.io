SQL에서는 집계 함수를 사용하여 데이터를 조회하고 필요한 정보를 얻을 수 있습니다. 이 글에서는 일반적으로 사용되는 집계 함수와 함께 데이터를 조회하는 방법에 대해 알아보겠습니다.

## 집계 함수란
집계 함수는 데이터 집합에서 요약된 정보를 반환하는 함수입니다. 주요 집계 함수로는 `COUNT`, `SUM`, `AVG`, `MIN`, `MAX` 등이 있습니다.

## 데이터 조회
데이터베이스의 특정 테이블에서 집계 함수를 사용하여 데이터를 조회하는 방법은 다음과 같습니다.

### 1. COUNT
```sql
SELECT COUNT(*) FROM 테이블명;
```
위의 쿼리는 특정 테이블의 전체 레코드 수를 조회합니다.

### 2. SUM
```sql
SELECT SUM(컬럼명) FROM 테이블명;
```
위의 쿼리는 특정 컬럼의 합계를 조회합니다.

### 3. AVG
```sql
SELECT AVG(컬럼명) FROM 테이블명;
```
위의 쿼리는 특정 컬럼의 평균값을 조회합니다.

### 4. MIN, MAX
```sql
SELECT MIN(컬럼명), MAX(컬럼명) FROM 테이블명;
```
위의 쿼리는 특정 컬럼의 최솟값과 최댓값을 조회합니다.

## 결론
집계 함수를 사용하여 데이터를 조회하면 데이터의 요약된 정보를 쉽게 확인할 수 있습니다. 필요에 따라 다양한 집계 함수를 활용하여 데이터를 분석하고 원하는 정보를 효율적으로 얻을 수 있습니다.
