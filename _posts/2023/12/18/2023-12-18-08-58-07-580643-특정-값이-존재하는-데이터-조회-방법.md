다음은 특정 값이 존재하는 데이터를 조회하는 SQL 쿼리의 예시입니다:
```sql
SELECT * FROM 테이블명 WHERE 컬럼명 = '특정값';
```

위의 SQL 쿼리에서 `테이블명`은 데이터를 조회할 테이블의 이름이고, `컬럼명`은 조회할 값을 포함하고 있는 열의 이름입니다. `'특정값'`에 실제로 조회하려는 값을 입력하면 됩니다.

추가로, 조건으로 특정 값이 존재하는 데이터를 조회할 때는 `LIKE`를 사용하여 부분 문자열 검색 또는 와일드카드를 활용할 수도 있습니다. 

이렇게하여 SQL을 사용하여 특정 값이 존재하는 데이터를 조회할 수 있습니다.
