예시:
```javascript
// 모듈에서 디폴트로 익스포트할 값
const myDefault = 'This is the default export';

// 디폴트로 익스포트
export default myDefault;
```

위의 코드에서 `myDefault` 값을 디폴트로 익스포트하고 있습니다.
