## 바이너리 데이터에서 비트 설정

바이너리 데이터에서 특정 위치의 비트를 설정하려면 `OR` 비트 연산을 사용하여 해당 비트를 1로 설정합니다. Go 언어에서는 비트를 설정하기 위해 `|` 연산자를 사용할 수 있습니다.

예를 들어, 7번째 비트를 1로 설정하려면 다음과 같이 할 수 있습니다:

```go
// 원래 데이터
var data byte = 0b00000000

// 7번째 비트를 1로 설정
data = data | (1 << 7)
```

위 예제에서 `1 << 7`은 7번째 비트가 1이고 다른 비트는 0인 값을 생성합니다. 그런 다음, `data`와 해당 값을 `OR` 비트 연산하여 7번째 비트를 1로 설정합니다.

## 요약

Go 언어에서는 바이너리 데이터의 특정 위치에 비트를 설정하기 위해 `|` 비트 연산자를 사용합니다. 위 예제는 특정 위치의 비트를 설정하는 방법을 보여줍니다. 이를 통해 데이터를 조작하거나 비트 필드를 다루는 데 도움이 될 수 있습니다.
