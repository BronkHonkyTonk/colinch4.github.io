### AND 연산자
AND 연산자는 모든 조건이 참일 때만 전체 조건이 참이 됩니다. 일반적으로 `&&` 기호로 표시됩니다.

```java
if (조건1 && 조건2) {
    // 두 조건이 모두 참일 때 실행되는 코드
}
```

### OR 연산자
OR 연산자는 하나 이상의 조건이 참일 때 전체 조건이 참이 됩니다. 일반적으로 `||` 기호로 표시됩니다.

```java
if (조건1 || 조건2) {
    // 둘 중 하나 이상의 조건이 참일 때 실행되는 코드
}
```

### NOT 연산자
NOT 연산자는 조건을 부정합니다. 만약 조건이 참이면 거짓으로, 거짓이면 참으로 평가됩니다. 일반적으로 `!` 기호로 표시됩니다.

```java
if (!조건) {
    // 조건이 거짓일 때 실행되는 코드
}
```

논리 연산자를 사용하여 복잡한 조건을 평가하고, 프로그램이 필요에 맞게 동작하도록 할 수 있습니다.
