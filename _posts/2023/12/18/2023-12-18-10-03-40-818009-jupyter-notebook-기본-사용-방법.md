## 1. Jupyter Notebook 설치 <a name="installation"></a>

Jupyter Notebook을 설치하려면 Anaconda를 사용하는 것이 가장 쉬운 방법입니다. Anaconda를 설치하면 Jupyter Notebook과 함께 Python 및 다양한 데이터 과학 도구가 포함되어 있습니다.

## 2. 새 노트북 만들기 <a name="creating"></a>

Jupyter Notebook을 열고 메뉴에서 "새 노트북"을 선택하면 새로운 노트북이 생성됩니다. 노트북의 이름을 지정하고 Python 또는 다른 언어를 선택할 수 있습니다.

## 3. 셀 유형 <a name="cell-types"></a>

### 3.1. 코드 셀 <a name="code-cells"></a>

코드 셀은 Python 또는 다른 지원되는 언어로 코드를 실행할 수 있는 셀입니다. 코드를 입력하고 실행할 때 결과가 표시됩니다.

예시:

```python
# 두 숫자를 더하는 간단한 예제
a = 5
b = 3
print(a + b)
```

### 3.2. 마크다운 셀 <a name="markdown-cells"></a>

마크다운 셀은 텍스트, 제목, 목록, 링크, 이미지 등을 포맷팅할 수 있는 셀입니다. 이 문서도 마크다운 셀을 사용하여 작성되었습니다.

## 4. 셀 실행 <a name="running-cells"></a>

셀을 실행하려면 셀을 선택한 후 `Shift + Enter`를 누르면 됩니다. 코드 셀은 코드를 실행하고 마크다운 셀은 포맷팅된 결과를 표시합니다.

## 5. 저장 및 내보내기 <a name="saving-exporting"></a>

노트북을 저장하려면 메뉴에서 "저장"을 선택하거나 `Ctrl + S`를 누르면 됩니다. 내보내기 기능을 사용하여 HTML, PDF, 또는 다른 형식의 파일로 노트북을 저장할 수 있습니다.

이제 Jupyter Notebook의 기본 사용 방법을 익혔습니다. 데이터 과학 작업을 위해 노트북을 활용해 보세요!
