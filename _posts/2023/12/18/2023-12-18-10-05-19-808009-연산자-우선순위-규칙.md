연산자 우선순위는 프로그래밍에서 연산이 수행되는 순서를 결정합니다. 예를 들어, `3 + 4 * 2`와 같은 표현식에서 `*` 연산이 덧셈보다 먼저 이루어집니다.

## 1. 괄호 우선순위

괄호 안의 식은 다른 연산자보다 먼저 계산됩니다. 

```c
int result = (3 + 4) * 2; // result는 14가 됩니다.
```

## 2. 곱셈과 나눗셈 우선순위

곱셈과 나눗셈은 덧셈과 뺄셈보다 먼저 계산됩니다.

```c
int result = 3 + 4 * 2;  // result는 11이 됩니다.
```

## 3. 덧셈과 뺄셈 우선순위

덧셈과 뺄셈은 같은 우선순위를 갖고 서로 먼저 나온 것 부터 계산됩니다.

```c
int result = 3 + 4 - 2;  // result는 5가 됩니다.
```

프로그래밍 언어마다 조금씩 우선순위 규칙이 다를 수 있으니, 작성하는 코드의 우선순위를 명확히 이해하고 사용하는 것이 중요합니다.

이상으로 연산자 우선순위 규칙에 대해 알아보았습니다.
