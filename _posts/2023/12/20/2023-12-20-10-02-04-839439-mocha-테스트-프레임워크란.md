## Mocha 사용하기

Mocha를 사용하려면 먼저 npm을 사용하여 프로젝트에 Mocha를 설치해야 합니다.

```bash
npm install mocha --save-dev
```

프로젝트에 Mocha를 설치한 후에는 테스트 파일을 작성하고, 터미널에서 Mocha를 실행하여 테스트를 수행할 수 있습니다.

```bash
mocha test.js
```

## Mocha의 특징

Mocha는 다음과 같은 특징을 가지고 있습니다.
- 유연성: BDD 및 TDD 스타일의 테스트를 모두 지원합니다.
- 강력한 보고서: 실패한 테스트 케이스에 대한 자세한 보고서를 제공하여 디버깅을 용이하게 합니다.
- 다양한 기능: 비동기 코드 테스트, 코드 커버리지 분석과 같은 다양한 기능을 지원합니다.

Mocha는 JavaScript 프로젝트에서 테스트 주도 개발을 위한 강력한 도구로 활용될 수 있습니다.
