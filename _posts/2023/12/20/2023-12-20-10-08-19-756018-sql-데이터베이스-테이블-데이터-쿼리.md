SQL(Structured Query Language)을 사용하여 데이터베이스 테이블에서 데이터를 쿼리하는 방법을 설명하겠습니다.

## 테이블 데이터 조회

특정 테이블의 모든 데이터를 조회하기 위해서는 다음의 SQL 명령문을 사용합니다.

```sql
SELECT * FROM 테이블명;
```

이 명령문은 해당 테이블에 있는 모든 열 및 행을 가져옵니다.

## 특정 조건에 맞는 데이터 조회

특정 조건을 만족하는 데이터만을 조회하고 싶을 때는 `WHERE` 절을 사용할 수 있습니다. 예를 들어, 특정 컬럼의 값이 10보다 큰 경우의 데이터를 조회하려면 다음과 같이 작성합니다.

```sql
SELECT * FROM 테이블명 WHERE 컬럼명 > 10;
```

## 결과 정렬

데이터를 특정 기준에 따라 정렬하여 조회할 수도 있습니다. 예를 들어, 특정 컬럼을 기준으로 오름차순으로 정렬하려면 다음과 같이 작성합니다.

```sql
SELECT * FROM 테이블명 ORDER BY 컬럼명 ASC;
```

## 결과 제한

대량의 데이터가 있는 경우 결과를 제한하여 특정 개수만을 조회할 수도 있습니다. 이를 위해서는 `LIMIT` 절을 사용합니다.

```sql
SELECT * FROM 테이블명 LIMIT 10;
```

## 결론

위의 예시들을 통해 SQL을 사용하여 데이터베이스 테이블에서 데이터를 쿼리하는 방법을 간략히 알아보았습니다. SQL을 통해 다양한 방법으로 데이터를 가져올 수 있으며, 실제 상황에 맞게 적절한 쿼리를 작성하여 데이터를 조회할 수 있습니다.
