보통 `if` 조건문 다음에 `else`를 사용하여 다음과 같이 코드를 작성합니다:

```c
if (조건) {
    // 조건이 참일 때 실행할 코드
} else {
    // 조건이 거짓일 때 실행할 코드
}
```

위의 예제에서 `else` 블록은 `if` 조건이 거짓일 때 실행됩니다. `else` 블록은 생략될 수도 있으며, 조건이 거짓일 때 아무런 동작을 하지 않도록 할 수 있습니다.

`else`는 다양한 상황에서 프로그램의 흐름을 제어하는 데 유용하며, 여러분의 프로그램에 유연성을 부여하는 데 중요한 역할을 합니다.
