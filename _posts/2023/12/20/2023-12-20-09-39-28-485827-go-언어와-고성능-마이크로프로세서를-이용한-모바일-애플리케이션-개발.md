## Go 언어 소개

Go 언어는 **성능**, **간결한 문법**, **병렬성 지원** 등의 특징으로 유명합니다. 이 언어는 모바일 애플리케이션 개발에 적합한 **고성능**과 **효율성**을 제공합니다. 또한 Go 언어는 **가비지 컬렉션**, **동적 메모리 할당** 등으로 인한 성능 저하를 방지하며, 안정적인 성능을 유지할 수 있습니다.

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}
```

위의 예제는 간단한 Go 언어 코드로 "Hello, Go!"를 출력하는 프로그램입니다.
