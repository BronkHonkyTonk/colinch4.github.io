리눅스 시스템에서는 사용자의 패스워드를 변경하기 위해 `passwd` 명령어를 사용합니다. 이 명령어를 통해 사용자는 자신의 패스워드를 변경할 수 있고, 시스템 관리자는 다른 사용자의 패스워드를 변경할 수 있습니다.

### 사용법

사용자가 자신의 패스워드를 변경하려면 다음 명령어를 입력합니다.

```bash
passwd
```

시스템 관리자가 다른 사용자의 패스워드를 변경하려면 `sudo` 명령어를 사용하여 다음과 같이 입력합니다.

```bash
sudo passwd [사용자명]
```

### 예제

- 사용자 패스워드 변경:
  ```bash
  passwd
  ```

- 시스템 관리자가 다른 사용자의 패스워드 변경:
  ```bash
  sudo passwd johndoe
  ```

### 주의사항

- 패스워드는 보안을 위해 암호화되어 저장되므로 안전한 복잡한 패스워드를 사용하는 것이 좋습니다.
- 사용자는 다른 사용자의 패스워드를 변경할 때, 접근 권한이 필요합니다.

위와 같이 `passwd` 명령어를 사용하여 리눅스 시스템에서 패스워드를 변경할 수 있습니다.
