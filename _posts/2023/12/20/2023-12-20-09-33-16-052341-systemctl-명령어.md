### systemctl 명령어 사용 예시:

#### 1. 시스템 상태 확인
```shell
systemctl status
```

#### 2. 서비스 시작
```shell
systemctl start 서비스명
```

#### 3. 서비스 중지
```shell
systemctl stop 서비스명
```

#### 4. 서비스 재시작
```shell
systemctl restart 서비스명
```

#### 5. 서비스 활성화
```shell
systemctl enable 서비스명
```

#### 6. 서비스 비활성화
```shell
systemctl disable 서비스명
```

위와 같은 명령어들을 사용하여 시스템을 효율적으로 관리할 수 있습니다. systemd의 기능을 완전히 활용하기 위해서는 'systemctl' 명령어에 대한 이해와 숙달이 필요합니다.
