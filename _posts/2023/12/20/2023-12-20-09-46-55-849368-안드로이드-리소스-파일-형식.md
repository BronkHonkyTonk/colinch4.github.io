1. XML 파일 형식
    안드로이드 앱에서 사용되는 레이아웃, 문자열, 색상, 디자인 등의 정보를 담고 있는 XML 파일 형식이 있습니다. 이들은 각각 `layout.xml`, `strings.xml`, `colors.xml`과 같은 형식으로 저장됩니다.

2. 이미지 파일 형식
    PNG, JPEG, GIF 등과 같은 이미지 파일 형식들이 안드로이드 앱의 이미지 리소스로 사용됩니다. 

3. 미디어 파일 형식
    비디오 파일이나 사운드 파일 같은 미디어 리소스는 MP4, MP3, WAV와 같은 형식으로 저장됩니다.

4. 따로 정의된 포맷 파일
    AndroidManifest.xml과 같은 특별한 형식의 파일들 또한 안드로이드 앱 리소스에 의미 있는 파일 형식으로 분류됩니다.

이처럼 안드로이드 앱의 리소스 파일은 다양한 형식으로 제공되며, 각각 다른 목적으로 사용됩니다.
