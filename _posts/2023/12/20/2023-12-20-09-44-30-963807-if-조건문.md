C언어에서 if 조건문은 해당 조건이 참(true)일 때 특정한 코드 블록을 실행하는 데 사용됩니다. if 조건문은 다음과 같은 형식을 갖습니다.

```c
if (조건) {
    // 조건이 참일 때 실행할 코드
}
```

조건은 참(true) 또는 거짓(false)으로 평가되는 식이며, 중괄호 안에 있는 코드 블록은 조건이 참일 때 실행됩니다. 예를 들어, 사용자가 입력한 값이 특정 조건을 만족하면 특정 코드를 실행하도록 할 수 있습니다.

아래는 if 조건문을 사용한 간단한 예제입니다.

```c
#include <stdio.h>

int main() {
    int num = 10;

    if (num > 5) {
        printf("숫자는 5보다 큽니다.");
    }

    return 0;
}
```

위 예제에서는 변수 `num`이 5보다 큰지를 확인하고, 조건이 참이면 "숫자는 5보다 큽니다."를 출력합니다.

if 조건문은 프로그램에서 특정 조건에 따라 실행 흐름을 제어할 때 유용하게 활용됩니다. 여러 개의 조건을 확인해야 하는 경우에는 if-else 문이나 switch 문을 사용할 수 있습니다.
