리눅스 시스템에서 `uname` 명령어는 현재 시스템의 정보를 확인하는 데 사용됩니다.

### **사용법**

- `uname` 명령어를 사용하면 시스템의 정보를 볼 수 있습니다.

- 예를 들어, 다음 명령어를 사용하여 커널 버전을 확인할 수 있습니다.

```bash
uname -r
```

### **옵션**

- `-a` 옵션을 사용하면 모든 시스템 정보를 확인할 수 있습니다.

- `-s` 옵션을 사용하면 커널 이름을 확인할 수 있습니다.

- `-n` 옵션은 네트워크 노드 호스트 이름을 확인합니다.

- `-r` 옵션은 커널 릴리즈를 확인합니다.

### **예제**

- 커널 버전 확인:

```bash
uname -r
```

- 모든 시스템 정보 확인:

```bash
uname -a
```

- 커널 이름 확인:

```bash
uname -s
```

### **요약**

`uname` 명령어는 리눅스 시스템에서 현재 시스템의 정보를 확인하는 데 유용합니다.
