## 리소스 디렉토리
안드로이드 프로젝트의 리소스는 `res` 디렉토리에 저장됩니다. 각 리소스 유형은 별도의 하위 디렉토리에 저장됩니다.

### drawable 디렉토리
이 디렉토리에는 이미지 파일이 저장됩니다. 여러 dpi(dots per inch)에서 다양한 밀도의 화면에 대응하기 위해 각각의 하위 디렉토리(mdpi, hdpi, xhdpi 등)에 이미지를 저장할 수 있습니다.

### layout 디렉토리
화면의 UI 레이아웃을 구성하는 XML 파일들이 저장되는 곳입니다. 각 액티비티 또는 프래그먼트마다 별도의 레이아웃 파일을 생성할 수 있습니다.

### values 디렉토리
여기에는 문자열, 색상, 스타일 등과 같은 값들이 정의된 XML 파일들이 저장됩니다. `strings.xml`, `colors.xml`, `styles.xml` 등이 여기에 속합니다.

### raw 디렉토리
이 디렉토리에는 리소스로 사용되는 원시 파일들이 저장됩니다. 주로 오디오나 비디오 파일 등이 여기에 속합니다.

### anim, xml, menu, mipmap 등
기타 유형의 리소스는 각각의 디렉토리에 저장됩니다.

앱의 리소스 디렉토리 구조를 잘 이해하여, 앱의 UI 및 기능을 보다 쉽게 관리할 수 있습니다.
