## 1. 정수 타입
C#에서는 여러 종류의 정수 데이터 타입을 제공합니다. 예를 들어, `int`는 32비트 정수를 표현하고 `long`은 64비트 정수를 표현합니다.

```csharp
int myInt = 10;
long myLong = 1000000000;
```

## 2. 부동 소수점 타입
C#에서의 부동 소수점 타입은 실수를 나타냅니다. `float`는 32비트, `double`은 64비트 부동 소수점 값을 저장합니다.

```csharp
float myFloat = 3.14f;
double myDouble = 3.14159265359;
```

## 3. 문자 타입
문자 데이터 타입은 `char`를 사용하여 표현하며, 유니코드 문자를 저장합니다.

```csharp
char myChar = 'A';
```

## 4. 부울 타입
부울 데이터 타입은 `bool`로 표현되며, `true` 또는 `false` 값을 가집니다.

```csharp
bool isTrue = true;
bool isFalse = false;
```

C#은 이외에도 다양한 데이터 타입을 지원하며, 이러한 기본 데이터 타입들은 프로그래밍에서 매우 중요한 역할을 합니다.
