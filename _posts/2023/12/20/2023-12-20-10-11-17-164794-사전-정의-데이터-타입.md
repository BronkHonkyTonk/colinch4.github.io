# 사전 정의된 데이터 타입

C#은 많은 사전 정의된 데이터 타입을 제공하여 다양한 데이터를 저장하고 처리할 수 있습니다.

## 1. int
int는 정수를 저장하는 데 사용됩니다. C#에서는 4바이트로 표현되며, 대략 -2,147,483,648부터 2,147,483,647까지의 값을 저장할 수 있습니다.

```csharp
int myNumber = 10;
```

## 2. string
string은 문자열을 저장하는 데 사용됩니다. 문자들의 시퀀스로 구성되며, 큰따옴표(")로 둘러싸여 있습니다.

```csharp
string myName = "John";
```

## 3. bool
bool은 불리언 값을 저장하는 데 사용됩니다. true 또는 false 값을 가질 수 있으며, 일반적으로 조건문에서 사용됩니다.

```csharp
bool isTrue = true;
```

## 4. double
double은 부동 소수점 숫자를 저장하는 데 사용됩니다. 8바이트로 표현되며, 높은 정밀도를 가집니다.

```csharp
double myValue = 3.14;
```

이러한 사전 정의된 데이터 타입을 사용하여 C# 프로그램에서 다양한 유형의 데이터를 다룰 수 있습니다.

더 많은 내용을 찾고 싶다면 [Microsoft 공식 문서](https://docs.microsoft.com/en-us/dotnet/csharp/language-reference/builtin-types/built-in-types)를 확인해보세요.
