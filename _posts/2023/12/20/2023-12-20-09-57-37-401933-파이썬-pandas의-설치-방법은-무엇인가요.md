1. 먼저, 터미널 또는 명령 프롬프트를 열고 다음 명령어를 입력하여 pip를 업그레이드합니다.
```
pip install --upgrade pip
```

2. 이제 아래의 명령어를 사용하여 pandas를 설치합니다.
```
pip install pandas
```

위 두 단계를 따르면 파이썬 환경에 pandas를 설치할 수 있습니다.
