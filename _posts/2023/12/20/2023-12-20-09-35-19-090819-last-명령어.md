리눅스 운영 체제에서 `last` 명령어는 시스템에 로그인한 사용자들의 세션 로그인 기록을 표시하는 데 사용됩니다. 이로써 시스템 관리자는 사용자들의 로그인/로그아웃 동작 및 접속 시간 등을 추적하여 시스템 보안에 도움이 됩니다.

## `last` 명령어 사용법

`last` 명령어의 기본적인 사용법은 다음과 같습니다:

```bash
last
```

이 명령은 기본적으로 `/var/log/wtmp` 파일을 읽어 최근 로그인 활동을 기록하며, root 권한이 필요합니다. 특정 사용자의 로그인 기록을 확인하려면 아래와 같이 사용합니다:

```bash
last <username>
```

명령을 통해 특정 일자의 로그인 기록을 확인할 수도 있습니다:

```bash
last -t YYYYMMDDHHMMSS
```

여기서 `-t` 옵션은 특정 일자 이후의 로그인을 표시하며, `YYYYMMDDHHMMSS`는 일자와 시간을 의미합니다.

## `last` 명령어의 결과 해석

`last` 명령어의 결과는 다음과 같은 컬럼들을 표시합니다:

- 사용자명
- 터미널
- IP 주소 또는 호스트명
- 로그인 시각 (시스템 시간 기준)
- 로그아웃 시각 또는 현재 로그인 상태

`last` 명령어는 옵션을 통해 더 많은 정보를 표시하거나 필터링하여 보여주도록 구성할 수도 있습니다.
