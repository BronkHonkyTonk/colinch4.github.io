### 1. systemctl 명령어를 사용한 확인
```bash
systemctl list-unit-files --type=service
```

위 명령어를 실행하면 자동으로 시작되는 모든 서비스 목록이 표시됩니다.

### 2. 서비스 목록 파일 확인
리눅스 시스템에서는 부팅 시 자동으로 시작되는 서비스 목록이 `/etc/rc.d/`, `/etc/init.d/`, `/etc/systemd/system/` 디렉토리에 저장되어 있습니다. 해당 디렉토리에서 서비스 목록을 확인할 수 있습니다.

### 3. 서비스 상태 확인
부팅 시 자동으로 시작되는 서비스 목록을 확인할 수 있는 다른 방법은 각 서비스의 상태를 확인하는 것입니다. `ps`나 `top` 명령어를 사용하여 현재 실행 중인 프로세스를 확인할 수 있습니다.

이러한 방법들을 통해 리눅스 시스템 부팅 시 자동으로 시작되는 서비스 목록을 확인할 수 있습니다.
