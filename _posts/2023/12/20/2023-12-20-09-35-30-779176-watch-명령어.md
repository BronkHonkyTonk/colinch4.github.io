리눅스 시스템에서 `watch` 명령어는 특정 명령어나 프로그램의 출력을 주기적으로 업데이트하여 다시 실행하고 표시하는 데 사용됩니다.

## 사용법

`watch` 명령어를 사용하여 특정 명령어나 프로그램의 출력을 주기적으로 업데이트하려면 다음과 같이 입력합니다:

```bash
watch [옵션] 명령어
```

예를 들어, 다음 명령어는 `ls` 명령어의 결과를 2초마다 업데이트하여 표시합니다:

```bash
watch -n 2 ls -l
```

## 옵션

`watch` 명령어에 사용할 수 있는 주요 옵션은 다음과 같습니다:

- `-n, --interval` : 업데이트 간격을 초 단위로 지정합니다. 기본 값은 2초입니다.
- `-d, --differences` : 변경점만 강조하여 표시합니다.
- `-h, --help` : 도움말을 표시합니다.

## 예제

`watch` 명령어의 간단한 예제는 다음과 같습니다:

```bash
watch -n 1 -d free -h
```

이 명령어는 1초마다 시스템의 메모리 사용량을 표시하고, 변경된 부분만 반영하여 표시합니다.

## 요약

`watch` 명령어는 주기적으로 특정 명령어나 프로그램의 출력을 업데이트하여 다시 실행하여 표시하는 유용한 도구입니다. 변경된 부분만 표시하거나 특정 시간 간격으로 업데이트하는 등의 옵션을 활용하여 시스템 모니터링이나 디버깅 작업에 활용할 수 있습니다.
