## 템플릿 언어 간단 소개

Django 템플릿 언어는 HTML 템플릿 안에서 Python 코드를 실행하고, 동적으로 컨텐츠를 생성할 수 있게 해줍니다. 이렇게 하면 서버 측 코드와 클라이언트 측 템플릿이 분리되어 유지보수가 용이해집니다.

## 변수 출력

템플릿 언어를 사용하면 {{ 변수 }}와 같은 형태로 파이썬 변수를 출력할 수 있습니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```python
{{ user.name }}
```

위의 코드는 `user` 객체의 `name` 속성을 템플릿에 출력하는 예시입니다.

## 조건문과 반복문

템플릿 언어에서는 파이썬과 비슷한 형식으로 조건문과 반복문을 사용할 수 있습니다. 다음은 간단한 조건문과 반복문의 예시입니다.

```python
{% if user.is_authenticated %}
    <p>Welcome, {{ user.name }}!</p>
{% else %}
    <p>Please log in</p>
{% endif %}

{% for item in items %}
    <li>{{ item }}</li>
{% endfor %}
```

## 템플릿 상속

Django의 템플릿 상속은 공통 레이아웃을 만들고, 해당 레이아웃을 다른 템플릿에서 확장하여 사용할 수 있게 해줍니다. 기본 틀을 만들고 변하는 부분만 따로 작성함으로써 코드 중복을 최소화할 수 있습니다.

이제 템플릿 언어를 활용하여 Django 애플리케이션을 보다 동적으로 만들 수 있는 방법에 대해 알아보았습니다. Django 공식 문서 및 튜토리얼을 참고하여 실전에서 템플릿 언어를 활용해보시기를 권장합니다.
