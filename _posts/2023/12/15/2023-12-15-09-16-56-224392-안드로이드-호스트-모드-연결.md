## 호스트 모드 연결을 위한 요구 사항
안드로이드 기기가 호스트 모드로 작동하려면 안드로이드 운영 체제가 USB 호스트 모드를 지원해야 합니다. 또한 Micro-B 또는 USB-C 타입의 USB OTG (On-The-Go) 케이블이 필요합니다. 

## 안드로이드 호스트 모드 활성화
안드로이드 기기의 호스트 모드를 활성화하려면 개발자 모드를 활성화해야 합니다. 개발자 모드를 활성화하는 방법은 다음과 같습니다.

```java
1. 설정 앱을 엽니다.
2. 정보(일반 관리) > 소프트웨어 정보를 선택합니다.
3. 빌드 번호를 여러 번 연속해서 탭하여 개발자 모드를 활성화합니다.
```

이제 안드로이드 기기는 호스트 모드로 작동할 수 있게 되었습니다.

## 외부 USB 장치 연결
이제 USB OTG 케이블을 사용하여 안드로이드 기기의 USB 포트에 외부 USB 장치를 연결할 수 있습니다. 안드로이드 기기는 연결된 장치를 자동으로 감지하고 연결하여 사용할 수 있습니다.

안드로이드 호스트 모드를 사용하여 외부 USB 장치와 안드로이드 기기를 연결하는 방법에 대해 간단히 살펴보았습니다. 호스트 모드를 통해 안드로이드 기기와 외부 USB 장치를 편리하게 연결하여 데이터를 주고받을 수 있습니다.
