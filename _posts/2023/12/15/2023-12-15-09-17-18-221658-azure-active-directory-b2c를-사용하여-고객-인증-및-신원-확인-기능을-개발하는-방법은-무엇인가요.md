### Azure AD B2C 소개
Azure AD B2C는 클라우드 기반 ID 관리 서비스로, 고객에게 회원가입, 로그인, 비밀번호 재설정 등을 제공합니다. 소셜 ID 공급자와의 연동을 통해 외부 ID로 로그인하고 사용자 지정 프로필 특성을 관리할 수 있습니다.

### Azure AD B2C 구성
Azure Portal에서 Azure AD B2C 인스턴스를 생성하고 구성합니다. 정책, 속성 및 ID 공급자 설정을 구성하여 사용자 인증 및 관리를 모두 구축합니다.

### 애플리케이션에 Azure AD B2C 통합
Azure AD B2C와 애플리케이션을 연동하여 인증 코드 및 액세스 토큰을 통해 사용자를 인증하고 앱 접근 권한을 부여합니다.

### 사용자 관리
사용자 관리를 통해 신규 사용자 등록, 기존 사용자 프로필 편집 등을 할 수 있습니다.

### 사용자 자격 증명 및 암호화
사용자 자격 증명을 관리하고, 암호화된 데이터를 사용하여 보안 개선을 시도합니다.

### 사용자 정의 프로필
Azure AD B2C에서는 사용자 지정 프로필 특성을 정의하여, 애플리케이션에서 필요로 하는 사용자 정보를 추가로 수집할 수 있습니다.

### 정책 및 권한 부여
Azure AD B2C에서는 회원가입, 로그인, 비밀번호 재설정 등을 정의하는 다양한 정책을 설정하고, 사용자에 대한 접근 권한을 통제합니다.

### 보안 및 인증
멀티 팩터 인증, 조건부 액세스 등의 기능을 통해 보안을 강화하고 인증 요구사항을 충족시킵니다.

이러한 방법을 통해 Azure AD B2C를 사용하여 고객 인증 및 신원 확인 기능을 개발할 수 있습니다. Azure AD B2C의 강력한 기능과 유연성을 활용하여 애플리케이션의 사용자 경험과 보안을 향상시킬 수 있습니다.
