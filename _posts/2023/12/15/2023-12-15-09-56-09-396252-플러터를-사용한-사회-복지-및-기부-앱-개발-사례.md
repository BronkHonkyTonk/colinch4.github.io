# 1. 소개
사회 복지와 기부를 촉진하기 위해 **플러터(Flutter)**로 개발한 모바일 앱에 대해 다룰 것입니다. 이 앱은 사용자가 쉽게 기부를 할 수 있고 사회적 문제에 대한 정보를 제공합니다.

# 2. 기능
- 사용자 프로필 관리
- 기부 및 후원 기능
- 실시간 사회 문제 정보 제공
- 커뮤니티 기능

# 3. 기술 사용
개발에는 **Dart** 언어와 플러터 프레임워크를 사용했습니다. 또한 **Firebase**와 **인앱 결제** API를 통해 사용자 경험을 향상시켰습니다.

```dart
// Dart 코드 예시
class Donation {
  String title;
  double amount;
  String category;
  // ...
}
```

# 4. UX/UI 설계
사용자 친화적이고 직관적인 UI를 위해 **Material Design**을 따랐습니다. 플러터의 다양한 UI 위젯과 애니메이션을 활용하여 매끄러운 인터페이스를 구현했습니다.

# 5. 향후 계획 및 개선
다음 단계로는 **머신러닝**을 통해 사용자 관심에 맞는 기부 추천 기능을 개발할 예정입니다. 또한 **안전한 결제 시스템**과 **빠른 데이터 처리**를 위해 기술적인 개선을 계획하고 있습니다.

# 6. 결론
플러터를 이용한 사회 복지 및 기부 앱은 사용자들에게 쉽고 간편한 기부 경험을 제공하며 사회 문제에 대한 인식을 높일 수 있는 효과적인 도구입니다.
