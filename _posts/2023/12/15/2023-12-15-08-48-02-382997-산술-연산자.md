1. 더하기 `+`: 두 숫자를 더합니다.
2. 빼기 `-`: 왼쪽 피연산자에서 오른쪽 피연산자를 뺍니다.
3. 곱하기 `*`: 두 숫자를 곱합니다.
4. 나누기 `/`: 왼쪽 피연산자를 오른쪽 피연산자로 나눕니다.
5. 나머지 `%`: 왼쪽 피연산자를 오른쪽 피연산자로 나눈 나머지 값을 반환합니다.

```javascript
let x = 10;
let y = 5;

let sum = x + y; // 15
let difference = x - y; // 5
let product = x * y; // 50
let quotient = x / y; // 2
let remainder = x % y; // 0
```
