아래는 **WHERE** 절을 사용하여 특정 값의 상관관계를 조회하는 간단한 SQL 쿼리의 예시입니다.

```sql
SELECT *
FROM 테이블명
WHERE 칼럼명 = '특정값';
```

위의 쿼리에서 **테이블명**은 조회할 데이터가 있는 테이블의 이름을 나타내며, **칼럼명**은 특정 값과 관련된 칼럼의 이름을 나타냅니다. **'특정값'**은 원하는 상관관계를 조회하려는 특정 값입니다.

예를 들어, 만약 **고객** 테이블에서 특정 **도시**의 **고객**들을 조회하고 싶다면, 다음과 같은 **SQL** 쿼리를 사용할 수 있습니다.

```sql
SELECT *
FROM 고객
WHERE 도시 = '서울';
```

이렇게 하면 **도시**가 '서울'인 **고객**들의 데이터만을 조회할 수 있습니다.

**WHERE** 절을 사용하면 데이터베이스에서 특정 값을 기준으로 상관된 데이터를 쉽게 조회할 수 있습니다.
