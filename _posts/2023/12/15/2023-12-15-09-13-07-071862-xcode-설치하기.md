# Xcode 설치하기

Xcode는 iOS 및 macOS 앱을 개발하는 데 필요한 Apple의 통합 개발 환경(IDE)입니다. 이 글에서는 macOS에 Xcode를 설치하는 방법에 대해 알아보겠습니다.

## 1. App Store에서 Xcode 다운로드

macOS에서 **App Store**를 열어 **Xcode**를 검색합니다. **다운로드** 버튼을 클릭하여 Xcode를 다운로드하고 설치합니다.

## 2. Command Line Tools 설치

터미널을 열고 다음 명령어를 입력하여 Xcode Command Line Tools를 설치합니다.

```shell
xcode-select --install
```

이제 Xcode가 정상적으로 설치되었으며, iOS 및 macOS 앱을 개발할 준비가 되었습니다.

이제 Xcode를 사용하여 앱 개발을 시작할 수 있습니다. 문제가 발생하거나 관련된 질문이 있으면 Xcode 설치 문제 해결을 확인하거나 Apple 개발자 지원팀에 문의하십시오.
