자바에서는 다양한 산술 연산자를 사용하여 사칙 연산을 수행할 수 있습니다. 이번 글에서는 자바에서의 덧셈, 뺄셈, 곱셈, 나눗셈 연산을 살펴보겠습니다.

## 덧셈 연산

덧셈 연산은 `+` 연산자를 사용하여 수행됩니다. 아래는 덧셈 연산의 예시입니다.

```java
int a = 5;
int b = 3;
int sum = a + b;
System.out.println("덧셈 결과: " + sum);
```

위 코드는 변수 `a`와 `b`를 더한 결과를 `sum`이라는 변수에 저장하고, 그 값을 출력합니다.

## 뺄셈 연산

뺄셈 연산은 `-` 연산자를 사용하여 수행됩니다. 아래는 뺄셈 연산의 예시입니다.

```java
int a = 5;
int b = 3;
int difference = a - b;
System.out.println("뺄셈 결과: " + difference);
```

위 코드는 변수 `a`에서 `b`를 뺀 결과를 `difference`라는 변수에 저장하고, 그 값을 출력합니다.

## 곱셈 연산

곱셈 연산은 `*` 연산자를 사용하여 수행됩니다. 아래는 곱셈 연산의 예시입니다.

```java
int a = 5;
int b = 3;
int product = a * b;
System.out.println("곱셈 결과: " + product);
```

위 코드는 변수 `a`와 `b`를 곱한 결과를 `product`라는 변수에 저장하고, 그 값을 출력합니다.

## 나눗셈 연산

나눗셈 연산은 `/` 연산자를 사용하여 수행됩니다. 아래는 나눗셈 연산의 예시입니다.

```java
int a = 6;
int b = 3;
int quotient = a / b;
System.out.println("나눗셈 결과: " + quotient);
```

위 코드는 변수 `a`를 `b`로 나눈 몫을 `quotient`라는 변수에 저장하고, 그 값을 출력합니다.

이렇게 자바에서는 산술 연산자를 활용하여 간단한 사칙 연산을 수행할 수 있습니다.
