# Azure Portal에 로그인
Azure Portal에 로그인하려면 웹 브라우저를 열고 [Azure Portal](https://portal.azure.com/)에 접속합니다. Azure 계정 정보로 로그인합니다.

# 서비스 찾기
Azure Portal 대시보드에서 "모든 서비스"를 클릭한 다음 사용하려는 서비스를 검색합니다. 예를 들어, "가상 머신"을 검색하여 Azure 가상 머신 서비스를 찾을 수 있습니다.

# 서비스 구성
서비스를 선택하면 해당 서비스를 구성하고 관리할 수 있는 옵션을 제공합니다. 기본적으로 가상 머신의 경우 "만들기", "모니터링", "액세스 제어" 등의 옵션이 제공됩니다.

# 리소스 만들기
Azure Portal을 사용하여 새로운 리소스를 만들려면 해당 서비스의 "만들기" 옵션을 선택하고 필요한 구성 설정을 완료합니다.

이와 같이 Azure Portal을 통해 다양한 Azure 서비스를 쉽게 관리할 수 있습니다. Azure Portal은 사용자 친화적이며 강력한 도구로써 클라우드 리소스를 효율적으로 관리하는 데 도움을 줍니다.
