### 텍스트 서식

마크다운에서 텍스트 서식을 다루는 방법은 다음과 같습니다.

- **굵게**: `**이것은 굵은 텍스트입니다.**`
- _기울임_: `*이것은 기울인 텍스트입니다.*`
- `인라인 코드`: `` `인라인 코드` ``

---

### 코드 블록

코드 블록은 다음과 같이 작성합니다.

```go
package main

import "fmt"

func main() {
	fmt.Println("Hello, Markdown!")
}
```

---

### 링크

마크다운에서 링크를 생성하는 방법은 다음과 같습니다.

외부 링크: `[텍스트](링크 URL)`

내부 링크: `[텍스트](#section-id)`

---

### 이미지

이미지를 삽입하는 방법은 외부 및 내부 이미지에 따라 조금 다를 수 있습니다.

외부 이미지: `![대체 텍스트](이미지 URL)`

내부 이미지: `![대체 텍스트](상대 경로)`

이제 마크다운의 기본적인 항목에 대해 간단하게 살펴보았습니다. 이제 실제로 어떻게 사용되었는지를 더 자세히 살펴볼 수 있습니다.
