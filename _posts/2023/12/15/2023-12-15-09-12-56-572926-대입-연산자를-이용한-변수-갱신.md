변수를 갱신하거나 증감시킬 때는 대입 연산자를 사용합니다.  

## 대입 연산자
대입 연산자는 변수에 값을 할당하는 데 사용됩니다. 자바에서는 `=` 기호가 대입 연산자로 사용됩니다.

```java
int x = 5; // 변수 x에 5를 할당
```

## 변수 갱신
변수를 현재 값에 더하거나 뺄 때는 증감 연산자를 사용합니다.

### 증가 연산자
```java
x++; // x의 값을 1 증가
```

### 감소 연산자
```java
x--; // x의 값을 1 감소
```

## 복합 대입 연산자
변수에 값을 더하거나 빼는 등의 갱신 작업을 한 번에 처리하는 복합 대입 연산자도 있습니다.

### 덧셈 연산 후 대입
```java
x += 3; // x에 3을 더한 값을 할당
```

### 뺄셈 연산 후 대입
```java
x -= 2; // x에서 2를 뺀 값을 할당
```

이와 같이 대입 연산자를 이용하여 변수를 갱신하면 코드를 더 간결하게 작성할 수 있습니다.

변수 갱신에 대한 내용은 Oracle Java Documentation에서도 [참고할 수 있습니다](https://docs.oracle.com/javase/tutorial/java/nutsandbolts/op2.html).

위의 예시를 통해 변수를 갱신하는 다양한 방법을 알아보았습니다. Java에서는 대입 연산자와 각종 증감 연산자를 활용하여 효율적인 변수 갱신을 할 수 있습니다.
