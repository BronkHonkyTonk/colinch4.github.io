SQL에서는 데이터베이스의 테이블에서 정보를 조회하고 분석하기 위해 다양한 기능을 제공합니다. 이 중에서도 집합 함수와 WHERE 절은 데이터베이스에서 특정 조건에 맞는 데이터를 선택하거나 집계하는 데 사용됩니다. 이번 글에서는 각각의 기능과 사용 방법에 대해 알아보겠습니다.

## WHERE 절

`WHERE` 절은 SQL 쿼리에서 특정 조건을 지정하여 그 조건을 만족하는 행만을 결과로 반환할 수 있게 합니다. 예를 들어, 다음과 같은 방식으로 사용될 수 있습니다:

```sql
SELECT * FROM 테이블명
WHERE 조건식;
```

여기서 `조건식`은 특정한 조건을 나타내며, 해당 조건을 만족하는 행만이 결과로 반환됩니다.

## 집합 함수

집합 함수는 여러 행의 데이터를 하나의 값을 계산하여 표시하는 데 사용됩니다. 일반적으로 다음과 같은 함수들이 있습니다:

- `COUNT`: 특정 열의 행의 수를 계산
- `SUM`: 특정 열의 값을 합산
- `AVG`: 특정 열의 평균값을 계산
- `MIN`: 특정 열에서 가장 작은 값을 찾아냄
- `MAX`: 특정 열에서 가장 큰 값을 찾아냄

예를 들어, 다음과 같이 사용될 수 있습니다:

```sql
SELECT COUNT(열명) FROM 테이블명;
```

## 사용 방법의 차이

`WHERE` 절은 특정 조건을 만족하는 행을 선택하기 위해 사용되고, 집합 함수는 특정 열의 값을 집계하는 데 사용됩니다. 따라서, `WHERE` 절은 행을 필터링하는 데 사용되고, 집합 함수는 열의 값을 계산하는 데 사용됩니다.

데이터베이스 쿼리에서 `WHERE` 절과 집합 함수의 적절한 사용은 특정 업무 요건과 분석 목적에 따라 다르므로, 각 상황에 맞게 적절히 선택하여 사용해야 합니다.

이상으로 SQL의 `WHERE` 절과 집합 함수의 사용 차이에 대해 알아보았습니다. 감사합니다.
