Azure는 다양한 기능과 유연성을 제공하는 클라우드 플랫폼으로, **Azure Virtual Machines**를 사용하여 가상 컴퓨터를 생성할 수 있습니다. 이를 통해 안정적이고 안전한 환경에서 원격으로 가상 서버를 관리하고 사용할 수 있습니다.

이제 Azure Portal을 통해 Azure Virtual Machines를 생성하는 방법을 알아보겠습니다.

## 1. Azure Portal에 로그인

가장 먼저, Azure Portal에 로그인합니다. Azure 구독이 없는 경우에는 구독을 생성한 후 로그인해야 합니다.

## 2. 가상 머신 생성

Azure Portal에서 다음 단계를 따라 새로운 가상 머신을 만듭니다.

### 2.1. 리소스 만들기

Azure Portal에서 **"리소스 만들기"** 버튼을 클릭합니다.

### 2.2. 가상 머신 선택

검색 상자에 **"가상 머신"**을 입력하여 **가상 머신** 리소스를 선택합니다.

### 2.3. 가상 머신 구성

가상 머신의 이름, 지역, 가상 네트워크, 이미지, 크기 등 다양한 구성 옵션을 선택합니다.

### 2.4. 다른 구성 옵션

가상 머신에 대한 기타 구성 옵션을 선택하고 필요한 경우 네트워크 및 보안 구성을 추가합니다.

### 2.5. 가상 머신 만들기

마지막으로, 모든 구성 및 옵션을 확인한 후 가상 머신을 만듭니다.

이제 Azure Virtual Machines를 사용하여 가상 컴퓨터를 생성하는 방법을 숙지했습니다. Azure Portal을 통해 손쉽게 가상 서버를 만들고 관리할 수 있습니다.
