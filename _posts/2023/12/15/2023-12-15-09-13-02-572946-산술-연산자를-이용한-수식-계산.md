이번에는 자바에서 산술 연산자를 활용하여 기본적인 수식을 계산하는 방법에 대해 알아보겠습니다.

## 덧셈 연산

덧셈 연산을 하기 위해서는 `+` 연산자를 사용합니다. 예를 들어, 2와 3을 더하려면 다음과 같이 코드를 작성할 수 있습니다.

```java
int sum = 2 + 3;
System.out.println("Sum: " + sum);
```

## 뺄셈 연산

뺄셈 연산을 하기 위해서는 `-` 연산자를 사용합니다. 10에서 5를 빼는 코드는 다음과 같이 작성할 수 있습니다.

```java
int difference = 10 - 5;
System.out.println("Difference: " + difference);
```

## 곱셈 연산

곱셈 연산을 하기 위해서는 `*` 연산자를 사용합니다. 예를 들어, 4와 6을 곱하는 코드는 아래와 같습니다.

```java
int product = 4 * 6;
System.out.println("Product: " + product);
```

## 나눗셈 연산

나눗셈 연산은 `/` 연산자를 사용합니다. 예를 들어, 20을 5로 나누는 코드는 다음과 같이 작성할 수 있습니다.

```java
int quotient = 20 / 5;
System.out.println("Quotient: " + quotient);
```

이렇게 자바에서는 간단한 산술 연산을 위한 연산자를 제공하고 있으며, 이를 활용하여 수식을 계산할 수 있습니다.
