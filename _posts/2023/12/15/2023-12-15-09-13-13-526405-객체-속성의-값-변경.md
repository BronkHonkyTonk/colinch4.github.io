# **객체 속성의 값 변경**

자바스크립트에서 객체는 속성과 값으로 구성되어 있습니다. 때때로 객체의 속성 값을 변경해야할 필요가 있습니다. 이 기사에서는 기존 객체의 속성 값을 변경하는 방법에 대해 살펴보고자 합니다.

## **객체 속성에 접근**

객체의 속성 값은 점 표기법이나 대괄호 표기법을 사용하여 접근할 수 있습니다.

```javascript
const person = {
  name: 'John',
  age: 30
};

// 점 표기법 사용
console.log(person.name); // John

// 대괄호 표기법 사용
console.log(person['age']); // 30
```

## **객체 속성 값 변경하기**

객체의 속성 값을 변경하려면 점 표기법이나 대괄호 표기법을 사용하여 해당 속성에 새로운 값을 할당하면 됩니다.

```javascript
// 점 표기법 사용
person.age = 25;

// 대괄호 표기법 사용
person['name'] = 'Mike';
```
