## SQL 데이터베이스란?
SQL 데이터베이스는 구조화된 쿼리 언어(SQL)를 사용하여 데이터를 저장, 관리, 및 검색하는데 사용되는 시스템입니다. 데이터베이스는 테이블 형태로 구성되어 있고, 각 테이블은 관련된 데이터를 저장합니다.

## SQL 데이터베이스 생성 단계
SQL 데이터베이스를 생성하는 단계는 다음과 같습니다:
1. **데이터베이스 생성**: 데이터베이스 관리 시스템(DBMS)에서 새로운 데이터베이스를 생성합니다.
2. **테이블 생성**: 생성된 데이터베이스 내에서 필요한 테이블을 정의하고 생성합니다.
3. **필드 정의**: 각 테이블에 필요한 필드를 정의하고 데이터 형식을 지정합니다.
4. **제약 조건 추가**: 필요한 경우 테이블에 제약 조건(Constraints)을 추가하여 데이터 무결성을 보장합니다.

## 예제 코드
다음은 SQL 데이터베이스와 테이블을 생성하는 예제 코드입니다:

```sql
-- 데이터베이스 생성
CREATE DATABASE MyDatabase;

-- 사용할 데이터베이스 선택
USE MyDatabase;

-- 테이블 생성
CREATE TABLE Customers (
    CustomerID INT PRIMARY KEY,
    CustomerName VARCHAR(50),
    Email VARCHAR(100)
);
```
