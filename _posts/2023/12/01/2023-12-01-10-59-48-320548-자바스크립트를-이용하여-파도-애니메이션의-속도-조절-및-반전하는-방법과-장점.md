## 파도 애니메이션의 속도 조절하기

파도 애니메이션의 속도를 조절하기 위해서는 자바스크립트의 `setTimeout()` 함수를 사용할 수 있습니다. 이 함수를 사용하여 특정 시간 간격마다 원하는 동작을 수행할 수 있습니다. 

```javascript
function animateWave() {
  // 파도 애니메이션의 코드를 작성합니다.
}

function startAnimation(speed) {
  // 파도 애니메이션을 시작하고 주어진 속도로 동작하도록 설정합니다.

  // 주어진 속도마다 setTimeout 함수로 animateWave 함수를 호출합니다.
  // 예를 들어, speed 가 100이라면 0.1초 마다 animateWave 함수가 호출됩니다.

// 파도 애니메이션을 0.5초 마다 동작하도록 시작합니다.
startAnimation(500);
```

```javascript

  // 파도 애니메이션의 방향을 반전시킵니다.
```
