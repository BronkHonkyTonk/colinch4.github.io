Go 언어에서의 변수는 다른 언어들과 매우 유사하게 사용됩니다. 변수를 선언하고 값을 할당하는 방법을 살펴보겠습니다.

## 변수 선언

Go 언어에서 변수는 `var` 키워드를 사용하여 선언됩니다. 예를 들어, 정수형 변수 `age`를 선언하려면 다음과 같이 작성할 수 있습니다:

```go
var age int
```

여기서 `var`는 변수를 선언한다는 것을 나타내고, `age`는 변수의 이름이며, `int`는 변수의 타입입니다.

변수를 선언하는 동시에 초기값을 할당할 수도 있습니다. 예를 들어, `name`이라는 문자열 변수를 선언하며 동시에 초기값 "Alice"을 할당하려면 다음과 같이 작성할 수 있습니다:

```go
var name string = "Alice"
```

`string`은 변수의 타입을 나타내며, "=" 연산자를 통해 초기값을 할당합니다.

Go 언어에서는 타입을 생략하고 변수를 선언할 수도 있습니다. 타입이 자동으로 추론됩니다. 예를 들어:

```go
var height = 180
```

이 경우 `height` 변수는 정수형으로 추론됩니다.

## 변수 사용

변수를 사용하는 것은 변수의 이름을 사용하여 해당 변수에 액세스하는 것과 같습니다. 예를 들어, 변수 `age`에 값을 할당하고 출력해 보겠습니다.

```go
age = 30
fmt.Println(age)
```

여기서 `age`에 30을 할당하고, `fmt.Println` 함수를 사용하여 `age` 변수의 값을 출력합니다.

이제 Go 언어에서 변수를 선언하고 사용하는 기본적인 방법을 살펴보았습니다.
