데이터베이스에서 정보를 업데이트하고자 할 때 **UPDATE** 문을 사용합니다. UPDATE 문은 기존 레코드의 값을 업데이트하는 데 사용됩니다. 

다음은 **UPDATE** 문을 사용하여 데이터 일괄 수정을하는 방법에 대한 예제입니다.

## 1. UPDATE 문 구문

```sql
UPDATE 테이블명
SET 컬럼1 = 값1, 컬럼2 = 값2
WHERE 조건;
```

위의 구문에서:
- `테이블명`은 업데이트를 수행할 테이블의 이름입니다.
- `컬럼1`, `컬럼2`는 업데이트할 열(column)의 이름이며, `값1`, `값2`는 해당 값으로 업데이트됩니다.
- `WHERE` 절은 선택적이며, 조건을 지정하여 특정 레코드만을 업데이트할 수 있습니다. 조건이 생략되면 모든 레코드를 업데이트합니다.

## 2. UPDATE 예제

예를 들어, 아래와 같은 테이블이 있다고 가정해봅시다.

**사용자 (Users) 테이블**

| ID | 이름   | 나이 |
|----|-------|-----|
| 1  | 홍길동 | 30  |
| 2  | 홍계월 | 25  |
| 3  | 아이유 | 28  |

만약 나이가 30세 이상인 모든 사용자의 나이를 31세로 업데이트하고 싶다면 다음과 같은 UPDATE 문을 사용할 수 있습니다.

```sql
UPDATE 사용자
SET 나이 = 31
WHERE 나이 >= 30;
```

위의 예에서, **Users** 테이블의 나이가 30세 이상인 사용자들의 나이가 31으로 업데이트됩니다.

이제 **UPDATE** 문을 사용하여 데이터를 일괄 수정하는 방법에 대해 알게되었습니다. 데이터베이스에서 정보를 효율적으로 업데이트하는 데 도움이 되길 바랍니다.
