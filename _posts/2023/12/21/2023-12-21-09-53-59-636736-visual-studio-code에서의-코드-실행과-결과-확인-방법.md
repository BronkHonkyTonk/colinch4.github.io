Visual Studio Code(이하 VSC)는 인기있는 통합 개발 환경 중 하나로, Python 개발을 위한 다양한 기능을 제공합니다. 이번 포스트에서는 VSC에서 Python 코드를 실행하고 결과를 확인하는 방법에 대해 알아보겠습니다.

## Python 확장 프로그램 설치

VSC에서 Python 코드를 실행하기 위해서는 먼저 Python 확장 프로그램을 설치해야 합니다. VSC를 열고 왼쪽의 확장 아이콘을 클릭한 후 "Python"을 검색하여 해당 확장 프로그램을 설치합니다.

## Python 파일 열기

Python 파일을 VSC에서 열어야 코드를 실행할 수 있습니다. VSC를 열고 상단 메뉴에서 "파일" > "열기"를 선택하여 Python 파일을 엽니다.

## 코드 실행

Python 파일을 열었다면, 코드를 실행할 준비가 끝났습니다. 코드를 실행하려면 `Ctrl+Shift+P`를 누르고 "Python: Run Python File in Terminal"을 입력한 후 실행을 선택합니다.

## 결과 확인

코드 실행 후 결과를 확인하려면 VSC 하단에 나타나는 터미널을 확인하면 됩니다. 결과가 출력되고 에러가 발생한 경우 해당 정보를 확인할 수 있습니다.

이제 여러분은 VSC를 사용하여 Python 코드를 실행하고 결과를 확인하는 방법을 알게 되었습니다. VSC를 통해 효율적으로 Python 코드를 작성하고 실행할 수 있으며, 개발 작업을 더욱 편리하게 진행할 수 있을 것입니다.
