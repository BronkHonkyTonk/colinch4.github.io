SQL 데이터베이스에서 레코드를 수정하는 방법에 대해 알아보겠습니다.

## 레코드 수정 기본 구문

먼저, SQL에서 레코드를 수정하기 위해서는 `UPDATE` 문을 사용합니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```sql
UPDATE 테이블명
SET 컬럼1 = 값1, 컬럼2 = 값2
WHERE 조건;
```

위 코드에서 `테이블명`은 수정하고자 하는 테이블의 이름을, `컬럼1`, `컬럼2`는 수정할 컬럼의 이름을, `값1`, `값2`는 수정할 값들을 나타냅니다. `WHERE` 절을 사용하여 특정 조건을 만족하는 레코드만을 수정할 수 있습니다.

## 예제

예를 들어, `employees` 테이블에서 `employee_id`가 100인 레코드의 `salary`를 5000으로 수정하고자 한다면 다음과 같이 사용할 수 있습니다.

```sql
UPDATE employees
SET salary = 5000
WHERE employee_id = 100;
```

## 마무리

SQL에서 레코드를 수정하는 방법에 대해 살펴보았습니다. `UPDATE` 문을 사용하여 특정 테이븋의 레코드를 수정할 수 있으며, 조건을 추가하여 원하는 레코드만을 수정할 수 있습니다.
