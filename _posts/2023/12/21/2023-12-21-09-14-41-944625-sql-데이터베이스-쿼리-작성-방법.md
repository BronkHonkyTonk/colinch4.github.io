SQL(Structured Query Language)은 관계형 데이터베이스에서 데이터를 관리하기 위한 표준 언어입니다. SQL을 사용하여 데이터를 삽입, 수정, 삭제, 조회하는데 사용됩니다. 이번 포스트에서는 SQL 데이터베이스 쿼리 작성 방법을 알아보겠습니다.

## 1. 데이터베이스 접속

SQL 쿼리를 작성하기 전에 데이터베이스에 접속해야 합니다. 예를 들어 MySQL 데이터베이스에 접속할 경우, 다음과 같은 명령어를 사용합니다.

```sql
mysql -u username -p
```

위 명령어에서 `-u` 옵션은 사용자 이름, `-p` 옵션은 비밀번호를 입력하는 옵션입니다.

## 2. 데이터 조회

가장 기본적인 SQL 쿼리는 데이터를 조회하는 것입니다. `SELECT` 문을 사용하여 데이터를 조회할 수 있습니다. 아래는 모든 열을 선택하는 간단한 예제입니다.

```sql
SELECT * FROM 테이블이름;
```

## 3. 데이터 필터링

특정 조건을 만족하는 데이터를 조회하기 위해 `WHERE` 구문을 사용할 수 있습니다. 아래는 특정 조건을 만족하는 데이터를 조회하는 예제입니다.

```sql
SELECT * FROM 테이블이름 WHERE 조건;
```

## 4. 데이터 삽입

새로운 데이터를 삽입하기 위해 `INSERT INTO` 문을 사용합니다. 아래는 새로운 데이터를 삽입하는 예제입니다.

```sql
INSERT INTO 테이블이름 (열1, 열2, 열3) VALUES (값1, 값2, 값3);
```

## 5. 데이터 수정

기존 데이터를 수정하기 위해 `UPDATE` 문을 사용합니다. 아래는 데이터를 수정하는 예제입니다.

```sql
UPDATE 테이블이름 SET 열1 = 값1, 열2 = 값2 WHERE 조건;
```

## 6. 데이터 삭제

데이터를 삭제하기 위해 `DELETE FROM` 문을 사용합니다. 아래는 데이터를 삭제하는 예제입니다.

```sql
DELETE FROM 테이블이름 WHERE 조건;
```

이상으로 SQL 데이터베이스 쿼리 작성 방법에 대해 알아보았습니다. SQL을 사용하여 데이터를 효과적으로 관리할 수 있습니다. SQL 쿼리 작성에 대한 추가적인 내용은 [SQL Tutorial](https://www.w3schools.com/sql/)에서 확인할 수 있습니다.
