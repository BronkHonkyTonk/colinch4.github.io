C언어에서 데이터 타입은 변수가 저장할 수 있는 데이터의 유형을 정의합니다. C언어에는 여러 가지 데이터 타입이 있지만, 여기서는 가장 기본적인 "단일 데이터 타입"에 대해 알아보겠습니다.

## 1. int

**int**는 정수를 저장하기 위한 데이터 타입입니다. 일반적으로 시스템에 따라 2 또는 4바이트 크기를 가지며, 범위는 -32,768에서 32,767(2바이트) 또는 -2,147,483,648에서 2,147,483,647(4바이트)까지입니다. 

예시:
```c
int number = 10;
```

## 2. float

**float**는 부동 소수점 숫자를 저장하기 위한 데이터 타입입니다. 보통 4바이트 크기를 가지며, IEEE 754 표준에 따라 단정밀도 부동 소수점 숫자를 표현합니다.

예시:
```c
float pi = 3.14;
```

## 3. double

**double**은 더 큰 범위와 더 높은 정밀도의 부동 소수점 숫자를 저장하기 위한 데이터 타입입니다. 보통 8바이트 크기를 가집니다.

예시:
```c
double bigNumber = 123456.789;
```

이외에도 C언어에는 char, short, long 등 다양한 데이터 타입이 있지만, 여기서는 가장 기본적인 단일 데이터 타입에 대해 다뤘습니다.

다양한 데이터 타입을 활용하여 프로그램을 작성할 때, 변수의 선언과 초기화에 적절한 데이터 타입을 선택하는 것이 중요합니다.
