## 1. 환경 변수란 무엇인가?

환경 변수는 프로세스가 실행될 때 사용되는 동적인 값들입니다. 예를 들어, **PATH**나 **HOME**과 같은 변수들이 있습니다. 이러한 변수들은 실행 파일을 찾을 때나 사용자 디렉토리에 접근할 때 사용됩니다.

## 2. 환경 변수 확인하기

시스템에서 사용 가능한 환경 변수들을 확인하는 방법은 간단합니다. 터미널에서 다음 명령어를 실행합니다.

```bash
printenv
```

이 명령어는 시스템에서 사용 가능한 모든 환경 변수를 출력해줍니다.

## 3. 환경 변수 설정하기

셸 스크립트를 사용하여 환경 변수를 설정할 수 있습니다. 예를 들어, **JAVA_HOME** 환경 변수를 설정하려면 다음과 같이 작성합니다.

```bash
export JAVA_HOME=/usr/lib/jvm/java-11
```

위의 명령어를 실행하면, **JAVA_HOME** 환경 변수가 지정된 디렉토리로 설정됩니다.

## 4. 영구적으로 환경 변수 설정하기

환경 변수를 영구적으로 설정하기 위해서는 셸 스크립트 파일을 수정해야 합니다. 사용하는 셸에 따라서 `~/.bashrc`, `~/.bash_profile`, `~/.zshrc`와 같은 파일들이 존재합니다. 이러한 파일에 환경 변수를 설정하는 스크립트를 추가하여 영구적으로 사용할 수 있습니다.

## 5. 환경 변수 제거하기

환경 변수를 제거하기 위해서는 `unset` 명령어를 사용합니다.

```bash
unset JAVA_HOME
```

이 명령어를 사용하면 **JAVA_HOME** 환경 변수가 제거됩니다.

환경 변수를 적절하게 설정하고 관리하는 것은 리눅스 시스템에서 매우 중요합니다. 제대로 관리된 환경 변수는 시스템 작업을 보다 효율적으로 만들어주고, 프로그램이 정상적으로 실행될 수 있도록 지원합니다.
