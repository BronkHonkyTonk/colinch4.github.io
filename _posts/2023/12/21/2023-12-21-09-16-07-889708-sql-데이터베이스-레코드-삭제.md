데이터베이스의 레코드를 삭제하는 것은 매우 중요한 작업입니다. SQL에서 레코드를 삭제하는 방법을 설명하겠습니다.

## 1. DELETE 문 사용

가장 일반적인 방법은 `DELETE` 문을 사용하는 것입니다. 다음은 가장 기본적인 형태의 DELETE 문입니다.

```sql
DELETE FROM 테이블명 WHERE 조건;
```

여기서, 
- `테이블명`은 레코드를 삭제할 테이블의 이름입니다.
- `조건`은 삭제할 레코드를 식별하는 조건입니다. 예를 들어, `id = 1` 과 같은 형태로 사용됩니다.

## 2. TRUNCATE 문 사용

`TRUNCATE` 문은 테이블에 있는 모든 레코드를 삭제하는 데 사용됩니다. 다음과 같이 사용할 수 있습니다.

```sql
TRUNCATE TABLE 테이블명;
```

이 명령은 테이블의 모든 레코드를 한 번에 지웁니다.

## 3. DROP 문 사용

`DROP` 문은 테이블 자체를 삭제하는 데 사용됩니다. 다음과 같이 사용할 수 있습니다.

```sql
DROP TABLE 테이블명;
```

이 명령은 전체 테이블을 삭제하므로 주의해야 합니다.
