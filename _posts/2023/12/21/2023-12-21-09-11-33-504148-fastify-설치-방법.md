## Fastify 설치

Fastify를 설치하려면 npm을 사용하여 새로운 프로젝트를 만들고 Fastify 패키지를 추가해주어야 합니다. 아래의 단계를 따라 진행해보세요.

1. **새로운 프로젝트 폴더 생성**
   ```bash
   mkdir my-fastify-app
   cd my-fastify-app
   ```

2. **npm을 사용하여 Fastify 설치**
   ```bash
   npm install fastify
   ```

위와 같은 순서로 Fastify를 설치하면 프로젝트 폴더에 Fastify 패키지가 설치됩니다.

이제 Fastify를 사용하여 웹 애플리케이션을 개발하는 준비가 되었습니다. Fastify는 탁월한 성능과 간편한 사용성을 제공하여 Node.js 기반 웹 애플리케이션을 빠르게 개발할 수 있게 해줍니다.
