C언어에서 변수를 선언할 때 데이터 타입을 지정하고 초기값을 설정할 수 있습니다. 아래는 데이터 타입을 초기화하는 몇 가지 예시입니다.

1. 정수형 변수 초기화
```c
int num = 10;
```

2. 실수형 변수 초기화
```c
float pi = 3.14;
```

3. 문자형 변수 초기화
```c
char letter = 'A';
```

4. 문자열 초기화 (문자열은 char 배열로 처리됨)
```c
char name[] = "John";
```

이러한 방법을 사용하여 변수를 선언하고 초기화할 수 있습니다.
