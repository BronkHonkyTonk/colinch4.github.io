## 포인터란 무엇인가요?

포인터는 메모리 주소를 저장하는 변수입니다. 다른 변수의 주소를 저장하고 그 변수를 간접적으로 조작할 수 있도록 해줍니다.

```c
int num = 10;  // 정수 변수 선언
int *ptr;      // 포인터 변수 선언
ptr = &num;    // 포인터에 num 변수의 주소 저장
```

위 코드에서 `ptr`은 `num` 변수의 주소를 가리키고 있습니다.

## 문자열과 포인터

C언어에서 문자열은 문자들의 배열로 표현됩니다. 문자열은 `char` 데이터 타입과 포인터를 이용하여 처리됩니다.

```c
char str[] = "Hello";  // 문자열 선언
char *ptr = str;       // 포인터에 문자열의 주소 저장
```

위 코드에서 `ptr`은 `str` 변수의 첫 번째 문자를 가리키고 있습니다.

## 문자열 처리 함수

C언어에는 문자열을 처리하기 위한 다양한 함수들이 포함되어 있습니다. 

- `strlen`: 문자열의 길이를 반환
- `strcpy`: 하나의 문자열을 다른 문자열에 복사
- `strcat`: 문자열을 뒤에 이어 붙임
- `strcmp`: 두 문자열을 비교

```c
#include <string.h>

char str1[] = "Hello";
char str2[10];

strcpy(str2, str1);  // str1의 내용을 str2에 복사
```
