이 포스트에서는 리눅스 시스템의 상태를 확인하는 몇 가지 주요 명령어에 대해 살펴보겠습니다.

1. **top**: 시스템의 현재 상태와 실행 중인 프로세스 목록을 실시간으로 확인할 수 있는 터미널 기반 프로세스 모니터링 도구입니다.
   
   ```bash
   top
   ```

2. **htop**: top 명령어와 유사하지만 보다 친숙하고 사용하기 쉬운 인터페이스를 제공하는 프로세스 관리 도구입니다. 보다 많은 상세 정보와 조작 가능한 기능을 제공합니다.

   ```bash
   htop
   ```

3. **free**: 시스템의 메모리와 스왑 공간에 대한 요약 정보를 보여주는 명령어입니다.

   ```bash
   free -h
   ```

4. **df**: 디스크 공간에 대한 정보를 보여주는 명령어로, 사용 가능한 공간 및 마운트된 파일 시스템 등을 확인할 수 있습니다.

   ```bash
   df -h
   ```

이러한 명령어들은 시스템 관리자 또는 개발자가 리눅스 시스템의 상태를 모니터링하고 문제 해결에 도움을 줄 수 있습니다.
