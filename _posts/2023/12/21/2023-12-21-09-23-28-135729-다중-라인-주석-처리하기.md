첫 번째 방법으로는 `/*`와 `*/` 사이에 주석을 작성하는 방법입니다. 이 방법은 여러 줄에 걸쳐 주석을 작성할 때 주로 사용됩니다.

```php
/*
여러 줄에 걸친
주석 작성 예시
*/
```

또 다른 방법은 여러 줄에서 각 줄 맨 앞에 `//*`을 작성하여 단일 줄 주석을 사용하는 방법입니다.

```php
// 여러 줄에서 
// 각 줄에 단일줄 주석 작성 예시
```

이 두 가지 방법을 사용하여 다중 라인 주석을 손쉽게 처리할 수 있습니다.

이렇게 작성된 주석은 소스 코드를 이해하기 쉽게 만들어주고, 다른 사람들이 코드를 이해하고 유지보수하는 데 도움이 됩니다.
