Apache XML Security 라이브러리는 Apache Software Foundation의 소프트웨어 라이센스인 Apache License 2.0을 따릅니다. 이 라이센스는 상업적 및 비상업적인 사용, 배포, 변경, 특허 식별에 대한 명시등을 허용합니다. 따라서, 이 라이브러리를 사용하는 경우 해당 라이센스를 준수해야 합니다.

이러한 라이센스 정보를 확인하려면 라이브러리의 배포 파일 또는 소스 코드에 포함된 LICENSE 파일을 참조하시기 바랍니다.
