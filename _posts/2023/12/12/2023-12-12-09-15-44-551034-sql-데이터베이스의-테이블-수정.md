SQL 데이터베이스에서는 기존 테이블을 수정하거나 업데이트할 수 있는데, 이를 위해서는 몇 가지 기본적인 명령어와 절차를 알고 있어야 합니다. 테이블을 수정하는 방법에 대해 알아보겠습니다.

## 1. 테이블 구조 변경

테이블의 구조를 변경하는 경우에는 **ALTER TABLE** 문을 사용하게 됩니다. 이를 통해 컬럼 추가, 컬럼 이름 변경, 데이터 유형 변경, 제약 조건 추가 및 제거 등을 할 수 있습니다.

예를 들어, 새로운 컬럼을 추가하려면 다음과 같은 명령어를 사용할 수 있습니다.

```sql
ALTER TABLE 테이블이름
ADD 컬럼이름 데이터유형;
```

## 2. 데이터 업데이트

테이블의 데이터를 업데이트하는 경우에는 **UPDATE** 문을 사용합니다. 해당 테이블의 특정 레코드의 값을 변경할 수 있습니다.

예를 들어, 컬럼 A의 값을 100에서 200으로 변경하려면 다음과 같은 명령어를 사용할 수 있습니다.

```sql
UPDATE 테이블이름
SET A = 200
WHERE A = 100;
```

## 3. 기타 변경 사항

테이블 구조나 데이터를 변경할 때는 원본 데이터의 백업을 수행하는 것이 좋습니다. 또한, 변경 사항을 적용하기 전에 데이터베이스의 무결성 및 퍼포먼스에 미치는 영향을 고려해야 합니다.

위의 방법들을 활용하여 SQL 데이터베이스의 테이블을 수정하고 관리할 수 있습니다.
