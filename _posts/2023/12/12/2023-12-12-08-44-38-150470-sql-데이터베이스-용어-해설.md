## SQL
**SQL**은 **Structured Query Language**의 약자로, 데이터베이스에서 데이터를 관리하기 위해 사용되는 표준화된 언어입니다. SQL은 데이터베이스에서 데이터를 조작, 삽입, 삭제 및 검색하는 데 사용됩니다.

## 데이터베이스
**데이터베이스**는 관련된 데이터의 모음으로, 구조화된 형식으로 저장되며, 다시 사용하기 위해 검색 및 수정될 수 있습니다. 데이터베이스는 여러 테이블 및 다른 객체(인덱스, 뷰, 프로시저 등)들로 구성됩니다.

## 테이블
**테이블**은 데이터를 저장하는 데 사용되는 구조화된 데이터 요소입니다. 데이터베이스 내에서 테이블은 행과 열의 형태로 구성되며, 각 열은 특정한 유형의 데이터를 나타냅니다.

## 쿼리
**쿼리**는 데이터베이스로부터 정보를 요청하는 SQL 문장입니다. 쿼리는 데이터를 검색하거나 조작하기 위해 데이터베이스에 전달됩니다.

## 인덱스
**인덱스**는 테이블에서 데이터의 검색 속도를 향상시키기 위한 데이터베이스 객체입니다. 인덱스를 사용하면 데이터베이스가 더 효율적으로 데이터를 찾을 수 있습니다.

## 뷰
**뷰**는 하나 이상의 테이블에서 가져온 데이터를 가상의 테이블로 표현하는데 사용됩니다. 뷰는 사용자에게 필요한 데이터만 표시할 수 있도록 도와줍니다.

## 트랜잭션
**트랜잭션**은 데이터베이스 연산의 단위를 나타내는 개념으로, 여러 단계의 연산이 모두 성공하거나 실패할 때까지 데이터베이스를 일관된 상태로 유지합니다.

## 트리거
**트리거**는 특정 이벤트가 발생했을 때 자동으로 실행되는 일련의 작업들을 정의한 데이터베이스 객체입니다. 트리거는 데이터의 일관성을 유지하기 위해 사용됩니다.
