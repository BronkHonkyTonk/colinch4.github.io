## 튜플(Tuple)이란?

튜플은 리스트와 함께 파이썬에서 가장 많이 사용되는 자료형 중 하나입니다. **리스트와 달리 한 번 생성된 요소를 변경할 수 없는 값으로 구성**되어 있습니다. 예를 들어, 

```python
my_tuple = (1, 2, 3, 4, 5)
```

위 코드에서 `my_tuple`은 1부터 5까지의 값을 가지며, 이 값을 변경할 수 없습니다.

## 튜플의 활용

튜플은 데이터를 읽기 전용으로 저장할 때 주로 사용됩니다. 데이터 분석을 수행할 때, 튜플은 데이터를 변경할 수 없기 때문에 **데이터 무결성을 보장해 주는 역할을 합니다**. 또한, 튜플은 리스트보다 **적은 메모리를 사용**하므로 대용량 데이터를 다룰 때 유용합니다.

```python
# 데이터 분석 예시
data = (23, 35, 28, 29, 21)  # 온도 데이터
average_temp = sum(data) / len(data)  # 평균 온도 계산
print("평균 온도:", average_temp)
```

위 코드에서 `data`는 온도 데이터를 저장한 튜플이며, 튜플을 이용하여 **평균 온도를 간단히 계산**할 수 있습니다.

## 결론

튜플은 **데이터 무결성을 보장**하고, **적은 메모리를 사용**하기 때문에 데이터 분석 도구로써 매우 유용합니다. 데이터를 추가하거나 변경할 필요가 없는 경우에는 튜플을 활용하여 안정적이고 효율적인 데이터 분석을 수행할 수 있습니다.
