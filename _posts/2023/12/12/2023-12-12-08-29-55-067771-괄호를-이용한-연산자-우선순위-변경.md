C++에서는 괄호를 사용하여 연산자의 우선순위를 변경할 수 있습니다. 괄호를 사용하면 해당 연산이 다른 연산보다 먼저 수행됩니다. 예를 들어, `2 + 3 * 4`에서 `*` 연산자가 `+` 연산자보다 우선적으로 수행됩니다. 하지만 괄호를 사용하여 `(2 + 3) * 4`와 같이 표현하면, 덧셈 연산이 먼저 이루어지게 됩니다.

```c++
#include <iostream>
using namespace std;

int main() {
    int result = (2 + 3) * 4;
    cout << result;  // 출력값: 20
    return 0;
}
```

위 예제에서는 괄호를 이용하여 덧셈 연산을 먼저 수행하고, 그 결과에 곱셈 연산을 수행합니다. 이를 통해 연산자의 우선순위를 변경할 수 있습니다.

이처럼 C++에서는 괄호를 통해 연산자의 우선순위를 명확히 지정할 수 있습니다.
