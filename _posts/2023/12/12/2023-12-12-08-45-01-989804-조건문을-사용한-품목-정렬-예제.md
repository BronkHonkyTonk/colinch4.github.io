이 예제에서는 **조건문(If 문)**을 사용하여 품목을 정렬하는 방법을 설명합니다.

## 문제

가게에는 3가지 품목(사과, 바나나, 오렌지)이 있습니다. 품목명을 입력하면 해당 품목의 재고량을 확인할 수 있는 시스템을 만들어야 합니다. 재고량은 다음과 같습니다.

- 사과: 10개
- 바나나: 15개
- 오렌지: 20개

## 해결책

우리는 **조건문(If 문)**을 사용하여 사용자가 입력한 품목을 확인하고 해당하는 재고량을 출력할 것입니다. 다음은 이를 수행하는 JavaScript 코드입니다.

```javascript
const itemName = prompt('품목을 입력하세요:');

if (itemName === '사과') {
  console.log('사과의 재고량은 10개입니다.');
} else if (itemName === '바나나') {
  console.log('바나나의 재고량은 15개입니다.');
} else if (itemName === '오렌지') {
  console.log('오렌지의 재고량은 20개입니다.');
} else {
  console.log('해당 품목은 재고가 없습니다.');
}
```

위 코드에서 `prompt` 함수를 사용하여 사용자로부터 품목명을 입력받습니다. 그리고 `if`, `else if`, `else` 구문을 사용하여 입력값에 따라 적절한 재고량을 출력합니다.

이제 위 코드를 실행해보면 사용자가 입력한 품목에 따라 해당하는 재고량이 출력될 것입니다.

## 결과

사용자가 '사과'를 입력하면 '사과의 재고량은 10개입니다.'가 출력되고, '바나나'를 입력하면 '바나나의 재고량은 15개입니다.'가 출력될 것입니다.

이와 같이 **조건문(If 문)**을 사용하여 품목을 정렬하는 예제를 살펴보았습니다.
