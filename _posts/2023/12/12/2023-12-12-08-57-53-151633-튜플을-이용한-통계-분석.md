파이썬에서 통계 분석을 수행할 때 **튜플(tuple)** 자료형을 사용하는 경우가 많습니다. 튜플은 변경이 불가능한 "불변(immutable)" 자료형으로, 데이터를 안전하게 보관하고 효율적으로 처리할 수 있습니다. 이번 글에서는 튜플을 이용하여 기초적인 통계 분석을 수행하는 방법에 대해 알아보겠습니다.

## 1. 튜플 생성하기
튜플은 괄호 `()`를 사용하여 생성하며, 값들은 쉼표로 구분됩니다. 

예를 들어, 다음과 같이 튜플을 생성할 수 있습니다.
```python
data = (10, 20, 30, 40, 50)
```

## 2. 기초적인 통계 분석 수행하기
튜플을 활용하여 기초적인 통계 분석을 수행할 수 있습니다.

### 2.1. 데이터의 합 구하기
튜플 내의 값들의 합을 구할 수 있습니다.
```python
data = (10, 20, 30, 40, 50)
total = sum(data)
print(total)  # 출력: 150
```

### 2.2. 데이터의 평균 구하기
튜플 내의 값들의 평균을 구할 수 있습니다.
```python
data = (10, 20, 30, 40, 50)
average = sum(data) / len(data)
print(average)  # 출력: 30.0
```

### 2.3. 최댓값 및 최솟값 구하기
튜플 내의 최댓값과 최솟값을 구할 수 있습니다.
```python
data = (10, 20, 30, 40, 50)
maximum = max(data)
minimum = min(data)
print(maximum, minimum)  # 출력: 50 10

## 3. 결론
튜플을 이용하면 간단하게 데이터를 관리하고 통계 분석을 수행할 수 있습니다. 파이썬의 내장 함수들과 튜플을 활용하여 데이터 분석 작업을 효율적으로 처리할 수 있습니다.
