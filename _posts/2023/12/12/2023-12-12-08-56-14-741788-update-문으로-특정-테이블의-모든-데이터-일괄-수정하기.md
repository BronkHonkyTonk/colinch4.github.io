아래는 특정 테이블의 모든 데이터를 한 번에 수정하는 간단한 예제입니다.

```sql
UPDATE 테이블명
SET 필드명 = 수정할값
WHERE 조건;
```

여기서, 
- `테이블명`은 수정하려는 대상 테이블의 이름입니다.
- `필드명`은 수정하려는 열의 이름이며, 일괄적으로 수정하고자 하는 경우 해당 필드명을 사용합니다.
- `수정할값`은 해당 필드에 지정할 값을 나타냅니다.
- `조건`은 필요에 따라 특정 레코드를 대상으로 지정할 수 있는 조건입니다. 모든 레코드를 대상으로 수정하려면 조건을 지정하지 않습니다.

예를 들어, `customer` 테이블에서 모든 고객의 연령을 30으로 일괄 수정하려면 다음과 같이 실행할 수 있습니다.

```sql
UPDATE customer
SET age = 30;
```

위 코드를 실행하면 `customer` 테이블에 있는 모든 고객의 연령이 30으로 업데이트됩니다.

더 복잡한 조건이나 다양한 값으로 일괄 수정을 원하신다면 그에 맞는 적절한 `WHERE` 절을 추가하여 사용하시면 됩니다.

이와 같이 `UPDATE` 문을 사용하여 특정 테이블의 모든 데이터를 일괄 수정할 수 있습니다.
