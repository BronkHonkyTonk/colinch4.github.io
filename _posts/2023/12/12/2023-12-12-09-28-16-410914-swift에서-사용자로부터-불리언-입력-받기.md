# 사용자 입력 받기

사용자로부터 값을 입력받기 위해서는 `readLine()` 함수를 사용합니다. 이 함수를 사용하여 사용자로부터 입력을 받고, 이를 적절한 형식으로 변환하여 사용할 수 있습니다.

```swift
if let input = readLine() {
    // 입력값 사용
}
```

# 불리언 입력 받기

사용자로부터 불리언 값을 입력받으려면, 먼저 문자열을 입력 받은 다음, 이를 불리언 값으로 변환해야 합니다. 입력된 문자열이 "true" 또는 "false"인지 확인한 후, 이에 따라 불리언 값으로 변환할 수 있습니다.

다음은 입력된 문자열이 "true" 또는 "false"일 때 해당하는 불리언 값으로 변환하는 예제입니다.

```swift
if let input = readLine() {
    if let boolValue = Bool(input) {
        // 불리언 값 사용
    } else {
        print("잘못된 입력입니다.")
    }
}
```

이제 사용자로부터 입력 받은 불리언 값을 안전하게 사용할 수 있게 되었습니다. 만약 사용자가 예상 외의 값을 입력했을 때에 대비하여 적절한 에러 처리를 추가하는 것도 중요합니다.

이로써 Swift에서 사용자로부터 불리언 값을 입력받는 방법에 대해 알아보았습니다.
