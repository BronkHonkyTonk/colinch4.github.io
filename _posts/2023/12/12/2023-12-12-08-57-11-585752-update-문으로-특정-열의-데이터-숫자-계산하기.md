```sql
UPDATE 테이블명
SET 열이름 = 숫자계산식
WHERE 조건;
```

위의 코드에서:
- `테이블명`은 데이터를 업데이트할 테이블의 이름입니다.
- `열이름`은 숫자로 계산된 값을 업데이트할 열의 이름입니다.
- `숫자계산식`은 해당 열의 데이터를 계산하는 SQL 표현식입니다.
- `조건`은 데이터를 업데이트할 행을 선택하는 조건입니다.

예를 들어, 만약 `테이블명` 테이블에서 `열이름` 열의 값을 1씩 증가시키고 싶다면 아래와 같이 작성할 수 있습니다:

```sql
UPDATE 테이블명
SET 열이름 = 열이름 + 1;
```

이렇게 하면 테이블의 각 행마다 `열이름` 열에 있는 데이터가 1씩 증가됩니다.
