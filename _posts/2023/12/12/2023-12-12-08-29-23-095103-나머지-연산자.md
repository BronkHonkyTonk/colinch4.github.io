C++에서의 **나머지 연산자(%)**는 두 수를 나눈 뒤 남는 나머지를 반환하는 연산자입니다. 예를 들어, `5 % 2`를 계산하면 결과는 1이 됩니다. 

```c++
int a = 5;
int b = 2;
int result = a % b; // 결과: 1
```

나머지 연산자는 특히 홀수, 짝수를 판별하거나 주기적인 작업을 할 때 유용하게 사용될 수 있습니다.

나머지 연산자는 일반적으로 정수형 데이터에 대해 사용됩니다. 

자세한 내용은 [cppreference.com](https://en.cppreference.com/w/cpp/language/operator_arithmetic)를 참조하세요.
