다음은 뺄셈 연산자를 사용하는 간단한 예시입니다:

```cpp
int a = 10;
int b = 5;
int result = a - b; // result에는 5가 저장됩니다.
```

또한, 뺄셈 연산자는 변수와 함께 사용하여 값을 감소시키거나 변형하는 데에도 사용될 수 있습니다.

```cpp
int x = 10;
x -= 3; // x는 이제 7입니다.
```

더 많은 정보를 원하시면 [C++ 공식 문서](https://en.cppreference.com/w/cpp/language/operator_arithmetic)를 참고해주세요.
