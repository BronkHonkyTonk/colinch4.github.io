SQL은 데이터베이스에서 데이터를 검색하고 필터링하는 데 사용되는 강력한 언어입니다. 이를 통해 목적에 맞는 데이터를 찾을 수 있고, 원하는 조건에 맞는 데이터를 구체적으로 필터링할 수 있습니다. 이번 글에서는 SQL을 사용하여 데이터를 검색하고 필터링하는 방법에 대해 알아보겠습니다.

## 데이터 검색

SQL을 사용하여 데이터를 검색할 때에는 `SELECT` 문을 사용합니다. `SELECT` 문은 데이터베이스에서 원하는 열이나 행을 선택하는 데 사용됩니다. 예를 들어, 다음과 같은 방법으로 모든 열을 선택할 수 있습니다.

```sql
SELECT * FROM 테이블명;
```

또한 특정한 열만을 선택하고 싶을 때에는 열 이름을 지정하여 선택할 수도 있습니다.

```sql
SELECT 열1, 열2 FROM 테이블명;
```

## 데이터 필터링

SQL을 사용하여 데이터를 필터링하기 위해서는 `WHERE` 절을 사용합니다. `WHERE` 절을 통해 특정한 조건을 만족하는 데이터를 선택할 수 있습니다. 예를 들어, 다음과 같이 나이가 30살 이상인 데이터를 검색할 수 있습니다.

```sql
SELECT * FROM 회원테이블 WHERE 나이 >= 30;
```

또는 다음과 같이 여러 가지 조건을 조합하여 필터링할 수도 있습니다.

```sql
SELECT * FROM 주문테이블 WHERE 상품명 = '커피' AND 가격 <= 5000;
```

이처럼 SQL을 사용하여 데이터를 검색하고 필터링하는 기본적인 방법에 대해 알아보았습니다. SQL을 효과적으로 활용하여 원하는 데이터를 손쉽게 찾아내고 조건에 맞게 필터링할 수 있습니다.
