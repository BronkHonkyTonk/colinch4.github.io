Swift는 정적 타입 언어로, 모든 값은 **명시적인 데이터 타입**을 가지고 있습니다. 데이터 타입을 명시함으로써 코드의 가독성을 높이고 안정성을 유지할 수 있습니다. Swift에는 여러 가지 데이터 타입이 있으며, 이에 대해 알아보도록 하겠습니다.

## 기본 데이터 타입
Swift는 기본 데이터 타입으로 다음과 같은 것들을 제공합니다:
- **Int**: 정수 타입
- **Double** 및 **Float**: 부동 소수점 타입
- **Bool**: 불리언 타입
- **String**: 문자열 타입
- **Character**: 문자 타입

## 복합 데이터 타입
Swift는 아래와 같은 복합 데이터 타입도 제공합니다:
- **Array**: 같은 데이터 타입의 순서 있는 목록
- **Dictionary**: 키-값 쌍의 무서 있는 목록
- **Set**: 같은 데이터 타입의 순서 없는 목록

Swift는 데이터 타입을 명시적으로 선언하여 변수나 상수를 생성하거나, 타입 추론을 활용하여 코드를 더 간단하게 작성할 수 있습니다. 데이터 타입은 앞으로의 개발 작업에서 중요한 역할을 합니다.
