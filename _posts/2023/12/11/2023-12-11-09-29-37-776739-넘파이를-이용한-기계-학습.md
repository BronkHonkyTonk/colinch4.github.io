## 넘파이란 무엇인가요?

`넘파이(Numpy)`는 파이썬에서 수치 계산을 위한 핵심 라이브러리로, 다차원 배열과 행렬 연산에 유용한 기능을 제공합니다. 넘파이는 기본 데이터 구조 외에도 다양한 함수와 연산을 지원하여 빠르고 효율적인 계산이 가능합니다.

## 기계 학습에 넘파이 활용하기

기계 학습에서 데이터 처리와 분석은 매우 중요한데, 넘파이를 활용하면 데이터를 효율적으로 다룰 수 있습니다. 배열을 통해 데이터를 저장하고 다루는 것이 가능하며, 벡터나 행렬 연산을 통해 데이터를 변환하고 모델을 학습시키는 데에 적합합니다.

## 넘파이를 이용한 예시

```python
import numpy as np

# 1차원 배열 생성
arr1 = np.array([1, 2, 3, 4, 5])

# 2차원 배열 생성
arr2 = np.array([[1, 2, 3], [4, 5, 6]])

# 행렬 곱셈
result = np.dot(arr2, arr1)
print(result)
```
