Python에서 파일에 데이터를 쓰는 것은 매우 간단합니다. 기존 파일을 열고 데이터를 쓰기 모드로 추가하거나 덮어쓸 수 있습니다.

## 파일에 데이터 쓰는 방법

### 1. 파일 열기

파일을 쓰기 모드로 엽니다. 아래는 파일을 열고 쓰기 모드로 데이터를 쓰는 예제입니다.

```python
with open('파일명.txt', 'w') as 파일:
    # 코드에서 "파일"은 파일 객체를 가리킵니다.

    # 파일에 데이터 쓰기
    파일.write('여기에 쓸 내용')
```

### 2. 파일에 데이터 쓰기

파일이 열린 후에 `write()` 함수를 사용하여 데이터를 파일에 씁니다.

### 3. 파일 닫기

파일을 다 사용했으면 항상 파일을 닫는 것이 좋습니다.

```python
파일.close()
```

위의 코드에서 `with`문을 사용하면 파일이 자동으로 닫히므로 `close()`를 호출할 필요가 없습니다.

## 예제

다음은 "파일명.txt" 파일에 "안녕, 세상아!"를 쓰는 예제입니다.

```python
with open('파일명.txt', 'w') as 파일:
    파일.write('안녕, 세상아!')
```

Python은 파일 관리를 매우 쉽게 만들어주므로, 데이터를 파일에 쓰고 관리하는 것이 간단합니다.

위의 예제를 참고하여 파일에 데이터를 쓰는 데 도움이 되기를 바랍니다.
