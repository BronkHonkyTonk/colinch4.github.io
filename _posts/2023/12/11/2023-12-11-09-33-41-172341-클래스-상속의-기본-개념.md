이번에는 파이썬에서 클래스 상속의 기본적인 개념에 대해 알아보겠습니다.

## 클래스 상속이란 무엇인가요?

**클래스 상속**은 기존 클래스의 특성을 그대로 가져와 새로운 클래스를 만드는 것을 말합니다. 상속을 받는 클래스는 **하위 클래스** 또는 **자식 클래스**라고 불리고, 상속을 하는 클래스는 **상위 클래스** 또는 **부모 클래스**라고 합니다.

## 왜 클래스 상속을 사용하는 걸까요?

클래스 상속을 사용하면 코드의 재사용성이 높아지고, 유지보수가 쉬워집니다. 또한, 새로운 클래스에 기존 클래스의 기능을 추가하거나 변경하여 확장할 수 있습니다.

## 어떻게 클래스를 상속받을까요?

클래스를 상속받기 위해서는 다음과 같이 코드를 작성합니다.

```python
class 부모클래스:
    # 부모 클래스의 내용 정의

class 자식클래스(부모클래스):
    # 자식 클래스의 내용 정의
```

위와 같이 자식 클래스의 정의 부분에서 괄호 안에 상속받고 싶은 부모 클래스의 이름을 넣어주면 됩니다.

## 정리

클래스 상속은 기존 클래스의 특성을 그대로 유지하면서 새로운 클래스를 만들고자 할 때 사용됩니다. 이를 통해 코드의 재사용성이 높아지고 유지보수가 용이해집니다.

클래스 상속에 대한 더 자세한 내용은 [파이썬 공식 문서](https://docs.python.org/3/tutorial/classes.html#inheritance)를 참고하시기 바랍니다.
