파이썬을 사용하여 파일에서 알파벳만 추출하는 간단한 스크립트를 작성해보겠습니다.

## 파일에서 텍스트 읽기

먼저, 파일에서 텍스트를 읽어 들이는 방법을 알아봅시다. 아래의 코드 예제를 참고해주세요.

```python
with open('파일경로.txt', 'r') as file:
    text = file.read()
```

위 코드는 `open()` 함수를 사용하여 파일을 열고, `read()`로 파일의 내용을 읽어들입니다.

## 알파벳 문자만 추출

이제 읽어들인 텍스트에서 알파벳 문자만 추출하는 과정을 살펴보겠습니다. 아래의 코드 예제를 참고해주세요.

```python
alpha_text = ''.join(filter(str.isalpha, text))
```

위 코드에서 `filter()` 함수와 `str.isalpha` 메서드를 사용하여 알파벳 문자만 추출합니다.

## 결과 출력

마지막으로, 추출된 알파벳 문자를 출력하거나 다른 용도로 활용할 수 있습니다.

```python
print(alpha_text)
```

위 코드는 추출된 알파벳 문자를 콘솔에 출력합니다.

이제 위의 코드를 실행하면 파일에서 알파벳 문자만을 추출할 수 있습니다. 각 단계에서 분석하고 응용하여 원하는 결과를 얻을 수 있기를 바랍니다.

이상으로 파일에서 알파벳만 추출하는 Python 스크립트에 대한 간략한 안내였습니다. 부족한 부분이 있다면 언제든지 알려주세요!
