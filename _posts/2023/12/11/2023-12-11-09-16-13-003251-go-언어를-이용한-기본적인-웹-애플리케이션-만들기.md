## 간단한 웹 애플리케이션 만들기

다음은 Go 언어를 사용하여 간단한 "Hello, World!" 웹 애플리케이션을 만드는 예제 코드입니다.

```go
package main

import (
	"fmt"
	"net/http"
)

func handler(w http.ResponseWriter, r *http.Request) {
	fmt.Fprintf(w, "Hello, World!")
}

func main() {
	http.HandleFunc("/", handler)
	http.ListenAndServe(":8080", nil)
}
```

이 코드는 HTTP 요청을 처리하고 "Hello, World!"라는 메시지를 응답으로 반환하는 기본적인 웹 서버를 구현한 것입니다.
