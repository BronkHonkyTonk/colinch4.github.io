특정 패턴으로 끝나는 값을 찾으려면 다음과 같이 SQL 쿼리를 작성할 수 있습니다.

```sql
SELECT * FROM 테이블명
WHERE 컬럼명 LIKE '%특정패턴';
```

위의 예시에서 "%특정패턴"은 특정 패턴으로 끝나는 값을 나타냅니다. 여기서 "%"는 임의의 글자를 나타내고, "특정패턴"은 여러 문자로 이루어진 원하는 패턴을 나타냅니다. 예를 들어, 만약 "abc"로 끝나는 값을 찾고 싶다면 다음과 같이 작성할 수 있습니다.

```sql
SELECT * FROM 테이블명
WHERE 컬럼명 LIKE '%abc';
```

이렇게하면 특정 패턴으로 끝나는 값을 검색할 수 있습니다. 이 쿼리를 실행하여 결과를 확인할 수 있습니다.
