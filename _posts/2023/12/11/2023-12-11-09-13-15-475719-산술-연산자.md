산술 연산자는 숫자 값들 사이에 사용되는 연산자입니다. Kotlin은 여러 산술 연산자를 제공합니다. 

## 덧셈 연산자 (+)

덧셈 연산자는 두 개의 숫자 값을 더합니다. 

```kotlin
val sum = 10 + 5 // sum에 15 저장
```

## 뺄셈 연산자 (-)

뺄셈 연산자는 하나의 숫자 값을 다른 숫자 값에서 뺍니다. 

```kotlin
val difference = 10 - 5 // difference에 5 저장
```

## 곱셈 연산자 (*)

곱셈 연산자는 두 개의 숫자 값을 곱합니다. 

```kotlin
val product = 10 * 5 // product에 50 저장
```

## 나눗셈 연산자 (/)

나눗셈 연산자는 하나의 숫자 값을 다른 숫자 값으로 나눕니다. 

```kotlin
val quotient = 10 / 5 // quotient에 2 저장
```

## 나머지 연산자 (%)

나머지 연산자는 하나의 숫자 값을 다른 숫자 값으로 나눈 후 남은 나머지를 반환합니다.

```kotlin
val remainder = 10 % 3 // remainder에 1 저장
```

이것으로 Kotlin의 산술 연산자에 대해 간단히 알아보았습니다. 숫자 값들을 다룰 때 유용하게 사용할 수 있습니다.
