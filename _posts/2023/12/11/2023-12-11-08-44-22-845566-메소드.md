```swift
class MyClass {
    func myMethod() {
        // 메소드 구현
    }
}

let myInstance = MyClass()
myInstance.myMethod() // 메소드 호출
```

메소드는 다양한 형태로 정의될 수 있으며 매개변수를 받아들이고 값을 반환할 수도 있습니다. 또한, 메소드의 위치에 따라 인스턴스 메소드와 타입 메소드로 구분됩니다. 인스턴스 메소드는 특정 인스턴스에 속하며 해당 인스턴스의 속성에 접근할 수 있습니다. 반면에, 타입 메소드는 특정 형식에 속하며 인스턴스를 생성하지 않고도 호출될 수 있습니다.

메소드는 객체 지향 프로그래밍에서 중요한 역할을 맡고 있으며, Swift 언어에서도 다양한 기능을 제공하여 프로그래밍을 보다 유연하고 효율적으로 할 수 있도록 도와줍니다.

이상으로 Swift 언어의 메소드에 대해 간략히 알아보았습니다. 더 자세한 내용은 Swift 공식 문서를 참고하시기 바랍니다.
