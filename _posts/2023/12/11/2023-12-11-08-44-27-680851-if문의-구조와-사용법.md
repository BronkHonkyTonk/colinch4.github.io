if문은 C++에서 조건에 따라 코드 블록을 실행할지를 결정하는 제어문 중 하나입니다. if문의 구조와 사용법을 알아봅시다.

## if문의 구조

```c++
if (조건) {
    // 조건이 참일 때 실행될 코드
}
```

위의 예시는 `조건`이 참일 경우 중괄호 안의 코드 블록이 실행됩니다.

## if-else문

때로는 `조건`이 참일 때와 거짓일 때 각각 다른 코드 블록이 실행되어야 하는 경우가 있습니다. 이때 if-else문을 사용합니다.

```c++
if (조건) {
    // 조건이 참일 때 실행될 코드
} else {
    // 조건이 거짓일 때 실행될 코드
}
```

## else if문

여러 조건에 따라 실행할 코드 블록을 설정해야 하는 경우, else if문을 사용할 수 있습니다.

```c++
if (조건1) {
    // 조건1이 참일 때 실행될 코드
} else if (조건2) {
    // 조건2가 참일 때 실행될 코드
} else {
    // 모든 조건이 거짓일 때 실행될 코드
}
```

C++에서 if문과 함께 사용되는 논리 연산자로는 `&&` (and), `||` (or), `!` (not) 등이 있습니다.

if문의 구조와 사용법에 대해 간략히 알아보았습니다. if문은 프로그램에서 조건에 따라 실행 흐름을 제어하는 데에 매우 유용한 도구입니다.
