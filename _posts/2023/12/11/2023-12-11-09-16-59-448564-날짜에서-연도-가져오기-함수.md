보통 날짜 형식의 데이터의 연도를 추출하기 위해서는 데이터베이스 시스템에 따라 다른 함수를 사용합니다.

### MySQL

MySQL에서는 `YEAR()` 함수를 사용하여 날짜에서 연도를 가져올 수 있습니다. 아래는 예제 코드입니다.

```sql
SELECT YEAR('2022-01-15');
```

이 쿼리는 '2022-01-15'에서 연도를 추출하여 2022를 반환합니다.

### PostgreSQL

PostgreSQL에서는 `EXTRACT()` 함수를 사용하여 연도를 가져올 수 있습니다. 아래는 예제 코드입니다.

```sql
SELECT EXTRACT(YEAR FROM TIMESTAMP '2022-01-15');
```

이 쿼리는 '2022-01-15'에서 연도를 추출하여 2022를 반환합니다.

### SQL Server

SQL Server에서는 `YEAR()` 함수를 사용하여 연도를 가져올 수 있습니다. 아래는 예제 코드입니다.

```sql
SELECT YEAR('2022-01-15');
```

이 쿼리는 '2022-01-15'에서 연도를 추출하여 2022를 반환합니다.

위의 함수들을 사용하여 날짜에서 연도를 추출할 수 있습니다. 데이터베이스 시스템에 따라 해당 함수의 사용법이나 형식이 다를 수 있으니 참고 자료를 참조하시기 바랍니다.
