SQL에서 **스칼라 함수**는 하나의 입력 값을 받아 하나의 결과 값을 반환하는 함수를 말합니다. 이 함수를 사용하여 데이터베이스에서 값을 변환하거나 계산할 수 있습니다.

### 스칼라 함수의 기본 구문

스칼라 함수는 다음과 같이 사용됩니다.

```sql
SELECT function_name(column_name)
FROM table_name;
```

여기서 `function_name`은 스칼라 함수의 이름을 나타내고, `column_name`은 함수에 입력으로 제공되는 열의 이름을 나타냅니다.

### 예제

다음은 SQL에서 스칼라 함수를 사용하는 간단한 예제입니다. `employee` 테이블에 저장된 `salary` 열의 값을 두 배로 반환하는 함수를 적용하는 쿼리입니다.

```sql
SELECT DOUBLE(salary)
FROM employee;
```

이 쿼리는 `salary` 열의 값을 두 배로 반환하여 결과를 보여줍니다.

### 스칼라 함수의 활용

스칼라 함수는 데이터베이스에서 다양한 용도로 활용됩니다. 예를 들어, 수학 계산, 문자열 조작, 날짜 및 시간 포맷 변경 등의 작업에 사용될 수 있습니다.

SQL에서 제공하는 기본 스칼라 함수 이외에도 사용자 정의 스칼라 함수를 생성하여 필요에 맞게 활용할 수도 있습니다.

스칼라 함수는 SQL 쿼리를 보다 강력하고 유연하게 만들어주는 중요한 도구 중 하나입니다.

이상으로 SQL에서의 스칼라 함수 사용 방법에 대해 알아보았습니다.
