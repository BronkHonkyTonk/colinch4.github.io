코틀린은 간결하고 표현력이 뛰어나기 때문에 조건문과 반복문을 함께 사용하는 것이 매우 간단합니다. 이 예제에서는 if문과 for문을 함께 사용하는 방법을 알아보겠습니다.

## if문과 for문을 함께 사용하는 예제

다음은 1부터 5까지의 숫자 중에서 짝수인 숫자만 출력하는 예제입니다.

```kotlin
fun main() {
    for (i in 1..5) {
        if (i % 2 == 0) {
            println(i)
        }
    }
}
```

위 예제에서 `for (i in 1..5)`는 1부터 5까지의 숫자를 순차적으로 변수 `i`에 할당합니다. 그리고 `if (i % 2 == 0)`는 `i`가 짝수인지를 확인하고 짝수인 경우에만 `println(i)`를 통해 출력합니다.

## 결과

위 예제를 실행하면 다음과 같은 결과가 출력됩니다.

```
2
4
```

이렇게 코틀린에서는 조건문과 반복문을 함께 사용하여 간결하고 가독성 좋은 코드를 작성할 수 있습니다.

위의 예제를 통해 코틀린에서의 조건문과 반복문을 함께 사용하는 방법을 알아보았습니다.
