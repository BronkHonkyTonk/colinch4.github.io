### 문자열에서 대체 가능한 문자 쌍 찾기

문자열에서 대체 가능한 문자 쌍을 찾는 방법은 매우 간단합니다. 우리는 문자열에서 대체 가능한 문자 쌍을 찾기 위해 두 문자열을 비교하여 찾을 수 있습니다. 이를 위해 `NSString`의 `compare` 메서드를 사용할 수 있습니다. 

아래는 Swift에서 문자열에서 대체 가능한 문자 쌍을 찾는 예제 코드입니다.

```swift
let str1 = "café"
let str2 = "cafe\u{301}"

if str1.compare(str2) == ComparisonResult.orderedSame {
    print("대체 가능한 문자입니다.")
} else {
    print("대체 가능한 문자가 아닙니다.")
}
```

위의 코드에서 두 문자열 `str1`과 `str2`를 정의하고, `compare` 메서드를 사용하여 두 문자열을 비교합니다. 조합형 문자 (\u{301})를 사용하여 é를 표현합니다. 이를 비교하여 두 문자열이 대체 가능한 문자 쌍임을 확인할 수 있습니다.

### 결론

Swift에서 문자열에서 대체 가능한 문자 쌍을 찾는 방법은 매우 간단합니다. `compare` 메서드를 사용하여 두 문자열을 비교하여 대체 가능한 문자 쌍을 확인할 수 있습니다.

이상으로 Swift에서 문자열에서 대체 가능한 문자 쌍을 찾는 방법에 대해 알아보았습니다. 혹시 추가로 궁금한 점이 있으시면 언제든지 물어보세요!
