자바스크립트를 사용하여 중첩 반복문을 이용하여 다양한 크기의 별 삼각형을 출력할 수 있습니다.

## 삼각형 출력하기

다음은 중첩 반복문을 사용하여 삼각형을 출력하는 간단한 예제 코드입니다.

```javascript
function printTriangle(height) {
  for (let i = 1; i <= height; i++) {
    let row = '';
    for (let j = 1; j <= i; j++) {
      row += '* ';
    }
    console.log(row);
  }
}

printTriangle(5);
```

이 코드는 높이를 입력받아 해당 높이만큼의 삼각형을 출력합니다. 예를 들어, `printTriangle(5)`를 호출하면 다음과 같은 결과가 출력됩니다.

```
* 
* * 
* * * 
* * * * 
* * * * * 
```

이 예제에서는 외부 반복문이 행을, 내부 반복문이 각 행의 별 수를 나타내며, 각 행마다 해당하는 수만큼 별을 출력하는 방식으로 삼각형을 출력합니다.

## 마무리

위의 예제 코드는 중첩 반복문을 이용하여 삼각형을 출력하는 방법을 보여줍니다. 이를 응용하여 행마다 각기 다른 패턴의 별 삼각형을 출력할 수도 있습니다. 자바스크립트를 사용하여 중첩 반복문을 활용하면 다양한 형태의 삼각형을 출력할 수 있습니다.
