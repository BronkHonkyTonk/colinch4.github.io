## Tooltip의 역할 및 사용법

`Tooltip` 위젯은 사용자가 특정 위젯을 길게 누르거나 마우스를 올려놓을 때 해당 위젯의 기능이나 역할에 대한 간단한 설명을 표시하는 데 사용됩니다. 사용자에게 어떤 위젯이 어떤 목적을 가지고 있는지 이해하기 쉽도록 도와줍니다.

```dart
Tooltip(
    message: '이 버튼을 누르면 저장됩니다.',
    child: IconButton(
        icon: Icon(Icons.save),
        onPressed: () {
            // 저장 로직 추가
        },
    ),
)
```

위 예제는 `IconButton`을 감싸고 있는 `Tooltip` 위젯을 보여줍니다. 사용자가 이 아이콘 버튼 위에 마우스를 가져가면 "이 버튼을 누르면 저장됩니다."라는 설명이 나타납니다.

## 툴팁 설계 시 고려할 사항

- **간결하고 명확한 설명**: 툴팁의 내용은 매우 간결하고 정보 전달이 명확해야 합니다. 
- **과하게 사용하지 않기**: 과하게 많은 툴팁을 사용하면 사용자가 혼란스러워할 수 있으므로, 필요한 기능에만 사용하는 것이 좋습니다.

## 툴팁 사용 시 주의사항

- **접근성 고려**: 툴팁을 사용할 때 접근성을 고려하여 모든 사용자가 툴팁을 이해할 수 있도록 해야 합니다.
- **오래 노출되지 않도록 주의**: 툴팁이 오래 노출되면 화면을 가리거나 사용자 경험을 방해할 수 있으므로 짧은 시간 동안만 표시되도록 해야 합니다.

플러터 앱에서 `Tooltip`을 사용할 때 이러한 사항들을 고려한다면 사용자들에게 좀 더 향상된 사용자 경험을 제공할 수 있을 것입니다.
