```c++
for (int i = 0; i < 5; i++) {
    for (int j = 0; j < 3; j++) {
        // 중첩된 반복문의 본문
    }
}
```

위 예시에서 `for` 키워드를 사용하여 중첩 반복문을 표현하였습니다. 각 반복문은 중괄호로 둘러싸인 본문을 가지고 있습니다.

또한, `do-while`문은 조건을 뒤나중에 평가하는 반면에 루프 본문은 반드시 한 번 실행됩니다. 다음은 간단한 `do-while`문의 예시입니다.

```c++
int i = 0;
do {
    // 루프 본문
    i++;
} while (i < 5);
```

위 예시에서 `do`와 `while` 키워드를 사용하여 `do-while` 반복문을 작성하였습니다.

중첩 반복문 및 `do-while`문은 코드의 유연성을 높여주는 중요한 도구입니다. 그러나 과도한 중첩이나 무한 루프에 주의해야 합니다. 코드의 가독성과 효율성을 고려하여 사용하여야 합니다.
