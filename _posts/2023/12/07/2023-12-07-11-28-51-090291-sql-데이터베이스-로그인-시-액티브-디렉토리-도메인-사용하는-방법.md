SQL 데이터베이스에 액티브 디렉토리 도메인을 사용하여 로그인하려면 다음 단계를 따릅니다:

1. SQL Server 인스턴스에 액티브 디렉토리 확장 기능을 설치합니다. 이 확장 기능은 Windows 운영체제에서 제공되는 액티브 디렉토리 서비스와 상호 작용할 수 있도록 합니다.

2. SQL Server 인스턴스의 보안 설정에서 'Windows 인증 모드'를 사용하도록 설정합니다. 이렇게 하면 액티브 디렉토리 도메인 계정으로 로그인하는 것이 가능해집니다.

3. SQL Server 인스턴스에 액티브 디렉토리 도메인 계정을 등록합니다. 이를 통해 액티브 디렉토리 도메인의 사용자 계정으로 SQL Server에 로그인할 수 있습니다.

아래는 T-SQL 예제 코드로, 액티브 디렉토리 도메인을 사용하여 SQL Server에 로그인하는 방법을 보여줍니다:

```sql
CREATE LOGIN [Domain\Username] FROM WINDOWS;
```

위의 예제에서 `[Domain\Username]`은 액티브 디렉토리 도메인에 등록된 사용자 계정입니다. 이 명령은 해당 도메인 계정을 SQL Server에 등록하여 로그인할 수 있도록 합니다.

액티브 디렉토리 도메인을 사용하여 SQL 데이터베이스에 로그인하는 방법에 대해 알아보았습니다. 추가적인 설정이나 맞춤화가 필요한 경우, 관련 문서 또는 마이크로소프트의 지원 문서를 참조하시기 바랍니다.
