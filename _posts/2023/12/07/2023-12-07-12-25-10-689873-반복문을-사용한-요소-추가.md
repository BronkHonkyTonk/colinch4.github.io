이번 글에서는 C++의 반복문을 사용하여 배열에 요소를 추가하는 방법에 대해 알아보겠습니다.

## 배열 초기화

우선 배열을 선언하고 초기화해야 합니다. 예를 들어, `int`형 배열을 선언하고 초기화하는 방법은 다음과 같습니다.

```c++
int arr[5] = {1, 2, 3, 4, 5};
```

## 반복문을 사용한 요소 추가

이후, 반복문을 사용하여 배열에 요소를 추가할 수 있습니다. 아래는 `for` 반복문을 활용한 예시 코드입니다.

```c++
#include <iostream>

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    int numToAdd = 10;

    for (int i = 0; i < 5; i++) {
        arr[i] += numToAdd;
    }

    for (int i = 0; i < 5; i++) {
        std::cout << arr[i] << " ";
    }

    return 0;
}
```

위 코드에서는 `for` 반복문을 사용하여 `arr` 배열의 모든 요소에 `numToAdd` 값을 더하는 예제를 보여줍니다.

## 결론

C++에서는 반복문을 활용하여 배열에 요소를 추가하고 각 요소를 수정하는 것이 가능합니다. 이를 응용하여 다양한 배열 조작을 수행할 수 있습니다.
