자바는 강력한 데이터 타입 시스템을 가지고 있습니다. 이 문서에서는 자바의 데이터 타입에 대해 자세히 알아보겠습니다.

## 기본 데이터 타입

자바는 다음과 같은 기본 데이터 타입을 제공합니다:

- `byte`: 1바이트 크기의 정수를 표현합니다. (-128 ~ 127)
- `short`: 2바이트 크기의 정수를 표현합니다. (-32,768 ~ 32,767)
- `int`: 4바이트 크기의 정수를 표현합니다. (-2,147,483,648 ~ 2,147,483,647)
- `long`: 8바이트 크기의 정수를 표현합니다. (-9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807)
- `float`: 4바이트 크기의 부동소수점 숫자를 표현합니다.
- `double`: 8바이트 크기의 부동소수점 숫자를 표현합니다.
- `boolean`: 참 또는 거짓 값을 표현합니다.
- `char`: 2바이트 크기의 유니코드 문자를 표현합니다.

## 참조 데이터 타입

자바는 기본 데이터 타입 외에도 참조 데이터 타입을 제공합니다. 몇 가지 주요한 참조 데이터 타입은 다음과 같습니다:

- `String`: 문자열을 표현하기 위한 클래스입니다.
- `Array`: 여러 개의 값을 하나의 변수에 저장하기 위한 클래스입니다.
- `Class`: 클래스의 메타데이터를 표현하기 위한 클래스입니다.

## 데이터 타입의 문법

자바에서 변수를 선언할 때는 데이터 타입을 명시해야 합니다. 변수의 이름은 자유롭게 지을 수 있지만, 자바의 예약어와 충돌하는 이름은 사용할 수 없습니다.

다음은 자바에서 변수를 선언하는 문법의 예입니다:

```java
<데이터 타입> <변수 이름> = <초기값>;
```

예를 들어, 정수를 저장하기 위한 변수를 선언하고 초기값을 할당하는 방법은 다음과 같습니다:

```java
int number = 10;
```

## 데이터 타입 변환

자바는 타입 변환을 위한 몇 가지 규칙을 가지고 있습니다. 큰 크기의 데이터 타입은 작은 크기의 데이터 타입으로 자동 변환될 수 있지만, 작은 크기의 데이터 타입은 큰 크기의 데이터 타입으로 자동 변환될 수 없습니다. 이 경우에는 명시적인 타입 변환(casting)을 해야 합니다.

다음은 타입 변환의 예입니다:

```java
int number = 10;
double doubleNumber = (double) number; // int를 double로 변환
```
