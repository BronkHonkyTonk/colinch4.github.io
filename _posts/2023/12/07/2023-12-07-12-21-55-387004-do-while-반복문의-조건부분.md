do {
    // 반복할 코드
} while (조건);
```

`do-while` 반복문은 코드를 한 번 실행하고 나서 조건을 검사하여, 조건이 참이면 코드를 계속 반복합니다. 반복문의 조건부분은 괄호 안에 위치하며, `while` 키워드 뒤에 조건을 기술합니다.
```markdown
이는 미리 조건을 검사하는 `while` 반복문과 달리, 코드를 먼저 실행한 후에 조건을 검사하기 때문에 최소한 한 번은 코드를 실행하게 됩니다.
```

`do-while` 반복문을 사용할 때에는 코드가 실제로 실행된 후에 조건을 검사하므로, 반복 코드의 실행을 보장하면서 조건을 검사할 때마다 반복을 계속할 것인지를 결정할 수 있습니다.
이는 루프의 실행을 최소한 한 번은 보장해야 할 때 유용하게 사용됩니다.
