코틀린은 다양한 용도로 사용될 수 있는 스크립트 언어로도 사용할 수 있습니다. 코틀린 스크립트에서 변수를 선언하고 사용하는 방법을 알아보겠습니다.

## 변수 선언

코틀린에서 변수를 선언할 때는 `val`과 `var` 키워드를 사용합니다. `val`은 값을 변경할 수 없는 변수, `var`은 값을 변경할 수 있는 변수를 선언하는 데 사용됩니다.

```kotlin
val name = "John"  // 값 변경 불가능한 변수
var age = 25       // 값 변경 가능한 변수
```

변수를 선언할 때 자료형을 명시적으로 지정할 수도 있습니다.

```kotlin
val name: String = "John"
var age: Int = 25
```

## 변수 사용

코틀린 스크립트에서 변수를 사용하는 방법은 다른 언어와 유사합니다. 변수 이름을 사용하여 값을 참조하거나 변경할 수 있습니다.

```kotlin
val message = "Hello, World!"
println(message)  // 변수 값 출력

var count = 0
count = count + 1  // 변수 값 변경
```

변수의 값은 선언시 지정한 자료형과 일치해야 합니다. 만약 자료형이 다른 값을 할당하려고 하면 컴파일 오류가 발생합니다.

```kotlin
val name: String = "John"
name = 25  // 컴파일 오류 발생. String 타입이 아닌 값을 할당하려고 함.
```

## 요약

코틀린 스크립트에서 변수를 선언하고 사용하는 방법을 알아보았습니다. `val` 키워드를 사용하여 값을 변경할 수 없는 변수를 선언하고, `var` 키워드를 사용하여 값을 변경할 수 있는 변수를 선언할 수 있습니다. 변수의 값은 변수 이름을 사용하여 참조하고 변경할 수 있습니다.
