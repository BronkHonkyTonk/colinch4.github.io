자바스크립트에서는 객체의 속성 값을 기준으로 합계를 구하는 방법이 여러 가지가 있습니다. 

아래의 예제 코드는 객체의 속성 값을 기준으로 합계를 구하는 간단한 방법을 보여줍니다.

```javascript
// 객체 생성
let obj = {
  a: 10,
  b: 20,
  c: 30
};

// 속성 값을 기준으로 합계 구하기
let sum = 0;
for(let key in obj) {
  sum += obj[key];
}

console.log(sum); // 60
```

위의 코드에서는 객체 `obj`의 각 속성 값을 `for...in` 문을 사용하여 순회하면서 합계를 구합니다. `for...in` 문은 객체의 속성을 반복적으로 읽어올 수 있게 해주는 반복문입니다. `for...in` 문을 사용하여 객체의 속성 값에 접근하여 합계를 구합니다.

위의 코드에서는 `sum` 변수를 선언하고 초기값을 0으로 지정합니다. 그리고 `for...in` 문을 사용하여 `obj`의 각 속성을 순회하면서 `sum`에 속성 값을 더해줍니다. 마지막으로 `sum`의 값을 출력하면 객체의 속성 값의 합계가 나타납니다.

이와 같은 방법으로 자바스크립트에서 객체의 속성 값을 기준으로 합계를 구할 수 있습니다.
