반복문은 배열이나 리스트와 같은 자료구조의 모든 요소를 한 번에 처리할 때 유용합니다. 여기에서는 C++의 `for` 반복문을 사용하여 배열의 요소를 수정하는 방법을 설명하겠습니다.

### 배열의 요소 수정하기

아래는 `for` 반복문을 사용하여 배열의 각 요소를 수정하는 예제 코드입니다.

```c++
#include <iostream>
using namespace std;

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    
    for (int i = 0; i < 5; i++) {
        arr[i] = arr[i] * 2; // 배열의 각 요소를 2배로 수정
    }
    
    for (int i = 0; i < 5; i++) {
        cout << arr[i] << " "; // 수정된 배열의 요소 출력
    }
    
    return 0;
}
```

위 코드는 `arr` 배열의 각 요소를 2배로 수정한 후, 수정된 배열의 요소를 출력합니다.

### 결과

위의 코드를 실행하면 다음과 같은 결과가 출력됩니다.

```
2 4 6 8 10
```

이처럼 `for` 반복문을 사용하면 배열의 요소를 효과적으로 수정할 수 있습니다.

이상으로 C++에서 반복문을 사용하여 배열의 요소를 수정하는 방법에 대해 알아보았습니다.
