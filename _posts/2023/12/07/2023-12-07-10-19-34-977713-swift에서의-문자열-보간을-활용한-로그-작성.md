Swift는 문자열 보간이라는 기능을 제공하여 로그 작성을 간편하게 할 수 있습니다. 문자열 보간을 사용하면 변수나 상수를 문자열에 삽입하여 원하는 형태로 로그를 작성할 수 있습니다.

## 문자열 보간 문법

문자열 보간 문법은 백슬래시와 열린 중괄호, 닫힌 중괄호 사이에 변수나 상수를 넣는 형태로 사용됩니다. 예를 들어, 다음과 같은 코드가 있다고 가정해봅시다.

```swift
let name = "John"
let age = 25

let message = "이름: \(name), 나이: \(age)"
print(message)
```

위의 코드는 "이름: John, 나이: 25"라는 메시지를 출력합니다. 문자열 보간을 사용하여 변수 `name`과 `age` 값을 문자열에 삽입하였습니다.

## 로그 작성에 문자열 보간 활용하기

로그를 작성할 때 문자열 보간을 활용하면 변수나 상수의 값을 로그에 쉽게 넣을 수 있습니다. 예를 들어, 다음과 같은 코드가 있다고 가정해봅시다.

```swift
func login(username: String, password: String) {
    // 로그인 로그 작성
    print("사용자 \(username)이(가) 로그인 시도")
    
    // 로그인 로직 구현
    // ...
}
```

위의 코드에서 `login` 함수는 사용자 이름과 비밀번호를 인자로 받습니다. 로그인 로그를 작성하기 위해 문자열 보간을 사용하여 사용자 이름을 출력하고 있습니다.

## 주의사항

문자열 보간을 사용할 때 주의해야 할 점이 있습니다. 문자열 보간을 사용하는 경우 작은따옴표(') 대신 큰따옴표(")를 사용해야 합니다. 또한, 문자열 보간은 문자열 안에서만 사용할 수 있습니다.

## 정리

Swift에서의 문자열 보간을 활용하면 로그 작성을 간단하고 효율적으로 할 수 있습니다. 변수나 상수의 값을 로그에 삽입하고 싶을 때는 문자열 보간을 사용하면 됩니다. 하지만 주의해야 할 점도 있으니, 작은따옴표가 아닌 큰따옴표를 사용하고 문자열 안에서만 사용하도록 합시다.
