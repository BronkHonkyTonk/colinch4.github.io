1. 정수 타입:
   - byte: 8비트 크기의 부호 있는 정수를 표현합니다. 범위는 -128부터 127까지입니다.
   - short: 16비트 크기의 부호 있는 정수를 표현합니다. 범위는 -32,768부터 32,767까지입니다.
   - int: 32비트 크기의 부호 있는 정수를 표현합니다. 범위는 -2,147,483,648부터 2,147,483,647까지입니다.
   - long: 64비트 크기의 부호 있는 정수를 표현합니다. 범위는 -9,223,372,036,854,775,808부터 9,223,372,036,854,775,807까지입니다.

2. 실수 타입:
   - float: 32비트 크기의 부동 소수점 수를 표현합니다. 정밀도는 7자리입니다.
   - double: 64비트 크기의 부동 소수점 수를 표현합니다. 정밀도는 15자리입니다.

3. 문자 타입:
   - char: 16비트 크기의 유니코드 문자를 표현합니다.

4. 논리 타입:
   - boolean: true 또는 false 값을 가지며, 조건식의 결과로 주로 사용됩니다.

이러한 기본 데이터 타입은 자바 프로그래밍에서 변수의 선언과 함께 사용되며, 데이터의 종류와 크기에 따라 적절한 타입을 선택하여 사용해야 합니다.

식별자 앞에 데이터 타입을 선언하고 값을 할당하는 방식으로 변수를 초기화할 수 있습니다. 예를 들어, int 타입의 변수를 선언하고 값을 할당하는 코드는 다음과 같습니다:

```java
int myVariable = 10;
```

더 자세한 내용은 [Oracle Java Documentation](https://docs.oracle.com/en/java/javase/14/docs/api/java.base/java/lang/doc-files/ValueBased.html)을 참조하세요.
