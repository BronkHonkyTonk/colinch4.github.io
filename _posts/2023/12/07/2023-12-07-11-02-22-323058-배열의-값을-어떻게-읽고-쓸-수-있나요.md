Kotlin에서 배열의 값을 읽거나 쓰는 방법은 간단합니다. 배열은 일련의 값들을 저장하는 자료구조로, 각 값은 인덱스를 통해 접근할 수 있습니다. 배열의 인덱스는 0부터 시작합니다.

배열에 값을 쓰는 방법은 다음과 같습니다. 먼저 배열을 선언하고, 해당 인덱스에 값을 할당하면 됩니다.

```kotlin
val array = arrayOf(1, 2, 3, 4, 5)
array[0] = 10
```

위의 예시에서는 `array`라는 이름의 배열을 선언한 후, 첫 번째 인덱스에 값인 10을 할당하였습니다. 이제 배열 `array`의 첫 번째 값은 10이 됩니다.

배열의 값을 읽는 방법은 다음과 같습니다. 배열의 인덱스를 사용하여 해당 인덱스의 값을 가져오면 됩니다.

```kotlin
val value = array[0]
```

위의 예시에서는 배열 `array`의 첫 번째 인덱스에 있는 값을 `value` 변수에 할당하였습니다. 이제 `value` 변수에는 배열의 첫 번째 값인 10이 저장되어 있습니다.

배열의 길이를 확인하려면 `size` 속성을 사용하면 됩니다.

```kotlin
val length = array.size
```

위의 예시에서는 배열 `array`의 길이를 `length` 변수에 할당하였습니다. 이제 `length` 변수에는 배열의 길이인 5가 저장되어 있습니다.

Kotlin에서 배열을 다루는 방법은 간단하고 직관적입니다. 배열을 사용하여 데이터를 저장하고 읽을 수 있다는 점은 앱 또는 프로그램을 개발할 때 매우 유용한 기능 중 하나입니다.
