자바는 강력한 데이터 타입 시스템을 가지고 있어 다양한 형식의 데이터를 다룰 수 있습니다. 이번 포스트에서는 자바에서 기본적으로 제공되는 데이터 타입들과 입출력 방법에 대해 알아보겠습니다.

## 데이터 타입

자바에서 사용되는 데이터 타입은 크게 원시 데이터 타입과 참조 데이터 타입으로 나뉩니다.

### 원시 데이터 타입

원시 데이터 타입은 메모리에 직접 값을 저장하는 데이터 타입입니다. 다음은 자바에서 제공하는 원시 데이터 타입들입니다.

- **byte**: 8비트로 부호 있는 정수 값을 저장합니다. (-128부터 127까지)
- **short**: 16비트로 부호 있는 정수 값을 저장합니다. (-32,768부터 32,767까지)
- **int**: 32비트로 부호 있는 정수 값을 저장합니다. (-2,147,483,648부터 2,147,483,647까지)
- **long**: 64비트로 부호 있는 정수 값을 저장합니다. (-9,223,372,036,854,775,808부터 9,223,372,036,854,775,807까지)
- **float**: 32비트로 부동 소수점 값을 저장합니다.
- **double**: 64비트로 부동 소수점 값을 저장합니다.
- **char**: 16비트로 유니코드 문자를 저장합니다.
- **boolean**: true 또는 false 값을 저장합니다.

### 참조 데이터 타입

참조 데이터 타입은 객체를 가리키는 참조 값을 저장하는 데이터 타입입니다. 자바에서 제공하는 몇 가지 주요 참조 데이터 타입은 다음과 같습니다.

- **String**: 문자열을 저장하는 클래스로, 다양한 문자열 조작 기능을 제공합니다.
- **Array**: 배열을 저장하는 클래스로, 동적으로 크기를 조정할 수 있습니다.
- **Class**: 클래스의 정보를 저장하는 클래스로, 리플렉션과 같은 고급 기능을 사용할 때 사용됩니다.

## 입출력 형식

자바는 다양한 방법으로 입출력을 할 수 있습니다. 가장 기본적인 입출력 방법은 콘솔에서의 입출력입니다. 자바에서는 `System.in`과 `System.out`을 사용하여 콘솔에서의 입력과 출력을 처리할 수 있습니다.

다음은 콘솔에서 사용자로부터 입력을 받는 방법과 콘솔에 출력하는 방법의 예시입니다.

```java
import java.util.Scanner;

public class ConsoleExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("이름을 입력하세요: ");
        String name = scanner.nextLine();

        System.out.print("나이를 입력하세요: ");
        int age = scanner.nextInt();

        System.out.println("입력된 이름: " + name);
        System.out.println("입력된 나이: " + age);
    }
}
```

위 예시 코드에서 `Scanner` 클래스를 사용하여 콘솔에서 사용자로부터 입력을 받을 수 있습니다. 또한, `System.out.println()`을 사용하여 콘솔에 출력할 수도 있습니다.
