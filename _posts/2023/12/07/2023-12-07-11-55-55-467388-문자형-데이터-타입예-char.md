자바(Java)에서는 문자형 데이터를 처리하기 위한 데이터 타입인 `char`를 제공합니다. `char`는 2바이트(16비트) 크기의 유니코드 문자를 저장할 수 있는 데이터 타입입니다.

## `char` 타입 선언하기

`char` 타입 변수를 선언하려면 다음과 같이 사용합니다:

```java
char myChar;
```

## `char` 타입에 값 할당하기

`char` 타입 변수에 값을 할당하려면 작은 따옴표(`'`)를 사용하여 문자를 감싸면 됩니다. 예를 들어, 'A'라는 문자를 할당하려면 다음과 같이 작성할 수 있습니다:

```java
char myChar = 'A';
```

## 유니코드 문자 할당하기

`char` 타입은 유니코드 문자를 처리할 수 있습니다. 유니코드(Unicode)는 전세계의 모든 문자를 컴퓨터에서 일관되게 표현하기 위한 국제 표준 코드입니다. 예를 들어, 유니코드 문자 '한'을 할당하려면 다음과 같이 작성할 수 있습니다:

```java
char myChar = '한';
```

## 특수문자 사용하기

`char` 타입은 특수문자도 처리할 수 있습니다. 특수문자는 역슬래시(`\`)를 사용하여 표현합니다. 예를 들어, 개행 문자를 나타내는 특수문자 `\n`을 할당하려면 다음과 같이 작성할 수 있습니다:

```java
char myChar = '\n';
```

## 문자열과의 차이

`char` 타입은 단일 문자를 처리하는 데이터 타입이고, 문자열을 처리하기 위해서는 `String` 타입을 사용해야 합니다. 문자열은 큰 따옴표(`"`)로 감싸며, 여러 문자를 포함할 수 있습니다.

## 요약

- `char` 데이터 타입은 2바이트(16비트) 크기의 유니코드 문자를 저장할 수 있습니다.
- `char` 타입은 작은 따옴표(`'`)로 문자를 감싸면 됩니다.
- 유니코드 문자와 특수문자도 `char` 타입으로 처리할 수 있습니다.
- 문자열은 `String` 타입을 사용해야 합니다.
