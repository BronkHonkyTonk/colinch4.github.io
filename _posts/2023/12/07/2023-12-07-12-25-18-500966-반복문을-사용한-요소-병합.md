## 요소 병합
배열이나 리스트 등의 요소를 병합하는 작업은 컴퓨터 과학에서 일반적인 작업입니다. C++에서는 반복문을 사용하여 이러한 병합 작업을 수행할 수 있습니다. 이를 통해 두 개의 배열이나 리스트를 하나로 합칠 수 있습니다.

## C++에서의 반복문을 사용한 요소 병합 코드 예제
아래는 두 개의 벡터를 병합하는 C++ 코드 예제입니다.

```cpp
#include <iostream>
#include <vector>

int main() {
    std::vector<int> vec1 = {1, 2, 3};
    std::vector<int> vec2 = {4, 5, 6};

    for (int i = 0; i < vec2.size(); i++) {
        vec1.push_back(vec2[i]);
    }

    // 결과 출력
    for (int i = 0; i < vec1.size(); i++) {
        std::cout << vec1[i] << " ";
    }

    return 0;
}
```

위 코드는 `vec1`에 `vec2`의 요소를 반복문을 사용하여 병합한 후, 결과를 출력하는 예제입니다.

## 요약
C++에서는 반복문을 활용하여 요소를 병합하는 작업을 수행할 수 있습니다. 이를 통해 여러 컨테이너의 요소를 효과적으로 결합할 수 있습니다.
