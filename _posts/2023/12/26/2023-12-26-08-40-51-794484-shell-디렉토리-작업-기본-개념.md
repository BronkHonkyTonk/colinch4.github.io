많은 사용자들이 셸을 사용하여 파일 및 디렉토리를 조작하고 명령을 실행하는 데 익숙해 있습니다. 그러나 디렉토리를 조작하는 데 있어서 몇 가지 기본 개념을 알고 있다면 더 효율적으로 작업할 수 있습니다. 이번에는 셸에서의 디렉토리 작업에 관련된 몇 가지 기본 개념을 살펴보겠습니다.

## 1. 디렉토리 이동

쉘에서 디렉토리를 이동하는 것은 중요한 작업입니다. `cd` 명령어를 사용하여 다른 디렉토리로 이동할 수 있습니다. 예를 들어, 다음은 홈 디렉토리로 이동하는 명령어입니다.

```shell
cd ~
```

## 2. 현재 디렉토리 확인

현재 작업 중인 디렉토리를 확인하는 것은 중요합니다. `pwd` 명령어를 사용하여 현재 디렉토리의 경로를 확인할 수 있습니다.

```shell
pwd
```

## 3. 디렉토리 리스트 표시

디렉토리의 내용을 확인하는 것은 디렉토리를 조작하는 데 유용합니다. `ls` 명령어를 사용하여 디렉토리의 파일 및 서브 디렉토리를 리스트할 수 있습니다.

```shell
ls
```

## 4. 디렉토리 생성

새로운 디렉토리를 생성하는 것은 파일을 구성하고 관리하는 데 중요합니다. `mkdir` 명령어를 사용하여 새로운 디렉토리를 생성할 수 있습니다.

```shell
mkdir new_directory
```

## 요약

디렉토리 작업은 셸 사용 중 가장 기본적인 요소 중 하나입니다. 이 기본 개념들을 숙지한다면 파일 및 디렉토리를 더 효율적으로 관리할 수 있습니다.
