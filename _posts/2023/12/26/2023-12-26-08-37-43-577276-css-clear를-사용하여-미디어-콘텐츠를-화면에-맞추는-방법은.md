예를 들어, 다음과 같이 사용할 수 있습니다.

```css
.media-content {
  clear: both;
}
```

이 코드는 `.media-content`라는 클래스를 가진 요소가 양쪽으로부터 떨어지도록 설정합니다.

추가로, 미디어 콘텐츠를 화면에 맞출 때 `clear` 속성을 적절히 활용하여 레이아웃을 조정할 수 있습니다. 만약 적용 후에도 원하는 결과가 나오지 않는다면 다른 레이아웃 속성들도 함께 검토해보는 것이 좋습니다.

더 자세한 내용은 [MDN web docs](https://developer.mozilla.org/ko/docs/Web/CSS/clear)를 참고하시기 바랍니다.
