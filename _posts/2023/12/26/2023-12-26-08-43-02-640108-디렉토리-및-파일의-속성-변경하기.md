## 디렉토리 및 파일의 속성 확인하기

먼저, 디렉토리와 파일의 속성을 확인해야 합니다. 이를 위해 `ls` 명령어를 사용합니다. 

```shell
ls -l
```

위 명령어를 실행하면, 각 파일 또는 디렉토리의 속성 정보를 확인할 수 있습니다. 

## 속성 변경하기

### 1. 소유자 변경

만약 파일의 소유자를 변경하려면, `chown` 명령어를 사용합니다.

예를 들어, `file.txt` 파일의 소유자를 `newuser`로 변경하려면:

```shell
chown newuser file.txt
```

### 2. 권한 변경

파일의 권한을 변경하기 위해서는 `chmod` 명령어를 사용합니다.

예를 들어, `file.txt` 파일의 소유자에 대한 실행 권한을 추가하려면:

```shell
chmod u+x file.txt
```

이렇게 하면 해당 파일의 소유자에 대한 실행 권한이 추가됩니다.

위 방법을 활용하여 디렉토리 또는 파일의 속성을 변경할 수 있습니다. 기타 속성 변경에 대한 자세한 내용은 매뉴얼이나 온라인 자료를 참조하세요.

이상입니다! 디렉토리와 파일의 속성 변경에 대한 간단한 소개였습니다.
