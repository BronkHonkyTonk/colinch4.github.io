이번에는 쉘에서 사용되는 기본적인 디렉토리 이동 관련 명령어에 대해 알아보겠습니다. 디렉토리 이동은 쉘에서 매우 중요한 작업 중 하나이며, 다음과 같은 명령어들을 사용하여 디렉토리를 이동할 수 있습니다.

## 1. `cd` 명령어

`cd` 명령어는 Change Directory의 약자로, 현재 작업 디렉토리를 변경하는데 사용됩니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```shell
cd /path/to/directory
```

위의 예시에서 `/path/to/directory`는 이동하고자 하는 디렉토리의 경로를 나타냅니다.

## 2. `ls` 명령어

`ls` 명령어는 현재 위치한 디렉토리의 내용을 나열해 주는 명령어입니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```shell
ls
```

## 3. `pwd` 명령어

`pwd` 명령어는 Print Working Directory의 약자로, 현재 작업 중인 디렉토리의 경로를 출력해 주는 명령어입니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```shell
pwd
```

디렉토리를 이동하고 싶을 때, 위의 명령어들을 사용하여 현재 위치를 확인하고, 원하는 디렉토리로 이동할 수 있습니다.
