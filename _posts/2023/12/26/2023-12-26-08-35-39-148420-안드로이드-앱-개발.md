## Kotlin의 기본 문법
Kotlin은 자바보다 훨씬 간결하고 실용적인 문법을 제공합니다. 변수 선언, 제어 구조, 함수 정의 등을 자바보다 훨씬 더 간단하게 작성할 수 있습니다.

```kotlin
// 변수 선언
val name: String = "John"
var age: Int = 25

// 제어 구조
if (age >= 20) {
    println("You are an adult")
} else {
    println("You are a minor")
}

// 함수 정의
fun greet(name: String) {
    println("Hello, $name!")
}
```
