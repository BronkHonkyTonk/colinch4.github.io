보다 구체적인 내용은 아래의 HTML 코드 예제를 확인해 주세요:

```html
<!DOCTYPE html>
<html>
<head>
    <title>테스트 페이지</title>
</head>
<body>
    <h1>환영합니다!</h1>
    <p>이것은 테스트 페이지입니다.</p>
</body>
</html>
```

해당 예제는 간단한 테스트 페이지를 작성하는 기본적인 HTML 구조를 보여줍니다. 이제 이 구조를 가지고 원하는 콘텐츠를 추가하고 스타일링을 적용하여 테스트 페이지를 완성할 수 있습니다.
