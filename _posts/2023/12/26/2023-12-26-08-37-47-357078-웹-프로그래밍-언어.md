## HTML (Hypertext Markup Language)
HTML은 웹 페이지의 구조를 정의하는 데 사용됩니다. 제목, 단락, 이미지, 링크 등을 정의하여 웹 페이지의 레이아웃을 결정합니다.

## CSS (Cascading Style Sheets)
CSS는 웹 페이지의 스타일과 디자인을 담당합니다. 색상, 레이아웃, 폰트 등을 조절하여 웹 페이지를 더욱 시각적으로 매력적으로 만듭니다.

## JavaScript
JavaScript는 웹 페이지에 동적인 기능을 추가하는 데 사용됩니다. 사용자와의 상호작용, 애니메이션, 데이터 처리 등을 수행할 수 있습니다.

이들 언어를 조합하여 웹 프로그래밍을 통해 멋진 웹 사이트를 구축할 수 있습니다.
