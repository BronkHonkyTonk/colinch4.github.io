### 시스템 리소스 모니터링
리눅스 데스크톱에서 가장 일반적으로 사용되는 **시스템 리소스 모니터링 도구** 중 하나는 **GNOME 시스템 모니터**입니다. GNOME 시스템 모니터는 CPU, 메모리, 디스크 및 네트워크 사용량을 실시간으로 모니터링할 수 있습니다.

```bash
sudo apt install gnome-system-monitor
```

### 프로세스 모니터링
프로세스 수준의 성능 모니터링을 위해 **htop**을 사용할 수 있습니다. htop은 실행 중인 프로세스 목록을 보여주고 CPU 및 메모리 사용량 등을 실시간으로 확인할 수 있습니다.

```bash
sudo apt install htop
```

### 네트워크 모니터링
**iftop**은 리눅스 데스크톱에서 네트워크 트래픽을 모니터링하는 데 유용한 도구입니다. 다양한 네트워크 관련 정보를 실시간으로 표시하여 네트워크 활동을 추적할 수 있습니다.

```bash
sudo apt install iftop
```

### 결론
리눅스 데스크톱에서 시스템 리소스, 프로세스, 그리고 네트워크를 모니터링하기 위한 여러 가지 도구를 사용할 수 있습니다. 이러한 도구들은 시스템의 성능을 실시간으로 추적하고 문제를 식별하는 데 도움이 됩니다. 적절한 성능 모니터링은 시스템의 안정성과 효율성을 향상시키는 데 중요한 역할을 합니다.

이제 리눅스 데스크톱에서 성능 모니터링에 대한 이해를 높이고, 필요에 따라 적절한 도구를 선택하여 시스템 성능을 최적화할 수 있을 것입니다.
