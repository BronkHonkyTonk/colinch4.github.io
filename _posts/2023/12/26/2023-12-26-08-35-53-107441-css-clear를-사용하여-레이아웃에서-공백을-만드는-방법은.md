예를 들어, 플로팅된 요소 뒤에 새로운 플로팅 요소를 추가하고 싶을 때 clear를 사용할 수 있습니다. 예를 들어, 다음과 같이 사용할 수 있습니다:

```css
.clear {
    clear: both;
}
```

이렇게 하면 해당 요소가 왼쪽과 오른쪽 모두 플로팅된 요소의 영향을 받지 않게 됩니다.

더 많은 정보가 필요하시다면 아래 링크를 참고해 주세요:
[MDN Web Docs - Clear](https://developer.mozilla.org/en-US/docs/Web/CSS/clear)
