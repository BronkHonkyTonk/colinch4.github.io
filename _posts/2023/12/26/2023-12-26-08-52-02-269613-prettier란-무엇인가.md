Prettier는 JavaScript뿐만 아니라 CSS, JSON, Markdown 등 다양한 언어의 코드를 자동으로 포맷팅할 수 있습니다. 코드포맷에 관한 수많은 논란과 시간낭비를 줄여주며, 코드 작성에 집중할 수 있도록 도와줍니다. 이를테면 Prettier의 사용 예는 다음과 같습니다.

```javascript
const add = (a, b) => {
  return a + b;
};

console.log(add(5, 3));
```

보시다시피, 코드의 형식은 Prettier에 의해 자동으로 정리되어 가독성과 일관성을 갖추게 됩니다.

더 많은 정보를 원하시면 Prettier 공식 홈페이지를 참고하세요: [Prettier 공식 페이지](https://prettier.io/)
