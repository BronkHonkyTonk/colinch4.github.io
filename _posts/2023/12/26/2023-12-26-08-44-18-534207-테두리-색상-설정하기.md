다음은 border-color를 사용하여 테두리 색상을 설정하는 예시입니다.

```css
.element {
  border: 1px solid #ff0000; /* 빨간색 테두리 */
}
```

위의 코드에서 #ff0000은 빨간색을 나타내는 16진수 색상 코드입니다. border-color 속성을 통해 다양한 색상을 사용할 수 있으며, RGB 색상, 색상 이름 등을 지정할 수도 있습니다.

자세한 내용은 CSS border-color 속성을 확인하세요.
