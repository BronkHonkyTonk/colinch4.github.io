### HTML이란 무엇인가

HTML(HyperText Markup Language)은 웹 페이지를 만들기 위한 기본적인 마크업 언어로, 웹 콘텐츠의 구조와 의미를 정의합니다. HTML은 웹 브라우저에게 각 요소의 의미를 알려주고, 적절한 방식으로 표시할 수 있도록 도와줍니다.

### HTML의 기본 구조

```html
<!DOCTYPE html>
<html>
<head>
    <title>문서 제목</title>
</head>
<body>
    <h1>제목</h1>
    <p>내용</p>
</body>
</html>
```

위는 가장 기본적인 HTML 문서의 구조입니다. `<!DOCTYPE html>`은 문서의 타입을 정의하고, `<html>`, `<head>`, `<title>`, `<body>` 등의 태그를 사용하여 문서의 구조를 정의합니다.

### 주요 HTML 요소

- `<h1>, <h2>, <h3>...<h6>`: 제목을 나타내는 태그
- `<p>`: 문단을 나타내는 태그
- `<a>`: 하이퍼링크를 생성하는 태그
- `<img>`: 이미지를 삽입하는 태그
- `<ul>, <ol>, <li>`: 목록을 나타내는 태그
