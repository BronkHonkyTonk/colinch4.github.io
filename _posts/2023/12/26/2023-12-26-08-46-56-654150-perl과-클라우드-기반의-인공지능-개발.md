### 2. Perl을 이용한 데이터 처리
Perl은 데이터 처리에 매우 유용한 언어로, 큰 규모의 데이터셋을 처리할 수 있는 강력한 기능을 제공합니다. 또한, 다양한 데이터 포맷을 처리하는 모듈도 많이 존재하여, 데이터를 파싱하고 가공하는데 매우 유용합니다. 

아래는 Perl을 사용하여 CSV 파일을 읽고 간단한 데이터 처리를 하는 예제입니다. 

```perl
use Text::CSV;

my $csv = Text::CSV->new({ binary => 1, auto_diag => 1, eol => $/ });

open my $fh, "<:encoding(utf8)", "data.csv" or die "data.csv: $!";
while ( my $row = $csv->getline( $fh ) ) {
    # 데이터 처리
    print "Name: $row->[0], Age: $row->[1]\n";
}
$csv->eof or $csv->error_diag();
close $fh;
```
