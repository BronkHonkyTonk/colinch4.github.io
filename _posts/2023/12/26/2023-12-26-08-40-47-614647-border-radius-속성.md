CSS의 `border-radius` 속성은 요소의 모서리를 둥글게 만들어주는 속성입니다.

## 기본 문법

```css
selector {
  border-radius: value;
}
```

## 속성 값

- `value`: 모서리를 둥글게 만들어주는 크기 값으로, 픽셀(px)이나 백분율(%)로 지정할 수 있습니다.

## 사용 예시

```css
.rounded-box {
  border-radius: 10px; /* 픽셀 단위로 모서리를 10px 만큼 둥글게 하는 예시 */
}
```

`border-radius` 속성을 사용하면 사각형 박스의 모서리를 둥글게 만들어 다양한 디자인을 적용할 수 있습니다.

더 자세한 정보는 [MDN web docs](https://developer.mozilla.org/ko/docs/Web/CSS/border-radius)에서 확인할 수 있습니다.
