## 변수
### 변수란 무엇인가?
프로그래밍에서 **변수**란 값이 저장될 수 있는 메모리 공간을 가리킵니다. 이 메모리 공간에는 어떤 종류의 데이터든지 저장할 수 있으며, 프로그램 실행 동안 그 값이 여러 번 변경될 수 있습니다.

### 변수의 사용
변수를 사용하기 위해서는 해당 프로그래밍 언어에서 변수를 선언하고 값을 할당해야 합니다. 아래는 JavaScript에서 변수를 선언하고 값을 할당하는 예시입니다.

```javascript
let count;      // 변수 선언
count = 10;     // 값 할당
```

## 상수
### 상수란 무엇인가?
**상수**는 프로그램 내에서 고정된 값을 가지는 식별자로, 한 번 값을 할당하면 프로그램 실행 중에는 그 값을 변경할 수 없는 변수를 가리킵니다.

### 상수의 활용
상수는 주로 변하지 않는 값을 표시하는데 사용되며, 프로그램의 가독성과 유지보수를 향상시킵니다. 아래는 Swift에서 상수를 선언하고 값을 할당하는 예시입니다.

```swift
let pi = 3.14159        // 상수 선언과 값 할당
```

변수와 상수는 어떤 프로그래밍 언어를 사용하더라도 기본적인 개념으로, 프로그램을 작성할 때 항상 다뤄야 하는 중요한 주제입니다.
