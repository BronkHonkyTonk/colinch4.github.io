## 쉘 스크립트란?

쉘 스크립트는 **리눅스나 유닉스 시스템에서 사용되는 스크립트 언어**로, 특정 작업을 자동화하거나 여러 명령어를 순차적으로 실행하는 데 사용됩니다. 쉘 스크립트는 일련의 명령어를 텍스트 파일에 작성하여 사용자가 실행할 수 있도록 하는 것입니다.

예를 들어, 다음은 파일 목록을 보여주는 간단한 쉘 스크립트의 예제입니다.

```bash
#!/bin/bash
ls
```

위의 예제에서 `#!/bin/bash`는 이 스크립트가 bash 쉘에서 실행되어야 함을 나타내는 헤더입니다. 그 다음 줄에 있는 `ls` 명령어는 현재 디렉토리의 파일 목록을 출력하는 역할을 합니다.

## 리눅스 명령어 사용하기

리눅스 명령어는 터미널을 통해 시스템을 제어하고 파일 및 디렉토리를 관리하는데 사용됩니다. 아래는 자주 사용되는 리눅스 명령어 몇 가지의 예제입니다.

### 파일 및 디렉토리 관리

- **`ls`**: 현재 디렉토리의 파일 목록을 보여줍니다.
- **`cd`**: 디렉토리를 변경합니다.
- **`mkdir`**: 새 디렉토리를 생성합니다.

예제:

```bash
ls
cd Documents
mkdir new_folder
```

### 파일 내용 보기 및 편집

- **`cat`**: 파일의 내용을 출력합니다.
- **`vi`**: 파일을 편집하는 텍스트 편집기입니다.

예제:

```bash
cat file.txt
vi new_file.txt
```
