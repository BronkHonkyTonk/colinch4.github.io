## 프로그램 실행

대부분의 운영 체제에서는 명령 프롬프트나 터미널을 통해 프로그램을 실행할 수 있습니다. 여기에는 명령을 입력하여 실행하는 방법이 포함됩니다.

### Windows

```bash
> program_name
```

### MacOS/Linux

```bash
$ program_name
```

## 프로그램 종료

### Windows 또는 MacOS의 일반적인 단축키

- **Windows**: Ctrl + C
- **MacOS**: Command + C

이러한 단축키를 사용하여 실행 중인 프로그램을 종료할 수 있습니다.

프로그램을 종료하는 방법은 프로그램 종류나 운영 체제에 따라 다를 수 있습니다. 여기서는 몇 가지 기본적인 방법을 알아보았습니다.
