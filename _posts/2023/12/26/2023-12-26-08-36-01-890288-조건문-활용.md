조건문은 프로그램에서 특정 조건이 충족될 때 코드 블록을 실행하는 데 사용됩니다. 예를 들어, 특정 조건이 참이면 특정 작업을 수행하고, 거짓이면 다른 작업을 수행할 수 있습니다.

조건문을 사용하여 프로그램은 입력 데이터나 상황에 따라 다양한 동작을 수행할 수 있습니다. 이를 통해 프로그램이 유연하고 응용 프로그램의 요구에 맞게 동작할 수 있게 됩니다.

### if 문

```python
if 조건:
    # 조건이 참일 때 실행하는 코드
else:
    # 조건이 거짓일 때 실행하는 코드
```

### if-elif-else 문

```python
if 조건1:
    # 조건1이 참일 때 실행하는 코드
elif 조건2:
    # 조건1이 거짓이고 조건2가 참일 때 실행하는 코드
else:
    # 위의 모든 조건이 거짓일 때 실행하는 코드
```

조건문을 적절히 활용하여 프로그램이 다양한 상황에 대응할 수 있도록 만들어 보세요.
