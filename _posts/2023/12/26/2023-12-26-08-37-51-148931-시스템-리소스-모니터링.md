이 기술 블로그는 리눅스 환경에서 시스템 리소스를 모니터링하는 방법에 대해 다룹니다.

## 1. CPU 모니터링

CPU 사용률을 모니터링하기 위해 아래 명령어를 사용할 수 있습니다:

```shell
top
```

위 명령어는 시스템 내 CPU 사용률, 프로세스 목록 및 기타 시스템 리소스 관련 정보를 보여줍니다.

## 2. 메모리 모니터링

시스템 메모리 사용량을 모니터링하기 위해 다음 명령어를 사용할 수 있습니다:

```shell
free -m
```

위 명령어는 시스템의 총 메모리, 사용 중인 메모리, 사용 가능한 메모리 등의 정보를 표시합니다.

## 3. 디스크 공간 모니터링

디스크 사용량을 모니터링하기 위해 아래 명령어를 사용할 수 있습니다:

```shell
df -h
```

위 명령어는 시스템의 각 디스크 마운트 지점 및 해당 디스크의 사용 용량, 사용 가능한 용량 등을 보여줍니다.

이러한 명령어들은 시스템에서 리소스를 모니터링하고 문제를 해결하는 데 도움을 줄 수 있습니다.
