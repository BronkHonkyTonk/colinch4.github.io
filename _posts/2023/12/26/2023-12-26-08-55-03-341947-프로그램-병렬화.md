## 병렬화란

**병렬화**란 하나의 작업을 여러 개의 부분으로 나누어 동시에 처리하는 것을 말합니다. 이를 통해 작업을 빠르게 처리할 수 있으며, 다양한 병렬화 기술을 활용하여 프로그램의 성능을 향상시킬 수 있습니다.

## C언어에서의 병렬화 기술

C언어에서는 **스레드** 또는 **프로세스**를 활용하여 병렬화를 구현할 수 있습니다. 스레드를 사용하면 하나의 프로세스 내에서 여러 개의 작업을 동시에 처리할 수 있으며, 프로세스를 사용하면 각각의 프로세스가 독립적으로 작업을 처리합니다.

또한, C언어에서는 **OpenMP**와 같은 병렬 컴퓨팅 라이브러리를 활용하여 코드를 병렬화할 수도 있습니다. 이러한 라이브러리를 활용하면 코드의 일부분을 병렬로 실행하여 성능을 향상시킬 수 있습니다.

## 병렬화 예시

아래는 C언어로 작성된 간단한 예시 코드를 통해 병렬화 기술을 적용하는 방법을 보여줍니다.

```c
#include <stdio.h>
#include <omp.h>

int main() {
    int i;
    #pragma omp parallel for
    for (i = 0; i < 10; i++) {
        printf("현재 스레드: %d, 현재 인덱스: %d\n", omp_get_thread_num(), i);
    }
    return 0;
}
```

위 코드는 OpenMP를 사용하여 for 루프를 병렬화하는 예시입니다.
