특히, clear 속성을 사용하면 요소의 오른쪽이나 왼쪽, 또는 양쪽의 부동을 제거할 수 있습니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```css
.clear {
  clear: both;
}
```
