## HTML 웹 템플릿이란?

**HTML 웹 템플릿**은 웹사이트를 만들 때 사용할 수 있는 미리 디자인된 HTML 파일의 형태를 말합니다. 이러한 템플릿은 프론트엔드 개발자가 기본적인 레이아웃과 디자인을 구축하는 데 도움을 줍니다.

## HTML 웹 템플릿 활용 방법

HTML 웹 템플릿은 간단한 텍스트 에디터로 열어서 수정하고, 필요에 따라 이미지나 스타일시트를 수정하여 사용할 수 있습니다. 아래는 간단한 HTML 템플릿의 예시입니다.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>웹 템플릿 예시</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>웹 템플릿 예시</h1>
    </header>
    <nav>
        <ul>
            <li><a href="#">홈</a></li>
            <li><a href="#">서비스</a></li>
            <li><a href="#">포트폴리오</a></li>
            <li><a href="#">연락하기</a></li>
        </ul>
    </nav>
    <main>
        <h2>제목</h2>
        <p>내용</p>
    </main>
    <footer>
        <p>Copyright © 2022</p>
    </footer>
</body>
</html>
```
