# 디스크 확인

리눅스에서는 `fdisk`나 `parted`와 같은 툴을 사용하여 디스크의 파티션과 파일 시스템을 확인할 수 있습니다. 예를 들어, 다음 명령어를 사용하여 디스크의 상세 정보를 확인할 수 있습니다.

```shell
fdisk -l /dev/sda
```

# 파티션 조작

디스크 파티션은 파일 시스템을 저장하는 영역으로, 파티션을 추가하거나 수정하는 등의 작업이 가능합니다. `fdisk`나 `parted`를 사용하여 파티션을 생성하고 수정할 수 있습니다.

예를 들어, 새로운 파티션을 생성하려면 다음과 같이 실행합니다.

```shell
fdisk /dev/sda
```

# 파일 시스템 관리

리눅스에서는 다양한 파일 시스템을 지원하며, `mkfs`나 `resize2fs`와 같은 명령어를 사용하여 파일 시스템을 생성하고 크기를 조정할 수 있습니다. 

예를 들어, 새로운 ext4 파일 시스템을 생성하려면 다음과 같이 실행합니다.

```shell
mkfs.ext4 /dev/sda1
```

디스크 관리는 시스템 안정성을 유지하는 데 중요한 요소이므로, 디스크 관리에 대한 이해는 시스템 관리자에게 필수적입니다.
