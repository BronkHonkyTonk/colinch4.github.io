일반적으로, `clear` 속성은 플로팅된 요소를 제어하기 위해 사용되며, 다음과 같이 사용될 수 있습니다.

```css
.clear {
  clear: both;
}
```

이 코드는 `.clear` 클래스가 지정된 요소는 다른 요소의 양 옆에 나타나지 않고 강제로 다음 행에서 시작하도록 합니다.

더 자세한 내용은 MDN 웹 문서를 참고하시기 바랍니다. [MDN clear 속성](https://developer.mozilla.org/en-US/docs/Web/CSS/clear)
