R은 통계 분석 및 데이터 시각화를 위한 프로그래밍 언어로 널리 사용되고 있습니다. 이번 포스트에서는 R 언어의 기본 문법에 대해 알아보겠습니다.

## 1. 변수 선언과 할당

R에서 변수는 `<-` 기호를 사용하여 선언하고 할당합니다. 변수에 값을 할당하기 위해서는 변수명 다음에 `<-` 기호를 사용한 후 값을 입력합니다.

예시:
```R
# 변수 x에 10을 할당
x <- 10
```

## 2. 데이터 구조

R에서는 다양한 데이터 구조를 지원합니다. 그 중에서도 가장 일반적으로 사용되는 것은 벡터(vector)입니다. 벡터는 하나의 데이터 타입으로 구성된 1차원 배열입니다.

예시:
```R
# 1부터 5까지의 정수로 이루어진 벡터 생성
my_vector <- c(1, 2, 3, 4, 5)
```

## 3. 조건문

조건문은 `if`와 `else` 키워드를 사용하여 정의됩니다. 조건이 참이면 if 블록이 실행되고, 거짓이면 else 블록이 실행됩니다.

예시:
```R
# x가 0보다 크면 "양수"를 출력
if (x > 0) {
  print("양수")
} else {
  print("음수 또는 0")
}
```

R 언어의 기본 문법은 위와 같은 형태를 가지고 있습니다. 데이터 분석 및 시각화를 위한 다양한 기능들과 함께 이러한 기본 문법을 숙지하면 효율적으로 프로그래밍할 수 있습니다.
