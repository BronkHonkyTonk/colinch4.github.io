## 개발 환경 구성

Angular를 사용하여 웹 애플리케이션을 개발하려면 Node.js와 npm을 설치해야 합니다. 그리고 Angular CLI(Command Line Interface)를 사용하여 프로젝트를 생성하고 관리할 수 있습니다.

### Node.js 및 npm 설치

다음과 같은 명령을 사용하여 Node.js와 npm을 설치합니다.

```bash
sudo apt install nodejs
sudo apt install npm
```

또는 윈도우에서는 공식 웹 사이트에서 설치 프로그램을 다운로드하여 설치합니다.

### Angular CLI 설치

npm을 사용하여 Angular CLI를 전역으로 설치합니다.

```bash
npm install -g @angular/cli
```

이제 Angular 프로젝트를 시작할 준비가 되었습니다. Angular CLI를 사용하여 프로젝트를 생성하고 개발 서버를 시작할 수 있습니다. 

Angular를 사용하여 웹 애플리케이션을 만들기 위한 기본적인 환경 구성을 완료했습니다. 이제 Angular의 강력함을 발휘하여 효율적으로 웹 애플리케이션을 개발할 수 있습니다.

이상으로 Angular 환경 구성에 대한 내용을 마치겠습니다. 감사합니다.
