1. **사용 알고리즘**: Prettier는 일관된 산술 연산 규칙을 적용하기 위해 내부적으로 AST(Abstract Syntax Tree)를 생성합니다. 이를 통해 코드를 분석하고 재구성함으로써 일관된 스타일을 적용합니다.

2. **공백 처리**: Prettier는 일관된 공백 처리를 통해 산술 연산을 균일하게 표현합니다. 연산자 주변의 공백, 콤마 후의 공백 등을 일관된 방식으로 포맷합니다.

3. **줄 바꿈**: Prettier는 산술 연산을 여러 줄에 걸쳐 표현할 때 일관된 방식으로 줄 바꿈을 처리합니다. 긴 식을 한 줄에 작성할지, 여러 줄에 나눠서 작성할지를 결정하여 일관성을 유지합니다.

4. **괄호 사용**: Prettier는 산술 연산 시 괄호의 사용에 일관성을 부여합니다. 괄호의 위치 및 사용 여부를 통해 코드를 더 명확하고 일관된 형태로 표현합니다.

Prettier를 사용하면 이러한 일관성을 수동으로 유지할 필요가 없어지므로 코드 작성 시간을 절약하고 일관된 형태의 코드를 유지할 수 있습니다. Prettier는 JavaScript 뿐만 아니라 여러 다른 언어에도 적용될 수 있는 툴이므로 프로젝트에 맞게 설정하여 사용할 수 있습니다.
