예를 들어, 다음과 같은 JavaScript 코드가 있다고 가정해봅시다.

```javascript
const result = 10 + 5 * 3;
```

Prettier를 사용하면 해당 코드가 다음과 같이 서식화됩니다.

```javascript
const result = 10 + 5 * 3;
```

Prettier는 연산자 우선순위를 고려하여 코드를 서식화하므로, 코드를 작성할 때 일관된 연산자 우선순위를 적용할 필요가 없습니다.

Prettier를 사용하면 코드 서식을 일관되게 유지하고 연산자 우선순위에 대해 걱정할 필요가 없으며, 코드의 가독성을 향상시킬 수 있습니다.
