파이썬은 함수의 실행 흐름을 조절할 수 있는 다양한 구문을 제공합니다. 이를 통해 조건에 따라 다른 코드 블록을 실행하거나 반복 작업을 수행할 수 있습니다. 이번 글에서는 파이썬에서 제공하는 조건문과 반복문에 대해 알아보겠습니다.

## 조건문 (if 문)
조건문은 주어진 조건식이 참일 때에만 특정 코드 블록을 실행하는데 사용됩니다. 다음은 간단한 if문의 예제입니다.

```python
x = 10
if x > 5:
    print("x는 5보다 큽니다.")
```

위 코드에서는 x가 5보다 크기 때문에 "x는 5보다 큽니다."가 출력됩니다.

## 반복문 (for 문)
반복문은 주어진 조건에 따라 코드 블록을 여러 번 실행하는데 사용됩니다. 다음은 간단한 for문의 예제입니다.

```python
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
```

위 코드에서는 fruits 리스트의 각 항목을 순회하면서 각각을 출력합니다.

## 기타 제어 구문
파이썬에서는 이외에도 다양한 제어 구문을 제공합니다. 예를 들어, while문을 사용하여 조건이 참일 때 코드를 반복 실행할 수 있고, break와 continue를 사용하여 반복문을 중단하거나 건너뛸 수도 있습니다.

함수의 실행 흐름을 제어하는 구문은 파이썬 프로그래밍에서 중요한 요소이며, 적재적소에 활용하여 코드를 효율적으로 작성할 수 있습니다.

이상으로 파이썬에서 함수의 실행 흐름을 제어하는 방법에 대해 알아보았습니다. 감사합니다.
