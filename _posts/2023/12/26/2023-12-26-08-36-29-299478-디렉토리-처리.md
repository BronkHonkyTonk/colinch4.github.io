### 디렉토리 목록 가져오기
쉘 스크립트로 디렉토리 내의 파일 목록을 가져오는 방법은 다양합니다. `ls` 명령어를 사용하여 디렉토리 내의 파일과 디렉토리 목록을 확인할 수 있습니다. 특정 확장자를 가진 파일만 가져오거나, 더 복잡한 필터 기능을 사용할 수도 있습니다. `find` 명령어를 이용하여 특정 조건에 맞는 파일을 검색하는 것도 가능합니다.

```shell
# 모든 파일과 디렉토리 목록 보기
ls

# 특정 확장자를 가진 파일 목록 보기
ls *.txt

# 특정 디렉토리 내의 모든 파일 목록 보기
find /path/to/directory
```

### 디렉토리 생성 및 삭제
디렉토리를 생성하거나 삭제하는 것은 쉘 스크립트에서 매우 흔한 작업 중 하나입니다. `mkdir` 명령어를 사용하여 새로운 디렉토리를 만들고, `rmdir` 명령어를 사용하여 비어있는 디렉토리를 삭제할 수 있습니다.

```shell
# 새로운 디렉토리 생성
mkdir new_directory

# 디렉토리 삭제
rmdir empty_directory
```

### 디렉토리 이동
디렉토리를 이동하는 것은 매우 중요하며, 쉘 스크립트에서 빈번하게 사용됩니다. `cd` 명령어를 사용하여 디렉토리를 이동합니다.

```shell
cd /path/to/directory
```

이를 통해여 쉘 스크립트를 사용하여 파일과 디렉토리를 효율적으로 처리할 수 있습니다. 여러분도 쉘 스크립트를 사용하여 디렉토리 처리를 원활하게 할 수 있을 것입니다.
