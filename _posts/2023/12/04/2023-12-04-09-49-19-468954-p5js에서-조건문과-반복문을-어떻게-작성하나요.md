P5.js는 JavaScript 기반의 프로그래밍 라이브러리로, 그래픽과 애니메이션을 쉽게 구현할 수 있도록 도와줍니다. 조건문과 반복문은 P5.js에서도 일반적인 JavaScript와 동일한 방식으로 작성할 수 있습니다.

## 조건문 (if 문)

조건문은 주어진 조건에 따라 다른 동작을 수행하는데 사용됩니다. 예를 들어, 만약 특정 조건이 참이라면 어떤 코드를 실행하고, 그렇지 않다면 다른 코드를 실행할 때 조건문을 사용합니다. P5.js에서는 다음과 같이 조건문을 작성할 수 있습니다:

```javascript
if (조건) {
    // 조건이 참일 때 실행할 코드
} else {
    // 조건이 거짓일 때 실행할 코드
}
```

아래는 조건문의 예시입니다:

```javascript
let num = 10;

if (num > 5) {
    console.log("num은 5보다 큽니다.");
} else {
    console.log("num은 5보다 작거나 같습니다.");
}
```

## 반복문 (for 문)

반복문은 동일한 동작을 반복적으로 수행할 때 사용됩니다. 예를 들어, 배열의 요소를 모두 출력하거나 특정 횟수만큼 반복적으로 어떤 코드를 실행할 때 반복문을 사용합니다. P5.js에서는 다음과 같이 반복문을 작성할 수 있습니다:

```javascript
for (초기값; 조건; 반복식) {
    // 반복적으로 실행할 코드
}
```

아래는 반복문의 예시입니다:

```javascript
for (let i = 0; i < 10; i++) {
    console.log(i);
}
```

위의 반복문은 0부터 9까지의 숫자를 출력합니다.

조건문과 반복문은 P5.js에서도 JavaScript의 기본 문법과 동일하게 작성할 수 있습니다. P5.js에서 추가로 제공하는 그래픽 관련 함수와 함께 사용하면 다양한 그래픽 및 애니메이션 효과를 구현할 수 있습니다.
