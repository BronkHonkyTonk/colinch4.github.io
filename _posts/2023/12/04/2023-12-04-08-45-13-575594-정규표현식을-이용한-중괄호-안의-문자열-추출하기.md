정규표현식을 사용하면 특정 패턴을 가진 문자열을 쉽게 추출할 수 있습니다. 이번 예제에서는 Python에서 정규표현식을 사용하여 중괄호 `{}` 안에 있는 문자열을 추출하는 방법을 알아보겠습니다.

```python
import re

# 추출할 문자열
text = "이것은 중괄호 안에 있는 {문자열}을 추출하는 예제입니다."

# 중괄호 안의 문자열을 추출하는 정규표현식 패턴
pattern = r"{(.*?)}"

# 정규표현식을 이용하여 문자열 추출
result = re.findall(pattern, text)

# 결과 출력
print(result)
```

위 코드를 실행하면 다음과 같은 결과가 출력됩니다.

```
['문자열']
```

위의 예제에서는 `re` 모듈의 `findall()` 함수를 사용하여 중괄호 안에 있는 문자열을 추출하였습니다. 정규표현식 패턴 `{(.*?)}`은 중괄호 안에 있는 어떤 문자열이라도 추출하도록 설정되어 있습니다.

정규표현식을 사용하여 문자열을 추출할 때는 패턴을 적절히 설정해야 합니다. 예제에서는 중괄호 `{}` 안에 있는 문자열을 추출하였지만, 다른 패턴을 사용하여 다른 문자열을 추출하는 것도 가능합니다.

이와 같이 정규표현식을 사용하여 문자열을 추출하면 원하는 패턴을 간편하게 찾을 수 있습니다.
