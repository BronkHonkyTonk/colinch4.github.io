현재 기준으로 Riverpod 패키지의 최신 버전은 1.0.1 입니다. Riverpod는 Flutter 앱에서 상태 관리를 용이하게 해주는 패키지로, Provider 패턴을 기반으로 한다는 특징을 가지고 있습니다. 최신 버전의 Riverpod 패키지를 사용하면 Flutter 앱의 상태 관리를 더욱 효율적으로 처리할 수 있습니다. 

Riverpod 패키지를 사용하기 위해서는 pubspec.yaml 파일에 다음과 같이 의존성을 추가해야 합니다:

```dart
dependencies:
  riverpod: ^1.0.1
```

위와 같이 의존성을 추가하고 나면  `flutter pub get` 명령어를 실행하여 패키지를 다운로드하고 사용할 수 있습니다.

자세한 내용은 Riverpod의 공식 문서 또는 GitHub 페이지를 참조하시기 바랍니다:

- [Riverpod 공식 문서](https://riverpod.dev/)
- [Riverpod GitHub 페이지](https://github.com/rrousselGit/river_pod)

Riverpod 패키지는 Flutter 개발에서 상태 관리를 용이하게 해주는 강력한 도구이니 새로운 버전을 확인하고 업데이트하는 것을 권장합니다!
