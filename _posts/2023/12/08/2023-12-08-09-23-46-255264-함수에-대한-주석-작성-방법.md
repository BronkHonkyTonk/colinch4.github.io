# C++ 함수에 대한 주석 작성 방법

C++에서 함수를 작성할 때 주석은 코드를 이해하고 유지보수하는 데 매우 중요합니다. 함수 주석은 함수가 하는 일, 매개변수의 역할, 반환 값 등을 설명하여 다른 프로그래머가 코드를 이해할 수 있도록 도와줍니다.

## 함수 주석 작성 규칙

함수 주석은 아래와 같은 내용을 포함해야 합니다:

1. **함수 설명**: 함수가 수행하는 작업에 대한 간결한 설명
2. **매개변수 설명**: 각 매개변수가 의도된 목적과 역할에 대한 설명
3. **반환 값 설명**: 함수가 반환하는 값에 대한 설명
4. **예외나 오류 처리**: 함수가 발생시킬 수 있는 예외나 오류에 대한 설명

아래는 함수 주석을 작성하는 간단한 예시입니다:

```cpp
/*
 * 함수 설명: 두 정수를 더하는 함수
 * 매개변수:
 *   a - 더할 첫 번째 정수
 *   b - 더할 두 번째 정수
 * 반환 값: a와 b의 합
*/
int add(int a, int b) {
    return a + b;
}
```

이렇게 작성된 함수 주석은 다른 프로그래머가 함수를 사용할 때 함수의 동작을 빠르게 이해할 수 있도록 도와줍니다.
