조건문을 작성할 때 주석을 추가하여 코드를 설명하고 가독성을 높일 수 있습니다. 이 포스트에서는 TypeScript에서 주석을 어떻게 작성하는지에 대해 알아보겠습니다.

## TypeScript에서의 주석 작성 방법

TypeScript에서 조건문의 주석을 작성하는 방법은 주석을 통해 조건의 목적을 설명하고 가독성을 높이는 것입니다. 아래는 TypeScript에서 if문과 함께 주석을 작성하는 예제입니다.

```typescript
// 사용자의 나이에 따라 인사말을 다르게 설정
let age: number = 25;

if (age < 18) {
    console.log('안녕하세요! 아이셔서 반가워요!'); // 18세 미만
} else {
    console.log('안녕하세요! 어서오세요!'); // 18세 이상
}
```

위의 예제에서는 주석을 사용하여 각 조건이 어떤 상황에서 실행되는지 설명하고 있습니다.

## 주석 작성 시 유의할 점

조건문의 주석을 작성할 때에는 다음과 같은 점에 주의해야 합니다.
- **명료한 설명**: 주석은 명확하고 간결하게 작성하여 코드의 의도를 명확히 전달해야 합니다.
- **코드 업데이트**: 코드를 업데이트할 때 주석도 함께 업데이트하여 일관성을 유지해야 합니다.
- **불필요한 주석**: 코드 자체가 명확한 경우, 불필요한 주석은 추가하지 않아야 합니다.

조건문의 주석은 코드를 이해하기 쉽게 만들어주고 유지보수를 용이하게 만들어줍니다. 올바르고 명확한 주석 작성은 프로젝트의 코드베이스 관리에 도움이 됩니다.
