코틀린에서는 다양한 콜렉션 타입을 활용하여 데이터를 다룰 수 있습니다. 콜렉션 타입은 여러 데이터를 모아놓은 것을 의미하며, 리스트나 세트와 같은 다양한 형태로 사용할 수 있습니다. 

콜렉션 타입을 활용하여, 데이터 개수를 셀 때는 `size` 속성을 사용하면 됩니다. 예를 들어, 리스트에 포함된 데이터의 개수를 세기 위해서는 다음과 같이 할 수 있습니다.

```kotlin
val myList = listOf(1, 2, 3, 4, 5)
val size = myList.size
println("리스트의 데이터 개수는 $size 입니다.")
```

이와 같이, 리스트의 `size` 속성을 이용하여 리스트에 포함된 데이터 개수를 쉽게 얻을 수 있습니다. 

세트나 맵과 같은 다른 콜렉션 타입에서도 비슷한 방식으로 데이터 개수를 얻을 수 있습니다. 이처럼, 콜렉션 타입을 활용하여 데이터 개수를 셀 때는 각 콜렉션 타입에 해당하는 방식으로 적용하면 됩니다.

## 총정리

콜렉션 타입을 활용하여 데이터 개수를 셀 때는 다음과 같이 간단하게 할 수 있습니다.

- 리스트: `list.size`
- 세트: `set.size`
- 맵: `map.size`

위와 같이, 각 콜렉션 타입에 해당하는 `size` 속성을 이용하여 데이터 개수를 얻을 수 있습니다. 이를 통해 콜렉션 타입을 효과적으로 활용하여 다양한 데이터를 다룰 수 있습니다.
