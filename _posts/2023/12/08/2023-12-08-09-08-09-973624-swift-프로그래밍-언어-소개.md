Swift는 Apple에서 개발한 **프로그래밍 언어**로, iOS, macOS, watchOS, tvOS 애플리케이션을 개발하기 위해 사용됩니다. Swift는 Objective-C의 대안으로 개발되었으며, **안전하고 빠른** 언어로 설계되어 있습니다. 

## Swift의 특징
1. **안전성**: Swift는 안전성을 지향하며, 컴파일 시 타입 안정성을 검증합니다.
2. **간결성**: 코드를 더 간결하게 작성할 수 있도록 설계되었으며, 다양한 기능을 제공합니다.
3. **성능**: C 및 Objective-C와 유사한 성능을 제공하며, 높은 실행 속도를 보장합니다.

## Swift의 문법 예시
```swift
// 변수 및 상수 선언
var greeting: String = "Hello, Swift!"
let maxLoginAttempts: Int = 3

// 기본 데이터 타입
var age: Int = 30
var pi: Double = 3.14159
var isActive: Bool = true

// 반복문과 조건문
for index in 1...5 {
    print("Index: \(index)")
}

if isActive {
    print("User is active")
} else {
    print("User is inactive")
}
```

Swift는 문법이 명확하고 간결하여 새로운 프로그래머가 빠르게 배우고 개발할 수 있도록 설계되었습니다.

Swift에 대한 자세한 정보는 [Swift 공식 문서](https://swift.org/documentation/)에서 확인할 수 있습니다.
