변수와 상수는 프로그래밍에서 데이터를 저장하고 나중에 사용하기 위해 사용됩니다. **변수**는 값이 변경될 수 있는 반면 **상수**는 한 번 정의되면 변경할 수 없습니다.

## 변수의 데이터 범위

변수는 주어진 범위 내에서 사용됩니다. 이 범위는 변수가 정의되고 사용될 수 있는 코드 블록을 나타냅니다. 코드 블록은 주로 중괄호 `{}`로 표시되는 함수나 조건문과 같은 것들을 의미합니다.

예를 들어, 다음은 변수 `age`가 `main` 함수 내에서 사용되는 경우입니다.

```kotlin
fun main() {
    var age = 25
    // 여기서 age 변수를 사용
}
// 여기서는 age 변수에 접근할 수 없음
```

위의 예제에서 `age` 변수는 `main` 함수 내에서만 사용될 수 있습니다. 함수 내의 중괄호를 벗어나면 `age` 변수에 접근할 수 없습니다.

## 상수의 데이터 범위

마찬가지로, **상수**도 정의된 범위 내에서만 사용됩니다. Kotlin에서 `val` 키워드로 상수를 정의할 수 있습니다.

```kotlin
fun main() {
    val pi = 3.14
    // 여기서 pi 상수를 사용
}
// 여기서는 pi 상수에 접근할 수 없음
```

위의 예제에서 `pi` 상수는 `main` 함수 내에서만 사용될 수 있습니다.

## 종합

변수와 상수는 코드 블록 내에서만 사용될 수 있는데, 이는 코드의 유지보수성을 더 높이고 예기치 못한 오류를 방지하는 데 도움이 됩니다. 데이터 범위를 잘 이해하고 변수와 상수를 올바르게 사용하는 것이 중요합니다.

이상으로 Kotlin에서 변수와 상수의 데이터 범위에 대한 내용을 살펴봤습니다.
