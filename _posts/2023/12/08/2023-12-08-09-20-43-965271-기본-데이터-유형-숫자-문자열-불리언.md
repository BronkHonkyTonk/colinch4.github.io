자바스크립트는 여러 가지 데이터 유형을 처리하고, 각각의 유형은 서로 다른 특징을 가지고 있습니다. 이러한 데이터 유형에는 **숫자**, **문자열**, **불리언** 등이 포함됩니다. 이번 글에서는 자바스크립트의 각각의 기본 데이터 유형에 대해 알아보겠습니다.

## 숫자 (Number)

숫자 데이터 유형은 숫자를 나타내는 데 사용됩니다. 이들은 정수뿐만 아니라 소수점 이하의 수도 포함하며, 산술 연산 및 수학적 계산 등에서 사용됩니다.

```javascript
let age = 30;
let pi = 3.14;
```

## 문자열 (String)

문자열 데이터 유형은 문자의 시퀀스를 나타냅니다. 작은따옴표('')나 큰따옴표("")로 감싸서 표현합니다.

```javascript
let name = 'John Doe';
let message = "안녕하세요!";
```

## 불리언 (Boolean)

불리언 데이터 유형은 논리적인 값을 나타내며, 참(true) 또는 거짓(false) 중 하나의 값을 가집니다.

```javascript
let isLoggedin = true;
let isWorking = false;
```

자바스크립트의 기본 데이터 유형은 프로그래밍에서 매우 중요한 역할을 하며, 코드를 작성하고 실행하는 데 있어서 항상 고려해야 합니다. 기본 데이터 유형을 활용하여 다양한 연산과 표현을 수행할 수 있기 때문에, 데이터 유형에 대한 이해는 프로그래밍 능력 향상에 도움이 됩니다.
