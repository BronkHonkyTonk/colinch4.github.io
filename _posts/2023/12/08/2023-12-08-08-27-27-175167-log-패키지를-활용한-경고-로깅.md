Go 언어에서 log 패키지는 간단한 로그 기능을 제공합니다. 로깅을 통해 애플리케이션의 상태 및 이벤트를 기록할 수 있습니다. 이 글에서는 log 패키지를 사용하여 경고 수준의 로깅을 하는 방법에 대해 알아보겠습니다.

## log 패키지의 Import

먼저, log 패키지를 사용하기 위해 아래와 같이 import문을 작성합니다.

```go
import "log"
```

## 경고 수준의 로깅

log 패키지에서는 Print, Panic, Fatal을 포함한 여러 메서드를 제공합니다. 이 중에서 경고 수준의 로깅을 하려면 `log.Println`을 사용합니다. 아래는 경고 메시지를 로깅하는 예제 코드입니다.

```go
log.Println("이것은 경고 메시지입니다")
```

이를 실행하면 표준 출력에 경고 메시지가 기록됩니다.

## 로깅 포맷 설정

log 패키지는 로그의 출력 형식을 변경할 수 있는 기능도 제공합니다. 예를 들어, 타임스탬프를 포함하려면 아래와 같이 설정할 수 있습니다.

```go
log.SetFlags(log.LstdFlags)
log.Println("타임스탬프를 포함한 경고 메시지")
```

## 결론

이렇게 log 패키지를 활용하여 경고 수준의 로깅을 할 수 있습니다. 로그를 적절히 활용하여 애플리케이션의 상태를 모니터링하고 디버깅하는 데 도움이 됩니다.

더 많은 log 패키지의 기능 및 사용법에 대해서는 [공식 문서](https://golang.org/pkg/log/)를 참고하시기 바랍니다.
