본 포스트에서는 Go 프로그래밍 언어에서의 형식화된 입출력과 다양한 데이터 타입에 대해 살펴보겠습니다.

## 형식화된 입출력

Go에서는 `fmt` 패키지를 이용하여 형식화된 입출력을 다룰 수 있습니다. 예를 들어, 정수를 출력하는 방법은 다음과 같습니다.

```go
package main

import "fmt"

func main() {
    var num int = 42
    fmt.Printf("정수: %d\n", num)
}
```

위 코드에서 `Printf` 함수는 첫번째 매개변수로 형식 지정자 `%d`를 받아들여서 변수 `num`을 해당 형식으로 출력합니다.

## 다양한 데이터 타입

Go 언어에는 다양한 데이터 타입이 있습니다. 가장 일반적으로 사용되는 몇 가지 데이터 타입은 다음과 같습니다.

- **정수형**: `int`, `int8`, `int16`, `int32`, `int64` 등
- **부동소수점형**: `float32`, `float64`
- **문자열**: `string`
- **부울형**: `bool`

다양한 데이터 타입을 사용함으로써 다양한 종류의 데이터를 다룰 수 있습니다.

이상으로 Go에서의 형식화된 입출력과 다양한 데이터 타입에 대해 알아보았습니다.
