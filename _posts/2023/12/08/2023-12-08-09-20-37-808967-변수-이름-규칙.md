## 의미 있는 이름 선택

변수 이름은 해당 변수가 하는 일을 설명해야 합니다. *의미 있는 이름을 선택*하여 코드의 가독성을 높일 수 있습니다.

좋은 예시:
```javascript
let userName = "John";
```

나쁜 예시:
```javascript
let x = "John";
```

의미 있는 이름을 선택하면 코드를 읽거나 수정할 때 더욱 쉬워집니다.

## 유효한 변수 이름

자바스크립트 변수 이름은 *유효한 식별자*여야 합니다. 유효한 변수 이름을 선택하려면 몇 가지 규칙을 따라야 합니다.

- 이름은 문자, 숫자, 밑줄(_) 또는 달러 기호($)로 시작해야 합니다.
- 이름은 대소문자를 구분합니다.
- 예약어(예: let, if, return)는 변수 이름으로 사용할 수 없습니다.

좋은 예시:
```javascript
let user_name = "John";
```

나쁜 예시:
```javascript
let 1user = "John"; // 숫자로 시작하여 유효한 변수 이름이 아님
```

## 코딩 규칙 준수

코드를 작성할 때 선택한 *코딩 규칙*을 따르는 것이 중요합니다. 일관된 스타일을 유지하면 여러 명의 개발자가 함께 작업할 때 혼란을 줄일 수 있습니다.

- 변수 이름에는 *캐멀 케이스*나 *스네이크 케이스*를 사용하는 것이 일반적입니다.

좋은 예시:
```javascript
let fullName = "John Doe"; // 캐멀 케이스
let user_name = "johndoe"; // 스네이크 케이스
```

의미 있는 이름을 선택하고 유효한 변수 이름과 코딩 규칙을 준수하는 것은 자바스크립트 코드를 더욱 가독성 있게 만들어줍니다.

## 마무리

의미 있는 이름 선택, 유효한 변수 이름, 코딩 규칙 준수는 자바스크립트 코드를 작성할 때 고려해야 하는 중요한 요소입니다. 올바른 변수 이름을 선택하여 코드를 더욱 이해하기 쉽게 만들어 보세요.
