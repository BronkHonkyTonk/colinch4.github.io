코드 자동 포맷팅은 프로그래머가 코드 작성 중 일관된 서식을 유지할 수 있도록 도와주는 중요한 도구입니다. IntelliJ IDEA에서 Kotlin 코드의 자동 포맷팅을 설정하는 방법에 대해 알아보겠습니다.

## IntelliJ IDEA 프로젝트 열기

먼저 IntelliJ IDEA를 열고 Kotlin 프로젝트를 선택합니다.

## 코드 스타일 설정

1. **Preferences(또는 Settings) 열기**: IntelliJ IDEA 상단 메뉴에서 `IntelliJ IDEA` > `Preferences` (또는 `Settings`)를 클릭합니다.
2. **Editor 메뉴 선택**: 왼쪽 패널에서 `Editor`를 선택합니다.
3. **Code Style 설정**: `Editor` 메뉴 하위에 있는 `Code Style`을 클릭합니다.
4. **Kotlin 선택**: `Code Style`에서 `Kotlin`을 선택합니다.
5. **Set from... 버튼 클릭**: 오른쪽 상단에 있는 `Set from...` 버튼을 클릭합니다.
6. **Predefined style 선택**: 원하는 코드 스타일을 선택합니다. 예를 들어, `Kotlin style guide`를 선택할 수 있습니다.
7. **OK 버튼 클릭**: 설정을 확인하고 적용하기 위해 `OK` 버튼을 클릭합니다.

이제 IntelliJ IDEA는 Kotlin 코드를 해당 스타일에 맞게 자동 포맷팅합니다.

## 자동 포맷팅 기능 사용하기

1. **코드 작성**: IntelliJ IDEA에서 Kotlin 코드를 작성합니다.
2. **자동 포맷팅 실행**: 작성한 코드에서 <kbd>Cmd</kbd>+<kbd>Alt</kbd>+<kbd>L</kbd> (Mac) 또는 <kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>L</kbd> (Windows/Linux)을 눌러 자동 포맷팅을 실행합니다.

코드 자동 포맷팅 설정을 통해 IntelliJ IDEA에서 일관된 코드 스타일을 유지할 수 있으며, 코드 작성 시간을 단축할 수 있습니다.

자세한 내용은 [IntelliJ IDEA 공식 문서](https://www.jetbrains.com/idea/)를 참조하십시오.
