변수에 저장된 값을 확인하고자 할 때, 주로 **반복문**을 사용합니다. 자바스크립트에서는 여러 가지 반복문을 활용하여 변수 값을 확인할 수 있습니다. 이번 글에서는 `for` 반복문을 사용하여 변수 값을 확인하는 방법에 대해 알아보겠습니다.

## `for` 반복문을 사용하여 변수 값 확인하기

다음은 변수에 저장된 값을 `for` 반복문을 이용하여 확인하는 예제 코드입니다.

```javascript
let numbers = [1, 2, 3, 4, 5];

for (let i = 0; i < numbers.length; i++) {
  console.log(numbers[i]);
}
```

위 코드에서는 `numbers`라는 배열에 저장된 값들을 `for` 반복문을 통해 하나씩 확인하고 출력합니다.

## 콘솔 출력 결과

위 예제 코드를 실행하면 다음과 같이 콘솔에 값들이 출력됩니다.

```
1
2
3
4
5
```

이제 **반복문을 사용하여 변수 값을 확인하는 방법**에 대해 간단히 소개해 보았습니다.

## 결론

변수에 저장된 값을 확인하기 위해서는 반복문을 활용할 수 있습니다. 위 예제처럼 `for` 반복문을 사용하면 변수의 값들을 순차적으로 확인할 수 있습니다. 이를 통해 변수에 저장된 값들을 효과적으로 다룰 수 있습니다.

반복문을 활용한 변수 값 확인은 프로그래밍에서 매우 유용하며, 다양한 상황에 활용할 수 있습니다.
