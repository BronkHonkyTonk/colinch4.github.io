자바에서는 **상속(inheritance)**을 사용하여 다른 클래스의 기능을 확장할 수 있습니다. 상속을 통해 새로운 클래스는 기존 클래스의 모든 기능을 상속받고, 이를 기반으로 새로운 기능을 추가하거나 기존 기능을 변경할 수 있습니다.

다음은 상속을 통한 다른 클래스의 기능 확장 방법에 대한 간단한 예제입니다.

```java
// 부모 클래스
class 부모클래스 {
    void 기존기능() {
        // 기존 기능의 구현
    }
}

// 자식 클래스
class 자식클래스 extends 부모클래스 {
    void 새로운기능() {
        // 새로운 기능의 구현
    }
}
```

위 예제에서 `자식클래스`는 `부모클래스`를 상속받았기 때문에 `부모클래스`의 `기존기능`을 사용할 수 있으며, 추가적으로 `새로운기능`을 정의할 수 있습니다.

이와 같이 상속을 통해 클래스를 확장함으로써 코드의 재사용성을 높이고, 유지보수를 용이하게 할 수 있습니다.

더 자세한 내용은 [자바 공식 문서](https://docs.oracle.com/javase/tutorial/java/IandI/subclasses.html)를 참조할 수 있습니다.
