## JAR 파일
JAR(Java Archive) 파일은 Java 클래스, 메타데이터 및 리소스를 포함하는 압축된 파일 형식입니다. JAR 파일의 명명 규칙은 보통 **.jar** 확장자를 사용합니다. 파일 이름에는 소문자만 사용하고, 단어 사이에는 하이픈(-)을 사용하는 것이 관례입니다. 예를 들어, **my-library.jar**와 같이 명명할 수 있습니다.

## WAR 파일
WAR(Web Application Archive) 파일은 Java 웹 애플리케이션을 패키징하는 데 사용됩니다. WAR 파일의 명명 규칙은 **.war** 확장자를 사용하며, JAR 파일과 마찬가지로 소문자와 하이픈을 사용하는 것이 관례입니다. 예를 들어, **my-webapp.war**와 같이 명명할 수 있습니다.

## EAR 파일
EAR(Enterprise Archive) 파일은 Java EE(EJB, Servlet, 등) 애플리케이션을 패키징하는 데 사용됩니다. EAR 파일의 명명 규칙은 **.ear** 확장자를 사용하며, 파일 이름에 소문자와 하이픈을 사용하는 것이 일반적인 관례입니다. 예를 들어, **my-enterpriseapp.ear**와 같이 명명할 수 있습니다.

Java의 JAR, WAR, EAR 파일의 명명 규칙은 일반적인 관례를 따르지만, 각 조직이나 프로젝트에서의 표준을 준수하는 것이 좋습니다.
