아래는 변수에 조건식을 적용하는 간단한 예제입니다.

```javascript
let 성인여부 = true;

if (성인여부 === true) {
  console.log('당신은 성인입니다.');
} else {
  console.log('당신은 미성년자입니다.');
}
```

이 예제에서, `if`문을 사용하여 `성인여부` 변수의 값이 `true`인지 확인합니다. 만약 조건이 참이면 '당신은 성인입니다.'를 출력하고, 그렇지 않으면 '당신은 미성년자입니다.'를 출력합니다.

이제 변수에 조건식을 적용하여 값을 설정하는 방법을 알게 되었으며, 이를 활용하여 프로그래밍을 할 수 있을 것입니다.
