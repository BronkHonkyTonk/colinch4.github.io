## 2. 실전 프로젝트

실전 프로젝트는 이론을 바탕으로 실제 문제를 해결하며 교육을 받는 데 효과적인 방법입니다. SQL 데이터베이스 결함 관리에 대한 프로젝트를 통해 학습자들은 실제 시나리오를 경험하고, 문제를 직접 해결하는 경험을 얻을 수 있습니다.

```sql
CREATE TABLE employee (
  id INT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  department VARCHAR(100)
);
```

위와 같은 예시를 활용하여 실제 데이터베이스를 구축하고, 데이터를 관리해보는 프로젝트를 통해 학습 효과를 높일 수 있습니다.
