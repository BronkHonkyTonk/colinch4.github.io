## 보고서

### 1. 결함 추적 보고서
결함 추적 보고서는 데이터베이스에서 발생한 결함을 추적하고 이를 기록하는 데 사용됩니다. 보통 결함의 상태, 우선 순위, 제보자, 담당자, 해결 일자 등을 포함합니다.

**예시 SQL 쿼리:**
```sql
SELECT * FROM 결함_테이블 WHERE 상태 = '미해결';
```

### 2. 성능 문제 보고서
데이터베이스 성능 문제에 대한 보고서는 성능 저하를 유발하는 쿼리, 인덱스 부재, 부하가 높은 테이블 등을 식별하여 제시합니다.

**예시 SQL 쿼리:**
```sql
SELECT 쿼리, 실행_시간 FROM 성능_로그 WHERE 실행_시간 > 1000;
```

### 2. 성능 대시보드
성능 대시보드는 데이터베이스의 성능 지표, 사용률, 트래픽 등을 모니터링하여 성능 문제를 시각적으로 확인할 수 있도록 도와줍니다.
