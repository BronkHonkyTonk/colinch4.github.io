JavaScript에서 변수를 사용하여 계산식을 작성하고 사용하는 것은 매우 간단합니다. 변수는 값을 저장하고 나중에 사용할 수 있게 해주며, 이를 통해 계산식을 더 효과적으로 작성할 수 있습니다.

## 변수 선언하기

먼저, 변수를 선언하고 값을 할당해보겠습니다. 다음은 변수 a와 b를 선언하고, 각각 5와 10을 할당하는 예제입니다.

```javascript
let a = 5;
let b = 10;
```

## 변수를 이용한 계산식 작성하기

이제, 변수를 사용하여 간단한 계산식을 만들어 보겠습니다. 예를 들어, 변수 a와 b를 더한 값을 다른 변수에 할당하고 출력해보겠습니다.

```javascript
let sum = a + b;
console.log(sum); // 15
```

또는, 변수를 사용하여 조금 더 복잡한 계산식을 작성할 수도 있습니다. 다음은 변수를 사용하여 곱셈과 나눗셈을 수행하는 예제입니다.

```javascript
let product = a * b;
let division = b / a;
console.log(product); // 50
console.log(division); // 2
```

위와 같이, JavaScript에서 변수를 사용하여 간단하거나 복잡한 계산식을 작성할 수 있습니다. 이를 통해 코드를 더 읽기 쉽고 유지보수하기 쉽게 만들 수 있습니다.

## 결론

변수를 사용하여 JavaScript에서 계산식을 작성하는 방법은 매우 간단합니다. 변수를 적절히 활용하면 코드를 더 유연하고 가독성 있게 작성할 수 있습니다.
