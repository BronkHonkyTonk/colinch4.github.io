딕셔너리는 여러 개의 키-값 쌍을 저장할 수 있는 파이썬의 자료구조입니다. 때로는 딕셔너리의 키와 값들을 튜플로 변환해야 하는 경우가 있습니다. 이번 포스트에서는 파이썬의 내장 함수를 사용하여 딕셔너리의 키와 값들을 튜플로 변환하는 방법을 알아보겠습니다.

## 딕셔너리 정의하기
우선 예시로 사용할 딕셔너리를 정의해보겠습니다.

```python
sample_dict = {'A': 1, 'B': 2, 'C': 3, 'D': 4}
```

## 딕셔너리의 키와 값들을 튜플로 변환하기
파이썬의 내장 함수 `items()`를 사용하면 딕셔너리의 키와 값들을 튜플로 변환할 수 있습니다.

```python
tuple_pairs = list(sample_dict.items())
print(tuple_pairs)
```

위의 코드를 실행하면 다음과 같은 결과가 출력됩니다.

```
[('A', 1), ('B', 2), ('C', 3), ('D', 4)]
```

이렇게 딕셔너리의 `items()` 함수를 사용하여 키-값 쌍을 튜플로 변환할 수 있습니다.

딕셔너리의 키와 값들을 튜플로 변환하는 방법에 대해 간단히 알아보았습니다. 파이썬의 내장 함수를 적절히 활용하면 다양한 데이터 구조 간 변환이 가능하므로, 유연한 데이터 처리를 위해 활용해보시기 바랍니다.
