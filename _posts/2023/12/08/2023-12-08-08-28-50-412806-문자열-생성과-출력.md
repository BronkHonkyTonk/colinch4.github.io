Python은 강력하고 유연한 문자열 처리 기능을 제공합니다. 이번 글에서는 Python에서 문자열을 생성하고 출력하는 방법을 알아보겠습니다.

## 문자열 생성

문자열을 생성하는 가장 간단한 방법은 작은 따옴표(')나 큰 따옴표(")로 감싸는 것입니다. 두 가지 방법 모두 동일하게 작동합니다.

```python
str1 = 'Hello, World!'
str2 = "Hello, Python!"
```

또한 여러 줄로 이루어진 문자열을 생성할 때는 세 개의 따옴표를 사용할 수 있습니다.

```python
multiline_str = '''This 
is a 
multiline 
string'''
```

## 문자열 출력

Python에서 문자열을 출력하는 방법은 간단합니다. `print` 함수를 사용하여 문자열을 화면에 출력할 수 있습니다.

```python
print("Hello, Python!")
```

위 코드를 실행하면 "Hello, Python!"이라는 문자열이 화면에 출력됩니다.

이상으로 Python에서 문자열 생성과 출력에 대해 알아보았습니다.
