변수는 프로그래밍에서 데이터를 저장하는 데 사용됩니다. **Kotlin**은 여러 종류의 데이터를 저장하고 다루기 위해 다양한 데이터 타입을 제공합니다. 다음은 **Kotlin**에서 사용되는 일반적인 변수의 데이터 타입 종류입니다.

## 1. 기본 데이터 타입
   1. **정수 타입**: `Byte`, `Short`, `Int`, `Long`
   2. **부동 소수점 타입**: `Float`, `Double`
   3. **문자 타입**: `Char`
   4. **불리언 타입**: `Boolean`
   
## 2. 참조 데이터 타입
   1. **문자열**: `String`
   2. **배열**: `Array`

## 3. 사용자 정의 데이터 타입
   1. **Enum(열거형)**
   2. **Class(클래스)**

각 데이터 타입은 특정 종류의 데이터를 저장하고 특정한 연산을 수행하기 위해 사용됩니다. **Kotlin**에서는 변수가 선언될 때 타입을 명시적으로 지정하거나, 변수의 값에 따라 타입을 추론할 수 있습니다.

변수를 사용할 때에는 해당 데이터 타입을 잘 고려하여 적절하게 사용해야 합니다.
