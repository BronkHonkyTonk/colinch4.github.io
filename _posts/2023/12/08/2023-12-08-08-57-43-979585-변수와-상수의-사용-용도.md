변수와 상수는 프로그래밍 언어에서 데이터를 저장하고 조작하는 데 중요한 개념입니다. Kotlin에서는 `var` 키워드를 사용하여 변수를 선언하고 `val` 키워드를 사용하여 상수를 선언합니다. 이들의 사용 용도에 대해 알아봅시다.

## 변수의 사용 용도

변수는 프로그램 실행 중에 값이 변경될 수 있는 데이터를 저장하는 데 사용됩니다. 예를 들어, 사용자의 입력, 계산 결과 등이 이에 해당됩니다. 변수의 값은 프로그램 실행 중에 여러 번 변경될 수 있습니다.

```kotlin
var counter: Int = 0
counter = counter + 1
```

위의 예시에서 `counter` 변수는 0으로 초기화되고, 이후에 1이 추가됨으로써 값을 변경합니다.

## 상수의 사용 용도

반면에, 상수는 한 번 정해지면 그 이후에 변경되지 않는 값을 저장하는 데 사용됩니다. 예를 들어, 수학 상수나 프로그램 실행 중 변하지 않는 설정 값 등이 상수에 해당합니다. 상수를 사용하면 값이 실수로 변경되는 경우를 방지하고 프로그램의 안정성을 높일 수 있습니다.

```kotlin
val pi: Double = 3.14
```

위의 예시에서 `pi` 상수는 초기화된 이후에 값이 변경되지 않습니다.

따라서, 변수와 상수를 올바르게 사용하여 프로그램의 목적과 안정성을 달성하는 데 도움이 됩니다.

이상으로 Kotlin에서 변수와 상수의 사용 용도에 대해 알아보았습니다.
