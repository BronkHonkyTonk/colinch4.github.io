코틀린(Kotlin)에서 주석은 코드를 설명하고 의도를 명확히 전달하는 데 사용됩니다. 주석은 코드를 이해하고 유지보수하는 데 도움이 되며, 다른 개발자들이 코드를 이해할 수 있도록 도와줍니다.

## 1. 한 줄 주석

```kotlin
// 이것은 한 줄 주석입니다. 코드 뒤에 "//"를 추가하여 한 줄에 작성할 수 있습니다.
val name: String = "Kotlin"  // 변수 name에 "Kotlin"을 할당합니다.
```

## 2. 여러 줄 주석

```kotlin
/*
   이것은
   여러 줄 주석입니다. 여러 줄에 걸쳐 주석을 작성할 수 있습니다.
   여러 줄 주석은 "/"와 "*"로 시작하여 "*"와 "/"로 끝납니다.
*/
val version: Double = 1.5  // 변수 version에 1.5를 할당합니다.
```

## 3. 문서화 주석

```kotlin
/**
 * 이것은 문서화 주석입니다. 주석 내용을 설명하는 데 사용됩니다.
 * 클래스, 함수 또는 변수의 기능, 매개변수, 반환 값 등을 문서화할 수 있습니다.
 */
fun greet(name: String) {
    println("Hello, $name!")
}
```

주석을 추가하여 코드를 설명하고, 이해하기 쉽도록 만들어 보세요.
