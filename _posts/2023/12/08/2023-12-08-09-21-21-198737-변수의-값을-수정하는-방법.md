예를 들어, 다음은 JavaScript에서 변수 값을 수정하는 방법입니다.

```javascript
let number = 5;
console.log(number); // 5

number = 10; // 변수의 값을 수정
console.log(number); // 10
```

위의 예제에서는 `let` 키워드를 사용하여 `number`라는 변수를 선언하고, 초기값으로 5를 할당했습니다. 그런 다음에는 할당 연산자를 사용하여 변수 `number`의 값을 10으로 수정했습니다.

변수의 값을 수정하는 방법은 프로그래밍 언어마다 조금씩 다를 수 있지만, 일반적으로는 할당 연산자를 사용하여 변수에 새 값을 할당하는 방식으로 값을 수정할 수 있습니다.
