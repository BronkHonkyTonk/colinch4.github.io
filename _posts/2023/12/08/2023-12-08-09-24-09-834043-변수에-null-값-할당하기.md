Javascript에서 변수에 null 값을 할당하는 방법을 알아보겠습니다.

## 예제

```javascript
let myVariable = null;
```

위의 예제에서는 `myVariable`이라는 변수에 null 값을 할당했습니다.

## null이란?

*null*은 값이 없음을 나타내는 원시 값입니다. 

## 왜 null을 사용하는가?

변수에 값을 할당하지 않을 때, 또는 변수로부터 이전에 할당된 값을 지우고 싶을 때 null을 사용합니다.

## 요약

변수에 null 값을 할당하면, 해당 변수를 초기화하고 값을 없애는데에 사용할 수 있습니다.

위의 예제와 설명을 참고하여 Javascript에서 변수에 null 값을 할당하는 방법을 알아보았습니다.
