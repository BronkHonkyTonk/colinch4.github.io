### 1. 문자열과 바이트 길이

Go 언어에서는 `len()` 함수를 사용하여 문자열의 바이트 길이를 구할 수 있습니다. 아래 예제를 통해 확인해 보겠습니다.

```go
package main

import (
	"fmt"
)

func main() {
	str := "안녕하세요"
	byteLen := len(str)
	fmt.Printf("문자열의 바이트 길이: %d\n", byteLen)
}
```

위 예제에서 `len(str)`을 호출하여 문자열 `str`의 바이트 길이를 확인할 수 있습니다.

### 2. 주의 사항

주의해야 할 점은 Go 언어에서 문자열은 UTF-8 인코딩으로 처리되므로, 문자열의 바이트 길이가 항상 문자 개수와 일치하지는 않는다는 점입니다. UTF-8에서는 문자당 가변적인 바이트 수가 할당될 수 있기 때문에, 문자열의 바이트 길이를 구할 때에는 이 점을 고려해야 합니다.

### 3. 결론

이상으로, Go 언어에서 문자열의 바이트 길이를 구하는 방법에 대해 알아보았습니다. `len()` 함수를 사용하여 문자열의 바이트 길이를 쉽게 확인할 수 있으나, UTF-8 인코딩의 특성을 고려하여 실제 문자 개수와 일치하지 않을 수 있다는 점을 주의해야 합니다.
