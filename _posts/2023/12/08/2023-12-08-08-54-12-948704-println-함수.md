```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

위의 예제에서는 "Hello, World!"를 출력하고 새로운 줄로 이동합니다. 해당 함수의 문서화를 보려면 [여기](https://pkg.go.dev/fmt#Println)를 참고하세요.
