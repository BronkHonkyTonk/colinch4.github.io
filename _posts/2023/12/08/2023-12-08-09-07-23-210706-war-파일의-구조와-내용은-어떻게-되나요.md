## **WAR 파일 구조**

일반적으로 WAR 파일은 다음과 같은 구조를 가지고 있습니다:

- **META-INF**: MANIFEST.MF 파일을 포함한 메타데이터 디렉토리
- **WEB-INF**:
  - **web.xml**: 웹 애플리케이션의 설정 정보를 담고 있는 배포 설명자 파일
  - **classes**: Java 클래스 파일들
  - **lib**: 웹 애플리케이션에서 사용하는 라이브러리(JAR 파일)들

그 외에도 웹 애플리케이션에 따라 추가적인 리소스들이 포함될 수 있습니다.

## **WAR 파일 내용**

WAR 파일은 웹 애플리케이션을 배포하기 위한 모든 파일과 리소스를 포함합니다. 이는 HTML, CSS, JavaScript, JSP, 이미지 파일 등과 같은 웹 자원 뿐만 아니라 Java 클래스 파일, 라이브러리 파일, 설정 파일 등을 모두 압축한 형태로 포함하고 있습니다.

Java에서 WAR 파일의 구조와 내용을 자세히 살펴보기 위해서는 WAR 파일을 해제하여 내부의 파일과 디렉토리를 살펴볼 수 있습니다. 일반적으로 이를 확인하기 위해 WAR 파일을 압축 해제한 후에 파일 탐색기나 터미널을 통해 내용을 확인할 수 있습니다.

WAR 파일은 Java 웹 애플리케이션을 쉽게 배포하고 관리할 수 있도록 도와주는 중요한 파일 형식입니다.

이상으로 WAR 파일의 구조와 내용에 대해 간단히 설명해 드렸습니다. 추가 질문이 있으시다면 언제든지 물어보세요!
