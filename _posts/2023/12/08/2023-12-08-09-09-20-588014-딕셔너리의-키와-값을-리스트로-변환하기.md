딕셔너리는 키와 값으로 이뤄진 자료구조입니다. 때때로 딕셔너리의 키나 값들을 리스트로 변환해야 하는 경우가 있습니다. 이번 포스트에서는 파이썬에서 딕셔너리의 키와 값을 리스트로 변환하는 여러 가지 방법에 대해 알아보겠습니다.

## 딕셔너리의 키를 리스트로 변환하기

딕셔너리의 `keys()` 메서드를 사용하여 딕셔너리의 키들을 리스트로 변환할 수 있습니다.

```python
# 딕셔너리
my_dict = {'a': 1, 'b': 2, 'c': 3}

# 키를 리스트로 변환
keys_list = list(my_dict.keys())
print(keys_list)
# 출력: ['a', 'b', 'c']
```

## 딕셔너리의 값 리스트로 변환하기

마찬가지로, 딕셔너리의 `values()` 메서드를 사용하여 딕셔너리의 값들을 리스트로 변환할 수 있습니다.

```python
# 딕셔너리
my_dict = {'a': 1, 'b': 2, 'c': 3}

# 값들을 리스트로 변환
values_list = list(my_dict.values())
print(values_list)
# 출력: [1, 2, 3]
```

## 딕셔너리의 키-값 쌍을 리스트로 변환하기

또한, `items()` 메서드를 사용하여 딕셔너리의 키-값 쌍을 (키, 값) 형태의 튜플로 이뤄진 리스트로 변환할 수 있습니다.

```python
# 딕셔너리
my_dict = {'a': 1, 'b': 2, 'c': 3}

# 키-값 쌍을 리스트로 변환
items_list = list(my_dict.items())
print(items_list)
# 출력: [('a', 1), ('b', 2), ('c', 3)]
```

딕셔너리에는 여러 가지 유용한 메서드와 기능들이 있어서, 필요에 따라서 적절한 메서드를 사용하여 키나 값을 리스트로 변환할 수 있습니다.

이상으로 파이썬에서 딕셔너리의 키와 값을 리스트로 변환하는 방법에 대해 알아보았습니다. 감사합니다!
