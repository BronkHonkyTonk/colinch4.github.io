자바스크립트에서 **변수 선언**과 **할당**은 프로그래밍에서 중요한 개념이다. 변수를 다룰 때 두 가지 작업을 분리하여 수행한다. **변수 선언**은 변수를 생성하는 것이고, **할당**은 변수에 값을 할당하는 것이다.

## 변수 선언

변수를 사용하기 전에 먼저 선언해야 한다. 변수를 선언하면 컴퓨터 메모리에 저장 공간이 확보되어 해당 변수를 사용할 수 있다. 자바스크립트에서 변수는 `let`, `const`, `var` 키워드를 사용하여 선언할 수 있다. 

```javascript
let x; // 변수 x를 선언
const PI = 3.14; // 상수 PI를 선언
```

## 변수 할당

변수에 값을 할당할 때는 `=` 연산자를 사용한다. 할당은 변수에 값을 넣어주는 것이다.

```javascript
x = 10; // 변수 x에 10을 할당
```

## 결론

변수 선언은 변수를 만들고, 할당은 변수에 값을 넣는 것이다. 두 작업을 이해하면 변수를 올바르게 다룰 수 있고, 프로그램을 효율적으로 작성할 수 있다.

위에 있는 예제를 보면, `let`과 `const` 키워드를 사용하여 변수를 선언하고, `=`를 사용하여 값을 할당하는 것을 확인할 수 있다.
