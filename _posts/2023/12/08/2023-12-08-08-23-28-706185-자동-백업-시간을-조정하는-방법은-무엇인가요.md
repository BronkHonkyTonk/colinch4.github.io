아래는 자동 백업 시간을 조정하는 단계입니다.

1. 먼저 SQL Server Management Studio(SSMS)에 연결합니다.
2. "SQL Server 에이전트" 노드를 확장하고 "작업" 폴더를 선택합니다.
3. 백업을 수행하는 작업을 선택하고 "속성"을 클릭합니다.
4. "일정" 탭을 선택하고 새로운 백업 일정을 만들거나 기존 일정을 수정합니다.
5. 원하는 시간에 작업이 실행되도록 설정하고 변경 사항을 저장합니다.

이러한 단계를 통해 SQL Server의 자동 백업 시간을 조정할 수 있습니다. 필요에 따라 빈도 및 일정을 조정하여 데이터를 안전하게 백업할 수 있습니다.
