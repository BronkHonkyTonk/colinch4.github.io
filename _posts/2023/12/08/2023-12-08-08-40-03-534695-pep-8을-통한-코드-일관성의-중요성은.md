```python
# 잘못된 예시
def my_function():
  return None

# 올바른 예시
def my_function():
    return None
```

PEP 8을 준수하여 코드를 작성하면 가독성이 향상되고 다른 개발자들이 코드를 이해하고 수정하기 쉬워집니다.
