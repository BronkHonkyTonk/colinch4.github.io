Swift는 강력한 데이터 타입 시스템을 가지고 있어서 안정적이고 안전한 코드 개발을 지원합니다. 이번 포스트에서는 Swift에서 지원하는 다양한 데이터 타입과 타입 추론에 대해 알아보겠습니다.

## 데이터 타입

Swift는 여러 종류의 데이터 타입을 지원합니다. 각 데이터의 종류에 따라 적합한 데이터 타입을 선택하여 사용함으로써 메모리를 효율적으로 활용할 수 있습니다. Swift의 기본 데이터 타입에는 다음과 같은 것들이 있습니다.

- Int: 정수형 데이터 타입
- Double, Float: 부동소수점 데이터 타입
- Bool: 불리언 데이터 타입
- String: 문자열 데이터 타입
- Array, Dictionary, Set: 컬렉션 데이터 타입

이 외에도 다양한 데이터 타입들이 존재하며, 필요에 따라 적절한 데이터 타입을 선택하여 사용할 수 있습니다.

## 타입 추론

Swift는 타입 추론을 통해 코드 작성을 더 간편하게 만들어줍니다. 변수나 상수를 선언할 때, 초기 값을 가지고 있다면 Swift는 자동으로 해당 값의 타입을 추론합니다. 이를 통해 타입을 명시적으로 선언하지 않아도 되므로 코드를 더욱 간결하고 가독성 좋게 작성할 수 있습니다.

예를 들어:

```swift
let name = "Swift" // name은 String 타입으로 추론됨
var count = 10 // count는 Int 타입으로 추론됨
```

## 결론

Swift는 다양한 데이터 타입을 지원하고 타입 추론을 통해 간편한 코드 작성을 가능하게 해줍니다. 이를 통해 안정적이고 효율적인 앱을 개발할 수 있습니다.
