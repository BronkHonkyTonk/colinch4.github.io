SQL에서 테이블을 생성할 때는 CREATE TABLE 문을 사용합니다. 이 문장을 사용하여 테이블을 생성하고, 필요한 속성들을 추가할 수 있습니다.

다음은 테이블을 생성하면서 속성을 추가하는 방법에 대한 예제입니다.

```sql
CREATE TABLE 테이블명 (
   컬럼1 데이터_타입,
   컬럼2 데이터_타입,
   ...
   컬럼N 데이터_타입
);
```

위 예제에서 "테이블명"은 생성할 테이블의 이름을 나타내며, "컬럼1"부터 "컬럼N"까지는 속성들의 이름을 나타냅니다. 각 속성은 "데이터_타입"으로 정의됩니다.

예를 들어, "users"라는 테이블을 생성하면서 "id", "name", "age"라는 속성을 추가하는 경우 다음과 같이 작성할 수 있습니다.

```sql
CREATE TABLE users (
   id INT,
   name VARCHAR(50),
   age INT
);
```

위 예제에서 "id" 속성은 정수형(INT)으로, "name" 속성은 문자열형(VARCHAR)으로, "age" 속성은 정수형(INT)으로 정의되었습니다.

이와 같이 CREATE TABLE 문을 사용하여 테이블을 생성하면서 속성을 추가할 수 있습니다. 필요에 따라 더 많은 속성을 추가하거나, 다른 데이터 타입을 사용할 수도 있습니다.
