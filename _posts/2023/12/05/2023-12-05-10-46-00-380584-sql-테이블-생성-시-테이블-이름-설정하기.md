SQL을 사용하여 데이터베이스에서 테이블을 생성할 때, 테이블의 이름을 설정하는 방법에 대해 알아보겠습니다.

## 문법

테이블을 생성할 때, `CREATE TABLE` 문을 사용합니다. 테이블 이름을 설정하는 부분은 `CREATE TABLE` 다음에 오는 `테이블 이름` 부분입니다. 테이블 이름은 대소문자를 구분하며, 보통 여러 단어로 이루어진 이름은 밑줄(_)이나 대시(-)를 사용하여 단어를 구분합니다.

다음은 테이블 이름을 설정하는 예제입니다.

```sql
CREATE TABLE 테이블_이름 (
  컬럼1 데이터_유형,
  컬럼2 데이터_유형,
  ...
);
```

## 예제

예를 들어, '고객'이라는 테이블 이름으로 고객 정보를 저장하는 테이블을 생성하고 싶다고 가정해봅시다. 다음은 이를 위한 SQL 문입니다.

```sql
CREATE TABLE 고객 (
  ID INT,
  이름 VARCHAR(50),
  이메일 VARCHAR(100),
  ...
);
```

위 예제에서 `CREATE TABLE` 다음에 오는 `고객`이라는 부분이 테이블 이름을 설정하는 부분입니다. 그리고 `고객` 테이블에는 `ID`, `이름`, `이메일` 등의 컬럼이 정의되어 있습니다.

## 주의사항

- 테이블 이름은 데이터베이스 내에서 고유해야 합니다. 즉, 이미 같은 이름의 테이블이 존재하는 경우 테이블 생성이 실패할 수 있습니다.
- 테이블 이름은 사용하는 데이터베이스의 제한에 따라 길이 제한이 있을 수 있습니다. 이에 유의하여 이름을 설정해야 합니다.

## 결론

SQL에서 테이블을 생성할 때, 테이블 이름은 `CREATE TABLE` 문의 다음에 오는 부분에서 설정합니다. 테이블 이름을 올바르게 설정하고, 데이터베이스 내에서 유일한 이름을 사용해야 합니다.
