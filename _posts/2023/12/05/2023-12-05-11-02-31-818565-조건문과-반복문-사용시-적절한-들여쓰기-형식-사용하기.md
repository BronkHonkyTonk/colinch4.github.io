코드를 작성할 때, 조건문과 반복문을 올바르게 사용하는 것은 중요합니다. 이를 위해 적절한 들여쓰기 형식을 사용해야 합니다. 

들여쓰기는 코드의 가독성을 높이고, 코드의 구조와 논리를 명확하게 전달할 수 있도록 도와줍니다.

### 조건문 사용시 들여쓰기 형식

조건문은 특정 조건이 참인 경우에만 특정 코드 블록을 실행하는데 사용됩니다. 이때, 조건문의 블록은 중괄호 `{}` 로 묶여야 합니다. 

```javascript
if (condition) {
    // 조건이 참일 때 실행되는 코드
} else {
    // 조건이 거짓일 때 실행되는 코드
}
```

반복문을 사용할 때에도 동일한 들여쓰기 형식을 사용합니다.

### 반복문 사용시 들여쓰기 형식

반복문은 특정한 조건이 참일 때, 특정 코드 블록을 반복적으로 실행하는데 사용됩니다. `for` 루프는 일반적으로 다음과 같은 형식을 가지며, 블록은 중괄호로 묶여야 합니다.

```javascript
for (initialization; condition; increment) {
    // 반복적으로 실행되는 코드
}
```

이때, 들여쓰기는 한 단계씩 늘어날 수 있습니다.

### 들여쓰기 규칙

들여쓰기는 보통 스페이스 2개 또는 탭으로 수행됩니다. 어떤 스타일을 사용하던지 간에 일관성을 유지하는 것이 중요합니다. 

중요한 점은 들여쓰기를 일관되게 적용하는 것입니다. 모든 조건문과 반복문 블록은 동일한 들여쓰기 수준을 가져야 하며, 내부 블록은 외부 블록보다 한 단계 더 들여써야 합니다.

### 요약

들여쓰기 형식을 적용하여 코드를 작성하면 가독성과 유지보수성이 향상됩니다. 개발자들 간의 협업을 간소화하고, 코드의 논리를 명확하게 전달할 수 있습니다.
