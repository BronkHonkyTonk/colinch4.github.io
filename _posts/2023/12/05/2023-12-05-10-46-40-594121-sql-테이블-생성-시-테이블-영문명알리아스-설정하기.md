SQL 데이터베이스에서 테이블을 생성할 때, 테이블의 영문명에 별칭(alias)을 설정할 수 있습니다. 이를 통해 테이블을 쉽게 구분하고, 간편하게 사용할 수 있습니다. 이번 블로그 포스트에서는 SQL 테이블 생성 시 테이블 영문명 설정하는 방법에 대해 알아보겠습니다.

## 테이블 생성 문법

```sql
CREATE TABLE [테이블명] AS SELECT * FROM [원본테이블]
```

위의 SQL 문법을 사용하여 테이블을 생성할 수 있습니다. '테이블명'에는 원하는 테이블 이름을 지정하면 됩니다.

## 테이블 영문명(알리아스) 설정하기

위의 테이블 생성 문법에서 '테이블명'에 별칭을 설정하여 영문명을 지정할 수 있습니다. 다음은 예시입니다.

```sql
CREATE TABLE `employees` AS SELECT * FROM `employees_data`
```

위의 예시에서는 테이블 생성 시 'employees'라는 테이블의 영문명(알리아스)로 'employees_data' 테이블을 사용하였습니다.

테이블 영문명을 설정하면, 해당 테이블을 사용할 때 별칭을 사용하여 간단하게 접근할 수 있습니다. 예를 들어, 아래와 같이 사용할 수 있습니다.

```sql
SELECT * FROM `employees`
```

위의 SQL 문장에서 'employees'는 실제로 'employees_data' 테이블을 의미합니다. 이런식으로 테이블 영문명(알리아스)를 활용하여 테이블을 사용할 수 있습니다.

## 정리

SQL 테이블 생성 시 테이블 영문명(알리아스) 설정은 테이블을 쉽게 구분하고, 사용하기 위한 방법입니다. 테이블 생성 문법에서 '테이블명'에 별칭을 설정하여 영문명을 지정할 수 있으며, 이를 활용하여 해당 테이블을 간단하게 사용할 수 있습니다.
