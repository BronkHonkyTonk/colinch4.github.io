변수의 주석 처리:
```swift
var age: Int = 25 // 사용자의 나이를 저장하는 변수
```

상수의 주석 처리:
```swift
let name: String = "John" // 사용자의 이름을 저장하는 상수
```

주석은 코드 옆에 설명을 추가함으로써 코드의 의도와 목적을 더 명확하게 전달할 수 있습니다. 이는 다른 개발자가 코드를 읽을 때 도움이 됩니다. 주석을 적절하게 사용하면 코드의 유지보수성과 협업에 큰 도움이 될 수 있습니다.

주석은 `//` 기호로 시작하며, 한 줄 주석 또는 여러 줄 주석으로 작성할 수 있습니다. 한 줄 주석은 해당 줄에서부터 줄의 끝까지 주석으로 처리되고, 여러 줄 주석은 `/*`로 시작하여 `*/`로 끝납니다.

예시:
```swift
// 이것은 한 줄 주석입니다.

/*
여러 줄 주석입니다.
주석 내용을 여러 줄에 걸쳐 작성할 수 있습니다.
*/
```

주석 처리는 코드를 명확하게 설명하고, 다른 개발자와의 협업을 쉽게하는 중요한 요소입니다. 주석을 적극적으로 활용하여 코드의 가독성을 높이고 개발 과정을 보다 효율적으로 관리할 수 있습니다.
