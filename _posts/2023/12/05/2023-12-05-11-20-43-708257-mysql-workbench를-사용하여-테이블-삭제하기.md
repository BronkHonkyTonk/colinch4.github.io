MySQL Workbench는 MySQL 데이터베이스를 관리하기 위한 그래픽 인터페이스 도구입니다. 이 도구를 사용하면 데이터베이스에 테이블을 생성, 편집 및 삭제할 수 있습니다. 이제 MySQL Workbench를 사용하여 테이블을 삭제하는 방법에 대해 알아보겠습니다.

## 1. MySQL Workbench 열기
먼저 MySQL Workbench를 엽니다. 일반적으로 "Applications" 또는 "프로그램" 폴더에서 찾을 수 있습니다.

## 2. 데이터베이스 연결
MySQL Workbench를 열었다면, 데이터베이스 서버에 연결해야 합니다. 연결 버튼을 클릭하고 호스트, 포트, 사용자 이름 및 암호를 입력하여 데이터베이스에 연결합니다.

## 3. 스키마 선택
연결에 성공하면 MySQL Workbench에서 사용 가능한 데이터베이스 스키마가 표시됩니다. 원하는 스키마를 선택하십시오.

## 4. 테이블 삭제
테이블을 삭제하려면 선택한 스키마에서 해당 테이블을 마우스 오른쪽 버튼으로 클릭하고 "Drop Table"을 선택하십시오. 이렇게 하면 선택한 테이블이 삭제됩니다.

![MySQL Workbench Drop Table](https://example.com/mysql-workbench-drop-table.png)

## 5. 변경 사항 적용
마지막으로, 테이블을 삭제한 후 변경 사항을 적용해야 합니다. 작업 창 상단의 "Apply" 버튼을 클릭하여 변경 사항을 적용하고 데이터베이스에서 테이블을 완전히 삭제합니다.

이제 MySQL Workbench를 사용하여 테이블을 삭제하는 방법을 알게 되었습니다. MySQL 데이터베이스를 관리하는 데 도움이 되는 이 도구를 사용하여 데이터베이스 작업을 수행할 수 있습니다.
