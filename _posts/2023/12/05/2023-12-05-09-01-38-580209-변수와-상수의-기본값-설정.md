변수와 상수는 값을 저장하기 위해 사용되며, 값을 변경할 수 있는 변수와 값을 변경할 수 없는 상수로 구분됩니다. 

변수의 기본값 설정은 var 키워드를 사용하여 변수를 선언한 후, 값을 할당하는 방식으로 이루어집니다. 예를 들어, 다음과 같이 변수를 선언하고 값을 할당할 수 있습니다.

```swift
var number: Int = 10
```

위의 코드에서는 number라는 변수를 선언하고, Int 형식의 기본값 10을 할당하였습니다. 이후에는 number 변수의 값이 변경될 수 있습니다.

반면에, 상수의 기본값 설정은 let 키워드를 사용하여 상수를 선언한 후, 값을 할당하는 방식으로 이루어집니다. 다음은 상수의 기본값 설정 예시입니다.

```swift
let name: String = "John"
```

위의 코드에서는 name이라는 상수를 선언하고, String 형식의 기본값 "John"을 할당하였습니다. 상수는 한 번 값을 할당하면 이후에는 변경할 수 없습니다.

또한, 변수와 상수의 경우에는 기본값을 할당하지 않을 수도 있습니다. 이 경우에는 변수나 상수를 선언한 후, 이후에 값을 할당할 수 있습니다. 다음은 값을 할당하지 않은 변수와 상수의 예시입니다.

```swift
var score: Int
let PI: Double
```

위의 코드에서는 score 변수와 PI 상수를 선언한 후, 초기값을 할당하지 않았습니다. 이후에 필요한 시점에 값을 할당할 수 있습니다.

변수와 상수는 Swift에서 많이 사용되는 개념으로, 기본값 설정을 통해 초기값을 설정하고 값을 유동적으로 변경하거나 변경할 수 없게 할 수 있습니다. 따라서 Swift에서 변수와 상수의 기본값 설정은 매우 중요한 개념이며, 잘 이해하고 사용해야 합니다.
