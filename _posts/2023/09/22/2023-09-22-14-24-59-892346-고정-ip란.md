하지만 고정 IP 주소는 ISP로부터 고정적으로 할당받는 주소입니다. 이는 인터넷에 항상 동일한 IP 주소를 사용할 수 있게 됩니다. 

고정 IP 주소의 장점 중 하나는 서버를 운영하는 경우에 매우 유용합니다. 고정 IP 주소를 가지고 있으면 동적 IP 주소와 달리 항상 같은 IP 주소로 서버에 접근할 수 있습니다. 이렇게 함으로써 외부 사용자가 서버에 접근하는 데에 편의성이 증가하며, DNS(Domain Name System) 설정에도 용이합니다.

그러나 고정 IP 주소를 사용하려면 추가 비용이 발생할 수 있습니다. 일부 ISP는 고정 IP 주소를 추가 요금을 받고 제공하기 때문에, 이를 고려하여 사용할 필요가 있습니다.
