인터넷에 연결된 디바이스들은 주로 동적 IP 주소를 할당받아 사용하게 됩니다. 하지만, 특정한 상황에서는 고정 IP 주소가 필요할 수 있습니다. 예를 들어, 웹서버나 네트워크 장비들은 항상 동일한 IP 주소로 접속되어야 한다는 요구사항이 있을 수 있습니다. 

아래는 IP 주소를 고정하는 방법에 대한 예제입니다. 

## 윈도우

1. 먼저 "제어판"을 실행합니다.
2. "네트워크 및 인터넷" 메뉴로 이동합니다.
3. "네트워크 및 공유 센터"를 클릭합니다.
4. "이더넷" 또는 "Wi-Fi" 옵션을 선택합니다.
5. "속성"을 클릭합니다.
6. "인터넷 프로토콜 버전 4 (TCP/IPv4)"를 선택한 후 "속성"을 클릭합니다.
7. "다음 IP 주소 사용"을 선택하고 원하는 IP 주소를 입력합니다.
8. "서브넷 마스크"와 "기본 게이트웨이"도 입력합니다.
9. 변경사항을 저장하고 나가면 IP 주소가 고정되었습니다.

## 맥

1. "시스템 환경 설정"으로 이동합니다.
2. "네트워크"를 클릭합니다.
3. "Wi-Fi" 또는 "이더넷"을 선택한 후 "고급"을 클릭합니다.
4. "TCP/IP" 탭을 선택합니다.
5. "IPv4 주소"를 "수동으로 설정"으로 변경합니다.
6. 원하는 IP 주소를 입력하고 "서브넷 마스크"와 "라우터"도 입력합니다.
7. 변경사항을 저장하면 IP 주소가 고정되었습니다.

이제 디바이스의 IP 주소가 고정되었으므로 네트워크 설정이 변경되지 않는 한 항상 동일한 IP 주소를 사용할 수 있습니다.
