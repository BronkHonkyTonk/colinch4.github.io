## 건강 데이터 수집

IoT 기술을 통해 체온, 심박수, 혈압 등의 건강 데이터를 수집할 수 있습니다. 이 데이터는 센서를 통해 실시간으로 측정되며, 블루투스나 와이파이를 통해 서버로 전송됩니다. 파이썬을 사용하여 센서와 통신하고, 데이터를 수신하는 코드를 작성해 보겠습니다.

```python
import bluetooth

# 블루투스 주소
address = '00:00:00:00:00:00'

# 블루투스 연결
socket = bluetooth.BluetoothSocket(bluetooth.RFCOMM)
socket.connect((address, 1))

# 데이터 수신
data = socket.recv(1024)

# 데이터 처리
# ...

# 연결 종료
socket.close()
```

위의 코드에서 `address`는 센서의 블루투스 주소를 나타내며, `socket`은 블루투스 소켓 연결을 의미합니다. `recv` 메서드를 사용하여 데이터를 수신하고, 이후에 데이터를 처리하는 코드를 작성해야 합니다.
