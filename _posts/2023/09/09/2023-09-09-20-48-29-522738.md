자바스크립트는 웹 개발에서 가장 널리 사용되는 프로그래밍 언어 중 하나입니다. 이 언어를 사용하여 변수를 선언하고 산술 연산을 수행 할 수 있습니다. 이 기사에서는 자바스크립트에서 변수와 산술 연산을 사용하는 방법을 알아보겠습니다.

## 변수 선언

자바스크립트에서 변수를 선언하기 위해 `var`, `let`, `const`를 사용할 수 있습니다. 각각의 키워드는 변수의 범위와 수정 가능 여부를 결정하는 데 사용됩니다.

- `var` : 함수 레벨 범위, 값 변경 가능
- `let` : 블록 레벨 범위, 값 변경 가능
- `const` : 블록 레벨 범위, 값 변경 불가능

변수를 선언할 때는 변수 이름을 지정하고 이에 대한 초기 값을 할당할 수 있습니다.

```javascript
var a = 5;
let b = 10;
const c = 15;
```

## 산술 연산

자바스크립트에서 산술 연산은 기본적인 수학 연산을 수행하는 데 사용됩니다. 다음은 자바스크립트에서 사용 가능한 산술 연산자 목록입니다.

- `+` : 더하기
- `-` : 빼기
- `*` : 곱하기
- `/` : 나누기
- `%` : 나머지
- `**` : 제곱

이제 이러한 연산자를 사용하여 변수 간에 산술 연산을 수행하는 방법을 살펴보겠습니다.

```javascript
var a = 5;
var b = 10;
var result = a + b; // 15

let x = 8;
let y = 3;
let multiplication = x * y; // 24

const p = 10;
const q = 3;
const division = p / q; // 3.3333333333333335
```

## 연산 순서

산술 연산의 순서는 수학의 PEMDAS 규칙을 따릅니다. 즉, 괄호를 먼저 처리하고, 지수를 다음으로 계산하고, 곱셈과 나눗셈을 마지막으로 덧셈과 뺄셈을 수행합니다.

```javascript
var result = (2 + 3) * 4; // 20
```

## 결론

자바스크립트에서 변수와 산술 연산을 사용하는 방법에 대해 알아보았습니다. 이를 통해 변수를 선언하고, 값을 할당하고, 산술 연산을 수행하는 기본적인 개념을 이해할 수 있습니다. 이러한 기능을 통해 더 복잡한 계산을 수행하고, 웹 애플리케이션에서 동적인 동작을 구현할 수 있습니다.
