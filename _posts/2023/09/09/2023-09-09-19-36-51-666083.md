자바스크립트에서 배열(array)은 여러 개의 값을 하나의 변수에 저장할 수 있는 데이터 구조입니다. 배열에는 숫자, 문자열, 객체 등 다양한 형태의 값을 저장할 수 있습니다. 이번 블로그 포스트에서는 자바스크립트 배열의 합과 평균을 계산하는 방법에 대해 알아보겠습니다.

## 배열의 합 계산하기

자바스크립트에서 배열의 합을 계산하는 가장 간단한 방법은 반복문을 사용하는 것입니다. 아래 예시 코드를 살펴보세요.

```javascript
let numbers = [1, 2, 3, 4, 5];
let sum = 0;

for (let i = 0; i < numbers.length; i++) {
  sum += numbers[i];
}

console.log(sum); // 결과: 15
```

위 코드에서는 `numbers` 배열의 요소를 반복하여 각 요소를 `sum` 변수에 더해줍니다. 반복문이 끝나면 `sum` 변수에는 배열의 합이 저장되어 있습니다.

## 배열의 평균 계산하기

배열의 평균을 계산하기 위해서는 배열의 합을 먼저 계산한 후, 배열의 길이로 나누어주어야 합니다. 아래 예시 코드를 살펴보세요.

```javascript
let numbers = [1, 2, 3, 4, 5];
let sum = 0;

for (let i = 0; i < numbers.length; i++) {
  sum += numbers[i];
}

let average = sum / numbers.length;

console.log(average); // 결과: 3
```

위 코드에서는 먼저 `sum` 변수를 사용하여 배열의 합을 계산한 후, `sum` 변수를 배열의 길이로 나누어 `average` 변수에 저장합니다. 결과적으로 `average` 변수에는 배열의 평균값이 저장되어 있습니다.

## 주의사항

배열의 합이나 평균을 계산할 때 주의해야 할 점이 있습니다.

- 배열에 값이 없는 경우(`numbers.length`가 0인 경우) 합과 평균은 0이 됩니다.
- 배열의 요소가 모두 숫자가 아닌 경우 합과 평균을 계산할 수 없습니다. 이 경우 추가적인 유효성 검사가 필요합니다.

## 마무리

자바스크립트에서 배열의 합과 평균을 계산하는 방법에 대해 알아보았습니다. 배열을 다룰 때는 주의하여 유효성 검사를 수행하고, 반복문을 사용하여 각 요소를 합하거나 평균을 계산하는 방법을 기억하면 더욱 효율적으로 작업할 수 있습니다. 합과 평균을 계산하는 방법을 활용하여 데이터 처리나 통계 분석 등 다양한 작업에 활용해보세요!
