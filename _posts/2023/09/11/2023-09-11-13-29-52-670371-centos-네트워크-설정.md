CentOS는 오픈 소스 리눅스 배포판 중 하나로, 기본적으로 네트워크를 구성하는 데 필요한 도구와 기능을 제공합니다. 이 블로그 포스트에서는 CentOS에서 네트워크 설정을 하는 방법을 알아보겠습니다.

## 1. 네트워크 설정 확인

네트워크 구성을 변경하기 전에, 현재 네트워크 설정을 확인하는 것이 좋습니다. 다음 명령을 사용하여 현재 설정을 확인할 수 있습니다.

```bash
$ ifconfig
```

이 명령을 실행하면 시스템의 네트워크 인터페이스 및 IP 주소 등의 정보가 표시됩니다.

## 2. 네트워크 설정 파일 확인

CentOS의 네트워크 설정은 주로 네트워크 설정 파일을 통해 관리됩니다. 다음 명령으로 설정 파일을 확인할 수 있습니다.

```bash
$ vi /etc/sysconfig/network-scripts/ifcfg-enp0s3
```

위 명령을 실행하면 해당 인터페이스(enp0s3)의 설정 파일이 열립니다.

## 3. 네트워크 설정 파일 편집

편집기가 열리면 다음과 같은 내용을 수정할 수 있습니다.

```bash
DEVICE=enp0s3
TYPE=Ethernet
ONBOOT=yes
BOOTPROTO=static
IPADDR=192.168.1.100
NETMASK=255.255.255.0
GATEWAY=192.168.1.1
DNS1=8.8.8.8
DNS2=8.8.4.4
```

위 예시에서는 인터페이스를 정적 IP로 설정하고 있습니다. 필요한 경우 해당 설정을 자신의 환경에 맞게 수정하십시오. 중요한 설정은 다음과 같습니다:

- `DEVICE`: 인터페이스 이름
- `TYPE`: 인터페이스 유형 (일반적으로 Ethernet)
- `ONBOOT`: 부팅 시 해당 인터페이스 활성화 여부
- `BOOTPROTO`: IP 할당 방식 (일반적으로 static 또는 dhcp)
- `IPADDR`: IP 주소
- `NETMASK`: 서브넷 마스크
- `GATEWAY`: 게이트웨이 주소
- `DNS1` 및 `DNS2`: DNS 서버 주소

위 내용을 수정한 후, 저장하고 편집기를 종료합니다.

## 4. 변경 사항 적용

변경한 네트워크 설정이 적용되려면 네트워크 서비스를 재시작해야 합니다. 다음 명령을 사용하여 네트워크 서비스를 재시작합니다.

```bash
$ systemctl restart network
```

네트워크 서비스가 재시작되면 변경한 네트워크 설정이 적용됩니다.

## 5. 네트워크 연결 테스트

마지막으로, 변경한 네트워크 설정이 올바르게 작동하는지 확인하기 위해 네트워크 연결을 테스트하는 것이 좋습니다. 다음 명령을 사용하여 네트워크 연결을 테스트할 수 있습니다.

```bash
$ ping google.com
```

위 명령을 실행하여 인터넷에 연결되는지 확인하십시오. 만약 연결이 성공하면, 네트워크 설정이 올바르게 구성된 것입니다.

## 정리

CentOS에서 네트워크 설정을 변경하는 방법에 대해 알아보았습니다. 현재 설정을 확인하고, 설정 파일을 수정한 후 변경 사항을 적용해야 합니다. 네트워크 연결을 테스트하여 설정이 올바른지 확인할 수 있습니다. 적절한 네트워크 설정으로 안정적인 네트워크 연결을 유지할 수 있습니다.
