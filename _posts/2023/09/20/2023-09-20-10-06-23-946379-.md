안녕하세요! 오늘은 마크다운 형식으로 기술 블로그 포스트 작성하는 방법에 대해 알려드리겠습니다.

## 마크다운 작성하기
마크다운은 간단하고 가독성이 좋은 텍스트 기반의 마크업 언어입니다. 다양한 포맷을 지원하며, 코드 블록, 헤더, 목록 등을 쉽게 작성할 수 있습니다.

### 헤더
헤더는 문서의 제목이나 소제목을 표시하는 역할을 합니다. 다음과 같이 사용할 수 있습니다.

```
# 제목 1
## 제목 2
### 제목 3
```

### 목록
목록은 순서가 없는 항목과 순서가 있는 항목으로 구성됩니다.

#### 순서가 없는 항목
```
- 항목 1
- 항목 2
- 항목 3
```

#### 순서가 있는 항목
```
1. 항목 1
2. 항목 2
3. 항목 3
```

### 코드 블록
코드 블록은 프로그래밍 코드를 강조하여 표시할 때 사용합니다. 다음과 같이 작성할 수 있습니다.

```python
def say_hello():
    print("안녕하세요!")
```
