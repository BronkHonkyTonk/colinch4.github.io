## 제목: 안드로이드 앱 개발을 위한 기본 가이드

안드로이드 앱 개발은 현재 가장 인기있는 분야 중 하나입니다. 이 글에서는 안드로이드 앱 개발을 시작하기 위한 기본 가이드를 제공하고자 합니다.

### 1. 개발 환경 설정하기

안드로이드 앱을 개발하기 위해서는 몇 가지 개발 도구와 환경을 설정해야 합니다. 다음은 개발을 위해 필요한 도구들입니다.

- **Android Studio**: 안드로이드 앱 개발을 위한 공식 IDE
- **Java Development Kit (JDK)**: 안드로이드 앱을 개발하기 위한 자바 개발 도구
- **안드로이드 에뮬레이터**: 실제 안드로이드 장치의 시뮬레이션을 제공하는 도구

이 도구들을 설치하고 설정한 뒤에야 안드로이드 앱 개발을 진행할 수 있습니다.

### 2. 프로젝트 생성하기

안드로이드 스튜디오에서 새로운 프로젝트를 생성하는 방법은 다음과 같습니다:

```
// Kotlin 코드로 새 프로젝트 생성하기
fun main() {

    // 새로운 안드로이드 프로젝트 생성
    val project = AndroidProject()
    project.create()

    // 기본 설정 및 구성 파일 생성
    val config = project.getConfig()
    config.create()

    // 앱 모듈 추가
    val appModule = project.addModule("app")
    appModule.create()

    // 필요한 라이브러리 의존성 추가
    appModule.addDependency("com.android.support:appcompat:1.0.0")
    appModule.addDependency("com.google.android.material:material:1.0.0")

    // 액티비티 생성
    val mainActivity = appModule.addActivity("MainActivity")
    mainActivity.create()

}
```

위의 코드는 Kotlin을 사용하여 안드로이드 프로젝트를 생성하고 기본 설정을 추가하는 예시입니다.

### 3. 빌드 및 실행하기

프로젝트를 생성하고 코드를 작성한 뒤에는 앱을 빌드하고 실행해야 합니다. 안드로이드 스튜디오에서는 빌드와 실행을 쉽게 할 수 있는 기능을 제공합니다. 다음은 빌드 및 실행하는 방법입니다:

1. 메뉴에서 **Build > Make Project**를 선택하여 프로젝트를 빌드합니다.
2. 프로젝트가 성공적으로 빌드되면, **Run > Run 'app'**을 선택하여 앱을 실행합니다.
3. 선택한 에뮬레이터 또는 연결된 안드로이드 장치에서 앱이 실행됩니다.

### 4. 추가 리소스 및 라이브러리 활용하기
