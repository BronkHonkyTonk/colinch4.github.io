Python은 데이터 분석 및 머신러닝 분야에서 가장 인기 있는 프로그래밍 언어 중 하나입니다. 그러나 파이썬을 처음 접하는 사람들에게는 해당 언어의 기본 개념과 문법을 이해하는 것이 중요합니다. 이 블로그 포스트에서는 데이터 분석을 위한 파이썬 기초에 대해 알아보겠습니다.

### 변수와 데이터 타입

파이썬에서 변수를 정의하려면 변수의 이름을 지정하고 할당 연산자(`=`)를 사용하여 값을 할당합니다. 예를 들어, 아래의 코드는 `age`라는 변수에 25라는 값을 할당하는 예시입니다.

```python
age = 25
```

파이썬은 동적 타이핑(dynamic typing) 언어이기 때문에 변수의 데이터 타입은 할당된 값에 따라 자동으로 결정됩니다. 또한 다양한 데이터 타입(문자열, 정수, 실수 등)을 지원합니다.

### 조건문과 반복문

조건문은 프로그램의 흐름을 제어하는 데 사용됩니다. 파이썬에서는 `if`, `elif`, `else` 키워드를 사용하여 조건문을 작성합니다. 아래의 코드는 `age` 변수의 값을 확인하고 조건에 따라 메시지를 출력하는 예시입니다.

```python
if age < 18:
    print("You are underage.")
elif age >= 18 and age < 65:
    print("You are of legal age.")
else:
    print("You are a senior citizen.")
```

반복문은 주어진 조건이 충족될 때까지 코드 블록을 실행하는 데 사용됩니다. 파이썬에서는 `for`와 `while` 두 가지 반복문을 제공합니다. 아래의 코드는 `numbers`라는 리스트의 각 항목을 반복하면서 출력하는 예시입니다.

```python
numbers = [1, 2, 3, 4, 5]

for number in numbers:
    print(number)
```

### 함수 정의

함수는 코드의 재사용성을 높이고 효율적인 프로그래밍을 위해 사용됩니다. 파이썬에서 함수를 정의하려면 `def` 키워드를 사용합니다. 아래의 예시는 "Hello, World!"를 출력하는 함수를 정의하는 코드입니다.

```python
def say_hello():
    print("Hello, World!")

say_hello()  # 함수 호출
```

위의 코드에서 `say_hello()`는 함수를 호출하는 구문입니다.
