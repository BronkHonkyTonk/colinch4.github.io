주석은 코드에 설명을 추가하고 코드의 이해를 돕는 중요한 도구입니다. 주석은 프로그래밍 언어에서 제공되며, 주로 코드의 목적, 작성자, 수정 날짜 등과 같은 정보를 제공합니다.

이번 포스트에서는 주석 작성 스타일과 규칙에 대해 알아보겠습니다.

## 1. 주석 유형

주석은 주로 두 가지 유형으로 나눌 수 있습니다:

### 1.1. 한 줄 주석

한 줄 주석은 코드 한 줄에 해당하는 주석입니다. 주로 코드 옆에 작성하며, 한 줄 주석은 해당 줄의 코드를 설명합니다. 

한 줄 주석은 `#` 기호로 시작합니다. 

```python
# 이 코드는 변수 x에 5를 할당하는 코드입니다.
x = 5
```

### 1.2. 여러 줄 주석

여러 줄 주석은 여러 줄의 코드에 대한 주석입니다. 일반적으로 코드 블록이나 함수 정의 위에 작성하며, 해당 코드 블록이나 함수의 목적, 동작, 사용법 등을 설명합니다.

여러 줄 주석은 `'''` 또는 `"""` 기호로 여닫습니다.

```python
'''
이 코드는 두 개의 숫자를 더하는 함수입니다.
두 개의 숫자를 더한 결과를 반환합니다.
'''
def add_numbers(num1, num2):
    return num1 + num2
```

## 2. 주석 작성 규칙

주석을 작성할 때는 일정한 규칙을 따르는 것이 좋습니다. 주석 규칙은 코드의 가독성과 유지보수를 위해 중요합니다. 아래는 몇 가지 주석 작성 규칙입니다:

- 코드를 간결하고 명확하게 설명하는 주석을 작성합니다.
- 주석은 올바른 철자와 문법을 사용하여 작성합니다.
- 주석은 코드에서 파악하기 어려운 부분을 설명하거나, 예상되는 결과를 나타냅니다.
- 코드가 변경될 때마다 주석도 함께 업데이트합니다.
- 주석은 일반적으로 영어로 작성하며, 다른 사람들도 이해할 수 있도록 작성해야 합니다.
- 긴 주석의 경우 여러 줄로 나눠 작성하고, 문단을 구분하기 위해 빈 줄을 추가할 수 있습니다.

```python
# 이 코드는 사용자로부터 입력을 받아서 출력하는 예제입니다.

username = input("사용자 이름을 입력하세요: ")  # 사용자로부터 이름을 입력받습니다.

# 사용자 이름을 화면에 출력합니다.
# 환영 메시지를 포함하여 출력합니다.
print("환영합니다,", username + "!")   
```

## 3. 주석의 활용

주석은 코드의 기능, 동작, 사용법 등을 설명하는 역할 외에도 다양한 용도로 활용될 수 있습니다:

- 코드를 임시로 비활성화할 때 주석을 사용하여 해당 코드를 무시하도록 할 수 있습니다.
- 코드의 변경 이력을 추적하거나 버전 관리할 때 주석을 사용하여 수정 내용을 기록할 수 있습니다.
- 작성 중인 코드에 대한 메모를 주석으로 작성하여 나중에 참고할 수 있습니다.
- 코드의 특정 부분을 강조하기 위해 주석을 사용하여 시각적으로나 이해하기 쉽게 표시할 수 있습니다.

주석은 코드의 가독성과 유지보수를 향상시키는 중요한 도구입니다. 주석을 올바르게 작성하고 사용한다면, 코드를 이해하기 쉽고 개선하기 편리한 코드로 만들 수 있습니다. 이러한 주석 작성 스타일과 규칙을 준수하여 코드를 작성하는 것을 권장합니다.
