주석은 코드에서 설명이나 메모를 작성하는 데 사용되며, 실행 중에는 무시됩니다. 주석은 코드의 가독성과 유지 보수성을 향상시키는 데 도움을 줍니다. 이 글에서는 주석이 어디에 위치해야 하고 어떻게 표현되어야 하는지에 대해 알아보겠습니다.

## 주석 위치

주석은 코드 바로 위에 작성하는 쪽이 좋습니다. 코드와 주석 사이에는 빈 줄을 넣어 구분하는 것이 가독성을 높일 수 있습니다. 주석은 간단하지만 이해하기 어려운 코드에 설명을 추가하는 데 유용하며, 함수나 메소드 위에도 작성하여 해당 함수 또는 메소드의 동작을 설명할 수 있습니다.

예를 들어, 다음은 함수의 위에 작성된 주석의 예입니다:

```python
# calculate_area 함수는 주어진 가로와 세로 길이를 사용하여 직사각형의 넓이를 계산합니다.
def calculate_area(width, height):
    area = width * height
    return area
```

위의 예시에서는 주석을 사용하여 `calculate_area` 함수가 어떤 일을 하는지 쉽게 이해할 수 있습니다.

## 주석 표현 방식

주석은 일반적으로 `#` 문자로 시작하며, 한 줄 주석으로 작성됩니다. 한 줄 주석은 해당 줄에 있는 코드에 대한 설명을 추가하는 데 사용됩니다.

여러 줄로 된 주석을 작성하려면 여러 줄 주석 표현 방식을 사용해야 합니다. 일부 프로그래밍 언어에서는 여러 줄 주석을 `/*`과 `*/` 사이에 작성하거나, `"""` 또는 `'''`로 감싸는 방식을 사용합니다. 예를 들어, 다음은 Python에서 여러 줄 주석을 작성하는 예입니다:

```python
"""
이것은 여러 줄 주석입니다.
여러 줄에 걸쳐서 설명이나 메모를 작성할 수 있습니다.
"""
```

또는 이와 같이 작성할 수도 있습니다:

```python
'''
이것도 여러 줄 주석입니다.
작은 따옴표(`'`)로 시작하여 작은 따옴표로 끝납니다.
'''
```

이러한 주석 형식은 코드의 특정 부분을 잠시 비활성화하거나, 다른 개발자와 주석을 공유하는 데에도 유용합니다.

## 정리

주석은 코드에서 설명과 메모를 작성하는 데 사용됩니다. 적절한 위치에 주석을 작성하고, 주석의 표현 방식을 알맞게 사용하여 코드의 가독성과 유지 보수성을 향상시킬 수 있습니다. 코드의 이해하기 어려운 부분에 주석을 작성하여 다른 개발자들과 코드를 공유하고, 더 나은 협업을 이룰 수 있습니다.
