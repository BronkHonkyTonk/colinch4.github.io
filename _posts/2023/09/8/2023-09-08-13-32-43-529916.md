정규 표현식(Regex)은 텍스트 패턴을 표현하고 검색, 추출, 대체 등 다양한 작업을 수행하는 강력한 도구입니다. 텍스트 분석을 수행할 때 정규 표현식은 매우 효과적이며, 파이썬에서도 `re` 라이브러리를 통해 간편하게 활용할 수 있습니다.

## 정규 표현식의 기본 구성 요소

정규 표현식은 패턴을 나타내는 문자열입니다. 이 패턴은 특정 문자 또는 문자열 집합의 일치 여부를 확인하고 추출하는 데 사용됩니다. 아래는 표준적인 정규 표현식 구성 요소입니다:

- `.` : 어떤 문자 하나와 일치합니다.
- `*` : 앞의 패턴이 0번 이상 연속해서 나타나는 경우와 일치합니다.
- `+` : 앞의 패턴이 1번 이상 연속해서 나타나는 경우와 일치합니다.
- `?` : 앞의 패턴이 0번 또는 1번만 나타나는 경우와 일치합니다.
- `[]` : 괄호 안에 있는 문자 중 하나와 일치합니다.
- `()` : 괄호 안에 있는 패턴을 하나의 그룹으로 처리합니다.

## 정규 표현식을 사용한 텍스트 패턴 매칭

정규 표현식은 문자열 안에서 원하는 패턴을 찾아내는데 사용됩니다. 예를 들어, 이메일 주소 검색을 위해 아래와 같은 정규 표현식을 사용할 수 있습니다.

```
import re

text = "Contact us at info@example.com for more information"
pattern = r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}\b'

result = re.findall(pattern, text)
print(result)
```

위의 예제에서, `re.findall()` 함수는 주어진 정규 표현식 패턴과 텍스트를 매칭하여 결과를 리스트로 반환합니다. 이 경우, 이메일 주소 패턴을 찾아내어 출력합니다.

## 정규 표현식을 사용한 텍스트 추출과 대체

정규 표현식은 텍스트에서 특정 패턴을 추출하거나 원하는 형식으로 대체하는 데에도 유용합니다. 예를 들어, 텍스트에서 숫자만 추출하고자 할 때 다음과 같은 정규 표현식을 사용할 수 있습니다.

```
import re

text = "The price of the product is $50.99"
pattern = r'\d+\.\d+'

result = re.findall(pattern, text)
print(result)
```

위의 예제에서, 숫자와 소수점만을 추출하기 위해 `\d+\.\d+` 패턴을 사용합니다. 결과로는 `['50.99']`라는 리스트가 출력됩니다.

정규 표현식은 더 복잡한 텍스트 처리와 변환 작업에도 사용될 수 있습니다. 예를 들어, 텍스트 안에 있는 모든 URL을 하이퍼링크로 변경하는 작업은 정규 표현식을 활용하여 간단히 수행할 수 있습니다.

## 결론

정규 표현식은 파이썬에서 텍스트 분석 작업을 수행할 때 매우 유용한 도구입니다. 이를 통해 텍스트에서 특정 패턴을 찾아내거나 추출하고, 원하는 형식으로 변환하는 작업을 간단하게 수행할 수 있습니다.

정규 표현식은 다양한 패턴 매칭 및 텍스트 처리 작업에 활용할 수 있으며, `re` 라이브러리를 통해 파이썬에서도 간편하게 사용할 수 있습니다.
