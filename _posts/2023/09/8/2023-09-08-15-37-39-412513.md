파이썬에서 파일 경로를 다루는 작업은 꽤 흔한 일입니다. 이를 위해 `os` 모듈이나 `glob` 모듈 등의 다양한 방법을 사용할 수 있지만, `pathlib` 모듈은 좀 더 간결하고 직관적인 방식으로 파일 경로를 다룰 수 있는 기능을 제공합니다.

`pathlib` 모듈의 `Path` 클래스는 파일 경로를 객체로 표현하는데 사용되며, 여러 가지 편리한 메서드를 제공합니다. `Path.rglob()` 메서드는 현재 경로 및 하위 디렉토리에서 특정 패턴을 가지는 파일들을 재귀적으로 찾아주는 기능을 제공합니다. 이를 통해 파일 시스템을 탐색하고 필요한 파일들을 손쉽게 찾을 수 있습니다.

## 사용 예제

다음은 `Path.rglob()` 메서드의 사용 예제입니다.

```python
from pathlib import Path

# 현재 디렉토리부터 시작하여 .txt로 끝나는 모든 파일을 찾음
files = Path().rglob("*.txt")

# 파일 경로 목록 출력
for file in files:
    print(file)
```

위 예제에서는 `Path()`를 호출하여 현재 작업 디렉토리를 나타내는 `Path` 객체를 생성합니다. `Path.rglob()` 메서드는 해당 객체에 적용되며, `.txt`로 끝나는 모든 파일을 찾기 위해 `"*.txt"` 패턴을 전달했습니다. 이렇게 하면 현재 디렉토리 및 하위 디렉토리에서 `.txt`로 끝나는 모든 파일을 재귀적으로 검색할 수 있습니다.

`Path.rglob()`은 모든 일치하는 파일의 `Path` 객체를 반환하므로, 이를 순회하면서 각 파일의 경로를 출력하거나 다른 작업을 수행할 수 있습니다.

## 추가 기능

`Path.rglob()` 메서드의 기능은 이 예제에서 설명한 내용 이외에도 더욱 다양합니다. 다음은 몇 가지 추가적인 기능입니다:

- 특정 패턴에 해당하는 파일만 검색하는 것이 아니라, 디렉토리 또는 심볼릭 링크까지도 재귀적으로 탐색할 수 있습니다.
- `Path.rglob()`을 사용할 때 여러 패턴을 함께 사용하여 여러 종류의 파일을 선택적으로 찾을 수 있습니다. 예를 들어 `Path.rglob("*.txt", "*.csv")`와 같이 사용하면 `.txt`와 `.csv` 파일을 모두 찾을 수 있습니다.
- `Path.rglob()` 메서드에 정규식 패턴을 사용하여 더욱 유연한 검색을 수행할 수도 있습니다.

## 마치며

파일 시스템에서 특정 패턴을 가진 파일을 찾는 일은 많은 경우에 필요한 작업입니다. `pathlib` 모듈의 `Path.rglob()` 메서드는 이를 간단하고 효율적으로 처리할 수 있는 방법을 제공합니다. 파일 시스템을 탐색하거나 원하는 파일들을 손쉽게 찾기 위해 `Path.rglob()` 메서드를 적절히 활용해보세요.
