# functools 오버라이드 가능한 함수 작성

함수를 작성할 때 종종 오버라이딩이 가능한 기능을 가진 함수를 만들어야 할 때가 있습니다. functools 모듈은 파이썬에서 이러한 기능을 구현하는 데 도움을 주는 유용한 도구들을 제공합니다. 이번 블로그 포스트에서는 functools 모듈을 사용하여 오버라이드 가능한 함수를 작성하는 방법에 대해 알아보겠습니다.

## 1. functools.wraps 데코레이터

첫 번째로 알아볼 기능은 functools.wraps 데코레이터입니다. 이 데코레이터를 사용하면 기존 함수의 속성들을 유지한 채로 새로운 함수를 정의할 수 있습니다. 예를 들어, 다음은 기존 함수를 래핑하여 새로운 함수를 정의하는 예제입니다.

```python
import functools

def my_decorator(func):
    @functools.wraps(func)
    def wrapper(*args, **kwargs):
        # 새로운 함수 구현
        pass
    return wrapper
```

## 2. functools.partial

두 번째로 알아볼 기능은 functools.partial입니다. 이 기능을 사용하면 하나 이상의 인수를 고정한 새로운 함수를 생성할 수 있습니다. 이 기능은 오버로딩과 비슷한 효과를 얻을 수 있으며, 인수를 일부 고정하여 부분 함수를 생성할 때 유용합니다. 다음은 functools.partial을 사용하여 부분 함수를 정의하는 예제입니다.

```python
import functools

def add(a, b, c):
    return a + b + c

add_5 = functools.partial(add, 5)
result = add_5(2, 3)  # 5 + 2 + 3 = 10
```
