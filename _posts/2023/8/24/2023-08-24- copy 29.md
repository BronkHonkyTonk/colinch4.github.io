Numpy의 Universal Functions (ufuncs)는 배열 요소에 대한 빠른 연산을 수행하는 함수입니다. 아래에는 몇 가지 ufuncs의 예제를 제공합니다.

1.  **덧셈 ufunc 예제:**

```python
import numpy as np

arr1 = np.array([1, 2, 3, 4])
arr2 = np.array([10, 20, 30, 40])

result = np.add(arr1, arr2)

print(result)
```
이 코드에서 `np.add()`는 두 배열 `arr1`과 `arr2`의 요소별로 덧셈을 수행합니다.

2.  **뺄셈 ufunc 예제:**

```python
import numpy as np

arr1 = np.array([10, 20, 30, 40])
arr2 = np.array([1, 2, 3, 4])

result = np.subtract(arr1, arr2)

print(result)
```
`np.subtract()` 함수는 두 배열 간의 요소별 뺄셈을 수행합니다.

3.  **곱셈 ufunc 예제:**

```python
import numpy as np

arr1 = np.array([1, 2, 3, 4])
arr2 = np.array([10, 20, 30, 40])

result = np.multiply(arr1, arr2)

print(result)
```
`np.multiply()` 함수는 두 배열 간의 요소별 곱셈을 수행합니다.

4.  **나눗셈 ufunc 예제:**

```python
import numpy as np

arr1 = np.array([10, 20, 30, 40])
arr2 = np.array([2, 4, 6, 8])

result = np.divide(arr1, arr2)

print(result)
```
`np.divide()` 함수는 두 배열 간의 요소별 나눗셈을 수행합니다.

5.  **지수 및 로그 ufunc 예제:**

```python
import numpy as np

arr = np.array([1, 2, 3, 4])

# 지수 계산
result_exp = np.exp(arr)

# 자연로그 계산
result_log = np.log(arr)

print("지수 계산 결과:", result_exp)
print("자연로그 계산 결과:", result_log)
```
`np.exp()` 함수는 배열 요소의 지수 값을 계산하고, `np.log()` 함수는 배열 요소의 자연로그 값을 계산합니다.

6.  **삼각함수 ufunc 예제:**

```python
import numpy as np

arr = np.array([0, np.pi/4, np.pi/2, np.pi])

# 삼각함수 계산
result_sin = np.sin(arr)
result_cos = np.cos(arr)

print("사인 함수 결과:", result_sin)
print("코사인 함수 결과:", result_cos)
```
`np.sin()`과 `np.cos()` 함수는 배열의 요소에 대한 사인과 코사인 값을 계산합니다.

이러한 ufuncs는 배열의 요소에 대한 간단한 연산부터 복잡한 수학 함수까지 다양한 작업을 빠르게 수행할 수 있도록 도와줍니다. Numpy의 ufuncs를 활용하면 배열 연산을 효율적으로 수행할 수 있으며 코드를 간결하게 유지할 수 있습니다.
