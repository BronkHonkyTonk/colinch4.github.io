내장 함수 `input()`을 사용하여 사용자로부터 입력을 받는 예제를 제공합니다.

**예제 1: 사용자 이름 입력 받기**

```python
name = input("이름을 입력하세요: ")
print("안녕하세요, " + name + "!")
```
이 코드는 사용자에게 이름을 입력하도록 요청하고, 사용자가 입력한 이름을 화면에 출력합니다.

**예제 2: 숫자 입력 받기**

```python
age = input("나이를 입력하세요: ")
age = int(age)  # 입력된 문자열을 정수로 변환
print("당신은", age, "살입니다.")
```
이 코드는 사용자에게 나이를 입력하도록 요청하고, 입력된 문자열을 정수로 변환하여 나이를 출력합니다.

**주의:** `input()` 함수로 입력 받은 값은 기본적으로 문자열로 처리됩니다. 따라서 숫자 등의 다른 유형으로 사용하기 전에 필요한 유형으로 변환해야 할 수 있습니다.

**예제 3: 사용자 선택 받기**

```python
print("좋아하는 과일을 선택하세요:")
print("1. 사과")
print("2. 바나나")
print("3. 딸기")

choice = input("번호를 입력하세요: ")

if choice == "1":
    print("사과를 선택했습니다.")
elif choice == "2":
    print("바나나를 선택했습니다.")
elif choice == "3":
    print("딸기를 선택했습니다.")
else:
    print("잘못된 번호를 입력했습니다.")
```
이 코드는 사용자에게 좋아하는 과일을 선택하도록 메뉴를 표시하고, 사용자의 선택을 받아 해당하는 과일을 출력합니다.

`input()` 함수를 사용하면 사용자와 상호작용하며 입력을 받을 수 있으므로 다양한 종류의 사용자 인터페이스를 구현하는 데 사용됩니다.
