함수는 다음과 같은 기본 구조를 가지고 있습니다:

```python
def 함수이름(매개변수1, 매개변수2, ...):
    # 함수 내용
    # ...
    return 반환값
```
이 구조를 요소별로 설명해 보겠습니다:

1.  **def 키워드**: 함수를 정의하기 위해 `def` 키워드를 사용합니다. 이 키워드는 파이썬에게 이것이 함수 정의임을 알려줍니다.
    
2.  **함수 이름**: 함수의 이름은 공백없이 문자와 밑줄(언더스코어)로 이루어져야 합니다. 함수 이름은 함수를 식별하는 데 사용됩니다.
    
3.  **매개변수 (Parameters)**: 함수에 입력값을 전달받을 매개변수를 정의합니다. 매개변수는 함수 내부에서 사용될 변수로, 함수 호출 시 인수(argument)를 전달받습니다. 매개변수는 괄호 `()` 안에 나열되며, 필요에 따라 여러 개의 매개변수를 정의할 수 있습니다.
    
4.  **콜론 (:)**: 함수 정의 헤더의 끝에는 콜론이 옵니다. 이 콜론은 함수 본문의 시작을 나타냅니다.
    
5.  **함수 내용 (Function Body)**: 함수 본문은 들여쓰기로 구분되는 부분으로, 실제로 함수가 수행할 작업을 정의합니다. 이 부분에는 코드 블록이 포함되며, 함수의 동작을 결정합니다.
    
6.  **return 문**: 함수가 값을 반환하는 경우, `return` 문을 사용하여 반환값을 지정합니다. `return` 문은 함수의 실행을 종료하고, 지정된 값을 함수 호출 지점으로 반환합니다. 함수가 값을 반환하지 않는 경우, `return` 문이 없거나 `return`만 사용됩니다.
    

함수의 구조를 이해하면 함수를 정의하고 호출할 수 있으며, 이를 통해 코드를 모듈화하고 재사용 가능한 조각으로 나눌 수 있습니다. 함수는 프로그램을 더 구조화하고 가독성을 향상시키며, 유지 보수와 협업을 용이하게 합니다.
