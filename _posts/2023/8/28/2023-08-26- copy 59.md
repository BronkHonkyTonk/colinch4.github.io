재귀 함수(Recursive Function)는 자신을 호출하는 함수로, 반복적인 작업을 처리할 때 유용합니다. 재귀 함수의 개념을 이해하고 예제를 살펴보겠습니다.

**재귀 함수의 개념**:

재귀 함수는 다음과 같은 특징을 가집니다:

1.  함수가 자기 자신을 호출합니다.
2.  재귀 함수는 종료 조건을 가져야 합니다. 종료 조건 없이 함수가 무한히 호출되면 스택 오버플로우와 같은 문제가 발생합니다.
3.  재귀 함수는 문제를 작은 부분으로 나누어 해결하고, 이를 결합하여 전체 문제를 해결합니다.

**예제: 팩토리얼 계산**

팩토리얼(factorial)은 양의 정수 n에 대해 n부터 1까지의 모든 정수를 곱한 값을 의미합니다. 팩토리얼은 재귀 함수를 사용하여 계산할 수 있습니다.

```python
def factorial(n):
    # 종료 조건: n이 0 또는 1일 때 1을 반환
    if n == 0 or n == 1:
        return 1
    else:
        # 재귀 호출: n! = n * (n-1)!
        return n * factorial(n - 1)

# 팩토리얼 계산 예제
result = factorial(5)
print("5! =", result)
```python
이 코드에서 `factorial()` 함수는 자기 자신을 호출하여 팩토리얼 값을 계산합니다. 종료 조건은 n이 0 또는 1일 때로 설정되어 무한 재귀 호출을 방지합니다. 함수는 작은 부분 문제를 해결하고 이를 결합하여 전체 팩토리얼 값을 계산합니다.

재귀 함수는 재귀 호출 스택을 사용하므로 큰 입력에 대해서는 스택 오버플로우 문제에 주의해야 합니다. 따라서 반복문을 사용하여 같은 작업을 수행하는 경우와 재귀 함수를 사용하는 경우를 비교하여 선택해야 합니다.
