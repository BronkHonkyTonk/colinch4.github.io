내장 함수 `input()`을 사용하여 사용자로부터 입력을 받는 예제를 제공합니다.

**예제: 사용자 입력 받기**

```python
name = input("이름을 입력하세요: ")
print("안녕하세요, " + name + "님!")
```python
이 코드는 사용자에게 이름을 입력하도록 요청하고, 입력된 이름을 변수 `name`에 저장한 후 환영 메시지를 출력합니다.

사용자에게 입력을 받을 때 `input()` 함수를 사용합니다. `input()` 함수는 사용자가 입력한 문자열을 반환합니다. 이 입력된 값은 변수에 저장하거나 다양한 연산에 활용할 수 있습니다.

사용자 입력을 받을 때 프롬프트 메시지를 출력하면 사용자에게 어떤 입력을 기대하는지 명확하게 알려줄 수 있습니다.
