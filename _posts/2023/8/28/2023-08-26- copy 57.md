지역 변수(Local Variable)와 전역 변수(Global Variable)는 변수의 범위(scope)에 따라 차이가 있습니다. 이 두 유형의 변수는 변수가 선언된 위치와 범위에서 차이가 있습니다.

1.  **지역 변수 (Local Variable)**:
    
    -   지역 변수는 함수 내부에서 선언된 변수로, 함수 내에서만 접근 가능합니다.
    -   함수가 호출될 때 생성되고 함수가 종료될 때 소멸합니다. 따라서 함수 내부에서만 유효한 범위를 가집니다.
    -   다른 함수 내에서 동일한 이름의 변수를 사용해도 서로 영향을 주지 않습니다.
    -   지역 변수는 함수 내부에서만 접근 가능하므로 함수 외부에서는 액세스할 수 없습니다.
    
    예제:
    
```
    def my_function():
        x = 10  # 지역 변수 x
        print(x)
    
    my_function()
    # print(x)  # 오류: 함수 외부에서 x에 접근할 수 없음
```    
2.  **전역 변수 (Global Variable)**:
    
    -   전역 변수는 함수 외부에서 선언된 변수로, 프로그램 어디에서나 접근 가능합니다.
    -   프로그램이 시작될 때 생성되고 프로그램이 종료될 때 소멸합니다.
    -   여러 함수에서 동일한 이름의 전역 변수를 참조할 수 있으며, 이 변수의 값을 변경하면 모든 함수에서 변경된 값을 공유합니다.
    
    예제:
    
```
    global_var = 20  # 전역 변수
    
    def func1():
        print(global_var)  # 전역 변수에 접근 가능
    
    def func2():
        global global_var  # 전역 변수를 함수 내에서 사용하려면 global 키워드를 사용해야 함
        global_var = 30  # 전역 변수 값 변경
        print(global_var)
    
    func1()
    func2()
    print(global_var)  # func2에서 변경된 값이 출력됨
```    

전역 변수는 프로그램의 여러 부분에서 공유되기 때문에 사용할 때 주의가 필요합니다. 변수의 범위를 잘 이해하고, 필요한 경우 함수 내에서 전역 변수를 수정하려면 `global` 키워드를 사용하여 명시적으로 선언해야 합니다.
