내장 함수 `write()`를 사용하여 파일에 데이터를 쓰는 예제를 제공합니다. 파일은 쓰기 모드 `"w"`로 열어야 합니다.

**예제: 파일에 데이터 쓰기**

```python
# 파일을 쓰기 모드로 열기
file = open("example.txt", "w")

# 파일에 데이터 쓰기
file.write("안녕하세요, 파일에 쓰는 예제입니다.")
file.write("\n")  # 새로운 줄 추가
file.write("두 번째 줄입니다.")

# 파일 닫기
file.close()
```python
이 코드는 `"example.txt"` 파일을 쓰기 모드 `"w"`로 열고, `write()` 메서드를 사용하여 파일에 문자열을 씁니다. `write()` 메서드를 호출할 때마다 데이터가 파일에 추가됩니다. 또한 `"\n"`을 사용하여 새로운 줄을 추가할 수 있습니다.

파일을 열고 데이터를 쓴 후에는 `close()` 메서드를 사용하여 파일을 닫아야 합니다. 파일을 닫으면 변경 내용이 저장됩니다.

**참고:** 파일을 다룰 때 `with` 문을 사용하여 파일을 열면, 블록이 종료될 때 자동으로 파일을 닫을 수 있습니다. 이렇게 하면 파일을 닫는 것을 잊어버리는 실수를 방지할 수 있습니다.
