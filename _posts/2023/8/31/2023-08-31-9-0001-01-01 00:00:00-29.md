Python에서 기본 산술 연산자는 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/) 입니다. 각각의 연산자는 다음과 같이 사용할 수 있습니다.

## 덧셈 (+)

덧셈 연산자는 두 개의 숫자를 더하는 연산자입니다.

```python
# 덧셈 연산자 사용 예

a = 3
b = 4

c = a + b

print(c)
# 7
```

## 뺄셈 (-)

뺄셈 연산자는 두 개의 숫자를 빼는 연산자입니다.

```python
# 뺄셈 연산자 사용 예

a = 3
b = 4

c = a - b

print(c)
# -1
```

## 곱셈 (*)

곱셈 연산자는 두 개의 숫자를 곱하는 연산자입니다.

```python
# 곱셈 연산자 사용 예

a = 3
b = 4

c = a * b

print(c)
# 12
```

## 나눗셈 (/)

나눗셈 연산자는 두 개의 숫자를 나누는 연산자입니다.

```python
# 나눗셈 연산자 사용 예

a = 3
b = 4

c = a / b

print(c)
# 0.75
```

Python에서 기본 산술 연산자는 덧셈(+), 뺄셈(-), 곱셈(*), 나눗셈(/) 입니다. 각각의 연산자는 위에서 설명한 대로 사용할 수 있습니다. 이러한 기본 산술 연산자는 프로그래밍을 하는 데 매우 중요한 역할을 합니다.
