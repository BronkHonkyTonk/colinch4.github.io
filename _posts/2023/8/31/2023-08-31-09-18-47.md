조건문은 프로그램이 어떤 작업을 수행하기 전에 특정 조건이 참인지 거짓인지를 확인하기 위해 사용됩니다. 프로그램이 조건을 확인하면, 그에 따라 다른 작업을 수행할 수 있습니다. Python에서는 **if**, **elif**, **else**라는 세 가지 조건문을 사용할 수 있습니다. 

## if 조건문

**if** 조건문은 특정 조건이 참인 경우 특정 작업을 수행하는 것을 의미합니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```python
if x > 0:
    print("x is positive")
```

위 예제에서, 변수 x가 0보다 큰 경우에만 출력문이 실행됩니다.

## elif 조건문

**elif** 조건문은 **if** 조건문 뒤에 따라 나오는 조건문입니다. **elif** 조건문은 여러 조건 중 하나를 확인하기 위해 사용됩니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```python
if x > 0:
    print("x is positive")
elif x < 0:
    print("x is negative")
```

위 예제에서, 변수 x가 0보다 큰 경우에는 첫 번째 출력문이 실행되고, 변수 x가 0보다 작은 경우에는 두 번째 출력문이 실행됩니다.

## else 조건문

**else** 조건문은 **if** 또는 **elif** 조건문 뒤에 따라 나오는 조건문입니다. **else** 조건문은 모든 조건이 거짓인 경우 실행됩니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```python
if x > 0:
    print("x is positive")
elif x < 0:
    print("x is negative")
else:
    print("x is zero")
```

위 예제에서, 변수 x가 0보다 큰 경우에는 첫 번째 출력문이 실행되고, 변수 x가 0보다 작은 경우에는 두 번째 출력문이 실행되고, 변수 x가 0인 경우에는 세 번째 출력문이 실행됩니다.

## 결론

조건문은 프로그램이 어떤 작업을 수행하기 전에 특정 조건이 참인지 거짓인지를 확인하기 위해 사용됩니다. Python에서는 **if**, **elif**, **else**라는 세 가지 조건문을 사용할 수 있습니다. **if** 조건문은 특정 조건이 참인 경우 특정 작업을 수행하는 것을 의미합니다. **elif** 조건문은 **if** 조건문 뒤에 따라 나오는 조건문입니다. **else** 조건문은 모든 조건이 거짓인 경우 실행됩니다.
