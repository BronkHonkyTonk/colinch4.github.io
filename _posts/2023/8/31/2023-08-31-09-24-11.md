Python에서 세트 연산은 두 개 이상의 세트를 다루는 데 사용됩니다. 이는 교집합, 합집합, 차집합 등의 연산을 수행하는 데 사용됩니다. 이러한 연산은 다음과 같이 수행할 수 있습니다.

## 교집합

교집합은 두 개의 세트가 공통적으로 가지고 있는 원소들의 집합을 말합니다. 예를 들어, 다음과 같은 두 개의 세트가 있다고 가정해 보겠습니다.

```python
A = {1, 2, 3, 4, 5}
B = {3, 4, 5, 6, 7}
```

그러면 교집합은 다음과 같이 구할 수 있습니다.

```python
A & B = {3, 4, 5}
```

## 합집합

합집합은 두 개의 세트가 가지고 있는 모든 원소들의 집합을 말합니다. 예를 들어, 다음과 같은 두 개의 세트가 있다고 가정해 보겠습니다.

```python
A = {1, 2, 3, 4, 5}
B = {3, 4, 5, 6, 7}
```

그러면 합집합은 다음과 같이 구할 수 있습니다.

```python
A | B = {1, 2, 3, 4, 5, 6, 7}
```

## 차집합

차집합은 두 개의 세트가 가지고 있는 원소들 중 한 쪽 세트에만 있는 원소들의 집합을 말합니다. 예를 들어, 다음과 같은 두 개의 세트가 있다고 가정해 보겠습니다.

```python
A = {1, 2, 3, 4, 5}
B = {3, 4, 5, 6, 7}
```

그러면 차집합은 다음과 같이 구할 수 있습니다.

```python
A - B = {1, 2}
B - A = {6, 7}
```

## 결론

Python에서 세트 연산은 교집합, 합집합, 차집합 등의 연산을 수행하는 데 사용됩니다. 이는 두 개 이상의 세트를 다루는 데 사용됩니다. 이러한 연산은 다음과 같이 수행할 수 있습니다. 교집합은 두 개의 세트가 공통적으로 가지고 있는 원소들의 집합을 말합니다. 합집합은 두 개의 세트가 가지고 있는 모든 원소들의 집합을 말합니다. 차집합은 두 개의 세트가 가지고 있는 원소들 중 한 쪽 세트에만 있는 원소들의 집합을 말합니다.
