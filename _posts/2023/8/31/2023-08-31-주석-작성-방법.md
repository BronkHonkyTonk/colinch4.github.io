Python은 인터프리터 언어로, 코드를 실행하기 전에 컴파일하지 않기 때문에 주석이 매우 중요합니다. 주석은 코드를 이해하기 쉽게 만들어줍니다. 이 블로그 글에서는 Python에서 주석을 작성하는 방법에 대해 알아보겠습니다.

## 주석 작성 방법

Python에서 주석은 `#` 문자로 시작합니다. 예를 들어 다음과 같이 작성할 수 있습니다.

```python
# This is a comment
```

주석은 코드를 이해하기 쉽게 만들어줍니다. 예를 들어 다음과 같이 주석을 사용하여 코드를 설명할 수 있습니다.

```python
# This is a function to calculate the sum of two numbers
def add(a, b):
    return a + b
```

## 줄 주석

Python에서는 줄 주석을 사용할 수 있습니다. 줄 주석은 한 줄에 하나의 주석만 작성할 수 있습니다. 예를 들어 다음과 같이 작성할 수 있습니다.

```python
# This is a line comment
```

## 블록 주석

Python에서는 블록 주석을 사용할 수도 있습니다. 블록 주석은 한 줄 이상의 주석을 작성할 수 있습니다. 예를 들어 다음과 같이 작성할 수 있습니다.

```python
"""
This is a block comment.
It can span multiple lines.
"""
```

## 결론

Python에서 주석을 작성하는 방법은 다양합니다. 주석은 코드를 이해하기 쉽게 만들어줍니다. 따라서 주석을 사용하는 것이 좋습니다.
