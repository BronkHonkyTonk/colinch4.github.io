Python은 다양한 데이터 타입을 지원하는 강력한 프로그래밍 언어입니다. 변수는 프로그램의 상태를 저장하기 위해 사용되며, 데이터 타입은 변수에 저장되는 값의 종류를 정의합니다. 이 둘은 Python 프로그래밍의 기본 개념이므로, 이 블로그 글을 통해 변수와 데이터 타입에 대해 알아보고 이해해보겠습니다.

## 변수

변수는 프로그램의 상태를 저장하기 위해 사용됩니다. 변수는 값을 저장하기 위해 사용하는 이름이며, 값이 변경될 수 있습니다. 변수는 다음과 같이 선언할 수 있습니다.

```python
# 변수 선언
my_variable = 10
```

## 데이터 타입

데이터 타입은 변수에 저장되는 값의 종류를 정의합니다. Python은 다음과 같은 데이터 타입을 지원합니다.

- **숫자** : 정수, 실수 등
- **문자열** : 문자 배열
- **리스트** : 순서가 있는 값의 배열
- **튜플** : 순서가 있는 값의 배열
- **딕셔너리** : 키-값 쌍으로 이루어진 배열
- **집합** : 유일한 값의 배열

예를 들어, 다음과 같이 변수에 데이터 타입을 지정할 수 있습니다.

```python
# 변수 선언
my_variable = 10 # 정수
my_variable = "Hello" # 문자열
my_variable = [1, 2, 3] # 리스트
my_variable = (1, 2, 3) # 튜플
my_variable = {'name': 'John', 'age': 20} # 딕셔너리
my_variable = {1, 2, 3} # 집합
```

## 결론

변수와 데이터 타입은 Python 프로그래밍의 기본 개념입니다. 변수는 프로그램의 상태를 저장하기 위해 사용되며, 데이터 타입은 변수에 저장되는 값의 종류를 정의합니다. Python은 숫자, 문자열, 리스트, 튜플, 딕셔너리, 집합 등 다양한 데이터 타입을 지원합니다.
