Python은 다양한 방법으로 변수에 값을 할당할 수 있습니다. 가장 기본적인 방법은 단일 값을 할당하는 것입니다.

```python
x = 5
```

하지만, 변수에 여러 값을 할당하고 싶은 경우가 있습니다. 이럴 때는 **튜플**을 사용하면 됩니다.

```python
x = (5, 10, 15)
```

튜플을 사용하면 변수 x에는 5, 10, 15이라는 세 개의 값이 할당됩니다. 각 값은 인덱스를 사용하여 접근할 수 있습니다.

```python
print(x[0]) # 5
print(x[1]) # 10
print(x[2]) # 15
```

또한, 변수에 여러 값을 할당하고 싶은 경우 **리스트**를 사용할 수도 있습니다.

```python
x = [5, 10, 15]
```

리스트를 사용하면 변수 x에는 5, 10, 15이라는 세 개의 값이 할당됩니다. 각 값은 인덱스를 사용하여 접근할 수 있습니다.

```python
print(x[0]) # 5
print(x[1]) # 10
print(x[2]) # 15
```

변수에 여러 값을 할당하기 위해 튜플과 리스트를 사용할 수 있습니다. 튜플과 리스트는 다양한 방법으로 활용할 수 있으며, 이 두 가지 방법을 사용하면 변수에 여러 값을 할당하는 데 도움이 됩니다.
