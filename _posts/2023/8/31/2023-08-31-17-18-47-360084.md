비교 연산자를 사용하여 조건문을 작성하는 것은 프로그래밍에서 중요한 부분입니다. 이 글에서는 Python에서 조건문과 비교 연산자를 어떻게 활용하는지 예제를 통해 알아보겠습니다.

## 1. if 문을 사용한 간단한 조건문

가장 기본적인 조건문은 `if`문입니다. 이 문장은 조건이 참(True)인 경우에만 실행됩니다. 예를 들어, 다음 코드는 숫자가 5보다 큰 경우에만 "숫자는 5보다 큽니다"라는 메시지를 출력하는 예제입니다.

```python
number = 7

if number > 5:
    print("숫자는 5보다 큽니다")
```

위의 코드를 실행하면 "숫자는 5보다 큽니다"라는 메시지가 출력됩니다.

## 2. else 문을 사용한 조건문

`else`문은 `if`문의 조건이 거짓(False)인 경우에 실행됩니다. 예를 들어, 다음 코드는 숫자가 5보다 크면 "숫자는 5보다 큽니다" 메시지를 출력하고, 그렇지 않은 경우에는 "숫자는 5보다 작거나 같습니다" 메시지를 출력하는 예제입니다.

```python
number = 3

if number > 5:
    print("숫자는 5보다 큽니다")
else:
    print("숫자는 5보다 작거나 같습니다")
```

위의 코드를 실행하면 "숫자는 5보다 작거나 같습니다"라는 메시지가 출력됩니다.

## 3. elif 문을 사용한 다중 조건문

`elif`문은 여러 개의 조건을 검사할 때 사용됩니다. 예를 들어, 다음 코드는 숫자가 5보다 큰 경우 "숫자는 5보다 큽니다" 메시지를 출력하고, 숫자가 5보다 작은 경우 "숫자는 5보다 작습니다" 메시지를 출력하며, 그렇지 않은 경우에는 "숫자는 5입니다" 메시지를 출력하는 예제입니다.

```python
number = 2

if number > 5:
    print("숫자는 5보다 큽니다")
elif number < 5:
    print("숫자는 5보다 작습니다")
else:
    print("숫자는 5입니다")
```

위의 코드를 실행하면 "숫자는 5보다 작습니다"라는 메시지가 출력됩니다.

## 4. 비교 연산자 활용하기

비교 연산자는 조건문에서 변수나 값을 서로 비교할 때 사용됩니다. Python에서 사용되는 주요 비교 연산자는 다음과 같습니다.

- `>`: 크다
- `<`: 작다
- `>=`: 크거나 같다
- `<=`: 작거나 같다
- `==`: 같다
- `!=`: 같지 않다

아래 예제는 변수와 비교 연산자를 사용한 조건문의 예제입니다.

```python
x = 10

if x > 5:
    print("x는 5보다 큽니다")
if x < 20:
    print("x는 20보다 작습니다")
if x == 10:
    print("x는 10입니다")
```

위의 코드를 실행하면 "x는 5보다 큽니다"와 "x는 10입니다"라는 메시지가 출력됩니다.

이상으로 Python의 조건문과 비교 연산자 활용 예제에 대해 알아보았습니다. 조건문을 사용하여 프로그램의 흐름을 제어하는 것은 매우 중요한 기술이므로 앞으로 더 다양한 활용 사례를 탐구해보시기 바랍니다.
