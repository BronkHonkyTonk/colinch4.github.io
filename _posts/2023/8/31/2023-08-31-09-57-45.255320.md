Python은 간결하고 읽기 쉬운 코드를 작성하는 것을 중요시합니다. 그래서, 변수명의 스타일을 정하기 위해 **Python Enhancement Proposal (PEP) 8**이라는 가이드라인이 있습니다. PEP 8은 변수명을 작성할 때 지켜야 할 규칙들을 제시하고 있습니다. 

## 변수명 작성 규칙

1. 변수명은 소문자로 작성합니다.
2. 단어를 연결할 때는 밑줄(_)를 사용합니다.
3. 변수명은 의미 있는 단어들로 구성합니다.
4. 상수는 모두 대문자로 작성합니다.

## 예시

아래는 PEP 8에 따라 변수명을 작성한 예시입니다.

```python
# 변수명 예시
name = "John"
age = 20
is_male = True

# 상수명 예시
PI = 3.14
GRAVITY = 9.8
```

PEP 8을 준수하면 코드를 읽고 쓰는데 도움이 됩니다. 변수명의 스타일을 가이드라인에 따라 작성하는 것은 프로그래밍을 하는 데 있어서 중요한 요소입니다.
