프로그래밍을 하다보면 변수명을 짓는 것이 가장 중요합니다. 변수명은 단수형과 복수형이 있습니다. 이 두가지는 어떤 차이가 있는지 알아보겠습니다.

## 단수형

단수형은 하나의 객체를 가리키는 변수명을 말합니다. 객체가 하나이기 때문에 변수명을 단수형으로 짓는 것이 일반적입니다. 예를 들어, 다음과 같이 단수형 변수명을 사용할 수 있습니다.

```python
user = 'John'
```

## 복수형

복수형은 여러 객체를 가리키는 변수명을 말합니다. 객체가 여러 개이기 때문에 변수명을 복수형으로 짓는 것이 일반적입니다. 예를 들어, 다음과 같이 복수형 변수명을 사용할 수 있습니다.

```python
users = ['John', 'Alice', 'Bob']
```

## 결론

변수명을 짓는 것은 프로그래밍을 하는데 있어 가장 중요한 요소입니다. 변수명은 단수형과 복수형이 있습니다. 단수형은 하나의 객체를 가리키고, 복수형은 여러 객체를 가리킵니다. 변수명을 짓는 데 있어 단수형과 복수형을 적절히 사용해야 합니다.
