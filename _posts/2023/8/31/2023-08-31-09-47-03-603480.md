코드 재사용은 프로그래밍을 하는 데 있어 중요한 개념입니다. 이는 같은 작업을 반복하지 않고 다른 작업에 사용할 수 있는 코드를 작성하는 것을 말합니다. 이는 DRY (Don't Repeat Yourself) 원칙과 관련이 있습니다. DRY 원칙은 중복된 코드를 피하고 단일 자리에서 코드를 업데이트하는 것을 의미합니다.

코드 재사용을 하는 가장 좋은 방법 중 하나는 함수를 사용하는 것입니다. 함수는 입력을 받고 출력을 내놓는 작은 기능의 단위입니다. 함수는 여러 번 사용할 수 있으며, 코드를 작성할 때 자주 사용합니다.

다음은 간단한 함수를 사용하는 예제입니다.

```python
def add_two_numbers(a, b):
    return a + b

# 함수 사용
sum = add_two_numbers(2, 3)
print(sum) # 5
```

위 예제에서 `add_two_numbers` 함수는 두 개의 숫자를 받아 더한 값을 반환합니다. 이 함수는 다른 작업에서도 사용할 수 있습니다. 

코드 재사용은 프로그래밍을 하는 데 있어 중요한 개념입니다. 이는 코드를 작성하고 유지보수하는 데 도움이 됩니다. DRY 원칙을 준수하면 코드를 작성하고 유지보수하는 데 많은 시간을 절약할 수 있습니다.
