Python에서 변수의 메모리 주소를 확인하는 방법은 다양합니다. 이 블로그 글에서는 `id()` 함수를 사용하여 변수의 메모리 주소를 확인하는 방법에 대해 알아보겠습니다.

## `id()` 함수

`id()` 함수는 객체의 고유한 주소 값을 반환합니다. 이는 객체가 메모리에 저장된 주소를 나타냅니다.

```python
a = 10
print(id(a))
```

위 코드를 실행하면 다음과 같은 결과가 나옵니다.

```
140725509022992
```

따라서, `a` 변수는 메모리 주소 `140725509022992`에 저장되어 있음을 알 수 있습니다.

## 결론

Python에서 변수의 메모리 주소를 확인하는 방법으로 `id()` 함수를 사용할 수 있습니다. `id()` 함수는 객체의 고유한 주소 값을 반환합니다. 따라서, 변수가 메모리에 저장된 주소를 확인하기 위해 `id()` 함수를 사용할 수 있습니다.
