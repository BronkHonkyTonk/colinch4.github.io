파이썬에서의 루프를 구현하기 위해 `for`와 `while` 두 가지 유형을 사용할 수 있습니다. 각각을 살펴보겠습니다.

### for 루프

`for` 루프는 주어진 범위 내에서 항목을 반복하면서 코드 블록을 실행합니다. 일반적으로 시퀀스(예: 리스트, 튜플, 문자열)의 각 항목을 반복할 때 사용됩니다.

```python
for item in sequence:
    # 실행할 코드 블록
    print(item)
```

위의 예제에서 `item`은 시퀀스의 각 항목을 나타내는 변수입니다. 루프가 실행되는 동안 코드 블록이 `item` 변수를 사용하여 작업을 수행합니다.

### while 루프

`while` 루프는 주어진 조건이 참인 동안 코드 블록을 반복적으로 실행합니다. 조건은 루프의 시작 부분에 배치되며, 매 반복마다 확인됩니다.

```python
while condition:
    # 실행할 코드 블록
    print("Hello")
```

위의 예제에서 `condition`은 루프가 반복되는 동안 검사할 조건을 나타냅니다. `condition`이 참인 경우, 코드 블록이 반복적으로 실행됩니다.

루프에서 실행할 코드 블록은 위와 같이 구현할 수 있습니다. 이를 통해 반복적인 작업을 효율적으로 수행하고 코드의 재사용성과 가독성을 높일 수 있습니다.
