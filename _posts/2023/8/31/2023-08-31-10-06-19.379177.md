프로그래밍 언어는 사람이 이해하기 쉽게 만들어진 것이 아니라, 컴퓨터가 이해하기 쉽게 만들어진 것입니다. 그래서 프로그래밍 언어는 사람이 이해하기 어려운 숫자나 문자를 사용하기도 합니다. 그러나 이러한 숫자나 문자는 사람이 이해하기 어려운 것이기 때문에, 프로그래밍 언어를 사용하는 사람들은 변수명의 숫자 대신 단어를 사용하는 것이 좋습니다. 

예를 들어, 다음과 같이 변수명의 숫자를 사용하는 것은 사람이 이해하기 어렵습니다.

```python
num1 = 10
num2 = 20
```

변수명의 숫자 대신 단어를 사용하면 다음과 같이 이해하기 쉬운 코드를 작성할 수 있습니다.

```python
first_number = 10
second_number = 20
```

또한, 변수명의 숫자 대신 단어를 사용하면 다음과 같이 코드의 의미를 더 명확하게 만들 수 있습니다.

```python
number_of_students = 10
number_of_teachers = 20
```

변수명의 숫자 대신 단어를 사용하는 것은 프로그래밍 언어를 사용하는 사람들에게 매우 중요합니다. 변수명의 숫자 대신 단어를 사용하면 코드를 이해하기 쉽고, 코드를 작성하기도 쉽게 됩니다. 따라서 프로그래밍 언어를 사용하는 사람들은 변수명의 숫자 대신 단어를 사용하는 것을 권장합니다.
