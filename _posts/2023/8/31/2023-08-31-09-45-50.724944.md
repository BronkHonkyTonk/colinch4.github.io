Python에서 변수는 값을 저장하고 참조하기 위해 사용됩니다. 변수는 이름으로 식별되며, 값을 할당하는 과정에서 이름을 지정합니다. 변수는 다른 언어에서 사용하는 개념과 비슷하지만, Python에서는 식별자 규칙이 있습니다. 이 글에서는 변수 할당과 식별자 규칙에 대해 알아보겠습니다.

## 변수 할당

Python에서 변수는 값을 할당하는 과정에서 생성됩니다. 변수는 값을 할당하기 위해 사용되며, 이름을 지정하는 과정에서 식별자 규칙이 적용됩니다. 변수는 다음과 같이 할당할 수 있습니다.

```python
# 변수 할당
name = "John"
age = 25
```

## 식별자 규칙

Python에서 식별자는 변수, 함수, 클래스 등을 식별하기 위해 사용됩니다. 식별자는 다음과 같은 규칙을 따라야 합니다.

1. 식별자는 반드시 영문자, 숫자, 밑줄(_)로 시작해야 합니다.
2. 식별자는 대소문자를 구분합니다.
3. 예약어는 사용할 수 없습니다.

예시를 통해 식별자 규칙을 알아보겠습니다.

```python
# 식별자 규칙 예시
my_name = "John" # 올바른 식별자
myName = "John" # 올바른 식별자
_my_name = "John" # 올바른 식별자
my-name = "John" # 잘못된 식별자
1name = "John" # 잘못된 식별자
```

## 결론

Python에서 변수는 값을 저장하고 참조하기 위해 사용됩니다. 변수는 값을 할당하는 과정에서 이름을 지정하며, 식별자 규칙이 적용됩니다. 식별자는 변수, 함수, 클래스 등을 식별하기 위해 사용되며, 영문자, 숫자, 밑줄로 시작하고 대소문자를 구분하며, 예약어는 사용할 수 없습니다.
