비교 연산자는 두 개의 값을 비교하고 그 결과를 논리값(True 또는 False)으로 반환하는 연산자입니다. Python에서는 다음과 같은 비교 연산자를 사용할 수 있습니다.

* **==** : 두 값이 같은지 비교합니다.
* **!=** : 두 값이 다른지 비교합니다.
* **<** : 왼쪽 값이 오른쪽 값보다 작은지 비교합니다.
* **>** : 왼쪽 값이 오른쪽 값보다 큰지 비교합니다.
* **<=** : 왼쪽 값이 오른쪽 값보다 작거나 같은지 비교합니다.
* **>=** : 왼쪽 값이 오른쪽 값보다 크거나 같은지 비교합니다.

이러한 비교 연산자를 사용하면 다양한 조건문을 작성할 수 있습니다. 예를 들어 다음과 같이 두 수가 같은지 비교하는 코드를 작성할 수 있습니다.

```python
a = 5
b = 5

if a == b:
    print("a와 b는 같습니다.")
```

결과는 다음과 같습니다.

```
a와 b는 같습니다.
```

또한 다음과 같이 두 수가 다른지 비교하는 코드를 작성할 수도 있습니다.

```python
a = 5
b = 10

if a != b:
    print("a와 b는 다릅니다.")
```

결과는 다음과 같습니다.

```
a와 b는 다릅니다.
```

비교 연산자는 조건문을 작성할 때 매우 유용합니다. 따라서 비교 연산자를 잘 이해하고 사용하는 것이 중요합니다.
