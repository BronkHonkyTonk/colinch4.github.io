Python에서 모듈과 라이브러리를 가져오는 것은 간단합니다. `import` 키워드를 사용하면 됩니다. 

## 모듈 가져오기

모듈은 하나의 파일 내에 여러 개의 함수나 변수를 모아놓은 것입니다. 하나의 파일 내에 여러 개의 함수나 변수를 모아놓으면 같은 기능을 하는 함수나 변수들을 한 번에 관리하기 쉽게 됩니다. 

모듈을 가져오기 위해서는 `import` 키워드를 사용합니다. 예를 들어 `math` 모듈을 가져오고 싶다면 다음과 같이 입력합니다.

```python
import math
```

이렇게 하면 `math` 모듈에 있는 함수들을 사용할 수 있게 됩니다. 예를 들어 `math` 모듈에 있는 `sqrt` 함수를 사용하고 싶다면 다음과 같이 입력합니다.

```python
import math

x = math.sqrt(25)
print(x)
```

결과는 다음과 같습니다.

```
5.0
```

## 라이브러리 가져오기

라이브러리는 모듈들의 집합입니다. 라이브러리를 가져오기 위해서는 `import` 키워드를 사용합니다. 예를 들어 `numpy` 라이브러리를 가져오고 싶다면 다음과 같이 입력합니다.

```python
import numpy
```

이렇게 하면 `numpy` 라이브러리에 있는 함수들을 사용할 수 있게 됩니다. 예를 들어 `numpy` 라이브러리에 있는 `array` 함수를 사용하고 싶다면 다음과 같이 입력합니다.

```python
import numpy

x = numpy.array([1, 2, 3, 4, 5])
print(x)
```

결과는 다음과 같습니다.

```
[1 2 3 4 5]
```

## 결론

Python에서 모듈과 라이브러리를 가져오는 것은 간단합니다. `import` 키워드를 사용하면 됩니다. 모듈은 하나의 파일 내에 여러 개의 함수나 변수를 모아놓은 것이고, 라이브러리는 모듈들의 집합입니다. 각각을 가져오기 위해서는 `import` 키워드를 사용합니다.
