프로그램에서 특정한 상황에 따라 다른 동작이 이루어져야 할 때, 조건문(condition)을 사용하여 프로그램의 흐름을 제어할 수 있습니다. Python은 다양한 조건문을 제공하므로, 다양한 상황에 맞게 효과적으로 프로그램을 작성할 수 있습니다.

## if문

가장 간단한 조건문은 `if`문입니다. `if`문은 주어진 조건이 참(True)인 경우에만 특정한 동작을 수행합니다. 예를 들어, 어떤 숫자가 양수인지 판별하는 프로그램을 작성해보겠습니다.

```python
num = 5

if num > 0:
    print("양수입니다.")
```

위의 코드에서는 `num` 변수가 0보다 큰지를 확인하고, 조건이 참이면 "양수입니다."라는 메시지를 출력합니다.

## if-else문

`if`문은 `else`문과 함께 사용하여 조건이 거짓(False)인 경우에도 다른 동작을 수행할 수 있습니다. 예를 들어, 양수인지 음수인지 판별하는 프로그램을 작성해보겠습니다.

```python
num = -2

if num > 0:
    print("양수입니다.")
else:
    print("음수입니다.")
```

위의 코드에서는 `num` 변수가 0보다 큰지를 확인하고, 조건이 참이면 "양수입니다."를 출력하고, 그렇지 않은 경우 "음수입니다."를 출력합니다.

## if-elif-else문

복잡한 조건문을 다룰 때는 `if-elif-else`문을 사용할 수 있습니다. 여러 개의 조건을 순차적으로 검사하고, 조건이 참인 경우에 해당하는 동작을 수행합니다. 예를 들어, 점수에 따라 학점을 부여하는 프로그램을 작성해보겠습니다.

```python
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print("학점은", grade, "입니다.")
```

위의 코드에서는 `score` 변수의 값에 따라 학점을 부여합니다. 각 조건을 순차적으로 검사하여 해당하는 학점을 `grade` 변수에 할당한 후, 결과를 출력합니다.

## 중첩된 조건문

`if`문은 다른 `if`문 내에 중첩하여 사용할 수 있습니다. 이를 통해 더 복잡한 조건문을 구성할 수 있습니다. 예를 들어, 어떤 숫자가 0보다 큰지, 작은지, 또는 0인지 판별하는 프로그램을 작성해보겠습니다.

```python
num = -5

if num > 0:
    print("양수입니다.")
else:
    if num < 0:
        print("음수입니다.")
    else:
        print("0입니다.")
```

위의 코드에서는 `num` 변수의 값에 따라 중첩된 `if`문을 통해 양수, 음수, 또는 0인지를 판별합니다.

조건문은 프로그램의 흐름을 제어하는 가장 기본적인 도구 중 하나입니다. 조건문을 적절하게 활용하여 다양한 상황에 맞는 동작을 수행하는 프로그램을 작성해보세요!
