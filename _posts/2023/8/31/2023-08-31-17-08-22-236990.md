파이썬은 많은 작업을 자동화하기 위해 루프와 반복문을 사용할 수 있는 강력한 기능을 제공합니다. 이를 통해 동일한 작업을 여러 번 반복하거나 조건이 충족될 때까지 동일한 작업을 계속 수행할 수 있습니다. 이번 블로그에서는 루프와 반복 횟수 제어에 대해 알아보겠습니다.

## 1. `for` 루프

`for` 루프는 시퀀스나 컬렉션의 각 요소에 대해 반복하는데 사용됩니다. 아래는 `for` 루프의 기본 구조입니다.

```python
for element in sequence:
    # 코드 블록
```

`sequence`는 반복해야하는 요소를 포함하는 시퀀스 또는 컬렉션입니다. `element`는 각 요소에 대한 변수로 사용됩니다. 코드 블록은 각 요소에 대해 실행되며 여러 줄의 코드를 포함할 수도 있습니다.

예를 들어, 1부터 5까지의 숫자를 출력하는 코드를 작성해보겠습니다.

```python
for num in range(1, 6):
    print(num)
```

결과는 다음과 같이 출력됩니다.

```
1
2
3
4
5
```

## 2. `while` 루프

`while` 루프는 특정 조건이 만족될 때까지 코드 블록을 반복적으로 실행합니다. 아래는 `while` 루프의 기본 구조입니다.

```python
while condition:
    # 코드 블록
```

`condition`은 실행되는 동안 평가해야하는 조건입니다. 조건이 `True`로 평가되면 코드 블록이 실행되고, 조건이 `False`로 평가되면 루프가 종료됩니다.

예를 들어, 1부터 5까지의 숫자를 출력하는 코드를 `while` 루프를 사용하여 작성해보겠습니다.

```python
num = 1
while num <= 5:
    print(num)
    num += 1
```

결과는 `for` 루프와 동일하게 출력됩니다.

```
1
2
3
4
5
```

## 3. 루프 제어문

루프에서 작업을 조작하려면 `break`와 `continue`와 같은 루프 제어문을 사용할 수 있습니다.

- `break`: 루프를 종료하고 다음 코드 블록으로 이동합니다.
- `continue`: 현재 반복을 건너뛰고 다음 반복으로 이동합니다.

예를 들어, 1부터 10까지의 숫자 중에서 짝수를 출력하는 코드를 작성해보겠습니다. `continue`를 사용하여 홀수일 때는 반복을 건너뛰도록 합니다.

```python
for num in range(1, 11):
    if num % 2 != 0:  # 홀수일 경우
        continue
    print(num)
```

결과는 다음과 같이 출력됩니다.

```
2
4
6
8
10
```

이제 파이썬에서 루프와 반복 횟수 제어를 할 수 있는 기본적인 개념에 대해 알아보았습니다. 이러한 기능을 효과적으로 사용하여 코드를 작성하면 반복적이고 복잡한 작업을 간단하게 처리할 수 있습니다.
