Python에서는 변수의 자료형을 확인하기 위해 `type()` 함수를 사용할 수 있습니다. 

`type()` 함수는 괄호 안에 변수를 입력하면 해당 변수의 자료형을 반환해줍니다. 

예를 들어, 다음과 같이 정수형 변수를 선언하고 `type()` 함수를 사용해 변수의 자료형을 확인해보겠습니다. 

```python
a = 10
print(type(a))
```

위 코드를 실행하면 다음과 같이 `<class 'int'>`가 출력됩니다. 이는 `a` 변수가 정수형 변수인 것을 의미합니다. 

```
<class 'int'>
```

또한, `type()` 함수는 문자열, 리스트, 튜플, 딕셔너리 등 다양한 자료형의 변수를 입력해도 정상적으로 동작합니다. 

예를 들어, 다음과 같이 문자열 변수를 선언하고 `type()` 함수를 사용해 변수의 자료형을 확인해보겠습니다. 

```python
b = "Hello World!"
print(type(b))
```

위 코드를 실행하면 다음과 같이 `<class 'str'>`이 출력됩니다. 이는 `b` 변수가 문자열 변수인 것을 의미합니다. 

```
<class 'str'>
```

따라서, `type()` 함수는 Python에서 변수의 자료형을 확인하기 위해 자주 사용되는 함수입니다.
