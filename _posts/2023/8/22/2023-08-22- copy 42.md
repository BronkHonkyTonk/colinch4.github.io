`clear()` 메서드는 리스트의 모든 원소를 제거하여 리스트를 빈 상태로 만드는 데 사용됩니다. 이를 통해 리스트의 내용을 비우고 초기화할 수 있습니다. 아래 예제를 통해 `clear()` 메서드의 사용법을 살펴보겠습니다.

**예제: `clear()` 메서드 사용**

```python
my_list = [1, 2, 3, 4, 5]

# 리스트의 모든 원소 제거
my_list.clear()

print(my_list)  # 출력: []
```
위의 예제에서 `my_list`에 `clear()` 메서드를 사용하여 모든 원소를 제거합니다. 결과적으로 리스트는 빈 리스트로 초기화됩니다.

`clear()` 메서드는 리스트를 비우고 초기화해야 할 때 사용됩니다. 리스트의 내용을 완전히 제거하고 빈 리스트로 만들어야 할 경우에 유용합니다.
