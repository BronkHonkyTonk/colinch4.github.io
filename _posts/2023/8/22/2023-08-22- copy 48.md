PEP 8는 파이썬 코드의 스타일 가이드로, 코드를 일관되고 가독성 있게 작성할 수 있도록 지침을 제공합니다. 리스트의 작성 방법에 관한 PEP 8 가이드를 따르는 예제를 보겠습니다.

**PEP 8에 따른 리스트 작성 방법:**

1.  리스트 내의 원소들은 `,`로 구분합니다.
2.  리스트 내의 원소들은 일반적으로 여러 줄에 나누어 작성하거나 한 줄에 작성할 수 있습니다. 긴 리스트는 여러 줄에 나누어 작성하는 것이 가독성을 높일 수 있습니다.
3.  리스트의 시작과 끝에는 `[`와 `]`를 사용합니다.

**예제: PEP 8에 따른 리스트 작성**

```python
# 여러 줄에 나누어 작성
fruits = [
    "apple",
    "banana",
    "orange",
    "grape"
]

# 한 줄에 작성
numbers = [1, 2, 3, 4, 5]

# 긴 리스트는 여러 줄에 나누어 작성
long_list = [
    "item_1",
    "item_2",
    "item_3",
    "item_4",
    "item_5",
    "item_6",
    "item_7",
    "item_8",
]

# 리스트를 함수 인자로 전달할 때
result = some_function([
    "arg1",
    "arg2",
    "arg3"
])
```
위의 예제는 PEP 8 스타일 가이드에 따라 리스트를 작성하는 방법을 보여줍니다. 코드의 가독성을 높이고 일관성을 유지하기 위해 스타일 가이드를 준수하는 것이 좋습니다.
