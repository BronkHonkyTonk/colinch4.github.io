Swift에서 배열을 오름차순으로 정렬하려면 `sort()` 메서드를 사용하거나 `sorted()` 함수를 호출하면 됩니다. 둘 다 배열의 요소를 기준에 따라 정렬된 새로운 배열을 반환합니다. 다음은 배열을 오름차순으로 정렬하는 예시입니다:

1.  `sort()` 메서드를 사용하는 방법:

```swift
var array = [3, 1, 4, 2, 5]
array.sort()
print(array) // 출력: [1, 2, 3, 4, 5]` 
```
위의 예시에서 `array`는 정렬할 배열입니다. `sort()` 메서드를 호출하여 배열을 오름차순으로 정렬합니다. 원본 배열이 변경되며, 정렬된 결과가 반영됩니다.

2.  `sorted()` 함수를 사용하는 방법:

```swift
let array = [3, 1, 4, 2, 5]
let sortedArray = array.sorted()
print(sortedArray) // 출력: [1, 2, 3, 4, 5]` 
```
위의 예시에서 `array`는 정렬할 배열입니다. `sorted()` 함수를 호출하여 배열을 오름차순으로 정렬한 새로운 배열을 반환합니다. 원본 배열은 변경되지 않습니다.

두 가지 방법 모두 배열의 요소를 오름차순으로 정렬할 수 있으며, 원하는 방법을 선택하여 사용하면 됩니다. `sort()` 메서드를 사용하면 원본 배열이 직접 변경되지만, `sorted()` 함수를 사용하면 원본 배열은 변경되지 않고 정렬된 새로운 배열을 반환합니다.
