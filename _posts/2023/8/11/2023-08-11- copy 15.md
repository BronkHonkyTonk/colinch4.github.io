배열에서 특정 인덱스의 요소를 가져오려면, 인덱스에 접근하여 해당 요소를 가져오면 됩니다. Swift에서 배열의 인덱스는 0부터 시작합니다. 다음은 배열에서 특정 인덱스의 요소를 가져오는 예시입니다:

```swift
let array = [1, 2, 3, 4, 5]
let index = 2

if index >= 0 && index < array.count {
    let element = array[index]
    print(element) // 출력: 3
} else {
    print("유효하지 않은 인덱스입니다.")
}` 
```
위의 예시에서 `array`는 원본 배열입니다. `index`는 가져오고자 하는 특정 인덱스입니다.

조건문을 사용하여 인덱스가 유효한 범위 내에 있는지 확인합니다. `index`가 0 이상이고 `array`의 요소 개수보다 작은지 검사합니다. 유효한 인덱스인 경우 해당 인덱스에 위치한 요소를 가져와 `element` 변수에 할당합니다.

`print(element)`를 사용하여 해당 요소를 출력합니다. 위의 예시에서는 2번 인덱스에 위치한 요소인 3을 출력합니다.

만약 인덱스가 유효하지 않은 경우, "유효하지 않은 인덱스입니다."라는 메시지를 출력합니다.

이와 같이 Swift에서는 배열의 특정 인덱스에 접근하여 해당 요소를 가져올 수 있습니다. 인덱스를 사용하여 배열의 원하는 위치에 있는 요소를 쉽게 가져와 활용할 수 있습니다.
