Foundation CSS는 반응형 웹 디자인을 쉽게 구현하기 위한 프론트엔드 프레임워크입니다. 이 글에서는 Foundation CSS의 주요 기능과 활용 방법에 대해 알아보겠습니다.

## 1. 그리드 시스템
Foundation CSS는 강력한 그리드 시스템을 제공하여 웹 페이지를 쉽게 구조화할 수 있습니다. 그리드 시스템은 컬럼과 로우를 조합하여 유연한 레이아웃을 만들 수 있도록 도와줍니다. 예를 들어, "row" 클래스를 사용하여 로우를 생성하고, "columns" 클래스로 컬럼을 지정할 수 있습니다.

```html
<div class="row">
  <div class="columns small-6">컬럼 1</div>
  <div class="columns small-6">컬럼 2</div>
</div>
```

## 2. 반응형 디자인
Foundation CSS는 모바일 기기부터 데스크탑까지 여러 장치에서 일관된 사용자 경험을 제공하기 위한 반응형 디자인을 지원합니다. 사전에 정의된 CSS 클래스를 사용하여 텍스트, 이미지, 레이아웃 등을 반응형하게 조정할 수 있습니다. 예를 들어, "hide-for-small-only" 클래스를 사용하면 작은 화면에서 요소를 감출 수 있습니다.

```html
<p class="hide-for-small-only">모바일에서는 보이지 않는 문장입니다.</p>
```

## 3. 컴포넌트
Foundation CSS는 다양한 컴포넌트를 제공하여 웹 페이지의 기능을 빠르게 구현할 수 있도록 도와줍니다. 예를 들어, 메뉴, 이미지 슬라이더, 탭 등의 컴포넌트가 포함되어 있습니다. 이러한 컴포넌트들은 사전에 정의된 CSS 클래스와 JavaScript를 사용하여 쉽게 구현할 수 있습니다.

## 4. 확장성
Foundation CSS는 확장성 있는 프레임워크로서, 컴포넌트를 추가하거나 사용자 정의 스타일을 적용하는 것이 가능합니다. 컴포넌트를 추가하려면 Foundation CSS의 Sass 버전을 사용하여 스타일 시트를 커스터마이징할 수 있습니다. 또한, JavaScript를 사용하여 웹 페이지에 동적인 기능을 추가할 수도 있습니다.

## 마무리
Foundation CSS는 웹 디자인과 개발을 위한 강력한 도구로서, 그리드 시스템, 반응형 디자인, 컴포넌트, 확장성 등 다양한 기능을 제공합니다. 이를 활용해 웹 페이지를 빠르게 구축할 수 있으며, 사용자 경험을 개선하는 데 도움이 됩니다.
