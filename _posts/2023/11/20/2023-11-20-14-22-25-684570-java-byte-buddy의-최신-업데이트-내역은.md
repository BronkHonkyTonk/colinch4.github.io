Java Byte Buddy는 Java 언어로 작성된 동적 클래스 생성 및 수정 라이브러리입니다. 최신 업데이트의 내용은 다음과 같습니다:

## 1.11.0 (2021년 4월 30일)

- Java 16과의 호환성 추가
- TypeReference 객체를 사용하여 메소드 시그니처에 대한 정보를 더 쉽게 제공할 수 있도록 개선
- Bug 수정 및 성능 향상

## 1.10.20 (2021년 3월 15일)

- Java 15과의 호환성 추가
- 클래스의 Generic Signature 추출을 위한 코드 변경
- Bug 수정 및 성능 개선

전체 업데이트 내역을 보려면 [Byte Buddy의 GitHub 레포지토리](https://github.com/raphw/byte-buddy/releases)를 확인해주시기 바랍니다.

Byte Buddy의 사용 방법과 예제 코드에 대한 자세한 내용은 [Byte Buddy 공식 문서](https://bytebuddy.net/#/tutorial)를 참조하실 수 있습니다.
