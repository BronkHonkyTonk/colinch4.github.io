Swift AcknowList는 오픈 소스 라이브러리로, 오픈 소스 라이브러리나 다른 기여자들에 대한 감사의 표시를 자동으로 생성해주는 도구입니다. 이를 통해 프로젝트에 사용된 오픈 소스 라이브러리들과 기여자들의 정보를 쉽게 확인할 수 있습니다.

API 문서를 확인하는 방법은 다음과 같습니다:

1. Swift AcknowList 저장소에 접속합니다. [GitHub 저장소](https://github.com/vtourraine/AcknowList)

2. README 파일에서 AcknowList에 대한 기본 정보와 사용법을 확인할 수 있습니다. 이 파일에는 어떻게 AcknowList를 사용하는지, 라이센스 정보 등이 포함되어 있습니다.

3. 저장소의 "Wiki" 탭을 클릭하면, AcknowList에 대한 더 자세한 문서를 확인할 수 있습니다. 여기에는 AcknowList의 API 문서와 사용 예제, 추가 기능들에 대한 설명 등이 제공됩니다.

4. "Code" 탭에서는 AcknowList의 소스 코드를 확인할 수 있습니다. 필요한 경우, 소스 코드를 참고하여 자신의 프로젝트에 적용할 수 있습니다.

5. 만약, AcknowList를 CocoaPods를 통해 사용 중이라면, `pod try AcknowList` 명령을 사용하여 예제 프로젝트를 실행해볼 수도 있습니다. 이를 통해 실제 사용 예제를 확인할 수 있습니다.

Swift AcknowList의 API 문서와 예제를 통해 세부적인 사용 방법과 추가적인 기능들을 자세히 확인할 수 있으니, 해당 문서를 참고하는 것이 좋습니다.
