## 1. 소개 {#introduction}

이번 포스트에서는 자연어로 마크다운 형식의 테크 블로그 포스트를 작성하는 방법에 대해 알아보겠습니다. 이 방법을 사용하면 텍스트를 더욱 강조하고 가독성을 향상시킬 수 있습니다.

## 2. 주요 특징 {#features}

- 자연어로 작성된 마크다운 형식
- 강조, 굵게 표시, 이탤릭체 등을 사용하여 중요한 내용을 표시할 수 있음
- 예제 코드를 작성하여 보다 실제적인 내용을 전달할 수 있음
- 관련 자료를 제공하여 참고할 수 있는 내용을 포함할 수 있음

## 3. 사용 방법 {#usage}

테크 블로그 포스트를 작성할 때는 다음과 같은 순서로 진행합니다.

1. 포스트 제목을 작성합니다.
2. 목차를 작성합니다.
3. 내용을 작성합니다.
4. 코드 블록을 작성합니다.
5. 참고 자료를 작성합니다.

## 4. 예제 코드 {#example-code}

아래는 자바스크립트 예제 코드입니다.

```javascript
function sayHello() {
  console.log("안녕하세요!");
}

sayHello();
```
