이번에는 파이썬을 사용하여 키 입력을 받아와서 스포츠 스코어를 확인하는 방법에 대해 알아보겠습니다.

## 코드 예시

```python
score = input("스코어를 입력하세요: ")

if score == "1:0":
    print("홈팀이 이기고 있습니다.")
elif score == "0:1":
    print("원정팀이 이기고 있습니다.")
elif score == "0:0":
    print("아직 경기가 시작되지 않았습니다.")
else:
    print("경기가 진행 중입니다.")
```

위의 코드는 사용자로부터 스코어를 입력받아 해당하는 조건에 따라 결과를 출력하는 간단한 예시입니다. 사용자에게 `스코어를 입력하세요:`라는 문구가 나타나고, 입력된 스코어에 따라 결과가 출력됩니다.

## 설명

- `input()` 함수를 사용하여 사용자로부터 스코어를 입력받습니다.
- 입력된 스코어를 `score` 변수에 저장합니다.
- `if` 조건문을 사용하여 입력된 스코어에 따라 다른 결과를 출력합니다.
- 만약 `score`가 "1:0"과 같다면 "홈팀이 이기고 있습니다."를 출력합니다.
- 만약 `score`가 "0:1"과 같다면 "원정팀이 이기고 있습니다."를 출력합니다.
- 만약 `score`가 "0:0"과 같다면 "아직 경기가 시작되지 않았습니다."를 출력합니다.
- 위의 조건에 해당하지 않는 경우 "경기가 진행 중입니다."를 출력합니다.

## 실행 결과 예시

```
스코어를 입력하세요: 1:0
홈팀이 이기고 있습니다.

스코어를 입력하세요: 0:1
원정팀이 이기고 있습니다.

스코어를 입력하세요: 0:0
아직 경기가 시작되지 않았습니다.

스코어를 입력하세요: 2:2
경기가 진행 중입니다.
```

위의 실행 결과 예시에서는 사용자가 "1:0"을 입력하면 "홈팀이 이기고 있습니다."가 출력되고, "0:1"을 입력하면 "원정팀이 이기고 있습니다."가 출력되었습니다.
