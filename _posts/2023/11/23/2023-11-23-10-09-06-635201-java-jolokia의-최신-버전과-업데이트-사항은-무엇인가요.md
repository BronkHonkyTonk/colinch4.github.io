Java Jolokia는 원격 JMX (Java Management Extensions)로컬 MBean 서버에 대한 HTTP/JSON 베이스의 프로토콜을 제공하는 라이브러리입니다. Jolokia를 사용하면 원격 MBean 속성에 대한 읽기/쓰기, 작업 수행, MBean 정보 가져오기 등을 포함한 JMX 작업을 원격으로 수행할 수 있습니다. 이를 통해 애플리케이션 모니터링 및 관리를 더 쉽게 할 수 있습니다.

Java Jolokia의 최신 버전은 1.7.1입니다. 이 버전은 2021년 5월 10일에 출시되었으며, 몇 가지 새로운 기능과 버그 수정이 포함되어 있습니다. 몇 가지 주요 업데이트 사항은 다음과 같습니다.

- 세션 유지 관련 인증을 위한 기능 개선
- JSON 요청 페이로드 크기 제한 설정 가능
- HTTP 클라이언트 구성 옵션 추가
- Jolokia JVM 에이전트의 오버헤드를 줄이기 위한 최적화

Java Jolokia의 상세한 변경 내역과 버전 업데이트 사항은 공식 Jolokia 웹사이트나 GitHub 저장소에서 찾을 수 있습니다. 또한 Jolokia를 사용하는 애플리케이션의 문서와 설명서도 도움이 될 수 있습니다.

[공식 Jolokia 웹사이트](https://jolokia.org/)
[GitHub 저장소](https://github.com/rhuss/jolokia)

참고로, Jolokia는 Java뿐만 아니라 다른 여러 프로그래밍 언어로도 사용할 수 있습니다. 따라서 다양한 언어로 개발된 애플리케이션에 대해 Jolokia를 사용할 수 있습니다.
