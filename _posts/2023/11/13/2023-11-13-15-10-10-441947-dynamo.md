## 소개
DynamoDB는 아마존 웹 서비스(AWS)에서 제공하는 완전 관리형 NoSQL 데이터베이스입니다. DynamoDB는 높은 확장성, 내구성, 가용성을 제공하며, 실시간 애플리케이션을 위한 필수 요소입니다.

## DynamoDB의 주요 특징
- 관계형 데이터베이스와 비교하여 비교적 정적인 데이터 모델이 있으며, 테이블, 항목 및 속성으로 구성됩니다.
- AWS Management Console, AWS SDK 또는 AWS CLI를 사용하여 쉽게 작업할 수 있다.
- 데이터베이스를 손쉽게 확장할 수 있으며, 필요에 따라 처리량과 저장 용량을 조정할 수 있습니다.
- 인덱스 기능을 제공하여 데이터에 대한 검색과 조회를 효율적으로 수행할 수 있습니다.

## DynamoDB 테이블 생성하기
DynamoDB 테이블을 생성하는 것은 간단한 작업입니다. 먼저 AWS Management Console에 로그인하고 DynamoDB 서비스로 이동합니다. 테이블을 생성하려는 AWS 리전을 선택하고 "테이블 생성"을 클릭합니다. 이어서 테이블의 이름, 항목 특성 등을 설정하고 "테이블 생성"을 클릭하면 테이블이 생성됩니다.

```python
import boto3

dynamodb = boto3.resource('dynamodb')

table = dynamodb.create_table(
    TableName='MyTable',
    KeySchema=[
        {
            'AttributeName': 'id',
            'KeyType': 'HASH'
        },
        {
            'AttributeName': 'timestamp',
            'KeyType': 'RANGE'
        }
    ],
    AttributeDefinitions=[
        {
            'AttributeName': 'id',
            'AttributeType': 'N'
        },
        {
            'AttributeName': 'timestamp',
            'AttributeType': 'N'
        }
    ],
    ProvisionedThroughput={
        'ReadCapacityUnits': 5,
        'WriteCapacityUnits': 5
    }
)

table.wait_until_exists()
```

## DynamoDB 항목 추가하기
DynamoDB 테이블에 항목을 추가하는 방법은 다양합니다. Python으로 작성된 예제 코드를 살펴보겠습니다.

```python
response = table.put_item(
    Item={
        'id': 1,
        'timestamp': 1634567890,
        'name': 'John Doe',
        'age': 30
    }
)
```

## DynamoDB 항목 조회하기
테이블에서 항목을 조회하는 방법은 다양합니다. 예를 들어, 특정 `id` 값에 해당하는 항목을 조회하는 경우에는 다음과 같이 코드를 작성할 수 있습니다.

```python
response = table.get_item(
    Key={
        'id': 1,
        'timestamp': 1634567890
    }
)
item = response['Item']
```

DynamoDB는 강력하고 확장성이 뛰어난 NoSQL 데이터베이스입니다. 애플리케이션의 요구에 맞게 데이터 모델링하고 DynamoDB를 활용해보세요!
