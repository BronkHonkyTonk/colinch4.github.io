SQLite는 오픈 소스 관계형 데이터베이스 관리 시스템(RDBMS)입니다. 경량이며 퍼블릭 도메인에 속하는 소프트웨어 라이브러리로, 특히 임베디드 시스템 및 모바일 애플리케이션에서 인기가 있습니다. SQLite는 ACID(원자성, 일관성, 고립성, 지속성)를 준수하는 트랜잭션을 지원하며, SQL 문법을 사용하여 데이터 검색, 추가, 수정 및 삭제 작업을 수행할 수 있습니다.

## SQLite의 주요 특징

### 1. 경량성

SQLite는 가벼운 라이브러리로, 많은 디스크 공간이나 메모리를 차지하지 않습니다. 이러한 특성으로 인해 모바일 디바이스와 임베디드 시스템에서 사용되며, 클라이언트-서버 모델의 전통적인 데이터베이스 시스템보다 가벼운 편입니다.

### 2. 독립성

SQLite는 클라이언트-서버 아키텍처를 가지지 않습니다. 데이터베이스 파일 자체에 모든 데이터가 저장되며, 다양한 응용 프로그램에 내장하여 사용할 수 있습니다. 따라서 독립적으로 작동되며 별도의 서버 프로세스가 필요하지 않습니다.

### 3. 크로스 플랫폼

SQLite는 크로스 플랫폼을 지원합니다. Windows, macOS, Linux 등 다양한 운영 체제에서 사용할 수 있으며, 다양한 프로그래밍 언어와도 호환됩니다. 또한 모바일 플랫폼인 Android 및 iOS에서도 많이 사용되고 있습니다.

### 4. 높은 성능

SQLite는 빠른 속도로 데이터를 처리할 수 있는 성능을 제공합니다. 데이터베이스 파일 내부에 B-트리 데이터 구조를 사용하여 빠른 검색 및 삽입 연산이 가능하며, 인덱싱을 통해 효율적인 데이터 액세스를 지원합니다.

## SQLite 사용 예제

```python
import sqlite3

# 데이터베이스 연결
conn = sqlite3.connect('example.db')

# 커서 생성
cursor = conn.cursor()

# 테이블 생성
cursor.execute('''CREATE TABLE IF NOT EXISTS products
                  (id INTEGER PRIMARY KEY AUTOINCREMENT,
                   name TEXT NOT NULL,
                   price REAL NOT NULL)''')

# 데이터 삽입
cursor.execute("INSERT INTO products (name, price) VALUES (?, ?)",
               ('Apple', 2.50))

# 데이터 조회
cursor.execute("SELECT * FROM products")
result = cursor.fetchall()
for row in result:
    print(row)

# 커밋 및 연결 종료
conn.commit()
conn.close()
```

위 코드는 Python에서 SQLite를 사용하는 간단한 예제입니다. 데이터베이스 연결, 테이블 생성, 데이터 삽입, 데이터 조회 등의 작업을 수행하는 방법을 보여줍니다.
