SwiftSoup은 Swift에서 HTML 파싱을 쉽게 할 수 있도록 도와주는 라이브러리입니다. SwiftSoup의 최신 버전과 업데이트 정보는 [SwiftSoup GitHub Repository](https://github.com/scinfu/SwiftSoup)에서 확인할 수 있습니다.

## 최신 버전

현재 최신 버전은 1.7.3입니다. 이는 2021년 7월 23일에 릴리스된 버전으로, [릴리스 페이지](https://github.com/scinfu/SwiftSoup/releases)에서 다운로드 및 업데이트 정보를 확인할 수 있습니다.

## 업데이트 정보

SwiftSoup의 업데이트 정보는 각 버전의 릴리스 페이지에서 확인할 수 있습니다. 최근 업데이트 내역은 다음과 같습니다:

### 버전 1.7.3

- XML 파서의 버그 수정
- CSS 선택자의 크래시 수정

### 버전 1.7.2

- CSS 선택자의 버그 수정
- 성능 향상

SwiftSoup의 GitHub 저장소를 주기적으로 확인하면 최신 버전 및 업데이트 정보를 놓치지 않을 수 있습니다.
