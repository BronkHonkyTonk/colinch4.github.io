SwiftyUserDefaults에서는 다음과 같은 값 타입을 사용할 수 있습니다:

1. Int: 정수 값을 저장하고 가져오는 데 사용됩니다.
2. Double: 실수 값을 저장하고 가져오는 데 사용됩니다.
3. Float: 단정도 부동 소수점 값을 저장하고 가져오는 데 사용됩니다.
4. Bool: 논리 값을 저장하고 가져오는 데 사용됩니다.
5. String: 문자열 값을 저장하고 가져오는 데 사용됩니다.
6. URL: URL 값을 저장하고 가져오는 데 사용됩니다.
7. Data: 바이트 데이터를 저장하고 가져오는 데 사용됩니다.
8. Date: 날짜와 시간 값을 저장하고 가져오는 데 사용됩니다.

이러한 값 타입을 사용하여 UserDefaults에 값을 저장하고, 다시 가져올 수 있습니다. SwiftyUserDefaults는 이러한 값을 간편하게 사용할 수 있는 메소드와 속성을 제공하여 개발자가 효율적으로 사용할 수 있도록 도와줍니다.

더 자세한 내용은 [SwiftyUserDefaults 공식 문서](https://github.com/radex/SwiftyUserDefaults)를 참조하시기 바랍니다.
