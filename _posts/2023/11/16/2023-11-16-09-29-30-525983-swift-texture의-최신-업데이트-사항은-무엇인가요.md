Texture의 최신 업데이트인 v3.0은 다음과 같은 주요 사항을 포함하고 있습니다:

1. **SwiftUI 지원 추가**: Texture v3.0은 SwiftUI 애플리케이션에 Texture 뷰를 통합하는 기능을 제공합니다. 이를 통해 Texture를 사용하여 성능 최적화된 사용자 인터페이스를 SwiftUI 앱에 쉽게 구현할 수 있습니다.

2. **Dark Mode 지원**: Texture v3.0은 다크 모드를 지원합니다. 이는 텍스처 노드를 사용하는 앱이 시스템의 다크 모드 설정을 자동으로 따를 수 있게 해줍니다.

3. **비동기 이미지 로딩 개선**: Texture v3.0은 비동기 이미지 로딩 성능을 향상시켰습니다. 이제 이미지를 비동기적으로 로딩할 때 더 빠르고 부드러운 스크롤 경험을 제공합니다.

4. **레이아웃 시스템 개선**: Texture v3.0은 레이아웃 시스템을 업그레이드하여 레이아웃 계산 속도를 향상시켰습니다. 이는 더 높은 성능을 제공하고 더 적은 메모리를 사용하는 애플리케이션을 만들 수 있게 해줍니다.

Texture의 최신 업데이트에 대한 자세한 내용은 [공식 GitHub 저장소](https://github.com/TextureGroup/Texture/releases)에서 확인할 수 있습니다.
