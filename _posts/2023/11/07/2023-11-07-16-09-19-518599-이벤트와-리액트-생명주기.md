리액트는 이벤트를 사용하여 사용자 상호작용에 대한 응답을 처리하는데 도움을 줍니다. 이벤트는 웹 애플리케이션의 동적인 동작에 중요한 역할을 합니다. 동시에, 리액트는 컴포넌트의 수명주기(lifecycle)를 갖고 있어 컴포넌트가 생성, 업데이트, 소멸되는 과정에서 특정한 작업을 수행할 수 있도록 합니다. 이 글에서는 리액트에서 이벤트를 다루는 방법과 컴포넌트의 생명주기에 대해 알아보겠습니다.

## 1. 이벤트 처리

### 1.1 이벤트 핸들러 등록

리액트에서 이벤트를 처리하기 위해서는 이벤트 핸들러를 등록해야 합니다. 이벤트 핸들러는 컴포넌트의 JSX 코드 내에서 이벤트가 발생했을 때 실행될 함수를 정의하는 것입니다. 아래는 버튼을 클릭했을 때 `handleClick` 함수가 실행되도록 이벤트 핸들러를 등록하는 예제입니다.

```jsx
import React from 'react';

function MyComponent() {
  const handleClick = () => {
    console.log('Button clicked!');
  };

  return (
    <button onClick={handleClick}>Click me!</button>
  );
}
```

### 1.2 이벤트 객체

이벤트 핸들러 함수는 일반적으로 이벤트 객체를 매개변수로 받아 사용할 수 있습니다. 이벤트 객체는 이벤트가 발생한 요소와 관련된 정보를 제공하며, 예를 들어 마우스의 좌표나 클릭한 요소의 값 등을 알 수 있습니다. 아래는 `handleClick` 함수에서 이벤트 객체를 활용하는 예제입니다.

```jsx
import React from 'react';

function MyComponent() {
  const handleClick = (event) => {
    console.log('Button clicked!', event.target);
  };

  return (
    <button onClick={handleClick}>Click me!</button>
  );
}
```

## 2. 리액트 생명주기

### 2.1 컴포넌트 생명주기 메서드

리액트 컴포넌트는 생성, 업데이트, 소멸과 같은 생명주기를 가지며 각 단계에서 호출되는 메서드를 활용해 특정 작업을 수행할 수 있습니다. 주요한 생명주기 메서드는 다음과 같습니다.

- `componentDidMount()` : 컴포넌트가 처음으로 렌더링된 직후에 호출됩니다. 주로 외부 라이브러리의 초기화나 데이터 로딩과 같은 작업에 이용됩니다.

- `componentDidUpdate(prevProps, prevState)` : 컴포넌트의 업데이트가 일어난 직후에 호출됩니다. 이전 props와 state를 매개변수로 받아 이전 값과 현재 값을 비교하여 필요한 업데이트 작업을 수행할 수 있습니다.

- `componentWillUnmount()` : 컴포넌트가 소멸되기 직전에 호출됩니다. 클린업 작업이나 이벤트 해제 등의 작업에 사용됩니다.

### 2.2 생명주기 메서드 예제

아래는 생명주기 메서드를 사용하여 컴포넌트의 동작을 제어하는 예제입니다.

```jsx
import React, { Component } from 'react';

class MyComponent extends Component {
  componentDidMount() {
    console.log('Component mounted!');
    // 외부 데이터 로딩 등 초기화 작업 수행
  }

  componentDidUpdate(prevProps, prevState) {
    if (prevProps.value !== this.props.value) {
      console.log('Component updated!');
      // 값이 업데이트 되었을 때 추가 작업 수행
    }
  }

  componentWillUnmount() {
    console.log('Component unmounted!');
    // 클린업 작업이나 이벤트 해제 등 수행
  }

  render() {
    return (
      <div>
        {/* 컴포넌트 렌더링 내용 */}
      </div>
    );
  }
}
```
