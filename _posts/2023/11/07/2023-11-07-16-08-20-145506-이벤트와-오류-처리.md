### 이벤트 처리

이벤트 처리는 컴퓨터 프로그래밍에서 중요한 부분입니다. 이벤트는 소프트웨어나 하드웨어에서 발생하는 사건을 의미합니다. 예를 들어, 버튼 클릭, 마우스 이동, 키 입력 등은 모두 이벤트입니다.

이벤트 처리는 프로그램이 사용자의 입력에 반응하고 적절한 액션을 수행할 수 있도록 해줍니다. 일반적으로 이벤트 처리는 이벤트를 감시하는 이벤트 핸들러를 사용하여 구현됩니다. 이벤트 핸들러는 이벤트가 발생할 때 실행될 코드를 정의하는 함수입니다.

예를 들어, 웹 페이지에서 버튼을 클릭할 때 어떤 작업을 수행하고 싶다면 해당 버튼에 이벤트 핸들러를 등록해야 합니다. 이벤트 핸들러를 등록하는 방법은 프로그래밍 언어나 프레임워크에 따라 다를 수 있습니다. 

```javascript
const button = document.querySelector('#myButton');
button.addEventListener('click', function() {
    // 버튼 클릭시 실행될 코드 작성
});
```

### 오류 처리

프로그램에서 오류가 발생하는 경우, 이를 적절하게 처리하는 것이 중요합니다. 오류 처리를 통해 프로그램의 안정성을 높일 수 있고, 사용자에게 적절한 안내 메시지를 제공할 수 있습니다.

대부분의 프로그래밍 언어는 오류 처리를 위한 예외(Exception) 처리 메커니즘을 제공합니다. 예외는 런타임 오류를 나타내는 객체로, 프로그램 실행 도중에 발생한 예외적인 상황을 나타냅니다. 예외 처리는 예외가 발생한 곳에서 예외 처리 블록을 사용하여 예외를 처리하는 것입니다.

아래는 자바스크립트에서 예외 처리를 하는 예시입니다.

```javascript
try {
  // 예외 발생 가능한 코드 작성
} catch (error) {
  // 예외 처리
  console.log(error.message);
}
```

이렇게 예외 처리를 함으로써 프로그램이 예상치 못한 오류에도 안정적으로 동작할 수 있도록 보장할 수 있습니다.
