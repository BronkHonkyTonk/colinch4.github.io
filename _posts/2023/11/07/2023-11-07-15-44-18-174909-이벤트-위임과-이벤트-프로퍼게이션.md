## 소개
이벤트 위임(Event Delegation)과 이벤트 프로퍼게이션(Event Propagation)은 웹 개발에서 자주 사용되는 기술입니다. 이들은 이벤트 처리를 더욱 효율적으로 관리하고 성능을 향상시키는 데 도움을 줍니다.

## 이벤트 위임(Event Delegation)
이벤트 위임은 상위 요소가 하위 요소들의 이벤트를 처리하는 방식입니다. 일반적으로, 하위 요소들이 동적으로 생성되는 경우에 이벤트 위임을 사용하는 것이 유용합니다. 이렇게 하면 각각의 하위 요소에 이벤트 핸들러를 등록하는 대신, 하위 요소들의 공통된 상위 요소에 이벤트 핸들러를 등록하여 처리할 수 있습니다.

예를 들어, 여러 개의 버튼이 있는 리스트를 가지고 있다고 가정해 봅시다. 각 버튼에 클릭 이벤트를 등록하는 대신, 상위 요소인 리스트에 한 번만 클릭 이벤트를 등록하고, 클릭된 버튼을 확인하여 처리할 수 있습니다. 이렇게 하면 메모리 사용을 줄이고 이벤트 핸들러의 수를 제한할 수 있습니다.

```javascript
const list = document.querySelector('.list');

list.addEventListener('click', function(event) {
  if (event.target.tagName === 'BUTTON') {
    // 버튼의 클릭 이벤트 처리
    console.log('버튼이 클릭되었습니다.');
  }
});
```

위의 예제에서는 상위 요소인 `.list`에 클릭 이벤트를 등록하고, 이벤트가 발생한 요소가 버튼인 경우에만 이벤트를 처리하고 있습니다.

## 이벤트 프로퍼게이션(Event Propagation)
이벤트 프로퍼게이션은 이벤트가 하위 요소로부터 상위 요소로 전파되는 방식입니다. 이벤트가 하위 요소에 발생하면, 해당 이벤트는 상위 요소로 전파되며, 이벤트가 발생한 요소부터 상위 요소까지의 모든 요소들에게 이벤트가 전달됩니다. 이렇게 함으로써 이벤트를 여러 요소에 각각 등록하지 않고, 상위 요소 하나로만 이벤트를 처리할 수 있습니다.

이벤트 프로퍼게이션은 일반적으로 이벤트 캡처링(Event Capturing)과 버블링(Bubbling) 두 가지 단계로 이루어집니다. 이벤트 캡처링은 이벤트가 상위 요소에서 하위 요소로 내려가는 과정을 말하고, 버블링은 이벤트가 하위 요소에서 상위 요소로 올라가는 과정을 말합니다.

```html
<div class="parent">
  <div class="child">
    <button class="button">Click me</button>
  </div>
</div>
```

```javascript
const parent = document.querySelector('.parent');
const child = document.querySelector('.child');
const button = document.querySelector('.button');

parent.addEventListener('click', function() {
  console.log('부모 요소가 클릭되었습니다');
}, true);  // 캡처링 단계 이벤트 핸들러

child.addEventListener('click', function() {
  console.log('자식 요소가 클릭되었습니다');
});  // 버블링 단계 이벤트 핸들러

button.addEventListener('click', function() {
  console.log('버튼이 클릭되었습니다');
});
```

위의 예제에서는 `.parent` 요소에는 캡처링 단계 이벤트 핸들러, `.child` 요소에는 버블링 단계 이벤트 핸들러를 등록하고 있습니다. 이렇게 하면 버튼을 클릭했을 때, `버튼이 클릭되었습니다`, `자식 요소가 클릭되었습니다`, `부모 요소가 클릭되었습니다` 순서로 이벤트가 실행됩니다.

## 결론
이벤트 위임과 이벤트 프로퍼게이션은 각각 다른 환경에서 유용하게 사용될 수 있습니다. 이벤트 위임은 동적으로 생성되는 요소들에 대한 이벤트 처리에 효과적이며, 이벤트 프로퍼게이션은 이벤트의 전파 경로를 이용하여 여러 요소에 하나의 이벤트 핸들러를 등록하는 데에 유용합니다.
