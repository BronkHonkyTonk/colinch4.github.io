이벤트 처리는 웹 개발에서 중요한 부분입니다. 이벤트가 발생하면 이벤트 객체와 이벤트 타겟이라는 개념이 사용됩니다. 이 둘은 이벤트를 이해하고 처리하는 데 도움을 줍니다.

## 이벤트 객체

이벤트 객체는 이벤트가 발생했을 때 자동으로 생성되는 객체입니다. 이 객체에는 이벤트와 관련된 정보를 담고 있는 프로퍼티와 메서드가 포함되어 있습니다.

```javascript
window.addEventListener('click', function(event) {
  console.log(event);
});
```

위의 예시에서 `event`는 이벤트 객체입니다. 이 객체를 사용하면 클릭 이벤트와 관련된 정보를 확인하고 처리할 수 있습니다. 이벤트 객체에는 클릭한 위치, 클릭된 요소, 발생한 이벤트의 타입 등 다양한 속성이 있습니다.

```javascript
document.getElementById('myButton').addEventListener('click', function(event) {
  console.log(event.target);
});
```

## 이벤트 타겟

이벤트 타겟은 이벤트가 발생한 요소를 가리킵니다. 이 요소는 이벤트 객체의 `target` 프로퍼티로 얻을 수 있습니다. 이벤트 타겟을 이용하면 이벤트를 발생시킨 요소를 정확히 식별할 수 있습니다.

```javascript
document.getElementById('myButton').addEventListener('click', function(event) {
  console.log(event.target);
});
```

위의 예시에서 `event.target`은 클릭 이벤트가 발생한 요소를 가리킵니다. 이를 통해 버튼이 클릭되었을 때 해당 버튼을 식별하고 원하는 처리를 할 수 있습니다.

## 마무리

이벤트 객체와 이벤트 타겟은 웹 개발에서 이벤트 처리에 필수적인 개념입니다. 이벤트 객체를 통해 이벤트와 관련된 정보를 확인하고 이벤트 타겟을 이용하여 이벤트를 발생시킨 요소를 식별할 수 있습니다. 이를 활용하여 웹 애플리케이션에서 다양한 이벤트를 처리할 수 있습니다.
