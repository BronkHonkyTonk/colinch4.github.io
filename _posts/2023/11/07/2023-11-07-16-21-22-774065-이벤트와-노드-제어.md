이벤트와 노드 제어는 웹 개발에서 매우 중요한 개념입니다. 이들은 사용자 상호작용 및 동적 웹 페이지 개발에 광범위하게 사용됩니다. 이번 블로그 포스트에서는 이벤트와 노드 제어에 대해 알아보도록 하겠습니다.

## 이벤트

이벤트는 사용자의 동작(클릭, 마우스 오버 등) 또는 시스템의 특정 동작(페이지 로드, 타이머 완료 등)에 의해 발생하는 신호입니다. 이벤트는 웹 페이지의 특정 요소에 바인딩되며, 해당 요소의 상태나 동작에 따라 어떤 코드가 실행될지 결정합니다.

### 이벤트 핸들러

이벤트 핸들러는 특정 이벤트가 발생했을 때 실행되는 함수입니다. 이벤트 핸들러를 등록하여 원하는 동작을 수행할 수 있습니다. 예를 들어, 버튼 클릭 이벤트의 핸들러를 등록하면 버튼이 클릭되었을 때 해당 핸들러 함수가 실행됩니다.

```javascript
const button = document.querySelector('#myButton');

button.addEventListener('click', function() {
    console.log('버튼이 클릭되었습니다!');
});
```

### 이벤트 전파

이벤트 전파는 한 요소에서 발생한 이벤트가 부모 요소나 자식 요소로 전파되는 것을 의미합니다. 이벤트 전파에는 캡처링(caputuring) 단계와 버블링(bubbling) 단계 두 가지 종류가 있습니다. 기본적으로 이벤트는 버블링 단계로 전파되며, 이벤트 핸들러는 이 단계에서 등록된 순서대로 실행됩니다. 이벤트 전파를 막는 방법도 제공되므로 필요에 따라 사용할 수 있습니다.

## 노드 제어

노드는 웹 페이지의 요소를 나타내며, HTML 요소, 텍스트 노드, 속성 노드 등이 있습니다. 노드 제어는 웹 페이지의 동적인 변화를 구현하기 위해 사용됩니다.

### 요소 선택

특정 요소를 선택하려면 `querySelector` 또는 `querySelectorAll` 메서드를 사용할 수 있습니다. `querySelector`는 첫 번째로 일치하는 요소만 선택하며, `querySelectorAll`은 모든 일치하는 요소를 선택합니다.

```javascript
const element = document.querySelector('#myElement');
const elements = document.querySelectorAll('.myClass');
```

### 요소 조작

선택한 요소를 조작하기 위해 다양한 메서드를 사용할 수 있습니다. 예를 들어, 요소의 내용을 바꾸려면 `innerHTML` 속성을 사용하거나, 특정 CSS 클래스를 추가 또는 제거하기 위해 `classList` 메서드를 사용할 수 있습니다.

```javascript
element.innerHTML = '새로운 내용';
element.classList.add('newClass');
element.classList.remove('oldClass');
```

### 이벤트와 노드 제어의 결합

이벤트와 노드 제어는 종종 결합되어 사용됩니다. 예를 들어, 특정 이벤트가 발생했을 때 요소를 조작할 수 있습니다. 아래는 버튼 클릭 이벤트 핸들러를 이용해 요소를 숨기거나 보이는 예제입니다.

```javascript
const button = document.querySelector('#myButton');
const element = document.querySelector('#myElement');

button.addEventListener('click', function() {
    if (element.classList.contains('hidden')) {
        element.classList.remove('hidden');
    } else {
        element.classList.add('hidden');
    }
});
```

이와 같이 이벤트와 노드 제어를 유연하게 조합하여 웹 페이지를 동적으로 제어할 수 있습니다.
