이번 글에서는 웹 개발에서 자주 사용되는 클릭 이벤트와 마우스 이벤트에 대해 알아보겠습니다. 

## 클릭 이벤트

### 동작 설명

클릭 이벤트는 사용자가 요소를 클릭할 때 발생하는 이벤트입니다. 일반적으로 마우스의 왼쪽 버튼을 클릭할 때 발생하지만, 모바일 기기에서는 터치 이벤트로 대체될 수도 있습니다.

### 예시 코드

```javascript
const button = document.getElementById('my-button'); // 버튼 요소 선택

button.addEventListener('click', function() {
  console.log('버튼이 클릭되었습니다!');
});
```

## 마우스 이벤트

### 동작 설명

마우스 이벤트는 마우스 커서의 동작과 관련된 이벤트입니다. 자주 사용되는 마우스 이벤트는 다음과 같습니다:

- `mousedown`: 마우스 버튼을 누를 때 발생
- `mouseup`: 마우스 버튼을 눌렀다가 뗄 때 발생
- `mousemove`: 마우스 커서가 요소 위에서 이동할 때 발생
- `mouseenter`: 마우스 커서가 요소에 진입했을 때 발생
- `mouseleave`: 마우스 커서가 요소를 벗어날 때 발생

### 예시 코드

```javascript
const box = document.getElementById('my-box'); // 박스 요소 선택

box.addEventListener('mouseenter', function() {
  console.log('마우스가 박스 안으로 진입했습니다!');
});

box.addEventListener('mouseleave', function() {
  console.log('마우스가 박스를 벗어났습니다!');
});
```

## 결론

클릭 이벤트와 마우스 이벤트는 웹 개발에서 자주 사용되는 이벤트 중 하나입니다. 이를 활용하여 사용자와 상호작용하고 동적인 웹 페이지를 구현할 수 있습니다.
