1. GFM (Github Flavored Markdown) 플러그인: 이 플러그인은 Markdown 문법을 GFM 형식에 맞게 처리합니다. GFM은 링크, 테이블, 체크박스 등과 같은 추가적인 기능을 제공하며, Marked에서도 이를 적절하게 렌더링할 수 있도록 지원합니다.

2. Emoji 플러그인: 이 플러그인은 문서에서 이모지를 사용할 수 있도록 해줍니다. 이모지는 감정이나 상태를 표현하는데 유용하며, Marked는 이모지 코드를 해당 이미지로 대체하여 렌더링합니다. 이 플러그인을 사용하면 문서에 이모지를 쉽게 추가할 수 있습니다.

3. Math 플러그인: 이 플러그인은 수학식을 TeX 형식으로 작성하여 렌더링할 수 있도록 해줍니다. Math 플러그인을 사용하면 Markdown 문서에서 수학 관련 내용을 포함할 수 있고, Marked는 이를 수식 이미지로 변환하여 표시합니다.

4. Custom Renderer 플러그인: 이 플러그인은 Marked가 Markdown 문서를 HTML로 렌더링할 때, 특정 요소의 렌더링 방식을 사용자 정의할 수 있도록 해줍니다. 사용자는 이 플러그인을 통해 자신만의 렌더링 규칙을 만들어 Markdown 문서의 렌더링을 커스터마이즈할 수 있습니다. 

Marked에는 위에 언급한 플러그인 외에도 다른 플러그인들이 존재하며, 이를 사용하여 Markdown 문서를 보다 다양하게 처리할 수 있습니다. 자세한 내용은 Marked의 공식 문서를 참고하시기 바랍니다.
