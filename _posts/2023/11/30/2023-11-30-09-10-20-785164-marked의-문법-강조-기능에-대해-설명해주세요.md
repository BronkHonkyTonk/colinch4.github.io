Markdown 문법 강조는 텍스트를 특정한 방식으로 강조하여 시각적으로 표현하는 기능입니다. Marked는 다음과 같은 문법 강조 기능을 제공합니다.

1. **굵게 강조하기** : 원하는 텍스트를 별표(\*\*) 또는 언더스코어(\_\_)로 감싸면 해당 텍스트가 굵게 표시됩니다. 예시: `**굵은 텍스트**`

2. *기울임체로 강조하기* : 원하는 텍스트를 별표(\*) 또는 언더스코어(\_)로 감싸면 해당 텍스트가 기울임체로 표시됩니다. 예시: `*기울인 텍스트*`

3. ~~취소선~~ : 원하는 텍스트를 물결표시(\~\~)로 감싸면 해당 텍스트에 취소선이 표시됩니다. 예시: `~~취소된 텍스트~~`

4. `인라인 코드` : 인라인으로 표현할 코드는 단일 백쿼트(\`)로 감싸면 됩니다. 예시: \`console.log('Hello, World!')\`

5. ```블록 코드``` : 여러 줄의 코드를 표현할 때는 코드가 속한 블록을 세 개의 백쿼트(\`\`\`)로 시작하고 끝내면 됩니다. 예시:

\```
console.log('Hello, World!');
console.log('안녕하세요!');
\```

위와 같이 Marked를 사용하면 간편하게 마크다운 문법을 HTML로 변환하여 웹 페이지에서 보여줄 수 있습니다. 더 자세한 내용은 [Marked 공식 GitHub 페이지](https://github.com/markedjs/marked)에서 확인할 수 있습니다.
