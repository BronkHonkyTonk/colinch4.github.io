PhotoSwipe은 JavaScript로 작성된 간단하면서 강력한 웹 포토 갤러리 라이브러리입니다. 다양한 환경과 장치에서 사용할 수 있어서 많은 웹 개발자들에게 인기가 있습니다. 그러나 PhotoSwipe을 사용하기 위해 몇 가지 추가적인 라이브러리를 설치해야 합니다.

### 1. PhotoSwipe CSS 파일

PhotoSwipe을 사용하기 위해 가장 먼저 필요한 것은 CSS 파일입니다. 이 파일은 PhotoSwipe 레이아웃과 디자인을 제공합니다. 아래 링크를 통해 PhotoSwipe CSS 파일을 다운로드 받을 수 있습니다:

[Download PhotoSwipe CSS](https://github.com/dimsemenov/PhotoSwipe/blob/master/dist/photoswipe.css)

```html
<link rel="stylesheet" href="path/to/photoswipe.css">
```

이 CSS 파일을 HTML 문서에 추가하여 PhotoSwipe 스타일을 적용할 수 있습니다.

### 2. PhotoSwipe JavaScript 파일

다음으로 필요한 것은 PhotoSwipe의 JavaScript 파일입니다. 이 파일은 PhotoSwipe을 초기화하고 갤러리 이미지를 불러오는 데 사용됩니다. 아래 링크에서 다운로드할 수 있습니다:

[Download PhotoSwipe JavaScript](https://github.com/dimsemenov/PhotoSwipe/blob/master/dist/photoswipe.min.js)

```html
<script src="path/to/photoswipe.min.js"></script>
```

HTML 문서에 위의 JavaScript 파일을 추가하여 PhotoSwipe 기능을 활성화할 수 있습니다.

### 3. PhotoSwipe UI JavaScript 파일

마지막으로 필요한 것은 PhotoSwipe UI JavaScript 파일입니다. 이 파일은 사용자 인터페이스 요소를 제공하고 터치 지원 기능을 활성화합니다. 다음 링크를 통해 다운로드할 수 있습니다:

[Download PhotoSwipe UI JavaScript](https://github.com/dimsemenov/PhotoSwipe/blob/master/dist/photoswipe-ui-default.min.js)

```html
<script src="path/to/photoswipe-ui-default.min.js"></script>
```

PhotoSwipe UI JavaScript 파일을 HTML 문서에 추가하여 사용자 인터페이스 기능을 활성화할 수 있습니다.

### 요약

PhotoSwipe을 사용하기 위해서는 위에서 언급한 세 가지 라이브러리를 HTML 문서에 추가해야 합니다. 이제 PhotoSwipe을 사용하여 멋진 웹 포토 갤러리를 만들어보세요!
