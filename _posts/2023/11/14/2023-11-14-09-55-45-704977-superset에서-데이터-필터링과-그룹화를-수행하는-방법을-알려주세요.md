Apache Superset은 데이터 시각화와 탐색을 위한 오픈 소스 비즈니스 인텔리전스 도구입니다. 이번 블로그 포스트에서는 Superset을 사용하여 데이터 필터링 및 그룹화를 수행하는 방법에 대해 알아보겠습니다.

## 데이터 필터링

데이터 필터링은 데이터 집합을 특정 조건에 따라 필터링하여 원하는 결과를 얻는 과정을 의미합니다. Superset에서 데이터 필터링을 수행하기 위해 다음 단계를 따를 수 있습니다:

1. **대시보드 편집**:
   먼저 Superset 대시보드를 편집하는 화면으로 이동합니다.

2. **차트 선택**:
   데이터 필터링을 적용하려는 차트를 선택합니다.

3. **차트 편집**:
   차트 선택 후 해당 차트의 편집 옵션으로 이동합니다.

4. **필터 추가**:
   필터를 추가하려는 컬럼을 선택합니다. 필터를 추가한 후 조건을 설정하고 저장합니다. 이제 차트에 적용된 필터가 보여질 것입니다.

## 데이터 그룹화

데이터 그룹화는 데이터를 특정 기준에 따라 그룹화하는 과정을 의미합니다. Superset에서 데이터 그룹화를 수행하기 위해 다음 단계를 따를 수 있습니다:

1. **대시보드 편집**:
   Superset 대시보드를 편집하는 화면으로 이동합니다.

2. **차트 선택**:
   데이터 그룹화를 적용하려는 차트를 선택합니다.

3. **차트 편집**:
   차트 선택 후 해당 차트의 편집 옵션으로 이동합니다.

4. **그룹화 설정**:
   그룹화를 적용하려는 컬럼을 선택합니다. 선택한 컬럼을 기준으로 데이터가 그룹화됩니다.

5. **조건 설정**:
   그룹화된 데이터를 특정 조건에 따라 필터링할 수 있습니다. 필요한 조건을 설정하고 저장합니다.

Superset을 사용하여 데이터 필터링 및 그룹화를 수행하는 방법을 알아보았습니다. 이를 통해 데이터를 더 쉽게 탐색하고 원하는 결과를 얻을 수 있습니다. 실험해보고 새로운 인사이트를 발견해보세요!
