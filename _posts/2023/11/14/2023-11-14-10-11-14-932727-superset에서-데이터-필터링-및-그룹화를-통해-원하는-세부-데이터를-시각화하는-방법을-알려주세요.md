Superset은 데이터 시각화를 위한 오픈 소스 도구입니다. 이 도구를 사용하면 원하는 세부 데이터를 시각화하고, 데이터를 필터링하고 그룹화 할 수 있습니다. 이번 블로그 포스트에서는 Superset에서 데이터를 필터링하고 그룹화하는 방법에 대해 알려드리겠습니다.

## 데이터 필터링

Superset에서 데이터 필터링을 하려면 다음 단계를 따르세요:

1. Superset 대시보드에서 원하는 데이터 시각화 패널을 선택합니다.

2. 패널의 오른쪽 상단에 있는 "필터" 아이콘을 클릭합니다. 

3. "필터 추가" 버튼을 클릭하여 필터를 생성합니다. 필터는 특정 데이터 컬럼을 기반으로 데이터를 제한하는 용도로 사용됩니다.

4. 필터를 구성합니다. 예를 들어, "주문일"이라는 컬럼을 기반으로 필터를 설정하여 특정 날짜 범위의 데이터만 시각화 할 수 있습니다.

5. 필터를 적용하려면 "적용" 버튼을 클릭하십시오.

## 데이터 그룹화

Superset에서 데이터를 그룹화하려면 다음 단계를 따르세요:

1. Superset 대시보드에서 원하는 데이터 시각화 패널을 선택합니다.

2. 패널의 오른쪽 상단에 있는 "축" 아이콘을 클릭합니다.

3. "축 추가" 버튼을 클릭하여 컬럼을 선택합니다. 이 컬럼을 기준으로 데이터가 그룹화됩니다.

4. 그룹화된 데이터를 원하는 방식으로 시각화합니다. 예를 들어, 특정 그룹의 데이터를 막대 그래프로 표시할 수 있습니다.

## 마치며

이렇게 Superset을 사용하여 데이터를 필터링하고 그룹화할 수 있습니다. 원하는 세부 데이터를 시각화하여 더 나은 인사이트를 얻을 수 있습니다. Superset의 다양한 기능을 활용하여 데이터 분석과 시각화를 더욱 효과적으로 수행해보세요.
