데이터베이스에서 중복된 데이터를 필터링하는 방법 중 하나는 JOIN을 사용하는 것입니다. JOIN을 사용하면 테이블 간의 관계를 설정하고, 조인 조건을 기준으로 중복된 데이터를 제거할 수 있습니다.

예를 들어, 주문 테이블과 상품 테이블이 있다고 가정해봅시다. 주문 테이블에는 주문 번호와 상품 ID가 포함되어 있고, 상품 테이블에는 상품 ID와 상품명이 포함되어 있습니다. 이 경우 주문 테이블과 상품 테이블을 JOIN하여 중복된 상품 데이터를 필터링할 수 있습니다.

다음은 SQL 쿼리를 사용하여 중복된 상품 데이터를 필터링하는 예제입니다.

```sql
SELECT 
    오더.주문번호, 
    상품.상품명
FROM 
    주문테이블 AS 주문
JOIN 
    상품테이블 AS 상품
ON 
    주문.상품ID = 상품.상품ID
```

위 예제에서는 주문 테이블과 상품 테이블을 JOIN하여 상품 ID를 기준으로 조인하고 있습니다. SELECT 문을 통해 주문 번호와 상품명을 가져오게 되며, 중복된 데이터는 필터링 됩니다.

이와 같은 방식으로 JOIN을 사용하여 데이터베이스에서 중복된 데이터를 필터링할 수 있습니다. 데이터베이스 종류에 따라 JOIN 구문이 약간 다를 수 있으므로, 해당 데이터베이스의 문서를 참조하여 정확한 JOIN 구문을 사용해야 합니다.
