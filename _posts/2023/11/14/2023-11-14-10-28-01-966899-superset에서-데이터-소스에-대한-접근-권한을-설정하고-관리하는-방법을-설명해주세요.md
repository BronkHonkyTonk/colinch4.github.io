Superset은 많은 데이터 소스들과 연동하여 데이터 시각화 및 대시보드 생성을 가능하게 해주는 오픈 소스 도구입니다. 이번 포스트에서는 Superset에서 데이터 소스에 대한 접근 권한을 설정하고 관리하는 방법에 대해 알아보겠습니다.

## 1. 권한 그룹 생성하기

먼저 데이터 소스에 대한 접근 권한을 관리하기 위해 권한 그룹을 생성해야 합니다. 권한 그룹은 특정 사용자 또는 사용자 그룹에 대한 권한을 지정하므로, 데이터 소스의 접근 권한을 그룹 단위로 관리할 수 있습니다.

Superset 대시보드에서 "관리" 메뉴로 이동한 후 "권한" 탭을 선택합니다. 그런 다음 "권한 그룹"을 클릭하여 새로운 권한 그룹을 생성합니다. 권한 그룹에는 이름과 설명을 포함하여 필요한 정보를 입력합니다.

## 2. 데이터 소스에 대한 접근 권한 설정하기

생성한 권한 그룹에 대한 접근 권한을 설정하기 위해 데이터 소스에 접근합니다. 데이터 소스의 "설정"으로 이동한 후 "액세스 제어" 탭을 선택합니다.

액세스 제어 탭에서 권한을 설정할 사용자 또는 사용자 그룹을 선택합니다. 그런 다음 "액세스 제어"를 설정하려는 데이터 소스에 대해 체크합니다. 이렇게 하면 선택한 사용자 또는 사용자 그룹에게 해당 데이터 소스에 대한 접근 권한이 부여됩니다.

## 3. 권한 그룹 별로 접근 권한 관리하기

이제 생성한 권한 그룹 별로 데이터 소스에 대한 접근 권한을 관리할 수 있습니다. "관리" 메뉴에서 "사용자" 탭을 선택한 후 원하는 사용자를 찾아 들어갑니다. 사용자 상세 페이지에서 "권한" 탭을 선택합니다.

권한 탭에서는 해당 사용자에 대한 권한 그룹을 설정할 수 있습니다. 선택한 권한 그룹에 따라 사용자는 해당 데이터 소스에 대한 접근 권한이 있거나 없을 수 있습니다.

## 마무리

Superset을 사용하면 데이터 소스에 대한 접근 권한을 세밀하게 설정하고 관리할 수 있습니다. 이를 통해 데이터 소스에 보안을 유지하면서도 필요한 사용자들에게만 접근 권한을 부여할 수 있습니다.

더 자세한 정보는 [Superset 공식 문서](https://superset.apache.org/docs/security/)를 참조해주세요.
