## 강조

텍스트에서 중요한 내용이나 문장을 강조하기 위해 다음과 같은 방법을 사용할 수 있습니다.

- **굵은 글씨** : 텍스트를 강조할 때 사용됩니다.
- *이탤릭체* : 강조하거나 감정을 표현하기 위해 사용됩니다.
- `하이라이트` : 코드 또는 특정 용어를 강조하기 위해 사용됩니다.

강조는 중요한 내용을 강조하거나 사용자의 주의를 끌기 위해 유용하게 사용될 수 있습니다.

## 코드 예제

프로그래밍 언어를 설명하거나 코드 예제를 표시하기 위해 다음과 같은 방법을 사용할 수 있습니다.

```python
def hello_world():
    print("Hello, world!")

hello_world()
```

위의 예제는 Python으로 작성된 간단한 "Hello, world!" 출력 코드입니다.
