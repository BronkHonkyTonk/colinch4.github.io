자바스크립트는 매우 인기있는 프로그래밍 언어로, 웹 개발에서 가장 많이 사용됩니다. 이 글에서는 자바스크립트를 시작하는 방법에 대해서 알아보겠습니다.

# 자바스크립트 개발 환경 설정

자바스크립트를 개발하기 위해서는 몇 가지 환경 설정이 필요합니다. 먼저, 코드를 작성하고 저장할 수 있는 텍스트 에디터가 필요합니다. 프로그램을 실행하고 결과를 확인하기 위해서는 웹 브라우저가 필요합니다.

자바스크립트 코드를 실행하려면 HTML 문서 내에서 `<script>` 태그를 사용해야 합니다. 이렇게 작성된 코드는 웹 브라우저에서 실행되며, 결과를 볼 수 있습니다.

# 자바스크립트 기본 문법

자바스크립트는 문법적으로 C와 유사한 면이 있습니다. 변수를 선언할 때는 `var` 키워드를 사용하고, 조건문을 작성할 때는 `if-else` 문을 사용할 수 있습니다.

```javascript
var age = 25;

if (age >= 18) {
  console.log("성인입니다.");
} else {
  console.log("미성년자입니다.");
}
```

반복문을 작성할 때는 `for`나 `while` 문을 사용할 수 있으며, 함수를 정의할 때는 `function` 키워드를 사용합니다.

```javascript
function greet(name) {
  console.log("안녕하세요, " + name + "님!");
}

greet("홍길동");
```
