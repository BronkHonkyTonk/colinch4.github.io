1. 먼저, HTTPie를 설치합니다. 터미널에서 다음 명령을 실행합니다:
```
$ pip install httpie
```

2. HTTPie를 사용하여 서버에 GET 요청을 보냅니다. 아래 명령을 터미널에 입력합니다:
```
$ http GET <URL>
```
여기서 `<URL>`은 요청을 보낼 서버의 주소입니다. 예를 들어, `http GET https://api.example.com`와 같이 입력합니다.

3. 서버의 응답을 받으면, HTTPie는 응답 상태 코드를 출력합니다. 응답 상태 코드는 요청이 성공했는지, 실패했는지 등을 나타내는 세 자리 숫자입니다. 일반적으로, 2xx는 성공, 4xx는 클라이언트 오류, 5xx는 서버 오류를 의미합니다.

예를 들어, 서버가 `200 OK` 상태 코드를 반환한다면, 요청이 성공적으로 처리되었다는 뜻입니다. 만약 `404 Not Found` 상태 코드를 반환한다면, 요청한 리소스가 서버에 존재하지 않는다는 뜻입니다.

HTTPie를 사용하면 터미널에서 간편하게 서버의 응답 상태 코드를 확인할 수 있습니다. 이를 통해 API 통신이나 웹 개발에서 문제가 발생했을 때 빠르게 원인을 파악하고 해결할 수 있습니다.
