```javascript
// 변수 선언
let name = "John";
let age = 30;

// 함수 선언
function greet() {
  console.log("안녕하세요, " + name + "님!");
  console.log("당신은 " + age + "살이시군요.");
}

// 함수 호출
greet();
```

위 코드는 이름과 나이 변수를 선언하고, greet 함수를 정의한 후 호출하는 간단한 JavaScript 코드입니다. 크롬 개발자 도구의 콘솔 창에서 실행할 수 있습니다.

출력 예시:
```
안녕하세요, John님!
당신은 30살이시군요.
```

더 많은 JavaScript 예제를 찾고 싶다면 [MDN 웹 문서](https://developer.mozilla.org/ko/docs/Web/JavaScript/Guide)를 참고하시기 바랍니다.
