1. Istanbul (nyc): Istanbul은 JavaScript 코드 커버리지 도구 중 가장 널리 사용되는 도구입니다. Jest와 호환되며, 추가적인 보고서 형식과 기능을 제공합니다.
2. Blanket.js: Blanket.js는 JavaScript 코드 커버리지를 측정하는 데 사용되는 도구입니다. Jest와 같이 사용할 수 있으며, 실시간으로 커버리지 데이터를 표시하는 기능을 제공합니다.
3. Coveralls: Coveralls는 여러 테스트 프레임워크에서 코드 커버리지를 측정하고 보고하는 데 사용되는 온라인 서비스입니다. Jest와 함께 사용할 수 있으며, CI/CD 파이프라인에 통합하여 커버리지 데이터를 자동으로 업로드할 수 있습니다.
4. Codecov: Codecov는 코드 커버리지 측정과 보고를 제공하는 온라인 서비스입니다. Jest와 연동할 수 있으며, 다양한 통계 및 보고서 형식을 제공합니다.

이러한 도구들은 Jest와 마찬가지로 JavaScript 코드 커버리지 측정에 도움을 줄 수 있습니다. 선택하는 도구는 프로젝트의 요구 사항과 개인적인 선호도에 따라 다를 수 있습니다.
