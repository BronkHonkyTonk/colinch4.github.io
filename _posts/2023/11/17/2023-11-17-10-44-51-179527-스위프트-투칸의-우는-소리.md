스위프트(Swift)는 애플(Apple)에서 개발한 프로그래밍 언어로, iOS, macOS, watchOS, tvOS 등 애플의 다양한 플랫폼에서 사용되고 있습니다. 스위프트는 Objective-C와 호환되도록 설계되어 있으며, 간결하고 안전한 코드 작성을 위한 다양한 기능을 제공합니다.

스위프트의 특징 중 하나는 간결하고 명확한 문법입니다. 예를 들어, 배열을 생성하고 값을 출력하는 코드를 살펴보겠습니다.

```swift
let fruits = ["apple", "banana", "orange"]

for fruit in fruits {
    print(fruit)
}
```

위 코드는 "apple", "banana", "orange" 세 가지 과일이 담긴 배열을 생성하고, `for-in` 루프를 사용하여 배열의 각 값을 출력하는 코드입니다. 스위프트의 문법은 간결하고 이해하기 쉬워서 프로그램을 빠르게 개발할 수 있습니다.

또한, 스위프트는 안전한 코드 작성을 위한 다양한 기능을 제공합니다. 예를 들어, 변수 선언 시 타입 추론으로 변수의 타입을 자동으로 추론할 수 있습니다.

```swift
let name = "John"
```

위 코드는 `name`이라는 변수를 문자열로 선언하고, "John"이라는 값을 할당하는 코드입니다. 스위프트는 타입 추론을 통해 `name` 변수의 타입을 자동으로 문자열로 추론합니다.

스위프트는 강력한 기능과 직관적인 문법을 갖춘 프로그래밍 언어입니다. 다양한 애플의 플랫폼에서 사용되고 있는 만큼, 스위프트를 익히면 애플의 다양한 앱 개발에 도움이 될 것입니다.
