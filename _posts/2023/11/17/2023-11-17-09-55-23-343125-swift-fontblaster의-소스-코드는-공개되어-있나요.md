Swift FontBlaster는 Swift 언어로 작성된 폰트 라이브러리입니다. 이 라이브러리는 앱에서 폰트를 쉽게 적용할 수 있도록 도와줍니다.

소스 코드에 대한 공식적인 레퍼지토리나 공개된 문서는 없지만, Swift FontBlaster의 GitHub 페이지에서 소스 코드를 참고할 수 있습니다.

1. Swift FontBlaster의 GitHub 페이지로 이동합니다: [GitHub Repository](https://github.com/ArtSabintsev/FontBlaster)
2. 페이지 상단의 "Clone or download" 버튼을 클릭하고 "Download ZIP"을 선택하여 ZIP 파일로 다운로드합니다.
3. ZIP 파일을 압축 해제하고, Swift 프로젝트에 필요한 파일만 사용할 수 있습니다.

이렇게 다운로드한 소스 코드를 참고하여 Swift FontBlaster를 사용할 수 있습니다. 이 라이브러리는 Xcode 프로젝트에 쉽게 통합할 수 있으며, 폰트를 앱에 추가할 때 유용한 기능을 제공합니다.

참고 문서:
- [Swift FontBlaster GitHub Repository](https://github.com/ArtSabintsev/FontBlaster)
