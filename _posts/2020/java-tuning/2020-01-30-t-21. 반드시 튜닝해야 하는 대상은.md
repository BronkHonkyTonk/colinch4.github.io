# 21. 반드시 튜닝해야 하는 대상은?

- 파레토 법칙? 20:80법칙
- 포틀릿? 웹 포탈 유저들이 사용할 수 있는 사용자 인터페이스 컴포넌트이다. 
- 유저가 마음대로 위치 수정 또는 추가, 삭제 할 수 있다.
- 성능을 분석할 대상을 정할 때, 모든 화면을 대상으로 하는 것은 상당히 무모한 행동이다.(자주 사용하는 화면/기능 위주의 튜닝 필요)
